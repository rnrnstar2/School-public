# School CM assets

Generated on 2026-04-27.

## Concept

School is positioned as a goal-first AI mentor workspace: it turns a user's goal into a clarifying chat, an actionable plan, and the next concrete action.

## Cut List

1. Night desk: too much information and too many AI options.
2. Goal input: the user gives School a concrete goal.
3. Chat: AI asks clarifying questions.
4. Plan: scattered context becomes a goal tree and timeline.
5. Action: the next task is selected and started.
6. Brand end: School shows the flow Goal -> Chat -> Plan -> Action.
