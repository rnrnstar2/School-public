# Curriculum Docs

This folder defines the first curriculum system for School.

- `curriculum-architecture.md`: canonical architecture for modular lessons, sequencing, and future track expansion
- `web-builder-track-outline.md`: first-pass lesson map for the "build and deploy a real website with AI" track
- `initial-rollout-plan.md`: practical rollout schedule and ownership plan

These docs are written to support three implementation streams in parallel:

1. curriculum design and authoring
2. admin/content model implementation
3. AI sequencing and recommendation logic
