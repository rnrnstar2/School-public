# 実践結果を振り返って改善する

料理を作ったあと、食べてみて「次はもう少し塩を減らそうかな」と思うこと、ありますよね。仕事でも同じです。やってみた結果をふり返り、AIの力を借りて次の一手を見つけるのがこのレッスンの目標です。

あなたがこれまでの実践で得た結果（商品説明文、キャンペーン、画像改善など）を振り返り、AIツールと一緒に具体的な改善プランを1つ作り上げます。

![振り返りの全体像](/lesson-assets/atom.ec-operator.review-results-and-improve/diagram.png)

## やったことを書き出す

まずは、前の実践でやったこととその結果を整理しましょう。ノートアプリでもメモ帳でも、使い慣れたものでOKです。

### 手順

1. 前の実践で「やったこと」を箇条書きで3〜5個書き出す
2. それぞれについて「うまくいったこと」「気になったこと」を1行ずつ書き添える
3. もし数字（売上、クリック数、問い合わせ件数など）がわかれば、あわせてメモする

**良い例:**
- 「商品説明文をAIで3パターン作った → 問い合わせが先月より2件減った」
- 「キャンペーン案内を2種類出した → B案のほうがクリック率が高かった」
- 「商品画像の背景をAIで除去した → 見栄えが良くなり、お客様から『見やすい』と言われた」

**悪い例:**
- 「なんとなくやったから結果は覚えていない」（振り返れる材料がない）
- 「全部うまくいった」（根拠がないと次に活かせない）

> 💡 ポイント: 数字がなくても「お客様の反応が良かった」「手間が30分減った」など、体感でもOKです。大事なのは「何かしら書き出すこと」です。

## AIに振り返りを手伝ってもらう

振り返りメモができたら、AIツールに分析と改善案を聞いてみましょう。ここでは ChatGPT または Claude を使います。

### ステップ1: 振り返りメモを送る

次のプロンプト（＝AIへの指示文）をコピーして、かっこ内をあなたのメモに書き換えて送信します。

```
私はネットショップを運営しています。
以下の実践結果を振り返って、改善アイデアを3つ提案してください。
それぞれ「なぜその改善が効果的か」も理由をつけてください。

【やったことと結果】
（ここにメモを貼る）
```

### ステップ2: 深掘りの質問をする

AIの最初の提案を受け取ったら、そのまま終わらずにもう1回質問しましょう。自分の状況に合った具体策を引き出すのがコツです。

```
ありがとうございます。この中で一番手軽に始められるものはどれですか？
私の店は【商品ジャンル】を扱っていて、スタッフは私1人です。
1週間以内にできる具体的な手順も教えてください。
```

**良い例:**
- 「売上が前月比で+12%だった。この伸びの要因をAIと一緒に分析する」
- AIの提案に対して「うちは1人運営なので、時間がかからない方法を優先したい」と条件を伝える

**悪い例:**
- 「もっと良くしたい」とだけ聞く（抽象的すぎて、抽象的な回答しか返ってこない）
- AIの提案をそのまま採用して、自分の状況に合うか検討しない

> 💡 ポイント: AIは「あなたのお店の事情」を知りません。商品ジャンル、スタッフ人数、使える時間など、条件を伝えるほど的確な提案が返ってきます。

## 改善プランを1つにまとめる

AIからの提案を参考に、次にやることを**1つだけ**決めます。欲張らず1つに絞るのが、着実に成果を出すコツです。

### 手順

1. AIの提案の中から「今すぐ着手できそうなもの」を1つ選ぶ
2. 選んだ改善策を「何を・いつまでに・どうやる」の3点セットで書く
3. 書いたプランをスクリーンショット（＝画面の写真）で保存する

![改善プランの記入例](/lesson-assets/atom.ec-operator.review-results-and-improve/screen_capture.png)

**良い例:**
- 「来週の金曜までに、商品説明文の見出しをAIで5パターン作り、一番クリックされそうなものに差し替える」
- 「3日以内に、問い合わせが多い質問トップ3をAIで回答テンプレート化する」

**悪い例:**
- 「商品説明を改善する」（何を・いつ・どうやるが不明で、結局動けない）
- 「AIの提案を全部やる」（多すぎて手がつけられない）

> 💡 ポイント: 改善プランは「完璧な計画」でなくて大丈夫です。小さな一歩を決めて動くことが大事です。次のふり返りでまた改善すればOK。

## 確認する

振り返りが終わったら、以下の3つをチェックしてください。すべてにチェックが入れば、このレッスンは完了です。

- [ ] やったことと結果を具体的に3つ以上書き出せた
- [ ] AIに改善案を聞き、さらに深掘り質問をして自分の状況に合った提案を引き出せた
- [ ] 「何を・いつまでに・どうやる」が入った改善プランを1つ作れた

## つまずきやすいポイント

| つまずき | 対処法 |
|---|---|
| 「結果が悪かった」と落ち込んでしまう | 悪い結果こそ改善のヒントです。「何が原因だったか」をAIと一緒に探ることが、まさにこのステップの目的です |
| AIの提案を全部やろうとする | まずは1つだけ選びましょう。1つ実行して振り返り、次の1つへ進むサイクルが一番確実です |
| 何を振り返ればいいかわからない | 「お客様からの反応」「かかった時間」「売上やアクセス数の変化」の3つだけ見れば十分です |
| AIの回答が抽象的すぎる | あなたのお店の具体的な情報（扱う商品、スタッフ数、予算）を追加で伝えてみましょう |
| 数字のデータがまったくない | 体感でOKです。「前より楽になった」「お客様の反応が良くなった」も立派な振り返り材料です |
