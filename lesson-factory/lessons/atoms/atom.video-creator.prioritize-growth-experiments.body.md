# 改善実験の優先順位を付ける

旅行の計画を立てるとき、行きたい場所が10あっても滞在は3日――「どこを優先するか」を決めますよね。動画づくりでも同じです。サムネイルを変えてみる、投稿時間をずらしてみる、動画の長さを短くしてみる……改善アイデアは次々浮かびますが、時間も体力も限られています。

この Atom では、**AIに相談しながら**「次にやるべき実験」を1つか2つに絞り込む方法を身につけます。最終的に「実験カード」というメモを1枚つくり、すぐ行動に移せる状態にします。

![優先順位付けフロー](/lesson-assets/atom.video-creator.prioritize-growth-experiments/diagram.png)

## 前提をそろえる

このレッスンを始める前に、次の2つを用意してください。

1. **改善アイデアのリスト**：あなたが「やってみたい」と思っている実験を3〜10個、箇条書きで書き出しておきます。メモアプリでもスプレッドシート（＝表計算ソフトのこと。Google スプレッドシートなど）でも構いません。
2. **現在の数字**：直近の動画の再生数やクリック率など、いまの成績が分かるものを手元に置きます。「何を改善したいか」の基準になります。

> まだアイデアが浮かんでいない場合は、まず「サムネイル」「タイトル」「動画の長さ」「投稿時間」の4つの観点から、それぞれ1つずつ思いつくものを書いてみましょう。

## AIにアイデアの壁打ちをする

アイデアを書き出したら、ChatGPT や Claude にそのリストを見せて「抜け漏れ」がないかチェックしてもらいましょう。以下のプロンプト（＝AIへの指示文）をコピーして、あなたのリストを貼り付けてください。

```
あなたは動画マーケティングの専門家です。
私はYouTubeチャンネルを運営しています。

【私の現状】
- 直近の動画の平均再生数：〇〇回
- チャンネル登録者数：〇〇人
- 一番伸びた動画の特徴：〇〇

【改善アイデアのリスト】
（ここにあなたのアイデアを貼る）

このリストを見て、以下を教えてください。
1. 見落としている改善アイデアがあれば3つまで追加してください
2. 各アイデアに「効果の高さ（1-5点）」と「すぐできる度（1-5点）」をつけて、表にまとめてください
3. 私が最初に試すべき実験を1つだけ選び、理由を教えてください
```

> **ポイント**：AIの回答はあくまで「たたき台」です。最終的に「これをやる」と決めるのはあなた自身です。AIのスコアを鵜呑みにせず、自分の感覚とすり合わせてください。

## アイデアにスコアを付ける

AIの回答も参考にしつつ、あなた自身の手でスコアを付けます。用意したアイデアリストの横に、2つの項目で1〜5点のスコアを付けましょう。

- **効果（効果が高そうか）**：これをやったとき、再生数やチャンネル登録者にどれくらい影響しそうか
- **手軽さ（すぐできそうか）**：今日から始められるか、特別な道具や費用が要らないか

各アイデアの2つのスコアを掛け算します（効果3 × 手軽さ4 ＝ 合計12点）。この合計点が高いものほど「すぐやる価値がある」という目安になります。

![スコアリング画面の例](/lesson-assets/atom.video-creator.prioritize-growth-experiments/screen_capture.png)

### 良い例・悪い例

**良い例**：
- 「サムネイルの文字を大きくする」→ 効果4・手軽さ5 ＝ **20点**
- 「動画の最初5秒で結論を言う」→ 効果4・手軽さ3 ＝ **12点**

**悪い例（避けたいスコア付け）**：
- 「新しいカメラを買う」→ 効果2・手軽さ1 ＝ **2点**（費用も手間も大きく、まずやるべきではない）
- 全部のアイデアに「効果5・手軽さ5」を付ける（差がつかず、絞り込めません）

> スコアは直感で構いません。「正確に測る」ことより「比べること」が目的です。

## 上位を選んで「実験カード」をつくる

スコア順にアイデアを並べ替えて、上位1〜2個を「次にやる実験」として決めます。

ここで大切なのは**一度にたくさんやらない**こと。料理で言えば、フライパンで同時に5つの料理を作ろうとすると、どれも焦げてしまいます。1つの実験に集中して結果を見るのがコツです。

選んだ実験を次の「実験カード」の形式でメモに書き残します。これが今回の成果物です。

```
【実験カード】
実験名：サムネイルの文字を大きくする
やること：次回の動画のサムネイルで、タイトル文字を今の1.5倍の大きさにする
スコア：効果4 × 手軽さ5 ＝ 20点
いつやる：〇月〇日の動画から
どう確認する：サムネイルのクリック率が前回より上がったか、1週間後に確認
成功の基準：クリック率が〇%から〇%に上がったら成功
```

> **AIに仕上げを頼むコツ**：実験カードを書いたら、ChatGPT や Claude に「この実験カードの内容に抜け漏れはありますか？」と聞いてみましょう。「成功の基準が曖昧です」「確認タイミングをもう少し具体的に」といったフィードバックがもらえます。

## 結果を確認する

実験を終えたら、次の2点をチェックします。

1. **数字は変わったか**：クリック率や再生数など、事前に決めた指標（＝測るための目安）が改善したか
2. **次の実験はどうするか**：結果が良ければ「この改善を続ける」、変わらなければ「別のアイデアを試す」と判断します

> 1回の実験で劇的に変わらなくても大丈夫。「変わりやすそうなこと」を少しずつ試していく積み重ねが成果につながります。

## つまずきやすいポイント

| つまずき | 対策 |
|---|---|
| アイデアが多すぎて絞れない | まず「手軽さ」が4以上のものだけ残してから考える |
| スコア付けで迷う | 直感で1〜5を付けて、後で調整する。完璧を目指さない |
| AIのスコアと自分の感覚が合わない | あなたの直感を優先する。AIはあくまで壁打ち相手 |
| 結果がすぐ出ない | 最低1週間は待つ。それでも分からなければ別の指標を見る |
| 一度に3つ以上試してしまう | 「今は1つだけ」と声に出してから始める |
| 実験カードを書いたのに行動に移せない | 「いつやる」を今日の日付にして、今すぐ最初の一歩だけ実行する |
