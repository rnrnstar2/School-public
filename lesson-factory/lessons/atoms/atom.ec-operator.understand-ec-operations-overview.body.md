# EC運営の全体像を理解する

レストランを思い浮かべてみてください。厨房で料理を作る人、ホールで注文を取る人、食材を発注する人、お客様の声を聞く人——みんなが役割を分担して、お店を回しています。

EC（＝インターネットで商品を売る仕組みのこと）も同じです。商品を用意する人、注文をさばく人、お客様と話す人、広告を出す人……。複数の役割が連携して、初めてお店が動きます。

このレッスンでは、EC運営に必要な**6つのエリア**を整理し、AIに相談しながら「あなたが今やるべきこと」をまとめたチェックリストを作ります。所要時間は約10〜15分です。

![EC運営の6エリア全体図](/lesson-assets/atom.ec-operator.understand-ec-operations-overview/diagram.png)

## 全体像を知る — EC運営の6つのエリア

EC運営は、大きく分けると次の6つのエリアで成り立ちます。

| # | エリア | ひとことで言うと | レストランで例えると |
|---|--------|------------------|----------------------|
| 1 | **商品管理** | 売るものを決めて、在庫を把握する | 食材の仕入れとメニュー決め |
| 2 | **受注・発送** | 注文が入ったら商品を梱包して送る | お料理を作ってテーブルに届ける |
| 3 | **顧客対応** | 問い合わせやクレームに答える | ホールでお客様の声を聞く |
| 4 | **集客** | お店に人を呼ぶ（SNS、広告、SEOなど） | チラシを配ったり看板を出す |
| 5 | **決済・送料** | 代金を受け取る仕組みと送料ルールを決める | レジとデリバリーの仕組み |
| 6 | **データ分析** | 売上や訪問者数を読んで次の手を考える | 日報を見て明日の仕込み量を決める |

それぞれのエリアは独立しているように見えますが、実際はつながっています。たとえば「集客」を頑張っても「受注・発送」が追いつかなければ、お客様を待たせてしまいます。

> 🍳 たとえるなら、料理の仕込み（商品管理）と接客（顧客対応）と宣伝（集客）を同時に回すのがEC運営。どれか一つが止まると、お店全体に影響が出ます。

## 自分の状況をAIに相談する

まずは、あなたが今どこまでできていて、どこが手薄かを知ることが大事です。AIツール（ChatGPTやClaudeなど）に次のような質問を投げてみましょう。

### やってみよう：AIに優先順位を聞く

以下のプロンプト（＝AIへの指示文）をコピーして、ChatGPTやClaudeに貼り付けてください。

```
ECサイトを運営しようとしています。
まだ何も始めていません。
EC運営の6つのエリア（商品管理、受注・発送、顧客対応、集客、決済・送料、データ分析）について、
最初に何から始めるべきか、優先順位と理由を教えてください。
各エリアの「最初の一歩」を具体的に1つずつ挙げてください。
```

**あなたの状況に合わせてカスタマイズするコツ：**

- すでに商品が決まっている場合 → 「売りたい商品は○○です」と1行追加する
- 予算に制限がある場合 → 「予算は月3万円以内です」と追加する
- 副業で始める場合 → 「本業があるので週末だけ作業できます」と追加する

### 良い例と悪い例

- ✅ **良い例**: 「ハンドメイドのアクセサリーを月10個くらい売りたいです。予算は月1万円以内で、土日だけ作業できます」と自分の条件を具体的に伝える
- ❌ **悪い例**: 「EC運営について教えて」とだけ聞く（条件がないので一般論しか返ってこない）

## AIでEC運営チェックリストを作る

それでは、AIを使って**あなたのEC運営チェックリスト**を作りましょう。これがこのレッスンの**成果物**です。

### ステップ1：AIにチェックリストを生成してもらう

次のプロンプトをAIツールに貼り付けて実行してください。

```
以下の形式で、EC運営の初心者向けに「EC運営チェックリスト」を書いてください。
各エリアごとに、最初にやるべきことを3つ以内で箇条書きにしてください。
それぞれの項目に「なぜ最初にやるべきか」の理由を1文で添えてください。

エリア：商品管理
エリア：受注・発送
エリア：顧客対応
エリア：集客
エリア：決済・送料
エリア：データ分析
```

### ステップ2：自分の状況に合わせて編集する

AIの回答をそのまま使うのではなく、以下のポイントで手を加えます。

1. **今月やることを3つだけ選ぶ** — 全部を同時にやろうとしない
2. **自分に関係ない項目は消す** — 例：実店舗がないなら「店頭受取」は不要
3. **具体的な数字を入れる** — 「商品を準備する」→「商品写真を5枚撮る」

もしAIの回答が抽象的すぎたら、こう聞き直してみましょう。

```
もっと具体的に教えてください。
私は個人でハンドメイト商品を売ろうとしている初心者です。
予算は月1万円以内、作業時間は週に5時間程度です。
この条件で、各エリアの「最初の一歩」を具体的なアクションで書き直してください。
```

### ステップ3：成果物を保存する

完成したチェックリストを保存します。

- **方法A**: メモアプリ（iPhoneのメモ、Googleドキュメントなど）にコピーして貼り付ける
- **方法B**: AIのチャット画面をスクリーンショットで保存する

### 良い例と悪い例

- ✅ **良い例**: AIの回答を読んで「今の自分には集客より先に商品写真の準備が必要だな」と判断し、優先順位を入れ替えて保存する
- ❌ **悪い例**: AIの回答をそのままコピーして保存して終わり（自分の状況に合っていない可能性がある）

## 確認する

次の3つのポイントを自分でチェックしてください。すべてにチェックが入れば、このレッスンは完了です。

- [ ] EC運営の6つのエリアの名前を見ずに言える（商品管理、受注・発送、顧客対応、集客、決済・送料、データ分析）
- [ ] 6つのエリアの中から「今の自分が最初に取り組むべきエリア」を1つ選び、その理由を一言で説明できる
- [ ] AIで作ったEC運営チェックリストを自分の状況に合わせて編集し、メモアプリなどに保存した

**確認のコツ**: 友人やAIに「EC運営って何をするの？」と聞かれたとき、6つのエリアを使って30秒で説明できれば合格です。

## つまずきやすいポイント

| つまずき | 原因 | 対策 |
|---|---|---|
| 6つのエリア全部を一度に始めようとしてしまう | 完璧にやりたい気持ちが先走る | まずは「商品管理」と「受注・発送」の2つだけに絞る。他は後回しでOK |
| AIの回答が抽象的すぎて使えない | 質問に具体的な条件を入れていない | 「個人運営・予算3万円以内・週5時間」のように条件を追加して聞き直す |
| どのエリアから手を付けるか迷う | 判断基準がない | 売る商品が決まっていなければ「商品管理」から。商品が決まっていれば「受注・発送」の準備から |
| AIに何を聞けばいいかわからない | プロンプトの書き方に慣れていない | このレッスンのプロンプト例をそのままコピペして、太字の部分だけ自分の状況に書き換える |
