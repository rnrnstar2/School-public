# 研修中のQ&A対応をAIで支援する

研修で受講者から質問が飛んできたとき、「えっと、どう答えよう…」と困った経験はありませんか？
このレッスンでは、AIチャットツール（＝ChatGPTやClaudeなどの、文章で対話できるAI）を"耳元のアシスタント"として使い、質問にスピーディに答えるコツを15分で身につけます。

たとえて言うなら、研修は「ライブの料理教室」です。生徒から「この代用食材で大丈夫ですか？」と聞かれたとき、隣にいるシェフの先生に小声で確認してから自信を持って答える——そんなイメージでAIを活用します。

![Q&A支援フロー](/lesson-assets/atom.training-designer.realtime-qa-support/diagram.png)

## 前提を確認する

このレッスンを進める前に、次の2つを用意してください。

- **AIチャットツールのアカウント** — ChatGPT、Claude、Microsoft Copilot など、無料で始められるもので大丈夫です
- **研修の資料** — スライドのテキストや配布資料の要点（あとでAIに教えるために使います）

プログラミングの知識は一切不要です。ブラウザでAIチャットを開ける環境があればOKです。

## 準備する — AIに研修の背景を伝える

研修が始まる前に、AIに「あなたはこの研修のアシスタントです」と教えておきましょう。このような事前の指示をプロンプト（＝AIへの指示文）と呼びます。

次の文をコピーして、AIチャットの入力欄に貼り付け、「［　］」の中を自分の研修内容に書き換えてから送信してください。

```
あなたは研修のアシスタントです。
以下の情報をもとに、受講者からの質問に答える手伝いをしてください。

・研修テーマ：［ここにテーマを書く。例：新入社員のビジネスマナー］
・受講者のレベル：［例：初心者／中級者／業務経験3年目以上］
・研修のゴール：［例：名刺交換・電話対応・メール作成を一通りできるようになる］
・回答のルール：平易な日本語で、専門用語には説明を添えてください。回答は3文以内で簡潔にお願いします。
```

**良い例：**
> あなたは研修のアシスタントです。研修テーマ：新入社員のビジネスマナー。受講者のレベル：初心者。研修のゴール：名刺交換と電話対応の基本を身につける。回答は平易な日本語で、3文以内で。

**悪い例：**
> 質問に答えて。（← 文脈が足りず、的外れな回答になりやすいです）

### さらに精度を上げる — 資料を読み込ませる

もし研修資料がテキストデータとして手元にあるなら、プロンプトの後にそのまま貼り付けてみましょう。AIが「教科書の中身」を理解した状態で質問に答えてくれるので、回答の精度がぐっと上がります。

```
【研修資料ここから】
（スライドや配布資料のテキストを貼り付ける）
【研修資料ここまで】

この資料の内容を踏まえて、受講者の質問に答えてください。
```

資料が長い場合は、今日使うスライドの部分だけでOKです。全部入れなくても十分効果があります。

## 質問をAIに投げる

研修中、受講者から質問が来たら、AIに次のように入力します。

```
受講者からの質問：「［質問内容をそのまま貼る／打つ］」
この質問に、研修の文脈に沿って3文以内で答えてください。
```

ポイントは「質問をそのまま入力する」こと。自分で言い換えると、元のニュアンスが変わってしまうことがあります。

**良い例：**
> 受講者からの質問：「名刺交換は右手で渡すのが正しいですか？」この質問に、研修の文脈に沿って3文以内で答えてください。

**悪い例：**
> 名刺の渡し方を教えて（← 質問のニュアンスが消え、的確な回答が得られません）

### 想定外の質問が来たときは

研修テーマから外れた質問が来ることもあります。そんなときは次のように聞きましょう。

```
受講者から研修テーマとやや外れた質問が来ました：「［質問内容］」
この質問が研修テーマと関連するなら簡潔に答えてください。関連が薄い場合は「この研修の範囲外ですが…」と前置きしたうえで一言だけ答えてください。
```

## 回答を確認して伝える

AIからの回答をそのまま読み上げるのは避けましょう。必ず一度目で確認してから、あなたの言葉で伝えます。これは「料理教室のたとえ」でいえば、シェフの助言をそのまま繰り返すのではなく、生徒がわかるように自分の言葉に直してから伝える、というステップです。

確認する3つのポイント：

1. **事実は正しいか** — 自分の知識や手元の資料と照らし合わせる。AIは自信満々に間違えることがあります
2. **トーンは適切か** — 丁寧すぎたり、逆にフランクすぎたりしないか
3. **研修のテーマに合っているか** — 関係のない話に逸れていないか

もし少しでも怪しいと感じたら、AIに「この回答の根拠を教えてください」と追加で聞いてみましょう。根拠を説明できない場合は、その情報は使わないのが安全です。

## 検証する — 自分のQ&A対応をふり返る

次のチェックリストで、あなたのQ&A支援が機能しているか確認しましょう。研修終了後、もしくは休憩時間に1分でチェックできます。

- [ ] AIに研修テーマとゴールを事前に伝えた
- [ ] 受講者の質問をそのままAIに入力できた
- [ ] AIの回答を自分の目で確認してから受講者に伝えた
- [ ] AIの回答が研修テーマとズレていなかった
- [ ] 回答に事実誤認がなかった（自分で確かめられた）
- [ ] 受講者に分かりやすい言葉で伝えられた
- [ ] 1問あたり2分以内で回答できた

4つ以上チェックがつけば、この方法はうまく回っています。3つ以下の場合は、次の「つまずき対策」を確認してみてください。

## つまずき対策

| つまずき | 原因 | 対処法 |
|---|---|---|
| AIの回答が的外れ | プロンプトに背景情報が不足 | 「研修テーマ：〇〇、ゴール：〇〇」をプロンプトに追加し直す |
| 回答が長すぎる | 文字数制限を指定していない | 「3文以内で」「100文字以内で」と制限を明記する |
| 専門用語が多い | 受講者レベルの指定が不足 | 「中学生にもわかる言葉で」と一文追加する |
| AIが「わかりません」と返す | 質問があいまいすぎる | 質問に背景を補足して「〇〇の場面で、△△について聞かれています」と状況を追加する |
| 回答の事実が怪しい | AIの知識が古い・不正確 | 「この回答の根拠を教えてください」と聞き、根拠が示せなければ使わない |
| 入力に時間がかかる | タイピングが追いつかない | スマホの音声入力を使う。質問をそのままマイクに向かって話すだけ |

## 成果物を確認する

このレッスンの成果物は、**あなたが作ったプロンプトのスクリーンショット**です。

1. AIチャット画面で、あなたが送信したプロンプト（研修テーマ・ゴール・回答ルールを含むもの）が表示されている状態にする
2. その画面のスクリーンショットを撮る
3. スクリーンショットの中に「研修テーマ」「回答ルール」が含まれていればOK

これが、あなたが「研修中にAIをQ&Aアシスタントとして使える準備ができた」証拠になります。
