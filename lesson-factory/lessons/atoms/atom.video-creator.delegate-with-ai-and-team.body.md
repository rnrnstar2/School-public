# AIと外注で分業する

レストランのキッチンを想像してみてください。シェフが一人で野菜を切り、ソースを作り、皿盛りまで全部やっていたら、1日に作れる料理は限られますよね。でも、野菜担当・ソース担当・盛り付け担当と分ければ、同じ時間で何倍もの料理が出せます。

動画制作も同じです。あなたが一人で全部やるより、AI（＝人工知能を使った便利ツール）と外注スタッフに作業を振り分けたほうが、より多くの質の良い動画を作れます。

このレッスンでは、「自分の作業をどう分けるか」「どの作業をAIに任せ、どれを外注するか」を15分で考え、実際に **分業計画** を1つ書き上げます。完了すると「次の動画から、誰に何を頼むか迷わない」状態になれます。

![分業フロー図](/lesson-assets/atom.video-creator.delegate-with-ai-and-team/diagram.png)

## 前提を確認する

このレッスンを進める前に、以下を用意してください。

- あなたが今制作している（または制作予定の）動画のテーマが1つ決まっていること
- ChatGPTやClaude（＝AIと対話できるウェブサービス）にブラウザからアクセスできること

特別なソフトウェアのインストールは不要です。メモアプリやノートが手元にあると便利です。

## 自分の作業を書き出す

まず、あなたの動画制作でやっている作業をすべて書き出しましょう。紙とペンでも、メモアプリでも構いません。

典型的な動画制作の作業例:

| 作業 | 内容の例 |
|------|----------|
| テーマ決め | 次に何を撮るか考える |
| 台本・原稿作成 | 話す内容を文字で書く |
| 撮影 | カメラの前で話す・映す |
| 編集（映像・音声） | いらない部分を切る、音を整える |
| サムネイル作成 | 動画の「表紙画像」を作る |
| 説明文の作成 | 概要欄に載せるテキストを書く |
| 投稿・公開 | 動画をアップロードする |

あなた独自の作業があれば、ぜひ追加してください。たとえば「BGM選び」「ロケハン（撮影場所の下見）」なども立派な作業です。

## 各作業を「AI向き」「外注向き」「自分向き」に分ける

書き出した作業を3つのグループに分けます。判断の目安は次の通りです。

**AI向きの作業** — コンピューターが得意なこと:

- 文字をたくさん書く（台本、説明文、ハッシュタグ案）
- アイデアを大量に出す（テーマ案、タイトル候補を10個出す、など）
- 下書きを要約・リライトする

**外注向きの作業** — 人間の専門スキルが必要なこと:

- 専門的な技術がいる（高度な映像編集、アニメーション、音楽作曲）
- 時間がかかる手作業（テロップ入れ、字幕の翻訳）

**自分向きの作業** — あなたにしかできないこと:

- あなたの顔や声が必要なもの（撮影、ナレーション）
- 最終的な方向性を決める（企画の承認、完成品のOK出し）
- あなたらしさが出るもの（企画のコンセプト、番組の世界観）

![作業分類の参考画面](/lesson-assets/atom.video-creator.delegate-with-ai-and-team/screen_capture.png)

### 良い例と悪い例

**良い例:**

- 「台本作成 → AIに任せる。プロンプト（＝AIへの指示文）に番組のトーンと長さを指定する」
- 「テロップ入れ → 外注する。動画ファイルとテロップ原稿を一緒に渡す」

**悪い例:**

- 「全部AIにやらせる」→ 動画にあなたらしさが残りません
- 「外注に丸投げする」→ 方向性がブレてやり直しが増えます

## AIへの指示文（プロンプト）を書く

AIに任せると決めた作業について、具体的な指示文を1つ書いてみましょう。

コツは **「何を」「どのくらい」「どんな雰囲気で」** の3点を必ず書くことです。この3点があるだけで、AIの出力がグッと良くなります。

### 台本作成を頼む場合のプロンプト例

```
YouTube動画の台本を書いてください。
テーマ: 「初心者向けの料理入門」
長さ: 5分程度（約1,500文字）
トーン: 明るく親しみやすい、友達に教えるような口調
話者: 一人称「私」
構成: オープニング → 本題3ポイント → まとめ
```

### サムネイルのキャッチコピーを頼む場合のプロンプト例

```
YouTubeサムネイル用のキャッチコピーを5案出してください。
動画テーマ: 「初心者向けの料理入門」
文字数: 10文字以内
狙い: 思わずクリックしたくなるもの
```

### 良い例と悪い例

**良い例:**
「5分の料理入門動画の台本を、明るいトーンで書いて。一人称は『私』で、構成はオープニング→本題→まとめ。」

**悪い例:**
「台本書いて」→ 何のテーマか、長さもトーンも分からず、使えない台本が出てきます

> **ヒント**: AIの出力が「なんか違うな」と感じたら、指示文に情報を足して再度お願いしましょう。一発で完璧な答えが出ることは少ないです。2〜3回やり取りするのが普通です。

## 外注先に依頼内容をまとめる

外注する作業について、依頼内容を短くまとめましょう。最低限、次の3点を含めます。

1. **成果物**: 何を作ってほしいか（例:「5分動画にテロップを入れる」）
2. **素材**: どんなものを渡すか（例:「動画ファイル(mp4)とテロップ原稿(txt)」）
3. **期限**: いつまでに必要か（例:「3営業日以内」）

この3点セットは「依頼の三点セット」と覚えておくと便利です。

### 良い例と悪い例

**良い例:**
「添付の動画(5分)に、テロップ原稿に沿ってテロップを入れてください。フォントは丸ゴシック、サイズは大きめでお願いします。参考動画のリンクも添付します。期限は12月20日です。」

**悪い例:**
「いい感じにテロップお願いします」→ 何をどう作るか分からず、やり直しの原因になります

> **ヒント**: 外注の仕上がりを自分のイメージに近づけるには、参考画像や参考動画を一緒に渡すのが最も効果的です。「こんな感じで」が一目でわかります。

## 分業計画を確認する

書き出した分類と指示文・依頼内容をまとめて、1枚の分業計画として確認しましょう。

チェックポイント:

- [ ] 全作業がいずれかのグループ（AI・外注・自分）に入っている
- [ ] AIへの指示文に「何を・どのくらい・どんな雰囲気で」が書かれている
- [ ] 外注依頼に「成果物・素材・期限」の三点セットが揃っている
- [ ] 「自分向き」の作業に、あなたにしかできないことが入っている
- [ ] 丸投げになっている作業がない（AIにも外注にも「確認ポイント」を設定した）

すべてチェックが付いたら、この分業計画が今回のレッスンの成果物です。

## つまずきやすいポイントを知る

- **「AIが書いた文章がなんか違う」**: 指示文に「トーン」「話者」「構成」が抜けていることが多いです。追加して書き直させましょう。「もっと明るい感じで」と追加指示するだけでもかなり変わります
- **「外注の仕上がりがイメージと違う」**: 参考画像や参考動画を一緒に渡すと、認識のズレがぐっと減ります。「言葉で伝える」より「見せる」方が早いです
- **「何を自分でやるべきか迷う」**: 迷ったら「自分の顔・声が入るか？」「最後のOKを出すのは誰か？」の2つで判断してください。この2つに当てはまるなら自分の担当です
- **「AIと外注、どちらに頼むか迷う」**: テキスト中心の作業はまずAIに試すのがおすすめです。費用がかからず、何度でもやり直せます。それで足りなければ外注を検討しましょう

## 復習する

このレッスンで身についたことを確認しましょう。

- 動画制作の作業を書き出し、AI・外注・自分の3つに分けられた
- AIへの指示文（プロンプト）に「何を・どのくらい・どんな雰囲気で」を含められた
- 外注先への依頼に「成果物・素材・期限」の三点セットを揃えられた

この3つができていれば、このレッスンはクリアです。次の動画制作から、さっそく今日作った分業計画を使ってみてください。
