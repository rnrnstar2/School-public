# 商品説明文をAIで量産する

「この商品、どう紹介すればいいんだろう？」——フリマアプリやネットショップに商品を出品するとき、説明文を書くのが苦手な人は多いですよね。でも実は、AIに「こんな商品なんだけど、魅力的に紹介して」とお願いすると、プロ級の説明文が数十秒で仕上がります。

この Atom では、あなたが手元の商品情報をAIに渡して、**そのまま使える説明文を3パターン作る**ところまで体験します。所要時間は約15分です。

## 理解する：なぜAIで商品説明文を書くのか

商品説明文を書く作業は、料理のレシピを書くのに似ています。材料（商品の特徴）を並べて、おいしそうに見せる（魅力を伝える）のがゴールです。

AIが得意なのは、この「おいしそうに見せる」部分です。あなたが材料リスト（商品名・特徴・ターゲット）を渡せば、AIが読み手の心に刺さる文章に組み立ててくれます。

**AIを使うメリット**
- 1商品あたり数十秒で説明文ができる（手書きなら10〜30分）
- 「カジュアル向け」「ビジネス向け」など複数トーンを一度に作れる
- 書き出しに悩む時間がゼロになる

## 準備する：始める前に用意するもの

- **AIツール**: ChatGPT、Claude、またはCursorのいずれか（無料プランでOK）
- **商品情報メモ**: 以下の4項目を事前にメモしておく
  1. 商品名
  2. 特徴やセールスポイント（3つ程度）
  3. 想定する買い手（例: 30代女性、ビジネスマンなど）
  4. 価格帯（任意）

> 💡 練習用の商品がない場合は、身の回りのもの（例: 使っているマグカップ、お気に入りの文房具）で試してみましょう。

## 実践する：AIに商品説明文を書いてもらう

![商品説明文生成の流れ](/lesson-assets/atom.ai-freelancer.product-description-ai/diagram.png)

### ステップ1：商品情報をプロンプトにまとめる

AIへの指示文（プロンプト）に、先ほどメモした商品情報を組み込みます。以下のテンプレートをコピーして、【 】の部分を自分の商品に書き換えてください。

```
あなたはECサイトのプロのコピーライターです。
以下の商品情報をもとに、商品説明文を作成してください。

■ 商品名: 【ステンレス製トラベルマグ 350ml】
■ 特徴:
- 【真空断熱で6時間保温】
- 【片手で開閉できるワンタッチ蓋】
- 【食洗機対応で手入れ簡単】
■ ターゲット: 【通勤・通学する20〜40代】
■ トーン: カジュアルで親しみやすい

200文字以内で、購入意欲が湧く説明文を書いてください。
```

**良い例（具体的で情報が揃っている）：**
> 商品名: ステンレス製トラベルマグ 350ml / 特徴: 真空断熱6時間、ワンタッチ蓋、食洗機OK / ターゲット: 通勤する30代

**悪い例（情報が曖昧すぎる）：**
> 「マグカップの説明文を書いて」——これだけでは、AIはどんなマグか分からず、ありきたりな文章しか出せません。

### ステップ2：AIの出力を確認して調整する

AIが出した文章をそのまま使うのではなく、以下の3点をチェックします。

1. **事実と合っているか**: 書いていない機能が追加されていないか確認する
2. **ターゲットに合った言葉遣いか**: カジュアルすぎ・堅すぎがないか
3. **文字数は適切か**: 指定した文字数の±20%以内に収まっているか

修正が必要な場合は、AIに追加で指示を出します。

```
もう少しフォーマルなトーンに変えてください。
「真空断熱」という言葉を冒頭に持ってきてください。
```

> 💡 **コツ**: 一度に大幅に変えようとせず、1回の指示で1つの修正だけ頼むとうまくいきます。

### ステップ3：3パターンのバリエーションを作る

1つの商品に対して、異なる切り口の説明文を3つ作ると、どれが一番反応が良いかテストできます。

以下のプロンプトをAIに送ってください。

```
先ほどの商品について、以下の3パターンで説明文を作ってください。

パターンA: 機能・スペックを重視した説明
パターンB: 使用シーンをイメージさせる説明
パターンC: 他商品との比較で優位性を伝える説明

それぞれ200文字以内でお願いします。
```

## つまずきポイントを知る

| よくある問題 | 原因 | 解決法 |
|---|---|---|
| 出力が長すぎる | 文字数指定を忘れた | 「200文字以内で」と明記する |
| 嘘の情報が混ざる | AIが知らない情報を補完した | 「記載した情報のみ使用してください」と追記する |
| 全パターンが似ている | 切り口の指示が曖昧 | パターンごとに「機能重視」「シーン重視」など具体的に指定する |
| 日本語が不自然 | 翻訳調になっている | 「自然な日本語の口語体で」と指定する |

## もっと活用するヒント

- **テンプレート化**: よく使うプロンプトをメモアプリに保存しておくと、次回から商品名を変えるだけで使い回せます
- **フリマアプリ向け**: メルカリやラクマ用なら「送料込み」「即購入OK」などのキーワードを追加指示するとさらに実用的です
- **量産のコツ**: 複数商品を一度に処理したい場合は「以下の5商品についてそれぞれ説明文を書いてください」とまとめて依頼できます
