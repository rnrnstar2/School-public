# カテゴリ文とメタ文を作る

## このレッスンでできるようになること

オンラインショップには「カテゴリページ」（商品を種類ごとにまとめたページ）があります。たとえば「キッチン雑貨」「アクセサリー」「ベビー用品」といったページです。

このレッスンでは、AIツールを使ってカテゴリページに必要な **2つの文章** を作ります。

- **カテゴリ文**：カテゴリページの上に表示する説明文
- **メタ文**（メタディスクリプション＝Google検索の結果に出る短い紹介文）

レストランにたとえるなら、カテゴリ文は「メニューの各セクションにある説明書き」、メタ文は「お店の外に出す看板」です。どちらもお客さんに「ここに欲しいものがありそう」と思ってもらうための文章です。

所要時間は **約15分** です。

## 前提を確認する

- AIチャットツール（ChatGPT・Claude・Geminiなど、どれか1つ）が使える状態になっている
- 自分のショップのカテゴリ名が決まっている（例：キッチン雑貨、アクセサリーなど）
- そのカテゴリにどんな商品があるか、3〜5個ほど思い浮かぶ

## カテゴリ文とメタ文の違いを理解する

![カテゴリ文とメタ文の違い](/lesson-assets/atom.ec-operator.write-category-and-meta-copy/diagram.png)

| | カテゴリ文 | メタ文 |
|---|---|---|
| **どこに表示される？** | カテゴリページの上部 | Google検索の結果一覧 |
| **文字数の目安** | 100〜200文字 | 70〜120文字 |
| **役割** | 「ここにどんな商品があるか」を伝える | 「クリックして見に行きたい」と思わせる |

### 良い例と悪い例を見る

**カテゴリ文の良い例：**
> 毎日使うキッチン雑貨を集めました。木の温もりを感じるカッティングボードから、食卓を彩る陶器のプレートまで。職人が丁寧に作った品々をお届けします。

**カテゴリ文の悪い例：**
> キッチン雑貨のカテゴリです。いろんな商品があります。

なぜ悪い？ → 「いろんな商品」では何があるかわかりません。良い例のように **具体的な商品名** と **魅力を伝える言葉** を入れるのがコツです。

**メタ文の良い例：**
> 毎日使いたくなるキッチン雑貨を豊富に取り揃え。木のカッティングボード、手作り陶器など職人品をお届け。

**メタ文の悪い例：**
> キッチン雑貨なら当店にお任せください。多数取り揃えております。

なぜ悪い？ → 「お任せ」「多数」では何を売っているか伝わりません。検索する人は **具体的な商品名** を見てクリックするか判断します。

## ステップ1：AIにカテゴリ文を作ってもらう

AIチャットツール（ChatGPT、Claude、Geminiなど）を開いて、以下のプロンプト（＝AIへの指示文）をコピーして貼り付けてください。

`[ ]` の中身だけ、あなたのショップに合わせて書き換えます。

```
オンラインショップのカテゴリページ用の説明文を3パターン作ってください。

カテゴリ名：[例：キッチン雑貨]
扱う商品の例：[例：木のカッティングボード、陶器のプレート、ガラスのグラス]
ターゲット層：[例：30代〜50代の料理好きな方]
お店の雰囲気：[例：温かくて丁寧、ナチュラルな暮らし]

条件：
- 100〜200文字
- 具体的な商品カテゴリや素材を含める
- 読者が「ここで買いたい」と思う表現にする
- 架空の商品名は使わない
```

### AIの出力を選んで微調整する

3パターン出てきたら、**一番しっくりくるもの** を1つ選びます。選ぶときの判断基準はこの3つです。

1. **自分のショップの雰囲気に合っている**（読み上げて違和感がないか）
2. **具体的な商品や素材が入っている**（「いろいろ」「多数」だけになっていないか）
3. **長すぎない**（200文字を超えていたら、AIに「150文字以内に短くして」と頼む）

選んだ文をさらに良くする一手間：

- 実際の商品名があれば差し替える（例：「木のカッティングボード」→「オリーブウッドのカッティングボード」）
- お店ならではのキーワードを1つ足す（例：「手作り」「無添加」「北海道産」）

> **💡 AIに修正を頼むプロンプト例：**
> 「2番目の案をベースに、商品名を〇〇と△△に差し替えて、150文字以内にまとめてください」

## ステップ2：AIにメタ文を作ってもらう

続けて、同じチャット画面で以下を貼り付けます。

```
以下のカテゴリ文をもとに、Google検索結果用のメタディスクリプション（紹介文）を3パターン作ってください。

カテゴリ文：
[ステップ1で完成したカテゴリ文を貼る]

条件：
- 70〜120文字
- 検索した人が「このページを見てみよう」と思う文にする
- 具体的な商品カテゴリ名を含める
- 文末は「。」で終える
```

3パターンから1つ選びます。判断のコツ：

- **120文字を超えていないか** → 超えると検索結果で途中で切れてしまいます
- **カテゴリ名や商品名が入っているか** → 検索する人が「これだ」と思える具体性が大事です

## 成果物を確認する

以下の2つが手元にできていれば完了です。

1. **カテゴリ文**（100〜200文字）：カテゴリページに貼れる完成形
2. **メタ文**（70〜120文字）：管理画面のメタディスクリプション欄に貼れる完成形

### 完了チェックリスト

- [ ] カテゴリ文に具体的な商品名や素材名が含まれている
- [ ] カテゴリ文が200文字以内におさまっている
- [ ] メタ文が120文字以内におさまっている
- [ ] どちらもあなたのショップのトーン（雰囲気）と合っている
- [ ] 声に出して読み上げて違和感がない
- [ ] カテゴリ文とメタ文の両方をテキストファイルやメモに保存した

## つまずいたときの対処法

| こんなとき | こうする |
|---|---|
| AIが長すぎる文を出す | 「〇〇文字以内で書き直して」と追加で指示する |
| 架空の商品名が入ってしまう | 「架空の商品名は使わず、カテゴリ名と素材名だけで書いて」と指示する |
| どのパターンを選ぶか迷う | 声に出して読んでみて、一番自然に聞こえるものを選ぶ |
| AIの文が硬すぎる・柔らかすぎる | 「もう少しカジュアルに」「もう少し丁寧に」とトーンを指定する |
| メタ文がカテゴリ文と似すぎる | 「カテゴリ文とは違う切り口で、検索する人の視点から書いて」と頼む |
