# 副業の法務・税務の基本を押さえる

「副業で稼ぎ始めたけど、税金ってどうすればいいの？」――これは副業を始めたほとんどの人が最初にぶつかる壁です。たとえるなら、料理を始めたのに食材の保存方法を知らないようなもの。せっかく作った成果物（＝収入）を台無しにしないために、最低限の「保存ルール」を知っておきましょう。

この Atom では、AIチャット（ChatGPT や Claude など）を使って、自分の状況に合った法務・税務の基本チェックリストを作ります。15分で「何を調べればいいか」がわかる状態を目指します。

## 知っておくべき3つの基本ルールを確認する

副業をするうえで、まず押さえるべきルールは次の3つです。

![法務・税務の3つの基本ルール](/lesson-assets/atom.ai-freelancer.legal-tax-basics/diagram.png)

### 1. 確定申告（＝1年間の収入と経費を国に報告すること）

- **20万円ルール**: 副業の所得（収入から経費を引いた金額）が年間20万円を超えたら、確定申告が必要です
- **所得税の確定申告が不要でも、住民税の申告は必要**な場合があります

**良い例**: 「副業収入50万円 − 経費15万円 ＝ 所得35万円 → 確定申告が必要」

**悪い例**: 「副業で30万円もらったけど、20万円以下じゃないから関係ないや」（経費を引く前の金額で判断してしまっている）

### 2. 開業届（＝「副業を始めました」と税務署に届けること）

- 提出は義務ですが、出さなくても罰則はありません
- ただし、**青色申告（＝節税できる申告方法）を使うには開業届が必須**です
- 開業届は国税庁のサイトから無料で提出できます

### 3. 契約まわりの基本

- クライアントとの仕事では、**業務委託契約書**を交わすのが安全です
- 口約束だけで始めると、報酬の未払いやトラブル時に困ります
- 契約書のひな型はAIに作ってもらうことができます（後述）

## AIに自分の状況を聞いてチェックリストを作る

ここからが本題です。ChatGPT や Claude を開いて、以下のプロンプト（＝AIへの指示文）を貼り付けてみましょう。

### プロンプト例①：確定申告が必要か確認する

```
私は会社員で、副業としてAIを使ったライティングの仕事を始めました。
以下の条件で、確定申告が必要かどうか教えてください。

- 本業の年収: 約400万円
- 副業の年間収入見込み: 約30万円
- 副業の経費（ツール代・通信費など）: 約5万円
- 副業の形態: 個人でクラウドソーシング経由

確定申告が必要な場合、いつまでに何をすればいいか、
初心者向けに手順を箇条書きで教えてください。
```

**ポイント**: 「約○万円」の部分をあなたの実際の金額に書き換えてください。AIは具体的な数字があるほど正確に答えられます。

### プロンプト例②：開業届を出すべきか判断する

```
副業でAIを使ったデザイン・ライティングの仕事をしています。
以下の状況で、開業届を出すメリット・デメリットを教えてください。

- 月の副業収入: 約3〜5万円
- 会社の副業規定: 届出制（届ければOK）
- 今後の見通し: 半年以内に月10万円を目指したい

メリットとデメリットを表形式でまとめて、
最終的にどちらがおすすめか理由とともに教えてください。
```

### プロンプト例③：契約書のひな型を作る

```
フリーランスとしてAIを使ったSNS投稿文作成の仕事を受けます。
以下の条件で、業務委託契約書のひな型を作ってください。

- 業務内容: SNS投稿文の作成（月20本）
- 報酬: 月額5万円（税込）
- 納品方法: Googleドキュメントで共有
- 契約期間: 3ヶ月（更新あり）
- 修正回数: 各投稿につき2回まで

初心者でもわかるように、各条項に簡単な解説コメントを
【】で付けてください。
```

## AIの回答を「たたき台」として使うコツを知る

AIが出してくれた回答は、あくまで**たたき台（＝最初の下書き）**です。以下の点を意識してください。

### やるべきこと

- **自分の状況に合っているか確認する**: AIは一般論を返すことが多いので、「自分のケースに当てはまるか」を必ずチェック
- **最新の情報か確認する**: 税制は毎年変わります。AIの回答にある数字や期限を、国税庁のサイト（nta.go.jp）で裏取りする
- **迷ったら専門家に相談する**: AIのチェックリストを持って税理士の無料相談に行くと、話がスムーズに進みます

### やってはいけないこと

- **AIの回答をそのまま確定申告に使う**: AIは法的責任を取れません。最終判断は自分か専門家で
- **契約書をAI出力のまま使う**: 必ず相手と確認し、できれば行政書士や弁護士にチェックしてもらう

## 成果物を仕上げる

以下の3つをMarkdownファイル（または普通のテキストファイル）にまとめて保存してください。これがこの Atom の成果物です。

1. **確定申告チェックリスト**: AIに聞いた結果をもとに、自分に必要な手続きを箇条書きでまとめたもの
2. **開業届の判断メモ**: 出す／出さないの判断と、その理由を1〜2行で書いたもの
3. **契約書テンプレートの保存先**: AIが生成した契約書ひな型をどこに保存したかのメモ

**良い例（成果物の書き方）**:
```
# 副業 法務・税務チェックリスト

## 確定申告
- 2026年の副業所得見込み: 約25万円（確定申告が必要）
- 申告期限: 2027年3月15日まで
- 必要書類: 源泉徴収票、経費の領収書、マイナンバーカード

## 開業届
- 判断: 提出する（青色申告で節税したいため）
- 提出先: 最寄りの税務署 or e-Tax

## 契約書テンプレート
- 保存先: Google Drive > 副業フォルダ > 契約書ひな型.docx
```

**悪い例**: AIの回答をコピペしただけで、自分の状況に合わせていないもの

- [ ] 確定申告が必要かどうか、自分の所得見込みで判断できた

## つまずきやすいポイントと対処法

| つまずき | 対処法 |
|---------|--------|
| 「所得」と「収入」の違いがわからない | 収入＝もらった金額の合計、所得＝収入から経費を引いた金額。AIに「所得と収入の違いを具体例で教えて」と聞いてみましょう |
| 経費にできるものがわからない | AIに「副業ライターが経費にできるものリスト」と聞くと一覧が出ます。ただし最終判断は税務署か税理士に確認を |
| 会社にバレるのが心配 | 確定申告時に住民税を「自分で納付（普通徴収）」にすると、会社の給与天引きに反映されません。AIに「副業 住民税 普通徴収 やり方」と聞いてみましょう |
| 契約書の内容が正しいか不安 | AIに「この契約書で不利になりそうな条項を指摘して」と追加で聞くと、リスクポイントを教えてくれます |
