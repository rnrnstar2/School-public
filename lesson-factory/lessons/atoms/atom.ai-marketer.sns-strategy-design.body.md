# SNS運用方針を設計する

あなたのビジネスや活動をSNSで広めたいけれど、「何を」「どこで」「どれくらいの頻度で」発信すればいいのか迷っていませんか？

このレッスンでは、AIチャット（ChatGPTやClaude）を相談相手にしながら、**SNS運用方針シート**を1枚つくります。料理にたとえるなら、レシピ（＝運用方針）を先に決めてから買い物（＝投稿作成）に行くイメージです。方針がないまま投稿を始めると、毎回ゼロから考えることになり疲れてしまいます。

完成すると「次に何を投稿すればいいか」が自分で判断できるようになります。所要時間は約15分です。

---

## 前提を確認する

- SNSアカウント（Instagram、X／Twitter、TikTokなど）をひとつ以上持っている、または作る予定がある
- AIチャットツール（ChatGPT、Claude、Geminiなど）をブラウザで開ける
- 運用方針シートを書くためのメモアプリやGoogleドキュメントを用意できる

---

## ステップ1：ターゲット読者を言語化する

まず「誰に届けたいか」をはっきりさせます。ここが曖昧だと、投稿の方向性がブレてしまいます。

### AIに聞いてみる

以下のプロンプト（＝AIへの指示文）をコピーして、ChatGPTやClaudeに貼り付けてください。【】の中はあなた自身の情報に書き換えます。

```
私は【あなたの職業・活動内容、例：フリーランスのWebデザイナー】です。
SNSで集客したいターゲットを考えています。
以下の情報をもとに、理想的なターゲット像を3パターン提案してください。
- 提供しているサービス：【例：小規模事業者向けのホームページ制作】
- 現在の顧客層：【例：30〜40代の個人事業主が多い】
- SNSで達成したいこと：【例：月に2件の問い合わせを得たい】
```

AIが3パターンを出してくれたら、いちばんしっくりくるものを1つ選んでメモしてください。

### 良い例と悪い例

- **良い例**：「30代の女性個人事業主で、自分でホームページを作りたいが時間がない人」→ 具体的なので投稿内容を決めやすい
- **悪い例**：「ビジネスに興味がある人」→ 広すぎて何を投稿すればいいかわからない

---

## ステップ2：発信テーマを3つ決める

次に「何について発信するか」のテーマを決めます。テーマがあると、ネタ切れになりにくくなります。

### AIに聞いてみる

```
ターゲット：【ステップ1で決めたターゲット像を貼り付け】

このターゲットに響くSNS発信テーマを5つ提案してください。
それぞれ、なぜそのテーマが有効かの理由も1行で添えてください。
```

AIが5つ提案してくれたら、あなたが「これなら継続して書ける」と思えるテーマを3つ選びます。得意なこと・日常的にやっていることに近いテーマを優先すると、投稿が楽になります。

## ステップ3：投稿カレンダーの骨格をつくる

テーマが決まったら、「いつ・何を投稿するか」のカレンダー骨格をつくります。

### AIに聞いてみる

```
以下の条件でSNS投稿カレンダーの雛形を作ってください。
- 投稿頻度：週【3】回
- 使用SNS：【Instagram】
- 発信テーマ：
  1. 【テーマA】
  2. 【テーマB】
  3. 【テーマC】
- 期間：1週間分

曜日ごとにテーマを割り振り、各投稿の内容案を1行ずつ書いてください。
```

AIが出してくれたカレンダーをそのまま運用方針シートに貼り付けます。

### 良い例と悪い例

- **良い例**：「月曜＝お客様の声紹介、水曜＝制作の裏側、金曜＝Tips共有」→ 曜日ごとにテーマが決まっているので迷わない
- **悪い例**：「気が向いたときに投稿する」→ 結局やらなくなるパターン

---

## ステップ4：運用方針シートにまとめる

ここまでの内容を1枚のシートに整理します。Googleドキュメントやメモアプリに、以下の項目を書き出してください。

```
【SNS運用方針シート】

■ ターゲット：（ステップ1の結果）
■ 発信テーマ：
  1.（テーマA）
  2.（テーマB）
  3.（テーマC）
■ 使用SNS：（Instagram / X / TikTok など）
■ 投稿頻度：週○回
■ 投稿カレンダー（1週間分）：
  （ステップ3の結果を貼り付け）
■ 成果の確認方法：（例：月末にフォロワー数と問い合わせ数を記録する）
```

## 完成を確認する

以下の3つが揃っていれば、このレッスンは完了です。

1. **ターゲット像**が1文で書かれている（「誰に届けるか」が明確）
2. **発信テーマ**が3つ決まっている（「何を発信するか」が明確）
3. **1週間分の投稿カレンダー**がある（「いつ投稿するか」が明確）

完成したシートのスクリーンショットを撮っておきましょう。これがあなたの成果物です。

---

## つまずきやすいポイント

### 「AIの提案がピンとこない」

AIへの入力情報が少ないと、一般的すぎる回答になります。プロンプトの【】部分をできるだけ具体的に書き換えてください。たとえば「Webデザイナー」ではなく「小規模飲食店専門のWebデザイナー」のように絞ると、提案の精度が上がります。

### 「テーマを3つに絞れない」

AIに追加で聞いてみましょう。

```
提案してくれた5つのテーマのうち、初心者が最も継続しやすい組み合わせ3つを選んで、理由とともに教えてください。
```

### 「投稿頻度が多すぎて続かなそう」

最初は週2回でも十分です。大事なのは頻度より「決めたルールを守ること」です。慣れてきたら増やしましょう。
