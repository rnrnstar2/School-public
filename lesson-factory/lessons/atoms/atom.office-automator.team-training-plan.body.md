# 非エンジニアの同僚にAIツールを教える研修を設計する

あなたの職場にも「AIって便利そうだけど、何から始めればいいかわからない」という同僚はいませんか？

このレッスンでは、あなたが**15分で「AI研修プラン」を1枚のドキュメントにまとめる**ことをゴールにします。料理のレシピを書くように、「誰に・何を・どの順番で教えるか」を整理していきましょう。

完成したドキュメントは、そのまま研修の台本として使えます。

---

## 前提を確認する

- **必要なもの**: パソコンとブラウザ（Chrome や Edge など）
- **AIツールのアカウント**: ChatGPT（無料版でOK）または Claude のアカウントがあると便利です。なくても手順どおりに進められます
- **所要時間**: 約15分

---

## ステップ1: 研修の「誰に・何を」を決める

まず、研修の対象者とゴールをはっきりさせます。これは旅行の計画と同じで、「誰と・どこに行くか」を決めないとスケジュールが立てられないのと同じです。

### やること

以下の3つの質問に答えてください。メモ帳やドキュメントに書き出しましょう。

1. **誰に教える？** → 例: 営業部の5人、経理チームの3人
2. **その人たちの今の状態は？** → 例: ChatGPTの名前は知っているが使ったことはない
3. **研修後にどうなっていてほしい？** → 例: 議事録の要約をAIでできるようになる

### 良い例と悪い例

| | 良い例 | 悪い例 |
|---|---|---|
| 対象者 | 営業部の新人5人（PC操作は問題なし） | 会社のみんな |
| ゴール | 議事録の要約をAIで30秒でできるようになる | AIを使えるようになる |

**ポイント**: 「みんな」「なんでも」は計画が曖昧になる原因です。具体的に絞りましょう。

### AIに手伝ってもらう（プロンプト例）

もし対象者やゴールを整理するのに迷ったら、ChatGPTやClaudeに以下のように聞いてみてください。

```
私は営業部の後輩5人にAIツールを教える研修を企画しています。
後輩たちはChatGPTの名前は知っていますが使ったことがありません。
15分の研修で達成できる、現実的なゴールを3つ提案してください。
```

AIが返してくれた候補から、一番実用的なものを1つ選びましょう。

---

## ステップ2: 研修の流れを3パートで組み立てる

研修の構成は、シンプルに**3パート**で考えます。サンドイッチのように「パン→具→パン」の構造です。

![研修3パート構成図](/lesson-assets/atom.office-automator.team-training-plan/diagram.png)

### 3パート構成

| パート | 時間 | 内容 | 例 |
|---|---|---|---|
| ① つかみ（導入） | 2分 | 「AIでこんなことができる」とデモを見せる | 議事録テキストをAIに貼って要約させる実演 |
| ② やってみよう（実践） | 10分 | 参加者が自分の手でAIを使う | 用意した練習テキストで要約を体験する |
| ③ まとめ（振り返り） | 3分 | 「明日からこう使おう」と次のアクションを決める | 「来週の会議で試してみよう」と宣言する |

### AIに手伝ってもらう（プロンプト例）

```
以下の研修ゴールに合わせて、15分の研修プログラムを「導入2分・実践10分・まとめ3分」の3パートで作ってください。
ゴール: 営業部の後輩が、会議の議事録をAIで要約できるようになる
対象者: ChatGPTを使ったことがない営業職5人
各パートでやることを箇条書きで書いてください。
```

---

## ステップ3: 研修ドキュメントにまとめる

最後に、ステップ1と2で決めた内容を1枚のドキュメントにまとめます。

### テンプレート

以下のテンプレートをコピーして、空欄を埋めてください。

```
# AI研修プラン

## 基本情報
- 研修タイトル: （例: はじめてのAI議事録要約）
- 対象者: （例: 営業部新人5人）
- 所要時間: 15分
- ゴール: （例: 会議の議事録をAIで要約できるようになる）

## 必要な準備
- 参加者に事前にやってもらうこと: （例: ChatGPTのアカウント作成）
- 当日用意するもの: （例: 練習用の議事録テキスト3本）

## プログラム
### ① 導入（2分）
- やること:
- 見せるデモ:

### ② 実践（10分）
- 参加者がやる作業:
- 使うプロンプト例:
- うまくいかないときの対処:

### ③ まとめ（3分）
- 振り返りの質問:
- 次のアクション:
```

### AIにドキュメント生成を手伝ってもらう（プロンプト例）

テンプレートを自分で埋めるのが面倒な場合、AIにたたき台を作ってもらうこともできます。

```
以下のテンプレートに沿って、AI研修プランのたたき台を作ってください。

対象者: 経理チーム3人（Excel は使えるがAIは未経験）
ゴール: 月次レポートの文章パートをAIで下書きできるようになる
時間: 15分

[上のテンプレートをここに貼り付ける]
```

## つまずきやすいポイントと対処法

| つまずき | 原因 | 対処法 |
|---|---|---|
| ゴールが大きすぎて15分に収まらない | 「AIを完全に使いこなす」のような曖昧なゴール | 「○○を1回やってみる」まで絞る |
| 実践パートで参加者が止まる | プロンプト（AIへの指示文）が用意されていない | コピペできるプロンプト例を3つ用意しておく |
| AIの回答が期待と違う | プロンプトが曖昧 | 「○○について、箇条書き3つで、100字以内で」のように条件を具体的に書く |
| 参加者のAIアカウントが準備できていない | 事前連絡が不十分 | 研修2日前にリマインドメールを送る。当日用に講師のアカウントでデモできる準備もしておく |

---

## 完了を確認する

以下の3つがすべてYesなら、このレッスンは完了です。

- [ ] 研修プランのドキュメントが1枚できている
- [ ] ドキュメントに「対象者」「ゴール」「3パート構成」「プロンプト例」が含まれている
- [ ] 15分で実施できる分量になっている（テンプレートの各パートが3行以内に収まっている）

おめでとうございます！これであなたは「AI研修を設計できる人」になりました。次は実際にこのプランで研修を実施して、参加者の反応をもとに改善してみましょう。
