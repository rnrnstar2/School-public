# 取材準備の質問リストをAIで生成する

取材前の質問づくり、意外と時間がかかりますよね。「何を聞けばいいかわからない」「質問が思いつかない」という悩みを、AI（＝人工知能にお願いして答えを出してもらう仕組み）があっという間に解決してくれます。

このレッスンでは、あなたが **15分以内** で「取材に使える質問リスト」をAIに作ってもらう方法を学びます。完成したリストはそのまま取材本番に持っていけます。

料理に例えると、取材の準備は「献立を考える」ようなものです。どんな食材（＝話題）があるかを知り、何品（＝何本の質問）作るかを決めて、順番を考えます。AIはあなたの「献立アシスタント」として、食材の候補をたくさん出してくれる頼もしい存在です。

## 使うAIツールを選ぶ

次のどれか1つを使います。すべて無料プランで始められます。

| ツール | 特徴 | おすすめの人 |
|---|---|---|
| **ChatGPT**（chat.openai.com） | 会話形式で使いやすい。最も利用者が多い | はじめてAIを使う人 |
| **Claude**（claude.ai） | 長い文脈を理解しやすく、丁寧な回答が得意 | じっくり相談したい人 |
| **Google Gemini**（gemini.google.com） | Google検索と連携して最新情報も拾える | 最新の話題について取材する人 |

どれを選んでも、このレッスンの手順はまったく同じです。迷ったら **ChatGPT** から始めてみましょう。

## 取材テーマを決める

まずは「誰に」「何について」聞きたいかを、1〜2文でメモしておきましょう。これがAIへの指示の土台になります。

**良い例:**
- 「地元のパン屋さんに、開業のきっかけと人気メニューのこだわりについて聞きたい」
- 「新入社員に、入社前の不安と入社後のギャップについて聞きたい」

**悪い例:**
- 「パン屋さんについて聞く」（だれに・何についてが曖昧）
- 「いろいろ聞きたい」（範囲が広すぎてAIも困ってしまう）

> **ポイント**: テーマが具体的であるほど、AIの回答も的確になります。「2文にまとめられるか？」を自分への確認にしてみてください。

## AIに質問リストを頼む

AIチャットツールを開き、次のテンプレート（＝決まった型）をコピペして、`[ ]` の中身を書き換えてください。

```
取材の質問リストを10本作ってください。

取材相手：[相手の役割や立場]
取材テーマ：[聞きたいこと]
読者層：[記事を読む人]
記事の長さ：[短い記事 / 長めの特集 など]

次の条件を守ってください：
- 質問はやさしい日本語で、相手が答えやすい表現にする
- 流れは「軽い質問→本題→深掘り→まとめ」の順番にする
- 各質問に【意図】を一言添える
```

**「何をしているか」の説明**: AIに対して、取材の設定情報と出力フォーマットの条件を伝えて、そのまま使える質問案を作ってもらっています。

### 具体例：パン屋さんへの取材

```
取材の質問リストを10本作ってください。

取材相手：地元で20年営むパン屋の店主
取材テーマ：開業のきっかけと看板メニューへのこだわり
読者層：地域の食に関心がある一般読者
記事の長さ：2000字程度の短めの記事

次の条件を守ってください：
- 質問はやさしい日本語で、相手が答えやすい表現にする
- 流れは「軽い質問→本題→深掘り→まとめ」の順番にする
- 各質問に【意図】を一言添える
```

### うまくいかなかったとき：追加でお願いする

AIの最初の回答が期待通りでないことはよくあります。そのときは、**同じチャットの続きに追加のお願い**を書きましょう。新しくチャットを始める必要はありません。

**よく使う追加のお願い（コピペOK）：**

| 困りごと | 追加で送る文 |
|---|---|
| 質問が一般的すぎる | 「具体的なエピソードを引き出せる質問に書き換えてください」 |
| 質問が多すぎる | 「重要度の高い上位5本に絞ってください」 |
| 堅い表現になっている | 「カジュアルなインタビュー向けの口調に直してください」 |
| 深掘りが足りない | 「3番の質問をもとに、深掘り用のサブ質問を3つ追加してください」 |

## 出てきた質問をふるいにかける

AIが出してくれた質問を、すべてそのまま使うのは危険です。次の3つの基準で選びましょう。

1. **相手が答えやすいか** — 専門用語や抽象的な聞き方になっていないか
2. **読者が知りたいことか** — 読者の関心に合っているか
3. **重複していないか** — 似た質問は一本にまとめる

ふるいにかけるコツは、「旅行の持ち物を減らす」感覚です。持って行きたいものはたくさんありますが、かばんに入りきらない分は思い切って減らしましょう。

**良い例:** 10本の質問を精査して7本に絞り、自分の言葉で手直しした

**悪い例:** AIの回答をそのままコピペして、中身を読まずに使う

> **大切なこと**: AIはあくまで「たたき台」を作る道具です。最終的に質問の良し悪しを判断するのは、あなた自身です。

## 質問の順番を並べ替える

質問を選んだら、取材がスムーズに進む順番に並べ替えます。基本の流れは次のとおりです。

1. **アイスブレイク（場を温める質問）** — 相手が答えやすい軽い質問から始める
2. **メインの質問** — 一番聞きたいことを真ん中に置く
3. **深掘り** — メインに対する「なぜ」や「どうやって」を聞く
4. **まとめ** — 今後の展望や読者へのメッセージで締める

並べ替えに迷ったら、AIにもう一度お願いできます：

```
この質問リストを、取材の流れに合う順番に並べ替えてください。
理由も一言ずつ添えてください。
```

## 成果物を確認する

これで質問リストが完成です。次のチェックリストで最終確認しましょう。

- [ ] 質問が **5〜10本** ある
- [ ] 順番に意味がある（軽い → 深い → まとめ）
- [ ] 相手の立場で答えられる内容になっている
- [ ] AIの回答をそのままコピペせず、**自分の言葉に書き換えた部分がある**
- [ ] 各質問を声に出して読み、不自然な表現がないか確認した
- [ ] 読者が読んで「この話、面白そう」と思える質問が少なくとも1つある

すべてチェックがついたら、このレッスンの成果物は完成です。スクリーンショット（＝画面の写真）を撮って保存しておきましょう。

## つまずきポイント

| つまずき | 原因 | 対策 |
|---|---|---|
| AIの質問が淡白すぎる | 指示の情報が足りない | テンプレートの `[ ]` をすべて埋めたか確認する。「具体的なエピソードを引き出す質問を含めて」と追加で頼む |
| 質問が多すぎて選べない | 絞り込み基準がない | 「読者が一番知りたいことは何か？」を軸に、上位5本をまず選ぶ |
| 順番が決まらない | 全体の流れが見えない | メイン質問を先に決め、その前後にアイスブレイクと深掘りを配置する |
| AIが英語で回答する | 指示に言語指定がない | テンプレートの冒頭に「日本語で回答してください」を追加する |
| 同じような質問が並ぶ | 条件に「重複排除」がない | 「似た質問は統合して、バリエーションを広げてください」と追加で頼む |
