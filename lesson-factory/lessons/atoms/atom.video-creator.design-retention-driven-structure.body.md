# 視聴維持を意識して構成する

YouTubeやTikTokを見ていて、「あ、気づいたら最後まで見ちゃった」という経験はありませんか？ その裏には、視聴者が途中で離脱しないように計算された「構成」があります。

料理に例えてみましょう。良いコース料理は、最初の前菜で「次も食べたい！」と思わせ、メインディッシュで満足させ、デザートで余韻を残します。動画も同じです。最初の数秒で引き込み、中盤で価値を届け、最後で次の行動を促す——この流れを意識して作るのが「視聴維持を意識した構成」です。

このレッスンでは、AIツール（＝人工知能を使った便利な道具。ChatGPTやClaudeなど）を活用して、視聴者が最後まで見たくなる動画の構成案を作ります。最終的に、あなた自身の手で「3パート構成の台本ドキュメント」を1本仕上げるのがゴールです。

## 前提を確認する

このレッスンを進める前に、次の2つが準備できていることを確認してください。

- AIチャットツール（ChatGPT、Claude、Geminiなど）のアカウントを持っていて、メッセージを送れる状態になっている
- 動画で伝えたいテーマが1つ決まっている（例：「初心者向け料理レシピ」「旅行Vlogの撮り方」など）

まだテーマが決まっていなければ、「自分が最近ハマったこと」をテーマに選んでみましょう。完璧なテーマである必要はありません。このレッスンの目的は「構成の作り方」を覚えることなので、テーマは練習用で大丈夫です。

## 視聴維持率を理解する

視聴維持率（＝動画を再生した人のうち、どのくらいの割合が最後まで見たかを示す数字）は、動画の「寿命」を決める最も重要な指標（＝良し悪しを判断するための数字）です。

- **良い例**：5分の動画で平均視聴維持率60％——半分以上の人が3分以上見ている状態
- **悪い例**：5分の動画で、最初の10秒以内に80%の人が画面を閉じている状態

![視聴維持率のイメージ図](/lesson-assets/atom.video-creator.design-retention-driven-structure/diagram.png)

YouTubeやTikTokなどの動画プラットフォーム（＝動画を投稿・閲覧できるWebサービス）は、視聴維持率が高い動画を「良い動画」と判断して、より多くの人のおすすめ欄に表示します。つまり、構成を工夫するだけで、同じ内容でもずっと多くの人に届くようになるのです。

## 3パート構成で考える

視聴維持を意識した動画は、大きく3つのパートで構成します。

1. **フック（＝視聴者を引き込む冒頭部分）**：最初の3〜5秒
2. **バリュー（＝視聴者に届ける価値・情報の本体）**：動画のメイン
3. **CTA（＝コールトゥアクション。「次にしてほしいこと」を伝える締め）**：動画の最後

![3パート構成の全体図](/lesson-assets/atom.video-creator.design-retention-driven-structure/diagram.2.png)

この3パートは、テレビCMでも映画の予告編でも使われている基本の型です。これを知っているだけで、動画の設計がぐっと楽になります。

### フックを設計する

フックは「この動画を見続ける理由」を視聴者に3〜5秒で伝えるパートです。旅行のパンフレットと同じで、「行った先で何が待っているか」が分かれば、人は最後まで付き合ってくれます。

- **良い例**：「今日、この動画を見終わる頃には、あなたの動画の視聴維持率を2倍にする方法が分かります」→ 見終わった後のメリットが具体的
- **悪い例**：「こんにちは、今日は動画構成について話します」→ 情報はあるが、見続ける理由が弱い

コツは**「見終わった後にどうなるか」を最初に伝える**こと。自己紹介はフックの後に回して大丈夫です。

### バリューを配置する

バリューパートでは、フックで約束した内容をしっかり届けます。一番大事なポイントは「情報を小刻みに届ける」ことです。

- **良い例**：1つの大きなテーマを「3つのポイント」に分けて、1つずつ丁寧に説明する
- **悪い例**：5分間ずっと同じ話題をだらだら説明し続ける（視聴者が途中で飽きてしまう）

食事でいうと、一口サイズの料理が次々に出てくるコースの方が、大きな塊の肉1枚どーんより食べやすいですよね。動画も同じで、情報は小分けにして「次は何だろう？」と思わせるのが大切です。

### CTAで次に繋げる

CTAは動画の最後に置く「視聴者への1つのお願い」です。

- **良い例**：「この内容が役に立ったら、ぜひチャンネル登録をお願いします。次回は〇〇について解説します」→ 1つの行動＋次回の予告
- **悪い例**：「登録して、コメントして、SNSでシェアして、あと商品も買ってください」→ お願いが多すぎて、結局どれもしてもらえない

**CTAは1つに絞るのが鉄則です。**

## AIツールで構成案を作る

ここからが実践パートです。AIツールを使って、あなたのテーマに合った3パート構成案を実際に作りましょう。

### ステップ1：プロンプト（＝AIへの指示文）を入力する

AIチャットツール（ChatGPT、Claudeなど）を開いて、次の文をコピーして貼り付けます。 **「［ここにあなたのテーマを入力］」の部分だけ**、あなたが決めたテーマに書き換えてください。

```
あなたは動画構成のプロです。次のテーマで、視聴維持を意識した3パート構成（フック・バリュー・CTA）の動画台本案を作成してください。

テーマ：［ここにあなたのテーマを入力］
動画の長さ：約5分
対象：プログラミング経験のない初心者

各パートについて、次の内容を日本語で書いてください。
- そのパートの目的（なぜこのパートが必要か）
- 具体的なセリフ案（話す言葉をそのまま書く）
- 想定する秒数
```

この指示を送ると、AIが「フック」「バリュー」「CTA」に分かれた構成案を返してくれます。

![AIツールでの構成案生成画面](/lesson-assets/atom.video-creator.design-retention-driven-structure/screen_capture.png)

### ステップ2：AIの出力を確認する

AIが返した構成案を読んで、次の3つの観点でチェックします。

| チェック観点 | 合格の基準 | ダメだったときのAIへの追加指示 |
|-------------|-----------|---------------------------|
| フック | 最初の5秒以内に「見る理由」を伝えている | 「フックを、見終わった後のメリットが具体的に分かるように書き直して」 |
| バリュー | 3〜5個の小さなポイントに分かれている | 「バリューパートを3つのポイントに分けて、各ポイントに見出しをつけて」 |
| CTA | 具体的な1つの行動を促している | 「CTAを1つの行動だけに絞って、次回予告も1文添えて」 |

### ステップ3：自分の言葉で書き直す

AIの案をそのまま使うと「どこかで聞いたことのある無難な動画」になりがちです。**必ず1回は自分の言葉で書き直しましょう。**

具体的には、次の手順で進めます。

1. AIの出力をテキストファイルやメモアプリにコピーする
2. フックのセリフを「自分が友達に話すなら、こう言うな」という言葉に置き換える
3. バリューに、自分の経験やエピソードを1つ追加する
4. CTAの文言を、自分のチャンネルの雰囲気に合わせて調整する

AIが作った「型」をベースに、あなたの個性を乗せる——これが一番効率よく、かつオリジナリティのある構成を作る方法です。

### ステップ4（任意）：AIに最終チェックしてもらう

書き直した構成案を、もう一度AIに見せてフィードバックをもらうこともできます。次のプロンプトを使ってみてください。

```
以下の動画構成案をレビューしてください。視聴維持の観点で、改善できるポイントがあれば3つ以内で教えてください。

---
［ここにあなたの構成案を貼り付ける］
```

この「AIで作る → 自分で直す → AIでチェックする」のサイクルを回すと、短時間で質の高い構成案が仕上がります。

## 構成を検証する

最終的な構成案ができたら、次のチェックリストで確認しましょう。すべてにチェックがつけば完成です。

- [ ] フックに「視聴者が得られる具体的なメリット」が含まれている
- [ ] フックは5秒以内に収まっている（自己紹介はフックの後にある）
- [ ] バリューパートが3〜5個の小さなセクションに分かれている
- [ ] 各セクションが30秒〜1分30秒以内に収まっている
- [ ] 最後のCTAが1つの行動に絞られている
- [ ] 動画全体の想定時間が5分前後に収まっている
- [ ] AIの出力をそのまま使わず、自分の言葉で1回以上書き直した

## つまずきポイントに対処する

| つまずき | よくある原因 | 解決策 |
|----------|------------|--------|
| フックが10秒以上になる | 自己紹介やあいさつから始めている | 自己紹介はフックの後に回す。最初の一文は「メリットの約束」から |
| バリューが1つの話で終わる | テーマが大きすぎる | AIに「このテーマを3つのサブトピックに分けて」と頼む |
| CTAが複数ある | あれもこれも伝えたい欲 | 「一番やってほしいこと」を1つだけ選ぶ。迷ったら「チャンネル登録」 |
| AIの出力がしっくりこない | プロンプトが曖昧 | テーマ・ターゲット・動画の長さを具体的に指定し直す |
| 構成は作れたが台本にならない | 構成と台本は別物 | このレッスンのゴールは「構成案」まで。台本化は次のステップ |

一番多いつまずきは「フックに自己紹介を入れてしまう」ことです。まずは視聴者への約束から始めましょう。自己紹介はそのあとで大丈夫です。

## まとめ

このレッスンで学んだことを振り返ります。

1. **視聴維持率**は動画がおすすめされるかどうかを決める重要な数字
2. **3パート構成**（フック・バリュー・CTA）で視聴者を引き込み、最後まで見てもらう
3. **AIツール**を使えば、構成案の「型」を短時間で作れる
4. AIの出力に**自分の言葉と個性**を乗せることで、オリジナリティが生まれる
