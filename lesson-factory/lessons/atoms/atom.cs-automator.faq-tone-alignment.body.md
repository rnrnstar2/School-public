# FAQの文体とトーンを揃える

レストランチェーンを想像してみてください。ある店舗のメニューには「お肉と野菜のセット」、別の店舗には「グリルドビーフと季節の野菜のプレート」と書いてあったら、同じチェーンなのに違うお店みたいに感じますよね。

FAQ（＝よくある質問とその回答のまとめ）も同じです。質問の書き方や答え方のテイストがバラバラだと、読む人は「これ、同じ会社の回答？」と不安になってしまいます。

このレッスンでは、AIチャットツールを使って、FAQ全体の文体（＝書き方のクセ）とトーン（＝声のニュアンス）をぴったり揃えます。15分で終わるよう、手順をシンプルに絞り込んでいます。

## 前提を確認する

このレッスンを始める前に、次の2つを確認してください。

- ✅ すでにFAQの質問と回答が **3ペア以上** 書いてある（少ないと「揃える」効果が見えにくいため）
- ✅ ChatGPT・Claude・Geminiなど、どれか1つのAIチャットツールにログインできる

まだFAQを1つも書いていない場合は、まずは質問と回答を3ペア書くことから始めましょう。

## トーンの基準を決める

まず「うちのFAQはこんな話し方」というルールを1つ決めます。このルールを **トーンガイド** と呼びます。料理でいえばレシピの「味付けの方針」にあたるものです。

トーンガイドには、次の3つの要素を含めましょう。

1. **丁寧さのレベル** — 「です・ます」か「だ・である」か
2. **絵文字・記号の使い方** — 使うか、使わないか
3. **専門用語の扱い** — そのまま使うか、やさしい言葉に置き換えるか

![トーンガイドの3要素](/lesson-assets/atom.cs-automator.faq-tone-alignment/diagram.png)

**良い例：**
> 「丁寧で落ち着いた話し方。です・ます調。絵文字は使わない。専門用語は避け、どうしても使う場合は括弧で意味を添える。」

**悪い例：**
> 「自然な感じで」（← 「自然」の基準が人によって違うため、AIも毎回違う結果を返してしまいます）

### AIにトーンガイドを作ってもらう

自分で決めにくいときは、AIに手伝ってもらいましょう。以下のプロンプト（＝AIへの指示文）をコピーして貼り付けてください。

```
私の会社のFAQのトーンガイドを作ってください。
以下の3点をそれぞれ1行で決めてください。
1. 丁寧さのレベル（です・ます or だ・である）
2. 絵文字・記号を使うか使わないか
3. 専門用語をそのまま使うか、やさしい言葉に言い換えるか

【参考：現在のFAQの一例】
（ここにあなたのFAQを1ペア貼り付けます）
```

AIが提案してくれたトーンガイドをそのまま使っても、自分で微調整してもOKです。

## AIにトーンを揃えさせる

トーンガイドが決まったら、AIにFAQを渡して書き直してもらいます。次のプロンプトをコピーして使ってください。

```
以下のトーンガイドに従って、FAQの回答を書き直してください。
変更した箇所には【変更】と印をつけてください。

【トーンガイド】
・丁寧で落ち着いた話し方
・です・ます調
・絵文字は使わない
・専門用語は避け、使う場合は括弧で意味を添える

【書き直すFAQ】
（ここにあなたのFAQを全て貼り付けます）
```

**ポイント**: 「変更した箇所に【変更】と印をつけて」と指示しておくと、AIがどこを直したか一目で分かります。そのまま受け入れるか、元に戻すか判断しやすくなります。

![AIでFAQを書き直した画面](/lesson-assets/atom.cs-automator.faq-tone-alignment/screen_capture.png)

### FAQが多いときは分割して渡す

FAQが10問以上あるときは、5問ずつに分けてAIに渡しましょう。一度に大量の文章を渡すと、後半の質問ほどトーンがぶれやすくなります。

**良い例：** 5問ずつ渡し、毎回トーンガイドを先頭に貼る
**悪い例：** 30問を一度に渡して「全部直して」と頼む

## 結果を確認する

AIが書き直したFAQを、元の文章と見比べてみましょう。確認ポイントは3つです。

| チェック項目 | 確認のしかた | OK / NG |
|---|---|---|
| 丁寧さが揃っているか | すべての回答が「です・ます」（or 選んだ文体）で統一されているか | □ |
| 絵文字・記号の有無が統一されているか | 使わないと決めたなら全篇で使われていないか | □ |
| 専門用語は処理されているか | 難しい言葉がやさしい言葉に置き換わっているか | □ |

3つとも ✅ がついたらOKです。もし気になるところがあれば、AIに追加で指示を出します。

```
以下の部分を修正してください。
・Q3の回答に「デプロイ」という言葉が残っています。
　「インターネットに公開する」に言い換えてください。
```

このように **何が問題で、どう直してほしいか** を具体的に書くのがコツです。

## よくあるつまずきに対処する

| つまずき | なぜ起きるか | 対策 |
|---|---|---|
| AIが毎回違うテイストで書く | トーンガイドを渡し忘れている | プロンプトの **先頭** にトーンガイドを毎回コピペする |
| 絵文字がどうしても混ざる | 指示が弱い | 「絵文字は **絶対に** 使わないでください」と強調する |
| 専門用語が残ってしまう | AIが「専門用語」の範囲を判断しきれない | 「〇〇という言葉は使わず、△△と言い換えて」と具体的に列挙する |
| 後半のFAQほどトーンがぶれる | 一度に渡す量が多すぎる | 5問ずつに分割して渡す |

## 成果物をまとめる

書き直したFAQを1つのファイルにまとめましょう。テキストファイルでも、Googleドキュメント（＝Googleが無料で提供する文章作成ツール）でも構いません。

ファイルの先頭にトーンガイドも一緒に貼り付けておくと、次回FAQを追加するときにそのままAIに渡せるので便利です。

```
===== トーンガイド =====
・丁寧で落ち着いた話し方
・です・ます調
・絵文字は使わない
・専門用語は避け、使う場合は括弧で意味を添える
========================

Q1. 〇〇はどうすればいいですか？
A1. 〇〇するには……

Q2. ...
```

最後に、先ほどの3つのチェック項目をもう一度確認して、すべて ✅ がついたら完成です。おつかれさまでした！
