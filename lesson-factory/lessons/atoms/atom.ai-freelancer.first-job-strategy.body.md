# 最初の1件を確実に獲得する

あなたが初めてお店を開いたと想像してみてください。看板はまだ小さくて、お客さんはまだ来たことがありません。でも、**最初の1人**さえ来てくれれば、「ここ、いいお店だよ」と噂が広まります。フリーランスの仕事も同じです。

この Atom では、AI フリーランス（＝AIツールを活用して個人で仕事を受ける働き方）における**「最初の1件」を確実に取るための最小の手順**を、AIツールをフル活用しながら学びます。終わるころには、提案文を1通書いて実際に応募できる状態になります。

![初案件獲得までの4ステップ](/lesson-assets/atom.ai-freelancer.first-job-strategy/diagram.png)

## 自分の「商品」をひとつ決める

まずはあなたが提供するサービスを**ひとつだけ**決めます。たくさん用意する必要はありません。メニューが1品だけの食堂でも、その1品が美味しければ人は来ます。

### AIに相談して決める

自分だけで考えるのが難しければ、AIツールに相談してみましょう。ChatGPT や Claude を開いて、次のように入力してみてください。

```
私はプログラミングの経験がない初心者です。AIツール（ChatGPTやClaude）を使って、クラウドソーシングで受けられそうな仕事を5つ提案してください。以下の条件でお願いします。
- 1件あたり2時間以内で終わる
- 特別なスキルがなくてもAIと一緒にできる
- 初心者でも品質を担保しやすい
```

AIが提案してくれたリストから、**一番「これならやれそう」と思えるもの**をひとつ選びましょう。

### 良い例

- 「ブログ記事の下書きをAIで作って、2時間で納品します」
- 「SNSの投稿文を10本、AIで作成してお渡しします」
- 「商品紹介文をAIで3パターン作成します」

### 悪い例

- 「何でもやります」（範囲が広すぎて、依頼する側が不安になる）
- 「AIでWebサイトを全部作ります」（非エンジニアには難易度が高すぎる）

> **ポイント**: 最初は「小さく・早く・確実に」が鉄則です。大きな仕事は実績ができてからで十分です。

---

## 受注場所をひとつ選ぶ

お店を開く場所を決めます。ネット上には、仕事を依頼したい人と仕事をしたい人をつなぐ**クラウドソーシング（＝ネット上で仕事を募集・受注できるサービス）**があります。

代表的な場所：

- **クラウドワークス** — 日本語の案件が豊富で、初心者が始めやすい
- **ランサーズ** — 初心者向けの案件が多く、サポートが手厚い

どちらか**ひとつだけ**選んでアカウントを作りましょう。

![クラウドソーシングサービスの登録画面](/lesson-assets/atom.ai-freelancer.first-job-strategy/screen_capture.png)

### 選び方のコツ

| 観点 | クラウドワークス | ランサーズ |
|------|----------------|----------|
| 案件数 | 多い | やや少ない |
| 初心者向け | ◎ | ◎ |
| 承認までの早さ | 早い | 早い |

> どちらでも大丈夫です。迷ったら**クラウドワークス**を選んでください。

### アカウント登録でやること

1. サイトにアクセスして「会員登録」をクリックする
2. メールアドレスとパスワードを入力する
3. プロフィール欄に「AIツールを活用したライティングが得意です」と書く

> **ポイント**: プロフィールは後からいくらでも直せます。最初は短くてOKです。完璧を目指さず、まず登録を済ませましょう。

## 案件を探して提案文を書く

案件（＝依頼されている仕事）を見つけたら、**提案文（＝「私がこの仕事をやりたいです」と伝える短い文章）**を書きます。ここでもAIツールが大活躍します。

### 案件の探し方

1. クラウドソーシングの検索欄で、先ほど決めた「商品」に関連するキーワードを入力する（例: 「ブログ記事」「SNS投稿」）
2. **予算が1,000円〜5,000円**くらいの小さな案件を探す
3. 依頼内容を読んで「これならAIと一緒にできそう」と思ったものをメモする

### AIで提案文を作る

案件が見つかったら、AIツールに提案文を作ってもらいましょう。ChatGPT や Claude に、次のように入力してください。

```
以下のクラウドソーシングの案件に応募する提案文を300字程度で作ってください。

【条件】
- 丁寧で誠実な印象にする
- AIツールを活用して効率的に作業できることを伝える
- 「まずは1件お試し」という気軽さを出す
- 具体的な納期（例: 2日以内）を提示する

【依頼内容】
（ここに案件の依頼内容をコピーして貼り付ける）
```

### 提案文を自分の言葉で仕上げる

AIが作った文章を**そのまま送ってはいけません**。次の3点を確認して、あなたらしさを加えてください。

1. **不自然な敬語がないか** — 読んで違和感があれば直す
2. **自分の経験を1文だけ加える** — 「普段から〇〇に興味があります」など
3. **嘘がないか** — できないことを「できます」と書いていないか確認

### 良い提案文の例

> 御社のブログ記事の下書きを担当させていただきたいです。AIツールを活用して、**2日以内**に品質の高い下書きをお渡しできます。まずは1件お試しでいかがでしょうか。普段からAIを使った文章作成に取り組んでおり、読みやすい記事を心がけています。

### 悪い提案文の例

> よろしくお願いします。

（短すぎて熱意が伝わらない）

> 私はどんな仕事でも完璧にこなせます。

（根拠がなく不自然。依頼する側が不安になる）

## 応募して結果を待つ

提案文を送ったら、あとは待つだけです。ただし、**1件だけに応募して待つのは危険**です。

- **同時に3〜5件**に応募するのがおすすめ（1件だと外れたときのショックが大きい）
- すべて同じサービス内容でOK（商品は1つで十分）
- 最初は安くても構わない（実績作りが目的）

### 結果の3パターンと対処法

| パターン | 次にやること |
|----------|------------|
| 採用された | おめでとう！納期を確認して仕事を仕上げる |
| お断りされた | 気にせず次の案件に応募する |
| 返信がない | 3日待って返信がなければ次に進む |

### 採用されたらAIと一緒に仕事を仕上げる

1. **納期（＝仕事を届ける期限）を確認する** — メッセージで「〇日までに納品します」と伝える
2. **AIツールを使って作業する** — 先ほどの提案と同じように、AIに指示を出して下書きを作る
3. **品質を自分でチェックする** — AIに「この文章に誤字脱字や不自然な表現がないか確認してください」と頼むと便利
4. **納品する** — 完成したらクラウドソーシングの画面から納品ボタンを押す

## つまずきやすいポイントと解決策

| つまずきポイント | 解決策 |
|----------------|--------|
| 「自分にはスキルがない」と感じる | AIツールがあなたのスキルです。文字が打てる人なら誰でも価値を提供できます |
| 提案文が書けない | 上の「AIで提案文を作る」のプロンプト（＝AIへの指示文）をそのままコピーして使ってください |
| 案件が見つからない | 検索キーワードを変えてみましょう。「ライティング」「記事作成」「文章」などで試す |
| 応募したけど全部落ちた | 普通です。10件応募して1件取れれば上出来。提案文をAIと一緒に改善して再チャレンジ |
| 採用後に何をすればいいかわからない | 依頼者にメッセージで「確認したいことリスト」を送りましょう。AIに「依頼者に確認すべき質問を5つ考えて」と頼めばOK |
