# 発信テーマを柱で設計する

レストランのメニューを想像してみてください。何でもかんでも1ページに詰め込んだメニューは、お客さんを迷わせてしまいますよね。でも「前菜・メイン・デザート」のようにいくつかの枠で整理されていれば、ぱっと見で何があるか分かります。

あなたの発信も同じです。あれもこれも伝えたいと全部バラバラに出すより、「私はだいたいこの3〜5本のテーマで発信しています」という柱（コンテンツピラー＝発信テーマをいくつかの大きな分類に整理する考え方）を決めておくと、作る側も見る側も迷いません。

このレッスンでは、AIチャットと対話しながら、あなたらしい発信の柱を3〜5本つくります。15分で完了できるように手順を絞っています。

## 自分の「好き・得意・需要」をAIに引き出してもらう

まずは、あなたが「楽しく話せること」と「人に役立てそうなこと」の交差点を見つけます。AIチャットツール（ChatGPTやClaudeなど、どれでもOKです）を開いて、次のプロンプト（＝AIへの指示文）をコピペしてください。

```
私がSNSで発信するテーマを決めたいです。
まず私に5つ質問して、私の好きなこと・得意なこと・趣味・仕事について聞いてください。
私の回答をもとに、発信に向いていそうなテーマを5個提案してください。
各テーマには「なぜ向いているか」の理由も1行つけてください。
```

このプロンプトが何をしているかというと、AIにインタビュアーの役割を頼んで、あなた自身の情報を整理してもらう、ということです。

AIがいくつか質問を返してきますので、思いつくままに正直に答えていきましょう。

### 良い答え方の例

- 「料理が好きで、最近は時短レシピにハマっています。ITは苦手です」→ 具体的なので、AIがテーマを絞りやすい
- 「子育て中で、離乳食の試行錯誤を記録したい気持ちがあります」→ 発信のモチベーションが明確

### 避けたい答え方の例

- 「特にありません、なんでもいいです」→ AIも手掛かりがなく、ありきたりな提案しかできない
- 「投資も料理も旅行も筋トレも全部やりたい」→ 柱が多すぎて、見る人が「この人は何の人？」と迷ってしまう

## 候補を3〜5本の柱に絞り込む

AIが提案してくれたテーマ候補を眺めて、あなたが「これなら週1回でも続けられそう」と思えるものを3〜5本選びます。

料理でたとえると「和食・洋食・中華」のように、守備範囲がなるべく重ならないように分けるのがコツです。全部「料理」では柱として広がりがありません。「時短レシピ・失敗談・食材の選び方」のように角度を変えると、それぞれの柱に深みが出ます。

選んだら、AIに次のプロンプトを送って整理してもらいましょう。

```
以下のテーマを3〜5本の「発信の柱」として整理してください。

- （あなたが選んだテーマ1）
- （あなたが選んだテーマ2）
- （あなたが選んだテーマ3）

それぞれの柱に次の3点をつけてください。
1. 柱の名前（短くキャッチーに）
2. 柱の説明文（1〜2文）
3. この柱で作れそうな投稿ネタの例を2つ
```

このプロンプトのポイントは「柱の名前・説明・ネタ例」の3点セットをお願いしているところです。名前だけだとあとで「これ何だっけ？」となりがちですが、説明とネタ例があればすぐ思い出せます。

## 柱が「立っているか」をチェックする

次の3つの観点で、できた柱が機能しているか確かめます。

| チェック項目 | OK の状態 | NG の状態 |
|---|---|---|
| 柱の本数 | 3〜5本 | 2本だと単調、6本以上だと散漫 |
| 柱どうしの重なり | テーマが明確に違う | 似た内容が2本に分かれている |
| 「あなた」らしさ | 自分の経験がにじんでいる | 誰でも書けそうな一般論だけ |

もし気になる点があれば、AIに次のように聞いてみてください。

```
以下の3本の柱に重複や弱い部分がないかチェックして、改善案があれば教えてください。

1. （柱1の名前と説明）
2. （柱2の名前と説明）
3. （柱3の名前と説明）
```

## つまずきやすいポイントを知っておく

- **「全部やりたい」と詰め込みすぎる** → まずは3本だけに絞りましょう。増やすのはいつでもできます。少ない方が続けやすいです
- **他の人と被るのが不安** → 同じテーマでも「あなたの体験や視点」が入ったとたんに別物になります。テーマが被ること自体は問題ありません
- **AIの提案をそのまま採用して終わる** → AIの提案はあくまで出発点です。しっくりこない言葉は遠慮なくあなたの言葉に書き換えてください。「自分の口から出そうな言い方」になっていればOKです
- **完璧な柱を最初から作ろうとする** → 発信を始めてから「やっぱりこっちの方がいいな」と変えるのは普通のことです。今日は「たたき台」を作る気持ちでOKです

## 成果物を保存する

最終的にできた3〜5本の柱のリスト（名前・説明・ネタ例つき）を保存してください。

保存先はどこでも構いません。メモ帳、Notion、Googleドキュメント、スマホのメモアプリなど、あなたが普段使っているものでOKです。

これがこのレッスンの成果物（＝あなたの手元に残る結果物）です。
