# 画像生成案件の市場と単価を調査する

## このレッスンで身につけること

あなたが「AIで絵を作れる」というスキルを、実際の収入に結びつけるための第一歩を踏み出します。具体的には、ネット上で画像生成の仕事がどれくらいあるか、ひとつあたりいくら報酬がもらえるかを、**15分でサッと調べる方法**を身につけます。

旅行に行く前に「ホテルの料金相場」をチェックしますよね。それと同じで、仕事を受け始める前に「市場の相場」を知っておくことが大切です。相場を知っていれば、安すぎる提案をして損をしたり、高すぎて案件をもらえなかったりするリスクを減らせます。

![市場調査の流れ](/lesson-assets/atom.ai-freelancer.image-gen-market-research/diagram.png)

## 前提

- パソコンまたはスマホでブラウザ（＝インターネットを見るアプリ）を開けること
- AI画像生成ツール（DALL·E、Midjourney、Canva など）の名前を聞いたことがあること
- ChatGPT や Claude など、AIチャットツールをひとつ使えること（無料版でOK）

## クラウドソーシング（＝ネット上の仕事マッチングサイト）を開く

日本でよく使われているサイトを3つ紹介します。すべて無料で登録できます。

| サイト名 | 特徴 |
|---|---|
| クラウドワークス | 日本最大級。案件数が多い |
| ランサーズ | デザイン系に強い |
| ココナラ | スキルを「サービス」として自分から出品できる |

まずはブラウザで **クラウドワークス（crowdworks.jp）** を開いてみましょう。

![案件検索画面の例](/lesson-assets/atom.ai-freelancer.image-gen-market-research/screen_capture.png)

## キーワードで案件を検索する

サイト上部の検索窓にキーワードを入れて、どんな案件が出てくるか見てみましょう。

### 良い検索キーワードの例

- 「イラスト作成 AI」
- 「画像生成」
- 「アイコン 作成」
- 「バナー AI」

→ 具体的で、あなたができそうな仕事に絞り込めています。

### 悪い検索キーワードの例

- 「デザイン」（広すぎて関係ない結果が大量に出る）
- 「AI」（何のAIか分からず、プログラミング案件も混ざる）

→ 範囲が広すぎると、調べる時間が倍増してしまいます。

検索結果の **案件名・報酬額・募集人数** をざっと眺めて、スクリーンショット（＝画面の写真）を3〜5枚撮ってください。

> 💡 **スクリーンショットの撮り方**: Windowsなら `Win + Shift + S`、Macなら `Command + Shift + 4` で画面の一部を写真として保存できます。

## AIに相場分析を手伝ってもらう

ここからがAI活用のポイントです。調べた案件情報をAIに整理してもらいましょう。

ChatGPT や Claude を開いて、次のようなプロンプト（＝AIへの指示文）をコピペしてください。

```
以下はクラウドソーシングサイトで見つけた画像生成の案件情報です。
これらの情報を整理して、次のことを教えてください。

1. 単価の相場（安い・中央・高いの3段階）
2. よく求められているスキルや納品形式
3. 初心者が狙いやすい案件の特徴

---
（ここにスクリーンショットを貼り付けるか、案件名と金額を箇条書きで入力）
```

> 💡 **ChatGPTの場合**: スクリーンショットの画像ファイルをそのままチャット欄にドラッグ&ドロップすれば、AIが画像の内容を読み取って分析してくれます。テキストで打ち直す必要はありません。

### AIの回答をチェックするポイント

AIの回答は便利ですが、100%正確とは限りません。次の点を確認しましょう。

- **金額が現実的か**: 「画像1枚で10万円が相場」などと出たら、実際のサイトの表示と見比べてください
- **古い情報が混ざっていないか**: AIは最新の案件情報を知らないことがあります。あなたが今日調べた実際のデータが一番正確です
- **具体的な数字があるか**: 「高い」「安い」だけでなく、「○○円〜○○円」のように数字で把握しましょう

## 単価の相場をまとめる

スクリーンショットとAIの分析結果を見返しながら、メモ帳に次の3項目を書き出します。

1. **案件名**（どんな絵を作る仕事か）
2. **報酬**（ひとつあたりいくらか）
3. **納期**（いつまでに届ければよいか）

5〜10個の案件を書き出せたら、報酬を **一番安いもの〜一番高いもの** の順に並べてみましょう。これがあなたの「単価の相場感」です。

> 💡 **ポイント**: 画像1枚あたりの報酬は、サイトや案件タイプによって500円〜30,000円と幅があります。最初は1,000円〜3,000円程度の案件から始めるのがおすすめです。

## 自分の強みを考える

相場が見えたら、「自分ならどのあたりで戦えるか」を考えます。ここでもAIに相談できます。

```
私はAI画像生成ツールを使って副業を始めたい初心者です。
以下の案件一覧を見て、私が最初に挑戦しやすい案件タイプを
3つ提案してください。理由も添えてください。

---
（先ほどまとめた案件一覧をここに貼り付け）
```

AIの提案を参考にしつつ、最終的には自分で「これなら楽しくできそう」と思えるものを選びましょう。

- **速さが武器**なら → 納期の短い案件を狙う
- **品質が武器**なら → 高単価でこだわりのある案件を狙う
- **ニッチ（＝他の人があまり手がけていない分野）** なら → その分野のキーワードで絞り込む

## 結果をまとめる（成果物を作る）

最後に、調べた内容をひとつのメモにまとめます。このメモが今回の **成果物（＝アウトプット）** です。テンプレートを用意しましたので、コピーして空欄を埋めてください。

```markdown
# 画像生成 案件調査メモ

調査日：2025-XX-XX
調査サイト：クラウドワークス / ランサーズ / ココナラ（使ったサイトを残す）
使ったAIツール：ChatGPT / Claude（使ったものを残す）

## 調べた案件

| 案件名 | 報酬 | 納期 | メモ |
|--------|------|------|------|
| （例）SNS用バナー作成 | 3,000円 | 3日 | AIで十分いけそう |
| （例）商品写真の背景変更 | 1,500円 | 2日 | 枚数が多いと大変 |
|  |  |  |  |

## 単価の相場
- 安い：〇〇円
- 中央：〇〇円
- 高い：〇〇円

## AIに聞いてわかったこと
- （AIが教えてくれた初心者向けの案件タイプや傾向を1-2行で書く）

## 自分の戦略
- （例）まずは1,000〜3,000円のアイコン作成案件を週3つ目指す
```

## つまずきやすいポイント

| つまずき | 対処法 |
|----------|--------|
| 案件が多すぎて迷う | まずは「画像生成」「イラスト AI」の2キーワードだけに絞りましょう |
| 報酬が安くて落ち込む | 最初は練習期間と思って、レビュー（＝お客さんからの評価）を稼ぐ目的で進めましょう |
| どのサイトがよいかわからない | まずはクラウドワークス1つを徹底的に見て、相場感を養いましょう |
| AIに何を聞いていいかわからない | 上の「プロンプト例」をそのままコピペして、自分の情報を貼り付けるだけでOKです |
| AIの回答が信用できるかわからない | あなたが実際にサイトで見た金額と比較してみましょう。大きくズレていたらAIの回答は無視してOK |

## 振り返る（確認チェックリスト）

このレッスンを終えたら、次の項目をすべて確認してください。

- ✅ クラウドソーシングサイトで画像生成の案件を検索できた
- ✅ 5個以上の案件の報酬を書き出せた
- ✅ 単価の相場を「安い・中央・高い」の3段階でまとめられた
- ✅ AIツールを使って案件情報を分析できた
- ✅ 自分の強みに合った戦略を1行で書けた
- ✅ 成果物テンプレートの空欄を埋めて保存できた

すべてにチェックが入れば、このレッスンは完了です！
