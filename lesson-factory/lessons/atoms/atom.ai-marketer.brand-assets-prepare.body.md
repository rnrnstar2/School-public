# ブランド情報を入力資産として整備する

このレッスンでは、あなたのブランドに関する情報を、AIツールにすぐ渡せる「ブランド資産ファイル」にまとめます。

料理のレシピを想像してみてください。材料がバラバラに置いてあると、いざ料理を始めるときに「あれどこだっけ？」と探すはめになります。冷蔵庫の前に材料を全部揃えてから料理を始めるとスムーズですよね。ブランド情報も同じです。あらかじめ決まった枠で整理しておけば、AIに「このブランドの広告文を書いて」と頼むとき、毎回ゼロから説明する必要がなくなります。

![ブランド資産整理の全体フロー](/lesson-assets/atom.ai-marketer.brand-assets-prepare/diagram.png)

## 必要なものをそろえる

このレッスンを始める前に、次のものを手元に用意してください。

- ブランドのロゴ画像（スマホの写真でもOKです）
- ブランド名と、あればタグライン（＝ブランドのキャッチフレーズ）
- 商品やサービスの説明文（ホームページからコピペで十分です）
- スマホまたはパソコンで使えるAIチャットツール（ChatGPT、Claude など。どれか1つあれば大丈夫です）

> **ヒント**: まだAIチャットツールを使ったことがない方は、ChatGPT（https://chat.openai.com）またはClaude（https://claude.ai）にアクセスして無料アカウントを作っておきましょう。

## ブランド情報を5つの枠に分ける

ブランド情報は、次の5つの枠（カテゴリ）に分けて書き出します。この5つが揃っていれば、AIはあなたのブランドらしい文章を書けるようになります。

| # | 枠 | 書く内容 | 例 |
|---|---|---|---|
| 1 | **ブランドの基本情報** | 名前、ロゴの雰囲気、始めたきっかけ | 「〇〇ベーカリー。2020年に地元の焼きたてパンを届けたくて創業」 |
| 2 | **ターゲット** | 誰に向けた商品・サービスか | 「30代の共働き女性。朝食の時短に関心が高い」 |
| 3 | **トーン＆マナー** | ブランドが話すときの口調 | 「です・ます調。親しみやすく、でも専門性も感じさせる」 |
| 4 | **競合との違い** | 他社と比べた特徴 | 「国産小麦100%。添加物不使用を徹底」 |
| 5 | **キーワード** | ブランドを表す5〜10個の言葉 | 「焼きたて、国産小麦、時短朝食、安心、手づくり」 |

### 良い例

> ターゲット：「30代の共働き女性。朝食の時短に関心が高い。Instagram をよく見る」

→ AIが「忙しい朝でもサッと食べられる」のような具体的な表現を提案できます。

### 悪い例

> ターゲット：「まあいろんな人」

→ AIが方向性を掴めず、誰にも刺さらないぼんやりした文章になってしまいます。

## AIに手伝ってもらいながら整理する

5つの枠を埋めるのは、自分だけで考える必要はありません。AIチャットツールに手伝ってもらいましょう。

### ステップ1: AIに質問を出してもらう

ChatGPT や Claude を開いて、次のプロンプト（＝AIへの指示文）をコピペして送信してください。

```
私のブランド情報を整理する手伝いをしてください。
ブランド名は「ここにブランド名」です。
以下の5つの枠について、1つずつ質問してください。
私が答えたら次の質問に進んでください。

1. ブランドの基本情報（名前、雰囲気、始めたきっかけ）
2. ターゲット（誰向けの商品・サービスか）
3. トーン＆マナー（ブランドの口調）
4. 競合との違い（他社と比べた特徴）
5. キーワード（ブランドを表す5〜10個の言葉）
```

> **大事なポイント**: 「ここにブランド名」の部分を、あなたの実際のブランド名に書き換えてから送信してください。

### ステップ2: AIの質問に1つずつ答える

AIが枠ごとに質問を出してくれます。思いつく範囲で答えてください。完璧でなくて大丈夫です。AIが追加で聞いてくれることもあります。

> **答えに迷ったときのコツ**:
> - ターゲットが思いつかない → 「一番最近買ってくれたお客さん」を1人思い浮かべてみてください
> - トーンが決められない → 競合他社のSNSを見て「これに近い」「これは違う」を3つずつ挙げると見えてきます
> - 全部書くのが面倒 → まず思いつくまま全部書き出して、AIに「5つの枠に振り分けて」と頼んでもOKです

### ステップ3: まとめを出力してもらう

5つの枠がすべて埋まったら、次のプロンプトを送ってください。

```
ここまでの内容を、以下のMarkdown形式でまとめてください。

# ブランド概要
- ブランド名：
- タグライン：
- 対象顧客：
- トーン：
- 差別化ポイント：
- キーワード：
```

AIがきれいにまとめてくれます。

## まとまった情報を保存する

AIがまとめてくれた内容を、あとで使いやすいようにファイルに保存します。

- **ファイル名**: `brand-brief-ブランド名.md`（例: `brand-brief-〇〇ベーカリー.md`）
- **保存先**: デスクトップやメモアプリなど、あとで見つけやすい場所

保存の方法は2つあります。お好きな方を選んでください。

| 方法 | やり方 |
|---|---|
| **方法A: コピペ** | AIの出力をそのままコピーして、メモ帳やテキストエディタに貼り付けて保存 |
| **方法B: AIに頼む** | 「この内容をダウンロードできるファイルにして」とAIに頼む（ChatGPTはファイル出力に対応しています） |

これがあなたの「ブランド資産」です。今後AIに広告文やSNS投稿を頼むとき、このファイルの内容を最初に貼り付けるだけで、ブランドに合った文章を生成してもらえます。

## 成果物を確認する

次のチェックリストで、あなたのブランド資産ファイルが完成しているか確認してください。

- [ ] ブランド名が書かれている
- [ ] ターゲットが「誰」「どんな関心」まで具体的に書かれている
- [ ] トーン＆マナーが「です・ます調」「カジュアル」のように具体的に書かれている
- [ ] 競合との違いが1つ以上、具体的に書かれている
- [ ] キーワードが5個以上書かれている
- [ ] ファイルとして保存できている（`brand-brief-ブランド名.md`）

すべてチェックがついたら、ファイルの中身が見えるようにスクリーンショット（画面の写真）を撮ってください。それがこのレッスンの成果物（＝あなたが完了した証拠）です。

## よくあるつまずきと解決策

| つまずき | 解決策 |
|---|---|
| AIが途中で話題を変えてしまう | 「5つの枠の整理に戻ってください」と一言添えれば、元の流れに戻ります |
| 答えが短すぎて枠が埋まらない | 「もう少し詳しく聞いてください」とAIに頼むと、深掘り質問をしてくれます |
| ターゲットが思いつかない | 「一番最近買ってくれたお客さん」を1人イメージして、その人の年代・関心・悩みを書き出してみてください |
| トーンが決められない | 競合他社のSNSを3社見て「これに近い」「これは違う」をそれぞれ挙げると方向性が見えてきます |
| ファイルの保存方法がわからない | パソコンなら「メモ帳」アプリを開いてコピペ→「名前を付けて保存」。スマホならメモアプリに貼り付けてください |
