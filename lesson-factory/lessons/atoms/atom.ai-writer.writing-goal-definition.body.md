# AI活用で目指す執筆スタイルを定義する

AIに文章を書いてもらうとき、「なんかイメージと違う…」と感じたことはありませんか？ それは、あなた自身が「どんな文章を書きたいのか」をまだ言葉にできていないからかもしれません。

たとえば、レストランで注文するとき「おいしいもの」とだけ言っても、シェフは困ってしまいますよね。「辛いものが好き」「魚料理がいい」「量は少なめで」と伝えるほど、ぴったりの料理が出てきます。AIへの指示もまったく同じです。

このレッスンでは、あなただけの「執筆スタイルシート」を1枚作ります。これがあれば、AIに毎回細かく説明しなくても、あなた好みの文章が出てくるようになります。

## このレッスンで手に入るもの

- 自分の執筆スタイルを言語化した「スタイルシート」（Markdownファイル1枚）
- AIに渡すだけで文体が安定する「スタイル指示テンプレート」

## 前提として必要なもの

- ChatGPTやClaudeなど、テキスト生成AIが使える状態であること
- 特別なソフトのインストールは不要です

## 全体の流れを確認する

![執筆スタイル定義の流れ](/lesson-assets/atom.ai-writer.writing-goal-definition/diagram.png)

スタイルシート作成は、以下の3ステップで進めます。

1. **好きな文章を集める** — お手本となる文章を2〜3本選ぶ
2. **スタイル要素を分解する** — 文体・語尾・構成などの特徴をリストアップする
3. **スタイルシートにまとめる** — AIに渡せるテンプレートに整理する

---

## ステップ1: 好きな文章を集める

まず、「こういう文章が書きたい」と思うお手本を2〜3本探しましょう。ブログ記事、SNS投稿、ニュースレターなど、ジャンルは問いません。

### 良い例

- 自分がつい最後まで読んでしまったブログ記事のURL
- 「この人の言い回し好きだな」と感じるSNSアカウントの投稿

### 悪い例

- 「なんとなく有名だから」で選んだ、自分の好みと関係ない記事
- 専門用語だらけで、自分のターゲット読者には合わない文章

お手本が見つかったら、テキストファイルやメモ帳にコピーしておいてください。

## ステップ2: スタイル要素を分解する

集めた文章を見ながら、以下の7つの観点で特徴を書き出します。AIに手伝ってもらいましょう。

### AIへのプロンプト例

以下のプロンプト（＝AIへの指示文）をコピーして、お手本の文章と一緒にAIに送ってください。

```
以下の文章の執筆スタイルを分析してください。
次の7つの観点で特徴を箇条書きにしてください。

1. 文体（です・ます調 / だ・である調 / 混合）
2. 語尾のバリエーション（どんな語尾が多いか）
3. 一文の長さ（短め / 普通 / 長め）
4. 漢字とひらがなのバランス
5. 読者への呼びかけ方（「あなた」「皆さん」「読者」など）
6. たとえ話や具体例の使い方
7. 全体の構成パターン（導入→本題→まとめ など）

--- 以下が分析対象の文章です ---
（ここにお手本の文章を貼り付け）
```

AIの分析結果を読んで、「たしかにそうだな」と思う特徴をメモしておきましょう。

## ステップ3: スタイルシートにまとめる

分析結果をもとに、以下のテンプレートを埋めて「スタイルシート」を完成させます。

```markdown
# 私の執筆スタイルシート

## 基本情報
- 名前（ペンネーム可）: 
- 想定読者: 
- 主な発信テーマ: 

## 文体ルール
- 語調: （例: です・ます調、やわらかめ）
- 一文の目安: （例: 40〜60文字）
- 漢字の開き: （例: 「事」→「こと」、「物」→「もの」に開く）
- 読者の呼び方: （例: 「あなた」）

## 構成パターン
- 導入: （例: 身近な疑問から始める）
- 本文: （例: 3つのポイントに分けて説明）
- 締め: （例: 次のアクションを提案して終わる）

## 好きな表現・避けたい表現
- 好き: （例: たとえ話を多用、「〜ですよね」で共感を誘う）
- 避けたい: （例: 「〜すべきです」のような断定、上から目線）

## AIへの指示文（このまま貼れる形）
以下のスタイルで文章を書いてください：
（上の内容を1段落にまとめたもの）
```

### AIにまとめを手伝ってもらうプロンプト例

```
以下の分析結果をもとに、「AIへの指示文」欄に入れる
1段落のスタイル指示を作ってください。
箇条書きではなく、自然な文章で書いてください。
100〜150字程度でお願いします。

--- 分析結果 ---
（ステップ2でメモした内容を貼り付け）
```

## 完成したスタイルシートを試す

スタイルシートが完成したら、実際にAIに使ってみましょう。

![スタイルシートを使ったAI出力の例](/lesson-assets/atom.ai-writer.writing-goal-definition/screen_capture.png)

### テスト用プロンプト例

```
（スタイルシートの「AIへの指示文」を貼り付け）

上記のスタイルで、以下のテーマについて300字程度の文章を書いてください。
テーマ: 朝の時間を有効に使う3つのコツ
```

出力された文章を読んで、以下を確認してください。

- [ ] 語調はスタイルシート通りか？
- [ ] 読者への呼びかけ方は合っているか？
- [ ] 一文の長さや漢字のバランスは期待通りか？
- [ ] 全体の構成パターンは指定した通りか？

もし「ちょっと違うな」と思ったら、スタイルシートの該当箇所を修正して、もう一度試してみましょう。2〜3回繰り返すと、かなりしっくりくるスタイル指示が完成します。

## よくあるつまずきと対処法

### 「自分の好みがよくわからない」

まずは「嫌いな文章」を思い浮かべてみてください。「こういうのは嫌だ」の裏返しが、あなたの好みです。

### 「AIの分析結果がピンとこない」

お手本の文章が1本だけだと、特徴が見えにくいことがあります。テイストが似ている文章を2〜3本集めて、共通点をAIに聞いてみましょう。

### 「スタイルシートを使っても出力がブレる」

指示文が抽象的すぎる可能性があります。「やわらかい文体」ではなく「です・ます調で、一文40字以内、読者を『あなた』と呼ぶ」のように具体的な数値や例を入れると安定します。

## まとめ

このレッスンで作った「執筆スタイルシート」は、今後のすべてのAIライティングの土台になります。新しい記事を書くたびにゼロから文体を説明する必要がなくなり、あなたらしい文章をAIと一緒に素早く生み出せるようになります。
