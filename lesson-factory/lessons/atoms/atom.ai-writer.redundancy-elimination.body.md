# 冗長な表現を削って簡潔にする

料理でお肉の余分な脂身を切り落とすと、味がぐっと引き立ちますよね。文章も同じです。余計な言葉が多いと、読者は「結局何が言いたいの？」と迷ってしまいます。

このレッスンでは、AIツールを使って文章の「余分な脂身」を見つけ、スッキリした文章に仕上げます。

![冗長削除の流れ](/lesson-assets/atom.ai-writer.redundancy-elimination/diagram.png)

## 前提を確認する

- ChatGPT・Claude などのAIチャットツールをブラウザで開ける
- すでに書いた文章（200〜500文字程度）を手元に用意している

特別なインストールは不要です。いつも使っているAIツールでそのまま進められます。

## ステップ1：AIに冗長な表現を見つけてもらう

まず、あなたの文章をAIに貼り付けて、ムダな部分を指摘してもらいます。

次のプロンプト（＝AIへの指示文）をコピーして、AIチャットに貼り付けてください。

```
次の文章の中で、重複している表現や削っても意味が通じる部分を箇条書きで指摘してください。それぞれ「なぜ削れるか」の理由も1行で添えてください。

[ここにあなたの文章を貼る]
```

> **コツ**：「重複」「削っても通じる」「理由も添えて」の3つのキーワードを入れると、AIが具体的に答えてくれます。

### 良い例・悪い例

- ✅ **良い例**：「重複している表現を箇条書きで指摘してください」→ AIが「『思いますが、感じます』は同じ意味の重複です」と具体的に返す
- ❌ **悪い例**：「文章を直して」→ 何をどう直したか分からず、学びにならない

![AIの指摘画面](/lesson-assets/atom.ai-writer.redundancy-elimination/screen_capture.png)

## ステップ2：提案を確認して取捨選択する

AIの指摘をすべて鵜呑みにしないことが大切です。料理で味見しながら調味料を調整するように、あなたの目で1つずつ確認しましょう。

確認する3つのポイント：

1. **意味が変わらないか** — 削っても読者に誤解されない？
2. **温度感が消えないか** — 丁寧さや親しみやすさが薄れていない？
3. **大事な強調が残っているか** — 伝えたかったポイントまで消えていない？

### 良い例・悪い例

- ✅ **良い例**：「非常にとても大きな問題」→「非常に大きな問題」にする。意味は同じでスッキリ
- ❌ **悪い例**：「ぜひご参加ください」→「参加ください」にする。丁寧なニュアンスまで消えてしまう

## ステップ3：書き直してAIに再チェックしてもらう

AIの提案を参考に、あなたの手で文章を書き直します。書き直したら、もう一度AIに確認してもらいましょう。

```
次の文章に、まだ重複や冗長な表現が残っていれば指摘してください。なければ「簡潔です」と答えてください。

[書き直した文章を貼る]
```

AIが「簡潔です」と答えたら完成です。もし追加の指摘があれば、ステップ2に戻って同じ流れで確認します。

## 結果を確認する

次の3点をセルフチェックしましょう。すべて「はい」ならこのレッスンは成功です。

| チェック項目 | はい／いいえ |
|---|---|
| 元の文章より文字数が減っている | |
| 伝えたかった内容が変わっていない | |
| 読みやすさが向上していると感じる | |

> **おすすめ**：元の文章と並べてスクリーンショットを撮っておくと、自分の成長が目に見えて実感できます。

## つまずきやすいポイント

- **AIが削りすぎる** — AIは「短いほど良い」と判断しがちです。あなたの意図を超えて削っていたら、遠慮なく元に戻しましょう。対処法：「この文は強調のために残したいです。それ以外で冗長な部分を探してください」とAIに伝え直す
- **全部まとめて直そうとする** — 一度に全部直すと混乱します。1文ずつ・1箇所ずつ確認するのがコツです
- **「簡潔＝冷たい文章」と思い込む** — 簡潔でも「です・ます」調や「！」を使えば温かみは残せます。短い＝素っ気ないではありません
