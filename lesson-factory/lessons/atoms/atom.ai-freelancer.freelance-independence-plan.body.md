# 副業から独立への移行計画を立てる

あなたは今、副業で少しずつ稼げるようになってきました。「いつかは独立したい」という気持ちはあるけれど、「具体的に何をすれば？」「お金は足りる？」と不安がいっぱい——そんな状態ではないでしょうか。

このレッスンでは、**AIチャットを相棒にして、15分で「副業から独立への移行計画書」を1枚つくり上げる**ことをゴールにします。完成した計画書は、引っ越しの段取り表のようなもの。独立という大きな決断を、数字とスケジュールで「見える化」します。

## たとえで理解する——引っ越し計画と同じです

賃貸（＝今の副業生活）からマイホーム（＝独立後の生活）に引っ越すとき、いきなり荷物をまとめる人はいませんよね。まず「いつまでに引っ越すか」「いくらかかるか」「何を準備するか」をリストにします。独立計画もまったく同じ。**4つのステップを順番に埋めるだけ**で計画書が完成します。

![移行計画の全体図](/lesson-assets/atom.ai-freelancer.freelance-independence-plan/diagram.png)

上の図のように、このレッスンは「現状整理 → 目標設定 → 安心マージン計算 → スケジュール作成」の4段階で進みます。

## 前提を確認する

このレッスンは次の状態を想定しています。

- すでに何らかの副業で**月1万円以上**を稼いだ経験がある
- **ChatGPT・Claude・Gemini**など、どれか1つのAIチャットツールをブラウザで開ける

まだ副業を始めていない場合は、先に「副業のカテゴリを知る」レッスンに取り組んでみてください。

---

## ステップ1: 現状を3つの数字で整理する

まず、あなたの「今」を数字で見える化します。AIチャットを開いて、次のプロンプト（＝AIへの指示文）をコピー＆ペーストしてください。

```
次の3つの情報をもとに、私の現状を表にまとめてください。
- 副業の月平均収入: ○○万円
- 本業（会社員など）の月収: ○○万円
- 毎月の固定費（家賃・食費・通信費など）: ○○万円

表には「収入合計」「固定費」「毎月の余裕額」も自動で計算して入れてください。
```

> **○○の部分をあなたの数字に書き換えてから送信してください。**

### 良い例 / 悪い例

| | 入力の仕方 | なぜ？ |
|---|---|---|
| ✅ 良い例 | 「副業3万円・本業25万円・固定費15万円」 | 具体的な数字がAIの回答精度を上げる |
| ❌ 悪い例 | 「そこそこ稼いでいる」 | あいまいな表現ではAIが正確な表を作れない |

### つまずきポイント

> **数字を書くのが怖いとき**: 「ざっくり」でOKです。千円単位の正確さは不要です。後でいつでも直せるので、まずは大体の数字を入れましょう。

---

## ステップ2: 独立の目標を決める

次に「いつ独立するか」と「独立後にいくら稼ぐか」を決めます。AIに続けて次のように入力しましょう。

```
次の条件で、私の独立目標を1文でまとめてください。
- 独立予定時期: 20XX年X月（例: 2027年4月）
- 独立後の目標月収: ○○万円
- こだわりたい条件: （例: リモート中心・週休3日・特定分野に集中 など）

1文にまとめた後、「現状との差」も計算して教えてください。
```

### 良い例 / 悪い例

| | 入力の仕方 | なぜ？ |
|---|---|---|
| ✅ 良い例 | 「2027年4月・目標月収30万円・リモート中心」 | 期限があるとAIが逆算スケジュールを作れる |
| ❌ 悪い例 | 「いつかは独立したい」 | 「いつか」では月ごとの計画を組めない |

> **独立時期が決まらないとき**: 「仮」でOKです。仮でも日付を入れることで、AIが具体的なスケジュールを提案できるようになります。

---

## ステップ3: 安心マージンを計算する

独立で一番怖いのは「収入が途切れること」。そこで**安心マージン（＝何かあったときのための貯蓄）**を計算しておきます。

AIに次のように聞きましょう。

```
毎月の固定費が○○万円の場合、次の3つを計算してください。
1. 6か月分の生活防衛資金（＝安心マージン）
2. 現在の貯蓄が○○万円だとして、あと何万円足りないか
3. 毎月○万円ずつ貯めたら何か月で達成するか

計算式もいっしょに見せてください。
```

> この金額が貯まっていなくても独立は可能です。大切なのは**「いくら足りないか」を数字で知っておくこと**です。

### つまずきポイント

> **計算結果が大きくて不安になったとき**: それは正常な反応です。数字を見て怖くなるのは「現実を直視している証拠」。計画書は不安を減らすためのツールなので、次のステップでスケジュールに落とし込めば道筋が見えてきます。

---

## ステップ4: 移行スケジュールを作る

いよいよ計画の仕上げです。ここまでの情報をもとに、月ごとのタイムライン（＝時系列の予定表）を作ります。

```
次の条件で、副業から独立への移行スケジュールを月ごとの表にしてください。

- 現在: 2026年4月
- 独立予定: 20XX年X月
- 現在の副業収入: ○○万円/月
- 独立後の目標月収: ○○万円/月
- 安心マージンの不足額: ○○万円

各月ごとに「やるべきアクション」「目標収入」「貯蓄の進捗」を入れてください。
アクションは「がんばる」ではなく「○○する」のように具体的にしてください。
```

![AIが出力したスケジュール表の例](/lesson-assets/atom.ai-freelancer.freelance-independence-plan/screen_capture.png)

### 良い例 / 悪い例

| | アクションの書き方 | なぜ？ |
|---|---|---|
| ✅ 良い例 | 「クラウドソーシングで月2件の案件を受注する」 | 具体的で翌月に振り返れる |
| ❌ 悪い例 | 「副業をがんばる」 | 何をすれば達成か判断できない |

### つまずきポイント

> **AIの回答が長すぎるとき**: プロンプトの末尾に「表は10行以内にまとめてください」と追加してみてください。AIは指示があれば簡潔にまとめてくれます。

---

## 計画書を1枚にまとめる

ここまでの4ステップの結果を、AIに1枚の計画書にまとめてもらいましょう。

```
これまでの会話をもとに、「副業から独立への移行計画書」を作ってください。
以下の見出しで構成してください。

1. 現状サマリー（収入・固定費・余裕額の表）
2. 独立目標（時期・目標月収・条件）
3. 安心マージン（必要額・不足額・貯蓄計画）
4. 月別移行スケジュール（表形式）
5. 最初の1か月でやること（3つ以内）

Markdown形式で出力してください。
```

出力されたテキストを**そのままコピー**して、次のどれかに保存してください。

- **Googleドキュメント**に貼り付ける
- **メモ帳（Windowsの場合）/ メモ（Macの場合）**に貼り付ける
- **Notion**に新しいページとして貼り付ける

これがあなたの**最初の独立計画書**です。

---

## 完成を確認する

計画書ができたら、次の5つのチェックポイントを確認しましょう。すべて「はい」なら合格です。

- ✅ 現在の収入と目標収入の**差の金額**が書いてあるか
- ✅ 安心マージン（生活防衛資金）の**金額**が書いてあるか
- ✅ 月ごとのアクションが「○○する」のように**具体的**か
- ✅ 独立予定の**日付**が入っているか（仮でもOK）
- ✅ 「最初の1か月でやること」が**3つ以内**に絞れているか

> 1つでも「いいえ」があったら、AIに「○○が足りないので追加してください」と伝えるだけで修正してもらえます。

---

## よくあるつまずきと対処法

| つまずき | 対処法 |
|---|---|
| 数字を書くのが怖い | 「ざっくり」でOK。後からいつでも修正できます |
| AIの回答が長すぎる | 「簡潔に」「表は5行以内で」と指示を追加する |
| 独立時期が決められない | 「仮で1年後」と入れる。日付があることで計画が具体的になります |
| 計画が現実離れしている気がする | AIに「この計画の弱点を3つ指摘してください」と聞いてみましょう |
| 途中でAIの会話がリセットされた | 保存した途中結果をコピペして「続きを作って」と伝えればOK |
