# ノーコードツールの違いを理解する

「ノーコードツール」と聞くと、どれも同じに見えるかもしれません。でも実は、ツールごとに得意なことがまったく違います。たとえるなら、包丁にも出刃包丁・菜切り包丁・ペティナイフがあるのと同じです。作りたいものに合った道具を選ぶことが、最短で完成にたどり着くコツです。

このレッスンでは、代表的なノーコードツールを「何が得意か」で整理し、あなた自身の目的に合ったツールを1つ選べるようになることをゴールにします。

## 知っておくこと：ノーコードツールの3つのタイプ

ノーコードツールは大きく3つのタイプに分けられます。

| タイプ | 得意なこと | 代表的なツール |
|--------|-----------|---------------|
| **Webサイト制作型** | 見た目のきれいなサイトやポートフォリオを作る | Wix、STUDIO、Squarespace |
| **業務アプリ型** | データの管理や社内の申請フローなどを仕組み化する | Glide、AppSheet、Kintone |
| **自動化・連携型** | 「Aが起きたらBをする」のように作業を自動でつなげる | Zapier、Make（旧Integromat）、n8n |

![ノーコードツール分類図](/lesson-assets/atom.nocode-builder.compare-nocode-platforms/diagram.png)

## 見分けるポイントを押さえる

「どれを使えばいいか」を判断するときは、次の3つの質問を自分に投げかけてみてください。

1. **誰に見せるもの？** → 社外のお客さん向け → Webサイト制作型が向いている
2. **データを管理したい？** → 社内でデータを入力・検索・集計したい → 業務アプリ型が向いている
3. **手作業をなくしたい？** → メール送信やファイル整理を自動にしたい → 自動化・連携型が向いている

### 良い例と悪い例

**良い選び方の例：**
- 「お客様向けの予約サイトを作りたい」→ STUDIO や Wix を選ぶ（Webサイト制作型）
- 「チームの在庫管理をスマホでやりたい」→ Glide や AppSheet を選ぶ（業務アプリ型）

**悪い選び方の例：**
- 「とりあえず有名だから」で選ぶ → 目的とツールの得意分野がズレると、途中で行き詰まりやすい
- 「全部入り」を探す → 1つのツールで何でもやろうとすると、どれも中途半端になりがち

## AIに相談して自分に合うツールを選ぶ

迷ったときは、ChatGPT や Claude に相談するのがおすすめです。以下のようなプロンプト（＝AIへの指示文）をコピペして聞いてみましょう。

```
私は [あなたの職種] です。
[やりたいこと] を実現するためにノーコードツールを使いたいです。
以下の条件で、おすすめのツールを3つ教えてください。
- プログラミング経験はありません
- 予算は月額 [金額] 円まで
- [期限] までに完成させたいです
```

`[ ]` の中をあなた自身の情報に書き換えて送ってみてください。AIが候補を絞ってくれるので、そこから1つ選べばOKです。

## 比較メモを作って確認する

ここまでの内容を整理するために、以下のテンプレートを使って「自分の比較メモ」を作りましょう。これがこのレッスンの成果物です。

```markdown
# ノーコードツール比較メモ

## 自分の目的
- やりたいこと：
- 届けたい相手：
- 期限：

## 検討したツール
| ツール名 | タイプ | 良さそうな点 | 気になる点 |
|---------|--------|------------|------------|
| （例）STUDIO | Webサイト制作型 | デザインが自由 | データ管理は苦手 |
|  |  |  |  |
|  |  |  |  |

## 選んだツール
- ツール名：
- 選んだ理由：
```

このメモをテキストファイルやメモアプリに書き出せたら、このレッスンは完了です。

## つまずきやすいポイント

- **「無料プランでどこまでできるか」がわからない** → 各ツールの公式サイトの料金ページを確認しましょう。多くのツールは無料プランでお試しできます
- **「ノーコード」と「ローコード」の違いがわからない** → ノーコードは一切コードを書かない前提、ローコードは少しだけコードを書く場面がある、という違いです。このレッスンではノーコード（コード不要）のツールだけを扱います
- **ツールが多すぎて比較しきれない** → まずは上の3タイプのうち「自分のやりたいことに一番近いタイプ」を1つ決めて、その中の2-3個だけ比べれば十分です

## まとめ

このレッスンでは、ノーコードツールを3つのタイプ（Webサイト制作型・業務アプリ型・自動化型）に分類し、目的に合ったツールの選び方を学びました。比較メモを書き出すことで、「自分で選べた」という実感が持てたはずです。次のレッスンでは、選んだツールを実際に触ってみましょう。
