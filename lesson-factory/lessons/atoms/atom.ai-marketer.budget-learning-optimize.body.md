# 少額予算でAIツールを使って学習を進める

料理を習いたいとき、最初から高級な包丁や万能オーブンを揃える必要はありませんよね。まずは手持ちのフライパンと菜箸で、レシピを1つ完成させるのが一番です。

AIツール（＝人工知能を使って文章を作ったり分析したりするソフト）を使った学習も同じです。月に数千円の予算でも、使い方を工夫すればしっかり学習を進められます。このレッスンでは「月3,000円以内」を例に、あなたに合った学習計画を立て、AIに相談しながら実行する方法を身につけます。

**完成イメージ**: レッスンの最後に「自分専用の月間学習プラン」のドキュメントが1枚できあがります。

![学習予算プランの全体図](/lesson-assets/atom.ai-marketer.budget-learning-optimize/diagram.png)

## 予算の範囲を決める

まず、毎月いくらまで学習に使えるかを数字で決めましょう。旅行の費用を決めてからホテルを探すのと同じです。

**良い例**: 「毎月2,000円まで。半年で合計12,000円」
**悪い例**: 「安く済ませたい（上限なし）」

上限を数字で決めておくと、後でツールを選ぶときに迷わずに済みます。

> **AIに聞いてみよう**: ChatGPT や Claude に次のように聞くと、予算感の参考になります。
>
> ```
> 私はマーケティングの仕事をしています。
> AIツールの学習に月2,000円まで使えます。
> この予算で使えるAIツールの組み合わせを3パターン提案してください。
> それぞれのパターンで何ができるかも教えてください。
> ```

## やりたいことを書き出す

学習したい内容を3〜5個の「やりたいこと」に分けます。スーパーで買い物リストを作る感覚です。

たとえば：

- AIを使ってブログ記事のアイデアを出したい
- 顧客へのメール文面をAIに手伝ってほしい
- 競合のWebサイトを分析したい

「何をしたいか」を先に書き出すと、不要な機能にお金を使わずに済みます。

> **AIに聞いてみよう**: やりたいことが漠然としているときは、AIに壁打ちしてもらいましょう。
>
> ```
> 私は中小企業のマーケティング担当です。
> AIで効率化したい業務を洗い出したいのですが、
> よくある活用例を10個リストアップしてください。
> それぞれ「初心者でもすぐできる度」を★1〜3で付けてください。
> ```

## ツールを選ぶ

書き出した「やりたいこと」を満たすツールを探します。多くのAIツールには無料の「お試し期間（＝最初の数日〜数週間は無料で使える期間）」があります。

| 予算感 | おすすめの進め方 | 具体例 |
|--------|-----------------|--------|
| 月0円 | 無料プランのみで組み合わせる | ChatGPT無料版 + Google Gemini無料版 |
| 月1,000〜3,000円 | メインツール1つの有料プラン＋無料ツール | ChatGPT Plus（月約3,000円）またはClaude Pro（月約3,000円） |
| 月3,000〜5,000円 | メインツール1つ＋サブツール1つの有料プラン | ChatGPT Plus + Canva Pro |

**良い例**: 「記事作成にChatGPTの無料プランを使い、分析は無料のGoogle Trendsで済ませる」
**悪い例**: 「全部入りの高額ツールに登録して、機能の半分しか使わない」

> **AIに聞いてみよう**: ツール選びに迷ったら、AIに比較表を作ってもらいましょう。
>
> ```
> 以下の条件でAIツールを比較する表を作ってください。
> - 予算: 月2,000円以内
> - やりたいこと: ブログ記事の作成補助、メール文面の作成
> - 比較項目: 月額料金、無料プランの有無、日本語対応、得意なこと
> ```

![ツール選びの比較画面](/lesson-assets/atom.ai-marketer.budget-learning-optimize/screen_capture.png)

## 学習スケジュールを組む

1日15分で終わる小さな単位に分けます。「今日は何をするか」を前日に決めておくのがコツです。

1. **1週目**: ツールに登録し、画面に慣れる（ボタンを押して、何が起きるか見る）
2. **2週目**: 簡単なタスクを1つやってみる（AIにメール文面を1通書いてもらうなど）
3. **3週目**: 成果物（ブログ記事やメール文面など）を1つ完成させる
4. **4週目**: 振り返りと次月の計画を立てる

> **AIに聞いてみよう**: スケジュール作りもAIに手伝ってもらえます。
>
> ```
> 以下の条件で、4週間の学習スケジュールを作ってください。
> - 1日15分、平日のみ
> - 目標: ChatGPTを使ってブログ記事を1本書けるようになる
> - 各日の「やること」を1行で具体的に書いてください
> ```

## 学習効果を確認する

月の終わりに、次の3つを振り返ります。

- **作った成果物の数**: ブログ記事、メール文面など、完成したものはいくつ？
- **予算の使い方**: 今月いくら使った？ 来月も同じペースで大丈夫？
- **次にやりたいこと**: 新しく覚えたいスキルは何？

この3つをメモしておくだけで、学習のペースが安定します。

> **AIに聞いてみよう**: 振り返りもAIと一緒にやると、次のアクションが明確になります。
>
> ```
> 今月のAI学習を振り返ります。以下の結果を見て、
> 来月のアドバイスをください。
> - 作った成果物: ブログ記事の下書き2本
> - 使った金額: 0円（無料プランのみ）
> - 困ったこと: AIの回答が長すぎて要点がわからなかった
> ```

## つまずきやすいポイントに対処する

| つまずき | 原因 | 対処法 |
|----------|------|--------|
| ツールが多すぎて選べない | 比較しすぎて決められない | まず1つだけ選んで2週間使う。合わなければ変える |
| お試し期間が終わって焦る | 終了日を把握していない | 登録したらカレンダーに「◯月◯日 お試し終了」とメモ |
| 何から始めればいいか迷う | やりたいことが広すぎる | 「一番やりたいこと」を1つだけ選んでスタート |
| AIの回答が期待と違う | 質問の仕方があいまい | 「誰が」「何を」「どんな形式で」を入れて質問する |
| 予算オーバーしそう | 複数ツールに課金 | メインツール1つだけ課金、残りは無料プランで |

## 完了を確認する

以下の3つがすべて「はい」なら、このレッスンは完了です。

- [ ] 月の予算上限を数字で決めた
- [ ] 「やりたいこと」を3〜5個書き出し、使うツールを1つ以上選んだ
- [ ] 4週間の学習スケジュール（各週の目標つき）を書き出した

これらをまとめた「自分専用の月間学習プラン」のドキュメントが、あなたの成果物です。スクリーンショットを撮って保存しておきましょう。
