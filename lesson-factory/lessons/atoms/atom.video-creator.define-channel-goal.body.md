# チャンネルの目的を定める

レストランを開くとしましょう。まず「どんな料理を、誰に提供するか」を決めますよね。和食の店なのにピザを出したり、お子様連れ歓迎なのに落ち着いた雰囲気にしたりすると、お客さんは迷ってしまいます。

動画チャンネルも同じです。何について、誰に向けて発信するのかを先に決めておくと、企画を考えるときや動画を撮るときに迷わず進められます。

このレッスンでは、AI ツール（＝人工知能を使った便利なサービス。ChatGPT や Claude など）を活用しながら、あなたのチャンネルの「目的」を一文で言えるようにします。15分程度で終わるよう、手順を絞って進めましょう。

![チャンネル目的の全体像](/lesson-assets/atom.video-creator.define-channel-goal/diagram.png)

## 前提を確認する

特別な準備は必要ありません。以下の2つだけ用意してください。

- インターネットにつながるパソコンまたはスマートフォン
- 無料で使える AI ツール（ChatGPT、Claude など）のアカウント

プログラミングの知識は一切不要です。

## ステップ1: 自分の「強み」を棚卸しする

料理にたとえると、冷蔵庫に何があるか確認するステップです。いきなり「何を発信しよう？」と考えると手が止まります。まずは自分がすでに持っているものを整理しましょう。

AI ツールを開いて、次のメッセージをコピー＆ペーストして送ってみましょう。`[ ]` の部分だけ、自分の状況に書き換えてください。

```
私は動画チャンネルを始めたいと思っています。
以下の情報をもとに、私が動画で発信できそうな「強み」を5つ挙げてください。

- 仕事や専門分野: [例: 経理事務を5年やっています]
- 趣味・好きなこと: [例: 料理とカフェ巡り]
- 人からよく聞かれること: [例: Excelの使い方をよく聞かれる]
- 最近ハマっていること: [例: AIツールで仕事を効率化すること]
```

**良い例**: 「仕事や専門分野: 30代会社員で、AIツールを業務に活用しています。趣味: カメラと旅行」

**悪い例**: 「特にありません、なんでもいいです」（情報が少ないと AI も的確な提案ができません）

AI から5つの強みが返ってきたら、「これなら毎週話せそう」と感じるものに丸をつけてください。

## ステップ2: ターゲットと提供価値をAIに提案してもらう

次は「お客さん」と「メニュー」を決めるステップです。ステップ1で選んだ強みをもとに、AI に聞いてみましょう。

```
私の強みは「[ステップ1で選んだ強み]」です。
この強みを活かした動画チャンネルについて、以下の形式で5パターン提案してください。

- ターゲット（誰に見てほしいか）:
- 提供する価値（見た人が何を得られるか）:
- チャンネルの方向性を一言で:
```

AI から5パターンの提案が返ってきます。「これだ！」と思うものを一つ選びましょう。迷ったら、**一番「自分が語りやすい」と感じたもの**を選ぶのがコツです。すべてピンとこない場合は「3番目が近いけど、もう少し○○寄りにして」のように AI に伝えると、調整した提案をもらえます。

![AIとのやり取り例](/lesson-assets/atom.video-creator.define-channel-goal/screen_capture.png)

## ステップ3: 目的を一文にまとめる

選んだ「誰に」「何を」をもとに、次のテンプレートに当てはめます。

```
[誰] に [何] を [どうやって] 伝えるチャンネル
```

具体例を見てみましょう。

- ✅ **良い例**: 「プログラミング未経験の会社員に、AIツールの活用方法を週1回の実践動画で伝えるチャンネル」
- ✅ **良い例**: 「子育て中のママに、15分でできるAI時短レシピを紹介するチャンネル」
- ❌ **悪い例**: 「AIについて幅広く情報発信するチャンネル」（誰に・何を伝えるかがぼんやりしている）
- ❌ **悪い例**: 「最新テクノロジーを追うチャンネル」（広すぎて、何を期待していいかわからない）

## ステップ4: AIにフィードバックをもらって磨く

書いた目的文を AI にチェックしてもらいましょう。次のように聞いてみてください。

```
以下のチャンネル目的文をチェックしてください。
次の3点について、それぞれ10点満点で採点し、改善案を教えてください。

1. ターゲットの明確さ（誰に向けているかはっきりしているか）
2. 提供価値の具体性（見た人が何を得られるか伝わるか）
3. 差別化（似たチャンネルと何が違うか伝わるか）

目的文: [ここに目的文を貼る]
```

AI の指摘を見て、気になった点を修正しましょう。**2回ほど修正すれば十分です**。完璧を目指して何度もやり直す必要はありません。後のレッスンでいつでも見直せます。

## 結果を保存する

完成した目的文をメモ帳（＝パソコンについている簡単なメモ機能のこと）やノートアプリに保存します。後のレッスンで何度も参照するので、次の内容をまとめて保存しておきましょう。

```
【チャンネル目的シート】

■ 目的文（一文）:
（ここに完成した目的文を貼る）

■ ターゲット:
（誰に向けたチャンネルか）

■ 提供する価値:
（見た人が何を得られるか）

■ 自分の強み:
（なぜ自分がこのテーマを発信するのか）
```

デスクトップなど、すぐに見つけられる場所に保存してください。

## 確認する

次のセルフチェックをやってみましょう。すべて「はい」なら完成です。

- [ ] 目的文に「誰に」が具体的に書いてある（「みんな」ではなく特定の層）
- [ ] 目的文に「何を」が具体的に書いてある（「情報」ではなく具体的な価値）
- [ ] 目的文に「どうやって」の要素がある（動画のスタイルや頻度のヒント）
- [ ] 家族や友人に目的文を見せたとき、チャンネルのイメージが湧くと言ってもらえそう
- [ ] AI に採点してもらい、各項目7点以上をクリアした

## つまずきやすいポイント

| つまずき | 対策 |
|---------|------|
| 「誰でもいい」と思ってしまう | まずは「3ヶ月前の自分」を一人だけ想像してみてください。その人が知りたかったことが、一番リアルなコンテンツになります |
| 目的文が長くなりすぎる | 30文字以内で言い直してみましょう。短いほど強い目的文になります。AI に「この目的文を30文字以内に要約して」と頼むのも手です |
| AI の提案がしっくりこない | 自分の状況をもっと具体的に書き直して、再度質問してみてください。「こういう雰囲気がいい」「こういうのは違う」と好みを伝えると、提案の精度が上がります |
| テンプレートに当てはまらない | 無理に全部埋めなくて大丈夫です。「誰に」「何を」の2つが埋まれば OK です |
| 「こんなチャンネル需要あるかな」と不安になる | 最初から完璧な目的は誰にも作れません。まず「仮決め」して動画を1本作ってみることが大切です。目的文はいつでも修正できます |
