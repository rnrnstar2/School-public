# 法規制チェックフローを運用する

## 広告の「安全ネット」を知る

旅行に行くとき、スーツケースの中身を最終確認しますよね。忘れ物がないか、機内持ち込み不可のものが混ざっていないか。広告を公開する前にも、同じような確認が必要です。

広告文のなかには、法律で禁止されている表現が知らずに混ざっていることがあります。あとから見つかると「回収」や「お詫び」の対応に追われ、信頼も傷つきます。

このレッスンでは、AIチャットツール（ChatGPT や Claude など、質問すると文章で答えてくれるサービス）を使って、あなたが作った広告文が法律に触れていないかをチェックする「フロー（＝一連の手順の流れ）」を組み立て、実際に運用できるようになります。

![チェックフローの全体像](/lesson-assets/atom.ai-marketer.review-flow-operate/diagram.png)

---

## 前提を確認する

このレッスンに取り組む前に、以下の準備が済んでいることを確認してください。

- AIチャットツール（ChatGPT、Claude、Gemini など）のアカウントを持っている
- チェックしたい広告文（テキスト）を1つ用意している

どちらもまだの方は、先にAIチャットツールの無料アカウントを作ってから戻ってきてくださいね。広告文は「仮の文章」でも大丈夫です。たとえば自社商品を30秒で紹介する文を書いてみましょう。

---

## ステップ1: チェック項目をリスト化する

まず「何をチェックするか」のリストを作ります。料理のレシピでいう「材料の一覧」にあたる部分です。

広告に関係しやすい法律は、主に次の3つです。

| 法律名 | ひとことで言うと | 注意すべき表現の例 |
|--------|------------------|--------------------|
| **景品表示法**（＝お客さんを誤解させる表示を禁止する法律） | 大げさなアピール禁止 | 「業界No.1」「最安値」 |
| **薬機法**（＝健康食品や化粧品で病気の治療・予防ができるかのように書くことを禁止する法律） | 医療っぽい効果を約束しない | 「1週間で5キロ痩せる」「シミが消える」 |
| **著作権法**（＝他人の文章や画像を勝手に使うことを禁止する法律） | 他人のものを無断で使わない | 他社のキャッチコピー転用、SNS写真の無断利用 |

AIにチェックリストを作ってもらいましょう。以下のメッセージをAIチャットに**そのまま**貼り付けてください。

```
あなたは広告の法務チェック担当者です。
以下の広告文を、①景品表示法 ②薬機法 ③著作権法 の3つの観点でチェックするための確認項目を、法律ごとに3〜4個ずつ、合計10個リストアップしてください。
各項目は「はい／いいえ」で答えられる質問の形にしてください。

【広告文ここから】
（ここにあなたの広告文を貼り付ける）
【広告文ここまで】
```

> **ポイント**: 「あなたは〇〇です」とAIに役割を伝えると、その分野に詳しい立場で答えてくれます。

**良い例（チェック項目）：**
- 「最安値」という表現に客観的な根拠（調査データなど）があるか → はい／いいえ
- 「1週間で5キロ痩せる」のように効果を保証する表現が含まれていないか → はい／いいえ

**悪い例（チェック項目）：**
- 「法律に違反していないか」→ 範囲が広すぎて、はい／いいえで答えられない
- 「大丈夫か」→ 何を判断すればいいかわからない

## ステップ2: AIに項目ごとに判定させる

リストができたら、次はAIに「審判役」をお願いします。以下のメッセージを続けて送ってください。

```
先ほど作ったチェック項目をひとつずつ判定してください。
- 問題がある場合：どの法律に触れる可能性があるか理由を書き、修正案を1つ提案してください
- 問題がない場合：「OK」とだけ書いてください
結果を表形式でまとめてください。
```

![AIでの判定結果の例](/lesson-assets/atom.ai-marketer.review-flow-operate/screen_capture.png)

AIの回答が返ってきたら、以下のポイントを確認します。

- **「問題あり」と言われた箇所** → どの法律に触れる可能性があるか、理由を読む
- **修正案が提示されているか** → そのまま採用してもOK、ヒントにして自分で書き直してもOK

**良い例（AIの回答）：**
> 「『1週間で5キロ痩せる』は薬機法に抵触する可能性があります。効果を保証する表現に当たるため、『健康的な食習慣をサポートします』のように言い換えることをお勧めします。」

**悪い例（AIの回答）：**
> 「特に問題ないと思います。」→ 根拠がなく、チェックとして不十分。この場合は次のように追加で聞きましょう。

```
根拠が不足しています。各項目について、なぜ問題ない／あると判断したのか、法律の条文や一般的な基準に照らして理由を添えてください。
```

---

## ステップ3: 結果を記録して修正する

AIの判定結果を、メモアプリやテキストファイルに記録します。「チェック記録シート」を作るイメージです。

記録の仕方：

1. **問題なし**の項目 → 「OK」と記録
2. **問題あり**の項目 → 指摘内容・修正前の表現・修正後の表現をセットで記録
3. 広告文を修正したら、**もう一度同じフローを回して再チェック**

再チェック時はAIに次のように伝えます。

```
以下は修正後の広告文です。先ほどと同じチェック項目で再度判定し、まだ問題が残っている箇所があれば指摘してください。

【修正後の広告文ここから】
（修正した広告文を貼り付ける）
【修正後の広告文ここまで】
```

修正→再チェックを **2〜3回** 繰り返すと、ほぼ安心できる広告文に仕上がります。

---

## フローが正しく動いているか検証する

あなたのチェックフローが実用レベルかどうか、次の5つの観点で確認してください。

- [ ] チェック項目が **5個以上** あるか
- [ ] 各項目が「はい／いいえ」で判定できる形になっているか
- [ ] AIが **具体的な理由と修正案** を返してくれたか
- [ ] 修正後の広告文で **少なくとも1回は再チェック** したか
- [ ] 結果を記録として **残せている** か（メモ・ファイルなど）

すべてチェックがついたら、あなたのフローは完成です！

## つまずきポイントに対処する

| つまずき | 原因 | 対処法 |
|---------|------|--------|
| AIが「わかりません」と返す | 広告文とチェック項目が別々のメッセージになっている | 広告文とチェック項目を **1つのメッセージ** にまとめて送り直す |
| チェック項目が多すぎて混乱する | 最初から3つの法律を全部扱おうとしている | まずは **景品表示法だけ** に絞り、3〜5個で始める。慣れたら法律を追加 |
| 修正しても同じ指摘が繰り返される | AIが前の文脈を忘れている | 「次のように書き換えました」と **修正後の全文** を貼り直してチェックを依頼する |
| 薬機法と景品表示法の区別がつかない | 対象が似ていて混同しやすい | 薬機法＝**健康・美容の効果**、景品表示法＝**価格や品質のアピール**、と覚える |
| AIの判定が正しいか不安 | AIは法律の専門家ではない | AIの指摘は「あたりをつける」ためのもの。最終判断は **公式ガイドラインや専門家** に確認する |

## まとめる

このレッスンでは、AIを使った法規制チェックフローを3ステップで組み立てました。

1. **チェック項目をリスト化する** — 3つの法律（景品表示法・薬機法・著作権法）を意識して、「はい／いいえ」で判定できる項目を作る
2. **AIに判定させる** — 広告文とチェック項目をまとめて渡し、理由と修正案をもらう
3. **結果を記録して修正する** — 判定結果をメモに残し、修正後は必ず再チェックする

このフローを広告文を作るたびに回す習慣をつけると、法律リスクをぐんと減らせます。次は、ぜひあなたの実際の広告文で試してみてください。

> **大事な注意**: AIによるチェックは「下書き段階のスクリーニング（ふるい分け）」です。重要な広告や判断に迷う場合は、必ず法務担当者や弁護士など専門家に相談してください。
