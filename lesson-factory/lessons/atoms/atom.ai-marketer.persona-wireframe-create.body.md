# ペルソナ別のワイヤーをつくる

レストランのメニューを考えてみましょう。子連れの家族には写真付きの「お子様セット」が嬉しいですが、ビジネスディナーの大人にはシンプルな黒文字のメニューが好まれます。同じ料理でも、見せる相手によって見せ方を変えるのが大事ですよね。

このレッスンでは、あなたのサービスの「こういう人に届けたい」という人物像（＝ペルソナ）ごとに、Webページの設計図（＝ワイヤー）をAIツールでつくる方法を学びます。完成すると、ペルソナの悩みと目標に合ったページ構成が1枚の文書になります。

![ペルソナからワイヤーへの流れ図](/lesson-assets/atom.ai-marketer.persona-wireframe-create/diagram.png)

## 準備する

次のものを用意してください。

- チャット形式のAIツール（ChatGPT、Claude、Gemini など）をブラウザで開いておく
- メモ帳などのテキストエディタ（＝文字を入力できるアプリ）

プログラミングの知識は一切不要です。AIと会話しながら進めます。

## ペルソナを1人に絞る

まずは「誰に向けたページか」を1人だけ決めます。ペルソナは、名前・年齢・悩み・目標の4つをまとめるだけでOKです。

**良い例：**
> 名前：佐藤まゆみ / 35歳 / フリーランスのグラフィックデザイナー / 悩み：自分の作品を魅力的に見せる方法がわからない / 目標：ポートフォリオサイトで月5件の問い合わせを獲得したい

**悪い例：**
> 「30代女性」→ 悩みも目標も書かれておらず、どんなページにすればいいか判断できない

### AIにペルソナを深掘りしてもらう

自分の頭だけでペルソナを考えるのは大変です。AIに手伝ってもらいましょう。次のプロンプト（＝AIへの指示文）をコピーして、AIツールに貼り付けてください。

```
私は[あなたのサービス内容]を提供しています。
ターゲットとなる顧客は[ざっくりした顧客像]です。
この人物の具体的なプロフィール（名前・年齢・職業）、悩みを3つ、目標を3つ挙げてください。
```

`[あなたのサービス内容]` と `[ざっくりした顧客像]` の部分を自分の言葉に書き換えるだけです。たとえば：

```
私はグラフィックデザインの受託制作を提供しています。
ターゲットとなる顧客はフリーランスや小規模事業主です。
この人物の具体的なプロフィール（名前・年齢・職業）、悩みを3つ、目標を3つ挙げてください。
```

AIが提案した内容を読んで、一番しっくりくるものを1つずつ選びましょう。「この悩みはちょっと違うな」と思ったら、AIに「もう少し〇〇寄りの悩みにしてください」と追加で頼めばOKです。

## ワイヤーの構成をAIに提案してもらう

ペルソナが決まったら、次はそのペルソナに最適なページ構成をAIに提案してもらいます。

```
以下のペルソナに向けたWebページのワイヤー（＝ページの設計図）を提案してください。

【ペルソナ】
名前：佐藤まゆみ
年齢：35歳
職業：フリーランスのグラフィックデザイナー
悩み：自分の作品を魅力的に見せる方法がわからない
目標：ポートフォリオサイトで月5件の問い合わせを獲得したい

【条件】
- セクションは5つ以内にしてください
- 各セクションに「何を置くか」と「なぜこのペルソナに必要か」を書いてください
```

![AI出力のワイヤー構成例](/lesson-assets/atom.ai-marketer.persona-wireframe-create/screen_capture.png)

AIの回答から、あなたが「これがいい！」と思う構成を選びます。

**良い例：**
> ヒーローセクション（名前とキャッチコピー）／作品ギャラリー（作品画像6枚）／プロフィール（経歴とスキル）／お客様の声（3件）／お問い合わせフォーム

**悪い例：**
> 「とりあえずホーム・about・contactの3ページ」→ ペルソナの悩み（作品の見せ方）に関係する要素が何もない

### 構成を改善するAI活用テクニック

AIが出してくれた構成に「もう一押し」したいときは、次のように追加で聞いてみましょう。

```
この構成で、ペルソナの「作品を魅力的に見せたい」という悩みが最も解決されるセクションはどれですか？そのセクションをもっと具体的にしてください。
```

こうすると、AIがペルソナの悩みに焦点を当てた改善案を出してくれます。

## 成果物を1枚の文書にまとめる

AIの提案をもとに、Markdown（＝見出しや箇条書きを「#」や「-」といった記号で表すシンプルな書き方）で1枚の文書にまとめます。テキストエディタに次のように書いてください。

```
# ペルソナワイヤー: 佐藤まゆみ

## ペルソナ
- 名前: 佐藤まゆみ
- 年齢: 35歳
- 悩み: 自分の作品を魅力的に見せる方法がわからない
- 目標: 月5件の問い合わせを獲得したい

## ページ構成
1. ヒーローセクション: 名前とキャッチコピー「あなたのブランドを、デザインで輝かせる」
2. 作品ギャラリー: 代表作品6点をカード形式で並べる
3. プロフィール: 経歴・使用ツール・得意ジャンル
4. お客様の声: 過去の依頼者3名のコメント
5. お問い合わせフォーム: 名前・メール・メッセージ

## 各セクションで伝えたいメッセージ
- ヒーロー: 「この人に頼めば安心」という第一印象
- ギャラリー: ジャンルの幅と品質を直感的に伝える
- プロフィール: 信頼感を数字と実績で裏付ける
- お客様の声: 実際の声で不安を解消する
- フォーム: 3項目で迷わず送信できる
```

この文書が、あなたの成果物です。AIに「この内容をMarkdown形式で整理してください」と頼めば、コピペするだけでまとまります。

## 成果物を確認する

次のチェック項目を確認しましょう。すべてにチェックがついたら完成です。

- [ ] ペルソナの「悩み」を解決する要素が1つ以上あるか
- [ ] ペルソナの「目標」を達成できる導線（＝ページを見てからアクションを起こすまでの道筋）があるか
- [ ] セクションが5〜7個に収まっているか（多すぎると読者が迷ってしまいます）
- [ ] 各セクションに「なぜこのペルソナに必要か」の理由が書かれているか

すべてチェックがついたら完成です。スクリーンショット（＝画面の写真）を撮って保存しておきましょう。

## つまずきやすいポイントと対策

| つまずき | 対策 |
|---------|------|
| ペルソナが抽象的すぎる | 「名前をつける」「年齢を決める」の2つを必ずやる。AIに「もっと具体的にして」と頼む |
| AIの回答が長すぎて迷う | プロンプトに「5つ以内で」と制限を加える |
| 構成がペルソナと合わない | ペルソナの悩みに戻って「これは解決に役立つか？」と問う |
| 何を書けばいいかわからない | AIに「このペルソナにとって最も重要なセクションは？」と聞く |
| AIの提案をそのまま使っていいか迷う | 自分のサービスや顧客を一番知っているのはあなたです。AIの案を「たたき台」として、しっくりこない部分は自分の言葉に書き換えましょう |
