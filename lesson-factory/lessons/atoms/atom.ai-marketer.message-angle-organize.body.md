# 訴求軸を整理する

商品やサービスには、いろいろな「見せ方」があります。
たとえば料理を例に考えてみましょう。

同じ「おにぎり」でも——

- **手軽さ**を強調する：「走りながら食べられる」
- **素材の良さ**を強調する：「新潟産こしひかり使用」
- **温かさ**を強調する：「店内で握りたて」

このように、同じものでも**どの面を見せるか**で伝わり方が変わります。
この「見せ方の方向」を**訴求軸（＝お客様に伝える切り口）**と呼びます。

このレッスンでは、あなたの商品・サービスの訴求軸を、AIチャットツールを使って15分で整理します。
終わったときには「自分の商品をどう伝えればいいか」が3パターンの文章としてまとまっている状態になります。

![訴求軸整理フロー](/lesson-assets/atom.ai-marketer.message-angle-organize/diagram.png)

## 用意する

始める前に、以下を準備してください。

- 整理したい商品・サービスの名前（頭の中にあるだけでもOK）
- AIチャットツールを開いた状態（以下のどれでも大丈夫です）
  - **ChatGPT**（chat.openai.com）— 無料プランでOK
  - **Claude**（claude.ai）— 無料プランでOK
  - **Gemini**（gemini.google.com）— Googleアカウントがあれば無料
- メモ帳やGoogleドキュメントなど、結果を貼り付ける場所

特別なインストールは不要です。ブラウザでAIツールを開ければ大丈夫です。

## ステップ1: 商品の特徴を洗い出す

まずはAIに、商品の特徴をざっくり書き出してもらいましょう。
AIチャットに次の文章をコピーして貼り付けてください。`[  ]` の中はあなたの商品名に置き換えます。

```
「[商品・サービス名]」という商品があります。
この商品の魅力や特徴を、5〜10個箇条書きで挙げてください。
一般のお客様向けに、わかりやすい言葉でお願いします。
```

**良い例：**
- 「季節のフルーツケーキ」「手作りアクセサリーの通販ショップ」のように、具体的な商品名を入れる
- 「30代女性向けの保湿クリーム」のように、ターゲット情報も添える

**悪い例：**
- 「何か」「うちの商品」とだけ書いて、AIにヒントを与えない
- 商品名なしで「マーケティング案を考えて」と丸投げする

AIから返ってきたリストを眺めてみてください。
なんとなく「これは似た要素だな」と感じるグループがあるはずです。

> **💡 うまくいかないとき**: AIの回答が抽象的すぎたら、「もっと具体的に、お客様が実感できる言葉で書き直してください」と追加で伝えてみましょう。

## ステップ2: グループに分ける

次に、洗い出した特徴を似ているもの同士でまとめます。
これは引っ越しのときに荷物を「キッチン用品」「衣類」「本」と箱分けするイメージです。

たとえば「美味しい」「新鮮」「素材にこだわっている」は**品質**というグループにまとめられます。
「安い」「お得」「コスパが良い」なら**価格**のグループです。

AIにグループ分けを頼んでみましょう。

```
先ほど挙げた特徴を、似ているもの同士で3〜5つのグループに分けてください。
それぞれのグループに、2〜4文字の短い名前をつけてください。
例：「品質」「価格」「手軽さ」「見た目」など
```

このグループ名こそが**訴求軸**です。
3〜5個にまとまっていればバッチリです。

![AIでグループ分けした画面例](/lesson-assets/atom.ai-marketer.message-angle-organize/screen_capture.png)

## ステップ3: 訴求軸を絞り込む

すべての訴求軸を一度に伝えると、お客様は混乱してしまいます。
料理で言うと、全部の調味料を一度にかけると味がわからなくなるのと同じです。

メインで伝える軸を**2〜3個**に絞りましょう。

```
この商品のターゲットは[ターゲットの具体的な描写]です。
ターゲットが一番惹かれそうな訴求軸を2〜3個選んでください。
選んだ理由も短く説明してください。
```

**良い例：**
- ターゲットを「朝が忙しい子育て家庭で、朝食の準備を時短したい人」と具体的に指定する
- 「20代後半の会社員で、週末だけ料理を楽しむ人」のように生活シーンまで書く

**悪い例：**
- 「一般層」「みんな」とだけ書いて、AIが絞り込めない
- ターゲットを書かずに「いい感じに選んで」とお願いする

> **💡 ターゲットが思いつかないとき**: まずは「自分自身」や「最近買ってくれたお客様1人」をイメージして書いてみましょう。完璧でなくて大丈夫です。

## ステップ4: まとめ文章を作って確認する

最後に、結果がちゃんと使える形になっているか確認しましょう。
AIに次のように頼んで、まとめ文書を作ってもらいます。

```
選んだ2〜3個の訴求軸を使って、
この商品の魅力を伝える短い文章（100字程度）を3パターン作ってください。
それぞれの文章で、どの訴求軸を強調しているかも書いてください。
最後に、以下の形式でまとめてください。

■ 商品名：
■ 訴求軸一覧：
■ 選んだ上位2〜3軸：
■ 各軸の紹介文（100字 × 3パターン）：
```

3つの文章を読んで、「確かにそう伝わるな」と思えれば完成です。

## ステップ5: 成果物を保存する

AIが返してくれたまとめを、以下のどちらかの方法で保存しましょう。

1. **コピー＆貼り付け**: AIの回答をコピーして、Googleドキュメントやメモ帳に貼り付ける
2. **スクリーンショット**: 画面をそのまま撮影して画像として保存する

これが今回の成果物（＝完成した作品）です。
次のレッスンでは、整理した軸を使って実際にキャッチコピー（＝目を引く短い文言）を作ってみます。

## つまずきポイント

| よくあるつまずき | こうすれば解決 |
|---|---|
| AIが10個以上のグループを作ってしまった | 「3〜5個に絞ってください。似ているグループは統合してください」と追加で伝える |
| どの軸も似たような名前になってしまった | 「それぞれの軸が重ならないよう、違う視点で分け直してください」と伝える |
| ターゲットが思いつかない | まずは「自分自身」か「最近買ってくれたお客様1人」をイメージして試す |
| AIの回答が難しい言葉ばかり | 「中学生でもわかる言葉に書き直してください」と追加で伝える |
| 商品の特徴が5個も出てこない | 「競合商品と比べた違い」「お客様の声でよく聞く感想」という切り口を追加で伝える |
