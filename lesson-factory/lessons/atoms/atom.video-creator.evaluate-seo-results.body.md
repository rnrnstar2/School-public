# SEO結果を振り返る

料理を作ってお客さんに出したあと、「おいしかった？」「どこで知ったの？」と聞きたくなりますよね。動画も同じです。公開した動画がどんな人に見られているか、どんな検索キーワードで見つけてもらえたかを知ることで、次の動画をぐっと良くできます。

この Atom では、あなたが公開した動画の SEO（＝検索エンジン最適化。ここでは「YouTubeで動画が見つかりやすくなる工夫」と考えてください）の結果をAIツールと一緒に振り返り、**「次の動画で何を変えるか」が書かれた振り返りメモ**を1枚完成させます。所要時間は15分程度です。

![SEO振り返りフロー](/lesson-assets/atom.video-creator.evaluate-seo-results/diagram.png)

## 前提を確認する

この Atom に取り組む前に、次の2つを満たしているか確認しましょう。

- YouTubeに動画を**1本以上公開していて、公開から1週間以上**経っている
- YouTube Studio（＝YouTubeの管理画面。動画の再生数などのデータが見られる場所）にログインできる

まだ動画を公開していない場合は、まず動画を1本公開してからこの Atom に戻ってきてください。公開して数日だとデータがほとんど貯まらないため、**最低1週間**は待ちましょう。

## YouTube Studioで数字を見る

まずは YouTube Studio を開いて、振り返りたい動画のデータを表示します。

1. ブラウザで **studio.youtube.com** にアクセスする
2. 左側のメニューから「**コンテンツ**」をクリックする
3. 対象の動画を選び、「**分析**」をクリックする

> **メニューが見つからないとき**: 画面左上の三本線のアイコン（ハンバーガーメニュー）をクリックすると、メニュー一覧が開きます。

![YouTube Studio分析画面](/lesson-assets/atom.video-creator.evaluate-seo-results/screen_capture.png)

## 3つの数字を読み取る

分析画面の中から、次の3つの数字をメモしましょう。あとでAIツールに渡すので、正確に書き写してください。

| 項目 | どこで見るか | 何を教えてくれるか |
|------|-------------|-------------------|
| **再生回数** | 「概要」タブの上部 | 動画が何回再生されたか |
| **CTR**（＝クリック率。サムネイルを見た100人のうち何人がクリックしたかの割合） | 「リーチ」タブ | サムネイルやタイトルが魅力的だったかの目安 |
| **視聴者維持率**（＝動画をどこまで見てもらえたかの割合） | 「エンゲージメント」タブ | 動画の中身が最後まで楽しめたかの目安 |

### 数字の目安を知る

- **再生回数**: 他の人と比べるより、あなたの過去の動画と比べるのがコツです
- **CTR は 5% 以上**ならまずまず。低いときはサムネイルやタイトルの見直しがヒントになります
- **視聴者維持率は 50% 以上**が目標。途中で離脱（＝見るのをやめること）が多い箇所があれば、そこを短くする工夫が考えられます

## 検索キーワードを確認する

次に、どんな言葉であなたの動画が見つけられたかを確認します。

1. 「**リーチ**」タブを開く
2. 「トラフィック ソース」のセクションで「**YouTube 検索**」を確認する
3. 表示される「検索キーワード」をメモする

### 良い例

- 動画のテーマに関連する言葉が並んでいる（例: 料理動画なら「簡単 レシピ」など）
- 想定していたキーワードが含まれている
- 意外と検索されている言葉があって、新しい動画のアイデアが浮かぶ

### 悪い例

- まったく関係ないキーワードばかり → タイトルや説明文が動画の中身とズレている可能性
- 検索からの流入がほぼ0件 → 検索されるキーワードがタイトルに入っていない可能性

## AIツールに分析を手伝ってもらう

ここからがこの Atom の一番大事なステップです。ChatGPT・Claude・Cursorなど、あなたが使い慣れたAIツールに、先ほどメモした数字をそのまま貼り付けて分析を頼みましょう。

### 使うプロンプト（そのままコピーしてOK）

```
私はYouTubeで動画を投稿しています。以下のデータを見て、
SEOの観点から改善点を3つ、具体的に教えてください。

## 動画データ
- 動画タイトル: （ここにあなたの動画タイトルを入れる）
- 再生回数: （ここに数字を入れる）
- CTR（クリック率）: （ここに数字を入れる）%
- 視聴者維持率: （ここに数字を入れる）%
- 主な検索キーワード: （ここにキーワードを入れる）

## お願い
- 各改善点について「なぜそうすべきか」の理由もつけてください
- 私は初心者なので、専門用語は避けてわかりやすく説明してください
```

### AIの回答をさらに深掘りする

AIの回答で気になる点があったら、追加で質問してみましょう。たとえば：

- 「CTRを上げるために、具体的にどんなサムネイルにすればいいですか？今のサムネイルは（特徴を説明）です」
- 「視聴者維持率が3分あたりで急に下がっています。何が原因だと思いますか？」
- 「『○○ 初心者』というキーワードで上位に出るためのタイトル案を3つ考えてください」

AIが提案してくれた改善点の中から、あなたが「次の動画で試せそう」と思うものを**1〜2個**ピックアップしてください。全部やろうとすると大変なので、まずは少しずつで大丈夫です。

## 振り返りメモを完成させる

最後に、今回の振り返り結果を1枚のメモにまとめます。AIに「メモのひな形を作って」と頼んでもOKです。

メモには次の4つを入れましょう。

1. **振り返った動画のタイトル**
2. **良かった点**（見つかりやすかったキーワード、高かった数字など）
3. **改善したい点**（低かったCTR、離脱が多かった箇所など）
4. **次の動画で試すこと**（AIの提案から選んだ1〜2個）

### 良いメモの例

```
動画: 「初心者でも10分で作れるペペロンチーノ」
良かった点: 「ペペロンチーノ 初心者」で検索上位に表示された
改善点: CTR 2.8%が低い。サムネイルが地味だったかも
次に試すこと: サムネイルに完成写真を大きく載せる + タイトルに「超簡単」を追加
```

### 悪いメモの例

```
再生回数が少なかった。もっと頑張る。
```

→ 「なぜ少なかったのか」「何を変えるのか」が書かれていないのが問題です。「頑張る」では次の行動が決まりません。

## つまずきやすいポイント

| つまずき | 対処法 |
|---------|--------|
| 数字が少なすぎて判断できない | 公開から1週間未満だとデータが不足します。もう少し待ちましょう |
| CTRの欄が表示されない | 動画の再生回数が少ないと表示されないことがあります。その場合は再生回数と維持率の2つだけで分析を進めましょう |
| CTRが極端に低い（2%以下） | タイトルとサムネイルを**セットで**見直しましょう。どちらか片方だけ変えても効果が薄いことがあります |
| AIの提案が抽象的すぎる | 「もっと具体的に教えてください。例を出してください」と追加で聞いてみましょう |
