# 案件を探せるプラットフォームを知る

## はじめに

あなたが「AIを使ってお仕事をしたい」と思ったとき、まず必要なのは**仕事を探す場所**を知ることです。

たとえば、美味しいパンを買いたいとき、近所のパン屋を知らないと困りますよね。仕事探しも同じで、どこに仕事が集まっているかを知るのが第一歩です。

このレッスンでは、日本と海外の代表的な「案件（＝依頼される仕事のこと）」を探せるプラットフォーム（＝仕事のマッチングサイトのこと）を知り、**AIを使って自分に合う場所を効率よく見つける**ところまでやります。

![プラットフォーム比較図](/lesson-assets/atom.ai-freelancer.platform-overview/diagram.png)

## 前提を確認する

このレッスンに必要なものは次の3つだけです。

- インターネットを見られるブラウザ（Chrome や Safari など）
- ChatGPT、Claude など、どれか1つのAIチャットツールが使える状態
- 15分の時間

特別なソフトのインストール（＝パソコンに新しいアプリを入れること）は不要です。

## 代表的なプラットフォームを知る

まずは、どんなサイトがあるかをざっと見ていきましょう。

### 日本のプラットフォーム

| プラットフォーム | 特徴 | AI案件の多さ |
|---|---|---|
| **クラウドワークス** | 日本最大級。案件の種類が豊富 | ★★★ 多い |
| **ランサーズ** | 自分のスキル一覧を載せてアピールしやすい | ★★☆ まあまあ |
| **ココナラ** | 自分のスキルを「出品」できる。初心者が始めやすい | ★★☆ まあまあ |

### 海外のプラットフォーム

| プラットフォーム | 特徴 | AI案件の多さ |
|---|---|---|
| **Fiverr** | ギグ（＝小さな仕事の単位）で出品。世界中の人に届く | ★★★ 多い |
| **Upwork** | 長めの案件に強い。英語力が必要 | ★★★ 多い |

![プラットフォーム画面例](/lesson-assets/atom.ai-freelancer.platform-overview/screen_capture.png)

## AIを使ってプラットフォームを比較する

ここからが、このレッスンの一番大事なポイントです。5つのプラットフォームの中から自分に合うものを選ぶために、**AIチャットに相談**してみましょう。

### AIに聞いてみるプロンプト例

次の文章をそのままコピーして、ChatGPT や Claude に貼り付けてください。

```
私はプログラミング経験がない会社員で、AIツール（ChatGPTなど）を使った副業を始めたいです。
以下の条件で、最初に登録すべきフリーランスプラットフォームを1つだけ教えてください。

条件:
- 日本語でやり取りしたい
- まだ実績がゼロ
- 平日の夜と週末に2〜3時間使える
- AI関連の案件が見つかるところがいい

各プラットフォーム（クラウドワークス、ランサーズ、ココナラ、Fiverr、Upwork）の
メリット・デメリットも簡単に比較してください。
```

> **ポイント**: 「条件」の部分をあなた自身の状況に書き換えると、より的確な回答が返ってきます。たとえば「英語が得意」なら海外プラットフォームが候補に入りますし、「デザインが好き」ならココナラが合うかもしれません。

### AIの回答を読むときのコツ

AIの回答は参考情報です。次の点に注意して読みましょう。

- **良い例**: 「AIが勧めてくれたクラウドワークスを、実際にサイトを開いて案件を検索してみてから決める」
- **悪い例**: 「AIの回答だけを信じて、サイトを見ずに登録する」→ 実際に自分の目で案件を確認しましょう

## 自分に合うプラットフォームを選ぶ

### 良い選び方の例

- 「まずはクラウドワークスに登録して、AI関連の案件を10件眺めてみる」
- 「ココナラでChatGPTを使った文章作成を出品してみる」

### 悪い選び方の例

- 「いきなりUpworkで英語の応募文を書こうとする」→ 英語に自信がなければまだ早いです
- 「全サイトに同時に登録して、どこに何を書いたか分からなくなる」→ まずは1サイトに絞りましょう

選び方のポイントは3つです。

1. **まずは日本のサイトから始める** — 日本語でやり取りできるので安心です
2. **AI関連の案件数を確認する** — サイト内で「AI」「ChatGPT」と検索して、どれくらい案件があるか見てみましょう
3. **報酬（＝仕事の対価としてもらえるお金）の相場を見る** — 最初は安くても、実績作りと割り切るのがコツです

## 成果物をつくる：プラットフォーム比較メモ

ここまで学んだことを、短いメモにまとめましょう。以下のテンプレート（＝ひな形）をコピーして、あなた自身の言葉で埋めてください。

```
# 私のプラットフォーム比較メモ

## 調べたプラットフォーム
- クラウドワークス: （感想を1行で）
- ランサーズ: （感想を1行で）
- ココナラ: （感想を1行で）
- Fiverr: （感想を1行で）
- Upwork: （感想を1行で）

## 最初に登録するところ
（プラットフォーム名）

## 選んだ理由
（AIに相談した結果と、自分で実際にサイトを見た感想をもとに1〜2行で）
```

> **AIを使った時短テクニック**: メモを書くのが苦手なら、AIチャットに「さっきの比較結果を、上のテンプレートに沿ってまとめて」と頼むこともできます。ただし、最終的な判断はあなた自身で行いましょう。

## 確認する

レッスンの終わりに、次の4つをチェックしてみましょう。

- [ ] プラットフォームを3つ以上、名前と特徴をセットで言える
- [ ] 自分がまず登録するプラットフォームを1つ選び、その理由を説明できる
- [ ] AIチャットを使ってプラットフォームの比較を試した
- [ ] 比較メモを1つ完成させた

## つまずきやすいポイント

| つまずき | 対処法 |
|---|---|
| 「どれも同じに見える」 | まずは「日本向けか世界向けか」で2グループに分けてみましょう。そのあと「出品型か応募型か」で絞ると違いが見えます |
| 「登録だけで終わってしまう」 | 登録後、必ずAI関連の案件を5件以上見ましょう。「AI」「ChatGPT」「自動化」で検索するのがコツです |
| 「どれを選べばいいか迷う」 | このレッスンで紹介したAIプロンプトをそのまま使ってみてください。それでも迷うなら、クラウドワークスから始めれば間違いありません |
| 「AIの回答が長すぎて読みきれない」 | 「3行で要約して」と追加で聞けば、短くまとめてくれます |
