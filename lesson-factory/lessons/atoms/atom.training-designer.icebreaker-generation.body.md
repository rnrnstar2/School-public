# アイスブレイクのアイデアをAIで出す

研修やワークショップの冒頭で場を和ませる「アイスブレイク」。毎回ゼロから考えるのは大変ですよね。料理にたとえると、レシピ本を開いて「今日の献立どうしよう…」と悩む時間を、AIという万能レシピ提案マシンに肩代わりしてもらうイメージです。

このレッスンでは、AIチャットツール（ChatGPTやClaude）に「こんな研修にぴったりのアイスブレイクを考えて」とお願いして、すぐ使えるアイデアリストを手に入れる方法を学びます。所要時間は約15分です。

## 用意するもの

- AIチャットツール（以下のどれか1つでOK）
  - **ChatGPT**（https://chat.openai.com を開くだけ）
  - **Claude**（https://claude.ai を開くだけ）
- メモ帳アプリ（Windowsの「メモ帳」やMacの「テキストエディット」など、文字を貼り付けられるものなら何でもOK）

## 全体の流れを確認する

このレッスンは3ステップで進みます。

![全体フロー](/lesson-assets/atom.training-designer.icebreaker-generation/diagram.png)

1. **研修の条件を整理する** — 参加人数・時間・目的をメモする
2. **AIにプロンプト（＝お願い文）を送る** — テンプレートに条件を埋めて送信する
3. **出てきたアイデアを選んで仕上げる** — 良いものを選び、自分の言葉で微調整する

## ステップ1: 研修の条件を整理する

AIに良いアイデアを出してもらうには、「どんな場で使うのか」を伝えることが大切です。まず、以下の3つをメモしましょう。

| 項目 | 記入例 |
|------|--------|
| 参加人数 | 20人 |
| 使える時間 | 5分 |
| 研修の目的 | 新入社員同士の緊張をほぐす |

**良い例**: 「新入社員20人、5分で緊張をほぐしたい」→ 具体的なので、AIが的確な提案をしやすい

**悪い例**: 「アイスブレイクを考えて」→ 条件がないので、的外れな提案が返ってきやすい

## ステップ2: AIにプロンプトを送る

プロンプト（＝AIへのお願い文）は、「役割」「条件」「出力形式」の3つを含めると精度が上がります。以下のテンプレートをコピーして、【 】の部分だけ書き換えてください。

```
あなたは研修ファシリテーションの専門家です。
以下の条件に合うアイスブレイクのアイデアを5つ提案してください。

## 条件
- 参加人数: 【20人】
- 使える時間: 【5分】
- 目的: 【新入社員同士の緊張をほぐす】
- 場所: 【会議室、机あり】
- 制約: 【大きな声を出せない環境】

## 出力形式
それぞれのアイデアについて以下を書いてください:
1. アイデア名
2. やり方（3行以内）
3. 盛り上がりやすいポイント
4. 注意点
```

![プロンプト送信画面](/lesson-assets/atom.training-designer.icebreaker-generation/screen_capture.png)

### 送信して結果を見る

テンプレートを貼り付けて送信ボタンを押すと、数秒でAIが5つのアイデアを返してくれます。

### もっと良くしたいときの追加プロンプト

AIの回答がいまひとつだったら、次のように追加で聞いてみましょう。

- 「もっと体を動かす系のアイデアに変えてください」
- 「オンライン研修でも使えるバージョンにしてください」
- 「3番のアイデアを、もう少し詳しく手順を書いてください」

このように「会話を重ねて絞り込む」のがAIチャットの強みです。一発で完璧な答えが出なくても、やりとりを繰り返せば欲しいものに近づきます。

## ステップ3: アイデアを選んで仕上げる

AIが出してくれた5つのアイデアから、実際に使うものを1〜2個選びましょう。選ぶときのチェックポイントは以下の通りです。

- **参加者が恥ずかしくないか?** — 初対面で歌を歌わせるようなアイデアは避ける
- **時間内に収まるか?** — 準備や説明の時間も含めて考える
- **道具が必要か?** — 当日すぐ用意できるものか確認する

選んだアイデアは、メモ帳アプリにコピー＆ペーストして、自分の言葉で微調整してください。AIの文章をそのまま読み上げるより、「自分の研修」に合わせた言い回しにすると参加者に伝わりやすくなります。

**良い例**: AIの「参加者はペアになり…」を「では、隣の方とペアになってください」に直す → 当日の進行台本になる

**悪い例**: AIの出力をそのまま印刷して配る → 堅い文体で場が和まない

## 確認する: 完成チェックリスト

以下の3つが揃っていれば、このレッスンは完了です。

- [ ] AIチャットにプロンプトを送信して、アイスブレイクのアイデアを5つ受け取った
- [ ] その中から1〜2個を選んで、自分の言葉でメモ帳に書き直した
- [ ] AIとのやりとり画面のスクリーンショットを1枚撮った（成果物の証拠として）

## つまずきやすいポイントと対処法

| つまずき | 対処法 |
|----------|--------|
| AIが英語で回答してきた | プロンプトの最後に「日本語で回答してください」と追加する |
| アイデアが自分の研修に合わない | 条件（人数・時間・目的）をもっと具体的に書き直して再送信する |
| 5つでは足りない / 多すぎる | 「あと3つ追加してください」「3つに絞ってください」と追加で伝える |
| どのAIツールを使えばいいかわからない | ChatGPTかClaudeのどちらか、ブラウザで開けるほうを使えばOK |

## まとめ

このレッスンで身につけたことは、「AIに条件を伝えてアイデアを引き出し、自分の判断で選んで仕上げる」という流れです。アイスブレイクに限らず、研修のプログラム設計やワークシート作りなど、さまざまな場面で同じ考え方が使えます。
