# レビューと口コミをAIで収集・整理する

旅行に行く前に「どのホテルが良いかな」とネットの口コミをいくつも読んで決めた経験はありませんか？ お店や商品を運営するあなたにとっても、お客様の声を知ることは売上アップや改善の大きなヒントになります。

このレッスンでは、AIチャットツール（ChatGPTやClaude など）を使って、ネットに散らばるレビューや口コミを集め、見やすい表に整理する方法を **15分** で体験します。完了すると「お客様の声リスト」が1枚できあがります。

![口コミ収集フロー](/lesson-assets/atom.ec-operator.collect-review-data/diagram.png)

---

## 目的を決める

まずは「どの商品の」「どんな声」を集めたいかをハッキリさせましょう。目的が決まっていると、AIへの頼み方がスッキリして、的確な結果が返ってきます。

**良い例**
- 「自社のハンドクリームについて、★3つ以下の低評価レビューだけを集める」
- 「競合A社のシャンプーの高評価コメントから強みを調べる」

**悪い例**
- 「色々なカテゴリーの商品レビューを一緒くたに集める」（目的がぼやけて整理しにくくなります）

> **ワンポイント**: 最初は1商品・1サイトに絞るのがコツです。慣れたら範囲を広げましょう。

## レビューをコピーして集める

目的の商品ページ（Amazon、楽天市場、Googleマップなど）を開き、気になるレビューの文章をコピーします。

1. 商品ページのレビュー一覧を開く
2. レビュー本文を **10〜30件** 程度、まとめて選択してコピーする
3. パソコンのメモ帳やGoogleドキュメントに貼り付けて保存しておく

> **ポイント**: 完璧に整形する必要はありません。星の数やユーザー名が混ざっていても、AIが読み取ってくれます。

## AIに整理を頼む

ChatGPT（またはClaude、Cursorなど好みのAIツール）を開き、集めた口コミを貼り付けます。そのあと、以下のようなプロンプト（＝AIへの指示文）を送ります。

### プロンプト例 1：良い点・悪い点に分類する

```
以下はある商品のレビューです。
「良い点」と「悪い点」に分けて、CSV形式（カンマ区切り）の表にまとめてください。
列は「分類, レビュー要約, 出現回数の目安」の3列でお願いします。

---
（ここに集めたレビューを貼り付ける）
```

### プロンプト例 2：改善ヒントまで出してもらう

```
上の表をもとに、商品の改善アイデアを3つ提案してください。
それぞれ「どのレビューの声に基づいているか」も添えてください。
```

AIが返してくれた表を確認して、内容がおかしければ「★2以下のレビューだけに絞ってやり直してください」のように追加で指示を出せます。

![AIチャット画面での整理イメージ](/lesson-assets/atom.ec-operator.collect-review-data/screen_capture.png)

## 結果を保存する

1. AIが作ってくれたCSV形式のテキストをコピーする
2. Googleスプレッドシート（またはExcel）を開き、セルA1に貼り付ける
3. ファイル名を「商品名_レビュー整理_日付」にして保存する

これであなた専用の **「お客様の声リスト」** が完成です！

## 確認する

以下の3つが揃っていれば、このレッスンは完了です。

- [ ] AIに口コミデータを渡して、「良い点」「悪い点」に分類された表が手元にある
- [ ] その表をスプレッドシート（またはExcel）に保存できた
- [ ] 表を見て「次にどこを改善できそうか」が1つ以上思いつく

## つまずきやすいポイント

| よくある困りごと | 対処法 |
|---|---|
| レビューが少なすぎて表がスカスカ | 同じ商品の別サイト（Amazon→楽天など）からも集めてみましょう |
| AIの分類がイマイチ | 「★2以下だけ」「配送に関するコメントだけ」のように条件を絞って再度お願いしましょう |
| コピーしたテキストが長すぎてAIに貼れない | 1回20件ずつに分けて貼り、「前回の続きです」と伝えれば大丈夫です |
| 表がCSV形式で返ってこない | 「CSV形式（カンマ区切り）で出力してください」と明示的に指示を追加しましょう |

## 次のステップ

完成した「お客様の声リスト」は、商品説明の改善や新商品企画の参考資料としてすぐに活用できます。悪い点の列をざっと眺めるだけでも「配送が遅い」「使い方が分かりにくい」といった改善ヒントが見つかるはずです。
