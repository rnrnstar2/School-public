# AIで広告文の案を作る

広告文を作るのは、料理に似ています。美味しい料理を作るには、レシピ（目的）と新鮮な食材（商品の魅力）が必要ですよね。AI（＝人工知能。ここではChatGPTやClaudeのようなチャットツールのこと）は、あなたの優秀なアシスタントシェフです。あなたが食材とレシピを渡せば、あっという間に美味しそうな料理（広告文）を何パターンも作ってくれます。

このレッスンでは、AIを使って**15分以内**に、あなたの商品の広告文の案を複数パターン作る方法を学びます。最終的に、「自分の商品の広告文を、AIを使っていつでも作れる」という状態を目指します。

![広告文作成の全体フロー](/lesson-assets/atom.ec-operator.create-ad-copy-variants/diagram.png)

## 準備する（3分）

まずは、商品の魅力を**3つ**ピックアップしましょう。メモ帳やスマホのメモで構いません。

**良い例:**
- 「無添加で敏感肌にも使える」
- 「保湿力が48時間続く」
- 「1本1,980円でコスパが良い」

**悪い例:**
- 「良い商品」（→ 何が良いのか具体的に書きましょう）
- 「人気」（→ 誰に人気？どう人気？を掘り下げましょう）

次に、AIツールをブラウザで開きましょう。以下のどれでもOKです。

- **ChatGPT**（https://chat.openai.com を開く）
- **Claude**（https://claude.ai を開く）
- **その他のAIチャットツール**

どれを使うか迷ったら、ChatGPTが最も広く使われているので、まずはそちらをおすすめします。

## 指示を出す（5分）

AIに広告文を作ってもらうには、プロンプト（＝AIへの指示文）を入力します。料理で言うなら、シェフに「こんな料理を作って！」とお願いする注文票です。

### プロンプトの組み立て方

良いプロンプトには、次の**4つの要素**を入れます。

| 要素 | 説明 | 例 |
|------|------|----|
| **誰に** | ターゲットとなるお客さん | 20代女性、乾燥肌に悩んでいる |
| **何を** | 商品名と特徴 | 無添加の化粧水「うるおいの泉」 |
| **どんな雰囲気で** | 文章のトーン | 親しみやすく、友達に話すように |
| **条件** | 文字数やパターン数 | 140字以内で3パターン |

### そのままコピペできるプロンプト例

以下のテンプレートの【　】部分を、あなたの商品に合わせて書き換えてください。

```
以下の条件で、広告文を3パターン作ってください。

■ ターゲット: 【20代女性、乾燥肌に悩んでいる】
■ 商品: 【無添加の化粧水「うるおいの泉」】
■ 商品の魅力:
  1. 【無添加で敏感肌にも使える】
  2. 【保湿力が48時間続く】
  3. 【1本1,980円でコスパが良い】
■ 雰囲気: 【親しみやすく、友達に話すような口調】
■ 文字数: 【140字以内】
■ 用途: 【SNS広告（Instagram）】
```

**良い例:**
「20代女性向けに、乾燥肌の悩みを解決する無添加の化粧水の広告文を3パターン作ってください。語り口は親しみやすく、140字以内でお願いします。」

**悪い例:**
「化粧水の広告文を作って。」
→ これだと誰に向けた文章なのかAIが判断できず、ぼんやりした広告文しか出てきません。

## 案を選ぶ（5分）

AIから複数の広告文案が返ってきたら、次の手順で選びましょう。

### ステップ1: まず読み比べる

3つの案を声に出して読んでみてください。「お客さんの立場で聞いたとき、どれが一番心に残るか？」が判断基準です。

### ステップ2: 気になる点を修正してもらう

そのまま使ってもOKですが、少し違うなと感じたら、AIに追加の指示を出しましょう。

**修正指示の例:**
- 「2番目の案をベースに、もっと若者向けの言葉遣いにしてください」
- 「『送料無料』というキーワードを入れてください」
- 「もう少し短く、100字以内にしてください」
- 「絵文字を少し入れて、SNS向けにしてください」

### ステップ3: 完成版をコピーして保存する

気に入った広告文をメモ帳やドキュメントにコピー＆ペーストして保存しましょう。これがあなたの成果物です。

## 確認する（2分）

最後に、作った広告文が以下の**3つのチェックポイント**を満たしているか確認しましょう。

- [ ] **魅力チェック**: 商品の魅力が具体的に伝わっているか？（「良い商品です」のような曖昧な表現になっていないか）
- [ ] **行動チェック**: 読んだ人が「詳しく見たい」「買いたい」と思えるか？
- [ ] **正確チェック**: 誤字脱字がないか？価格や商品名に間違いはないか？

1つでもチェックが付かない場合は、AIに「○○の部分を改善して」と指示を出して修正してもらいましょう。

## つまずいたときは

| こんなとき | こうしてみましょう |
|------------|--------------------|
| AIの返答がイマイチ | プロンプトに「ターゲット」「商品の魅力」「雰囲気」が入っているか確認。足りない要素を追加して再度お願いする |
| 案が似たり寄ったり | 「1つ目はフォーマル、2つ目はカジュアル、3つ目は疑問形で始めてください」とトーンの違いを指定する |
| 文字数が合わない | 「○○字以内に短くして」と具体的な数字で指示する |
| AIツールの画面がわからない | 画面下部のテキスト入力欄に文章を入力し、送信ボタン（紙飛行機マーク）を押すだけです |
