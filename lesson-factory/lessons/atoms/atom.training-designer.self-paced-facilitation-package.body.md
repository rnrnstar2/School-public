# 自習教材と進行支援資料を作る

## このレッスンでできるようになること

あなたが研修やワークショップを開催するとき、当日参加できない人や、あとから復習したい人のために「自習教材」があると助かりますよね。しかし、自習教材をゼロから作るのはかなり大変です。

このレッスンでは、AIを使って **自習教材**（受講者が一人で進められるガイド）と **進行支援資料**（講師やファシリテーターが使う手引き）を15分以内で作成する方法を学びます。

料理にたとえると、自習教材は「レシピカード」、進行支援資料は「料理教室の講師用メモ」のようなものです。レシピカードがあれば自宅で一人でも作れますし、講師用メモがあれば初めて教える人でもスムーズに進行できます。

![自習教材と進行支援資料の全体像](/lesson-assets/atom.training-designer.self-paced-facilitation-package/diagram.png)

## 準備する

以下のものを用意してください。

- **AIツール**: ChatGPT、Claude、Cursor のいずれか（ブラウザで使えるものでOK）
- **研修テーマ**: あなたが教えたい内容を1つ決めておく（例: 「ChatGPTの基本的な使い方」「議事録をAIで作る方法」など）
- **テキストエディタ**: メモ帳やGoogleドキュメントなど、文章を書ける場所

## ステップ1: 自習教材のたたき台をAIに作ってもらう

まず、AIに自習教材のたたき台を作ってもらいます。以下のプロンプト（＝AIへの指示文）をコピーして、あなたの研修テーマに合わせて書き換えてください。

### 使うプロンプト例

```
あなたは研修教材の専門家です。
以下のテーマについて、受講者が一人で学べる自習教材を作ってください。

【テーマ】（ここにあなたのテーマを書く。例: ChatGPTで議事録を作る方法）
【対象者】（ここに受講者の特徴を書く。例: パソコンは使えるがAIは初めての事務職の方）
【所要時間の目安】15分

以下の構成で出力してください：
1. はじめに（このガイドで何ができるようになるか）
2. 準備するもの
3. 手順（ステップごとに具体的に）
4. うまくいかないときのヒント
5. 確認テスト（自分でできたか振り返る質問を2-3個）
```

### 良い例と悪い例

**良い例**（具体的で明確）:
> 【テーマ】ChatGPTを使って会議の議事録を3分で作る方法  
> 【対象者】毎週の定例会議で議事録を担当している営業部の社員。パソコンは日常的に使うが、AIツールは使ったことがない

**悪い例**（あいまい）:
> 【テーマ】AIの使い方  
> 【対象者】社員

テーマと対象者が具体的であるほど、AIは的確な教材を出してくれます。

## ステップ2: 出力を確認して修正する

AIが出力した教材をそのまま使うのではなく、次の3つの観点でチェックしてください。

1. **正確性**: 手順どおりに実際にやってみて、問題なく進められるか
2. **わかりやすさ**: 初めての人が読んでも迷わないか。専門用語が説明なしで使われていないか
3. **過不足**: 余計な情報が多すぎないか。逆に足りない説明はないか

修正が必要な箇所を見つけたら、AIに追加で指示を出しましょう。

### 修正プロンプト例

```
ステップ3の「プロンプトを入力する」の部分をもっと詳しくしてください。
具体的には、どの画面のどこに入力するか、スクリーンショットの代わりに
言葉で場所を説明してください。
```

## ステップ3: 進行支援資料を作る

自習教材ができたら、次は「講師やファシリテーター向けの進行支援資料」を作ります。これは研修を対面やオンラインで実施するときに、進行役が手元に持っておくためのメモです。

### 使うプロンプト例

```
先ほど作った自習教材をもとに、ファシリテーター（進行役）向けの
進行支援資料を作ってください。

以下を含めてください：
1. タイムテーブル（各パートの所要時間の目安）
2. 各パートでの声かけ例（「ここまでできた方は手を挙げてください」など）
3. よくある質問と回答（受講者から出そうな質問を3つ）
4. つまずきポイントと対処法（受講者が詰まりやすい場所と、その助け方）
5. 準備チェックリスト（研修前に確認すべきこと）
```

## ステップ4: 成果物を仕上げる

最後に、自習教材と進行支援資料をそれぞれ別のファイルに整理して保存します。

- **自習教材**: `自習ガイド_（テーマ名）.md` または `.docx`
- **進行支援資料**: `進行支援メモ_（テーマ名）.md` または `.docx`

![完成した資料の例](/lesson-assets/atom.training-designer.self-paced-facilitation-package/screen_capture.png)

ファイル名にテーマ名を入れておくと、あとから探しやすくなります。

## 確認する

以下のチェックリストで、成果物ができているか確認しましょう。

- [ ] 自習教材に「はじめに」「準備するもの」「手順」「ヒント」「確認テスト」が含まれている
- [ ] 進行支援資料に「タイムテーブル」「声かけ例」「よくある質問」が含まれている
- [ ] 専門用語には説明が添えられている
- [ ] 手順を実際にやってみて、問題なく進められた

すべてにチェックが入れば、このレッスンは完了です。

## つまずいたときは

| 困りごと | 対処法 |
|---------|--------|
| AIの出力が長すぎる | 「箇条書きで簡潔にまとめ直してください」と指示する |
| 対象者に合わない表現がある | 「○○を知らない人向けに書き直してください」と追加指示する |
| 手順が抜けている | 自分で実際にやってみて、足りない手順をAIに追加してもらう |
| どのAIツールを使えばいいかわからない | ChatGPT（無料版）で十分です。ブラウザで chat.openai.com を開いてください |
