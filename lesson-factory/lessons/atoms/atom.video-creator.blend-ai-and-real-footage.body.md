# 実写とAI素材を自然に混ぜる

動画を作っていると、「ここにもう1カット欲しいけど、撮影し直すのは大変…」と感じることがありませんか？ そんなとき頼りになるのが、AIで映像を生成するツールです。

これを料理に例えてみましょう。実写の映像は「新鮮なお肉や野菜」、AIで作った映像は「プロの味付けがされた万能だし」です。この2つを上手に組み合わせれば、あなたの動画は格段にリッチでプロっぽい仕上がりになります。ただし、だしを入れすぎると味がぼやけるように、AI素材の使いすぎにも注意が必要です。

このレッスンでは、あなたが撮った映像とAIで作った映像を、違和感なく自然に混ぜるコツを15分で学びます。最終的に、実写とAI素材を1本の短い動画にまとめた「ビフォーアフター比較メモ」を作ります。

![実写とAI素材の合成フロー](/lesson-assets/atom.video-creator.blend-ai-and-real-footage/diagram.png)

## 撮影素材を用意する

まずは、あなたがスマートフォンなどで撮影した動画を1本用意します。5〜15秒程度の短いもので十分です。

**良い例:**
- 揺れが少なく、明るい場所で撮影されている動画
- 屋外の風景、机の上の商品、歩いている人など、シンプルな構図

**悪い例:**
- 手ぶれがひどく、暗い部屋で撮影された動画
- 被写体が多すぎて何がメインかわからない動画

実写の映像は、AI素材と混ぜたときの「リアルさの基準」になります。AIに寄せるのではなく、AIを実写に合わせるのがコツです。

> **つまずきポイント:** 「いい素材がない…」と思ったら、まずはスマホで窓の外の景色を10秒撮ってみてください。練習用にはそれで十分です。

## AIに映像を作ってもらう

次に、実写だけでは表現できない部分をAIツールで作ります。たとえば、夕焼けの空に鳥が飛ぶシーンや、商品が光に包まれるイメージ映像などです。

### ChatGPTで指示文を考える

いきなりAI映像生成ツールに指示を出すのは難しいので、まずChatGPTに相談しましょう。

**プロンプト例:**
```
私は実写で「カフェでコーヒーを注ぐ動画」を撮りました。
この動画に繋げる、おしゃれなイメージ映像をAIで作りたいです。
Luma Dream Machine用の英語プロンプトを3パターン考えてください。
映像の雰囲気は「温かみ・リラックス」でお願いします。
```

ChatGPTが出してくれたプロンプトの中から、気に入ったものを選んでそのまま使えます。

### AI映像生成ツールで動画を作る

Luma Dream Machine（ルマ・ドリームマシン）などのAI動画生成ツールに、先ほどのプロンプトを入力して映像を生成します。

**良い例:**
- 実写と同じ色味・明るさになるよう「warm lighting, natural colors」などの指定を加える
- 5秒以内の短いクリップに絞る（長いと不自然さが目立ちやすい）

**悪い例:**
- 実写が昼間なのに、AI映像がネオンカラーの夜景
- 30秒以上の長いAI映像をそのまま使う

> **つまずきポイント:** AI映像は1回で完璧にはなりません。2〜3回生成して、一番自然なものを選ぶつもりでいましょう。

## 編集ソフトで自然に繋ぐ

実写とAI素材が揃ったら、動画編集アプリ（CapCut など、スマホで無料で使えるもの）を使って、2つの映像を繋ぎます。

![編集画面のスクリーンショット](/lesson-assets/atom.video-creator.blend-ai-and-real-footage/screen_capture.png)

### 3つのコツで違和感をなくす

**1. トランジション（＝映像の切り替え効果）を使う**

映像の切り替わりに「クロスフェード（＝前の映像が薄れると同時に次の映像が浮かび上がる効果）」を入れると、自然に繋がります。CapCut の場合、2つのクリップの境目をタップすると、トランジションを選べます。

**良い例:** クロスフェードを0.5〜1秒で設定する
**悪い例:** 効果なしのベタ切り（＝瞬間切り替え）にして、映像がプツッと変わる

**2. 色味を揃える**

実写が暖色（オレンジっぽい）なのにAI素材が寒色（青っぽい）だと、見ている人は違和感を覚えます。CapCut の「フィルター」や「調整」で、AI素材の色温度を実写に近づけましょう。

**3. BGMを通しで流す**

1曲のBGMを動画全体にかけると、実写とAI素材の切り替わりが音で繋がって自然になります。BGMが途切れると、人間の耳は「あ、切り替わった」と感じてしまいます。

### AIに編集のアドバイスをもらう

繋ぎ方に迷ったら、ChatGPTに聞いてみましょう。

**プロンプト例:**
```
実写のカフェ動画（10秒）とAI生成のイメージ映像（5秒）を
1本の動画に繋ぎたいです。
CapCutで使えるトランジションのおすすめと、
自然に見えるコツを教えてください。
```

## 確認する

動画が完成したら、次の5つのチェックポイントを確認しましょう。すべて「はい」なら合格です。

| # | チェック項目 | はい/いいえ |
|---|---|---|
| 1 | 実写とAI素材の明るさ・色味が似ているか？ | |
| 2 | 映像の切り替わりにトランジションが入っているか？ | |
| 3 | BGMが途切れずに全体を通して流れているか？ | |
| 4 | AI素材だけを見ても不自然な動き（手足が溶ける等）がないか？ | |
| 5 | 全体を通して見たとき、1本の動画として統一感があるか？ | |

> **つまずきポイント:** チェック4で不自然な部分が見つかった場合は、AI映像を別パターンで再生成するか、その部分だけトリミング（＝不要な部分を切り取る）しましょう。

## 成果物を作る

ここまでの作業をまとめた「ビフォーアフター比較メモ」を作りましょう。以下のテンプレートをコピーして、あなたの内容に書き換えてください。

```
# ビフォーアフター比較メモ

## 使った素材
- 実写: （例: カフェでコーヒーを注ぐ動画 10秒）
- AI生成: （例: コーヒー豆が光に包まれるイメージ映像 5秒）

## 工夫したポイント
- （例: クロスフェードを0.8秒に設定して切り替えを自然にした）
- （例: AI素材の色温度を暖色寄りに調整して実写に合わせた）

## 完成動画の長さ
- （例: 合計15秒）
```

このメモのスクリーンショット（＝画面を撮影した画像）を撮って保存すれば、このレッスンは完了です。

## よくあるつまずきと対処法

| つまずき | 対処法 |
|---|---|
| AI映像の画質が実写より低い | AI生成時に解像度を上げるか、実写側もスマホ画質に統一する |
| トランジションを入れても不自然 | クロスフェードの長さを0.5〜1.5秒の範囲で調整してみる |
| AI映像の動きが不自然 | 短い部分（2〜3秒）だけ切り出して使う |
| 色味がどうしても合わない | ChatGPTに「実写が暖色のとき、AI素材の色温度をどう調整すべき？」と聞く |
