# ベネフィット訴求を書く

## このレッスンでできるようになること

このレッスンを終えると、あなたは商品の「特徴」と「ベネフィット（＝お客様が得られるうれしい結果）」をはっきり区別し、AIツールを使ってベネフィット中心の魅力的な訴求文を15分で書けるようになります。

---

## はじめに：レストランのメニューで考えてみよう

レストランのメニューにこう書いてあったとします。

> 「低温調理器（＝一定の温度でお湯を保つ機械）を使用しています」

これは事実ですが、お腹は鳴りませんよね。

では、こう書いてあったらどうでしょう。

> 「お肉は中までふんわり、ナイフがすっと入るやわらかさ」

こちらの方が「食べたい！」と思いませんか？

この違いこそが、**「特徴」と「ベネフィット」の違い**です。

- **特徴（フィーチャー）**：商品が「何であるか」「何を持っているか」という事実
- **ベネフィット**：お客様が「どうなるか」「何を感じるか」というお客様にとっての価値

![特徴とベネフィットの関係図](/lesson-assets/atom.ai-marketer.benefit-copy-write/diagram.png)

上の図のように、特徴はあくまで「商品側」の情報で、ベネフィットは「お客様側」の体験です。これからAIの力を借りて、この変換をサクッと行う方法を学びます。

---

## 前提を確認する

このレッスンを進める前に、次のものを用意してください。

- **AIチャットツール**：ChatGPT、Claude、Gemini のいずれかにアクセスできること（無料プランでOK）
- **書きたい商品やサービス**を1つ決めておくこと
  - 自分の仕事のサービスでも、身近な商品（お気に入りのコーヒーメーカーなど）でも構いません
- **メモ帳アプリ**（パソコンのメモ帳、Googleドキュメントなど何でもOK）

プログラミングの知識は一切不要です。AIとの会話だけで進めます。

---

## 手順をすすめる

### ステップ1：商品の「特徴」を3つ書き出す

まず、あなたが訴求したい商品の特徴を **3つだけ** メモ帳に箇条書きしてください。たくさん出す必要はありません。「事実」をそのまま書くだけでOKです。

**良い例（具体的な事実が書かれている）**
- 重さ350gで片手で持てる
- バッテリーが24時間持つ
- IPX7の防水仕様で水洗いできる

**悪い例（事実が書かれていない）**
- すごい商品です → 何がすごいのかわからない
- 競合より優れています → 比較だけで中身がない

> 💡 **コツ**：「何グラム？」「何時間？」「何に対応？」のように数字や具体名を入れると、後のステップでAIが良い変換をしてくれます。

### ステップ2：AIに「特徴→ベネフィット変換」を頼む

AIチャットツール（ChatGPT、Claude など）を開き、下のプロンプト（＝AIへの指示文）をコピーして貼り付けてください。`＜＞` の中だけ、ステップ1で書いた自分の商品の情報に書き換えます。

```
あなたはベテランのコピーライターです。
次の商品特徴を、お客様の日常生活でのベネフィットに変換してください。

ルール：
・お客様を主語にする（「あなたは〜」「朝起きたとき〜」など）
・具体的な生活シーン（時間帯、場所、行動）を含める
・各ベネフィットは2文以内にする

商品名：＜あなたの商品名＞
商品特徴：
・＜特徴1＞
・＜特徴2＞
・＜特徴3＞
```

> **何をしているか**：AIに「コピーライター役」を与え、特徴をベネフィットに変換してもらっています。「ルール」を明示することで、漠然とした出力を防いでいます。

![AIチャットでの操作画面](/lesson-assets/atom.ai-marketer.benefit-copy-write/screen_capture.png)

### ステップ3：AIの出力を「3つのチェック」で確認する

AIが返してきた文章を読みながら、次の3つをチェックしてください。1つでも「×」があれば、ステップ4で修正を依頼します。

| チェック項目 | ○ 良い例 | × 悪い例 |
|---|---|---|
| **お客様が主語か？** | 「あなたの朝が楽になります」 | 「当社の製品は優れています」 |
| **具体的な場面があるか？** | 「満員電車でも音楽に集中できる」 | 「快適な体験を提供します」 |
| **感情が動くか？** | 「安心して子供に持たせられる」 | 「安全性が高い仕様です」 |

### ステップ4：AIに修正を依頼する（必要な場合）

チェックで「×」があった場合は、以下のようにAIに追加の指示を送ります。そのまま会話を続けてOKです。

**「お客様が主語になっていない」場合のプロンプト例**
```
2番目のベネフィットが「当社の〜」で始まっています。
「あなたが〜」で始まるように書き直してください。
```

**「場面が抽象的すぎる」場合のプロンプト例**
```
「快適」だけでは伝わりにくいです。
朝の通勤電車の中での具体的なシーンに書き換えてください。
```

**「全体的にピンと来ない」場合のプロンプト例**
```
もう少し感情に訴える表現にしてほしいです。
「たとえば朝起きたときに〜」のように、
読んだ人が自分の生活を思い浮かべるような書き方にしてください。
```

> 💡 **コツ**：「もっと良くして」のような漠然とした指示より、「○番目の△を□に変えて」と具体的に伝える方が、AIは的確に応えてくれます。

### ステップ5：自分の言葉で仕上げて保存する

AIの出力をそのまま使うのではなく、最後にあなたの言葉で自然に読めるように整えます。

**良い例（あなたの言葉で調整した文）**
> 朝の準備、もう慌てなくて大丈夫。前の夜にセットしておけば、起きた瞬間に温かいコーヒーの香りがお部屋に広がります。

**悪い例（AIの出力をそのままコピーした文）**
> 本製品はタイマー機能を搭載しており、指定した時間に自動的に稼働を開始することが可能です。

整えるときの簡単なテスト：**声に出して読んでみてください**。友だちに話しかけるように聞こえればOK、報告書のように聞こえたら書き直しのサインです。

完成した訴求文をメモ帳やGoogleドキュメントに保存すれば、成果物の完成です。

---

## 完成した文章をセルフチェックする

以下の4項目すべてに ✅ が入れば合格です。

- [ ] 特徴がベネフィットに変換されている（「お客様がどうなるか」が書かれている）
- [ ] お客様が主語、またはお客様の視点で書かれている
- [ ] 具体的な生活シーンや感情が含まれている
- [ ] 声に出して読んで、自然に聞こえる

---

## つまずき対策

| よくあるつまずき | こうすれば解決します |
|---|---|
| AIの出力が「特徴」のまま戻ってくる | 「お客様が日常でどう変わるかを、具体的な場面つきで教えてください」と追加で伝える |
| ベネフィットが抽象的すぎる（「快適です」など） | 「朝起きたときにどう変わるか」のように **時間帯・場所・行動** を指定して聞き直す |
| 自分の商品の特徴が思い浮かばない | 商品のウェブサイトやパッケージの「特徴」「スペック」欄を見て、数字や固有名詞を拾い出す |
| AIが長すぎる文章を返す | 「各ベネフィットを2文以内、合計100文字以内にしてください」と文字数を指定する |
| どこを直せばいいかわからない | ステップ3のチェック表に戻り、「○×」をつけて×の項目だけAIに修正依頼する |

---

## まとめ

このレッスンでは、商品の「特徴」をお客様の「ベネフィット」に変換する練習をしました。

ポイントを振り返ります。

1. **特徴 = 商品の事実、ベネフィット = お客様の体験**
2. **AIにはルール付きで具体的に指示する**と、良い変換が返ってくる
3. **最後は自分の言葉で仕上げる**ことで、あなたらしさが出る

AIは「壁打ち相手（＝アイデアを出し合う相手）」として優秀です。次のレッスンでは、今回作ったベネフィット訴求を実際のSNS投稿や広告文に組み込む方法を学びます。
