# 適正価格を設定して値上げ交渉を成功させる

あなたが個人で仕事を請け負うとき、「いくらで請けよう？」と迷った経験はありませんか？
このレッスンでは、AIツールを活用して自分に合った価格を決め、既存の取引先に値上げを提案する方法を15分で学びます。

旅に出るとき、宿や食事にいくら使うか予算を決めますよね。料金設定も同じです——自分の「生活費」と「市場の相場」という2つのコンパスを使って、進むべき方向を決めます。

## 自分の「最低ライン」を計算する

レストランでメニュー価格を決めるとき、材料費・人件費・利益の3つを足しますよね。フリーランスの料金も同じ考え方です。

1. **1時間あたりの生活費**を計算する
   - 月の生活費（家賃・食費・保険など）を合計する
   - 1ヶ月に稼働できる時間（例：月80時間）で割る
   - これが「これ以下では生活できない」最低ラインです
2. **希望月収**に必要な時間単価を出す
   - 最低ラインに利益の上乗せ分（＋20〜30％程度）を加える
3. AIに市場相場を聞いてすり合わせる（次のステップで詳しく解説します）

### 良い例・悪い例を比べる

**良い例**: 「リサーチ記事1本（約2000字）を15,000円で提案。市場相場12,000〜18,000円の範囲内」

**悪い例**: 「なんとなく1,000円でOKしました。初めてだから安くていいかなと思って」

安すぎると「品質が低いのでは？」と不信感を持たれることがあります。自信を持って説明できる価格を探しましょう。

## AIで市場相場を調べて単価を決める

ChatGPTやClaudeなどのAIツールを使って、あなたの単価を具体的に計算します。

次のプロンプト（＝AIへの指示文）をコピーして、`[ ]` の中を自分の数値に書き換えて送信してください。

```
私はフリーランスで[仕事の種類（例：ブログ記事の執筆）]をしています。
月に稼働できる時間は[〇〇]時間、目標月収は[〇〇]万円です。
以下の3点を教えてください。
1. 日本のフリーランス市場での相場（時間単価・案件単価）
2. 私の目標月収を達成するための最低時間単価
3. 相場と比較して、私の目標が現実的かどうかのアドバイス
```

**AIの回答で確認すべきポイント**:
- 提示された相場の幅（下限〜上限）をメモする
- 自分の目標単価がその範囲内に入っているか確認する
- 範囲外なら、AIに「どうすれば相場上限に近づけるか」を追加で質問する

この回答があなたの料金の「ベースライン（＝基本となる基準）」になります。

## 料金表をAIと一緒に作成する

計算結果をもとに、取引先に見せられる料金表（プライスリスト）を作ります。以下のプロンプトをAIに送ってください。

```
以下の情報をもとに、取引先に提示できるシンプルな料金表を作ってください。
Markdown形式の表にしてください。

- 仕事内容: [例：ブログ記事執筆]
- 時間単価: [先ほど決めた金額]円
- 主な納品物: [例：2000字の原稿データ]
- 通常の納期: [例：5営業日]

表の列は「サービス名」「単価」「納品物」「納期の目安」の4つでお願いします。
```

AIが作った料金表を確認し、次の4項目がすべて入っているかチェックしましょう。

- **サービス名**: 何をするか（例：ブログ記事執筆）
- **単価**: 時間制 or 成果物ごとの価格
- **納品物**: 何を届けるか（例：2000字の原稿データ）
- **納期の目安**: いつまでに届けるか

### 良い例・悪い例を比べる

**良い例**: 「ブログ記事執筆：1本15,000円（2000字程度・SEO対応）5営業日以内」

**悪い例**: 「文章書きます：ご相談ください」（価格が不明確で信頼感が薄い）

## 値上げ交渉のメールをAIで下書きする

既存の取引先に値上げをお願いするのは緊張しますよね。でも、料金改定は仕事のプロとして当たり前のことです。引っ越し業者が繁忙期に料金を変えるのと同じで、条件が変われば価格も変わってしかるべきなのです。

値上げ交渉で大切な3つのポイントを押さえましょう。

1. **理由を伝える**（スキル向上・納品物の品質アップ・市場相場との比較など）
2. **先行投資として今の価格で始めたことに触れる**
3. **具体的な新価格と適用開始月を明示する**

AIに下書きを作ってもらいましょう。以下のプロンプトをコピーして使います。

```
以下の条件で、取引先への値上げ交渉メールの下書きを作ってください。

- 現在の価格: [〇〇]円
- 希望する新価格: [〇〇]円
- 仕事の種類: [〇〇]
- 値上げの理由: [スキル向上・品質改善・市場相場との乖離 など]
- 適用開始: [来月 or 〇月から]

条件：
- 丁寧でビジネスマナーに沿ったトーン
- 値上げの理由を2〜3行で説明する
- 相手との良好な関係を維持する表現にする
- メール件名も提案する
```

AIが出力した文面を読み返し、**自分の言葉に合うように少し書き換えて**完成させます。そのまま送るのではなく、「これは自分が書いた文章だ」と思える状態にすることが大切です。

### 良い例・悪い例を比べる

**良い例**: 「これまでスキルを磨いてまいりました。品質向上に伴い、来月から料金を15,000円に改定させていただきたく存じます。ご検討いただけますと幸いです」

**悪い例**: 「これからは15,000円でお願いします。よろしく」（唐突で理由がない）

## 成果物を確認する

以下のチェックリストですべてを確認しましょう。1つでも欠けていたら、該当のステップに戻って修正します。

- [ ] AIで市場相場を調べ、自分の目標単価と比較した
- [ ] 料金表に**サービス名・単価・納品物・納期**の4項目がすべて書かれている
- [ ] 値上げメールに**理由・新価格・開始月**が含まれている
- [ ] 価格がAIで調べた市場相場の範囲内に収まっている
- [ ] 値上げメールを自分の言葉で書き直した

すべてチェックが付いたら、料金表と値上げメールの画面のスクリーンショット（＝画面の写真）を撮って、完了の証拠として保存してください。

## つまずきポイントと対策

| つまずき | 対策 |
|---|---|
| 適正価格が高すぎる気がする | AIに「経験1年未満のフリーランスとして現実的な開始価格」を聞いてみましょう。最初は相場の下限から始めて、実績ができたら上げる戦略もあります |
| 値上げメールを送るのが怖い | AIに3パターンの文面を作ってもらい、一番自然なものを選びましょう。また「値上げメールを送った後、相手が断った場合の返信例も作って」と追加で聞くと安心です |
| 既存の取引先しかいない | 値上げは新規の取引先から適用し始め、既存の取引先には半年後に案内するのが安全です。AIに「段階的な値上げスケジュール」を相談してみましょう |
| 自分の稼働時間がわからない | 1日のうち「仕事に使えそうな時間」をざっくり書き出し、その半分を稼働時間として計算しましょう。AIに「副業で月〇時間使える場合の収入シミュレーション」を聞くと具体的になります |
| AIの回答が的外れだった | プロンプトに具体的な数字や条件を追加してみてください。「日本のクラウドソーシングサイトでの相場」のように範囲を絞ると精度が上がります |
