# 顧客・商品・競合を整理する

パーティーを開くとします。

- **誰を呼ぶか**（＝顧客）
- **何を出すか**（＝商品）
- **同じ日に行われる他のパーティーは何か**（＝競合）

この 3 つを知らないと、せっかくのパーティーも人は集まりませんよね。

マーケティングでもまったく同じです。あなたの商品が「誰に」「何を」「誰と比べられて」いるのか——この 3 つを整理することが、すべての出発点になります。

このレッスンでは、AI チャットツール（ChatGPT や Claude など）を使って、**15 分で市場整理シート**（＝顧客・商品・競合を 1 枚にまとめた文書）を作ります。完成すると「自分の商品の立ち位置がひと目でわかる地図」が手に入ります。

![市場整理の全体像](/lesson-assets/atom.ai-marketer.market-context-organize/diagram.png)

## 前提を確認する

このレッスンを始める前に、次の 2 つだけ準備してください。

1. **AI チャットツールを開ける状態にする** — ChatGPT（https://chat.openai.com ）や Claude（https://claude.ai ）など、どれでも OK です。ブラウザでアクセスして、文字を入力できる画面が出ていれば準備完了です
2. **売りたい（または売っている）商品・サービスを 1 つ決めておく** — まだ決まっていない場合は「自分が好きな商品」や「最近買ったもの」で練習しましょう

プログラミングの経験は一切不要です。AI に日本語で質問を書くだけで進められます。

## ステップ 1: 顧客の像を描く

まず「誰に売るのか」を整理します。AI チャットに次のプロンプト（＝AI への指示文）をコピーして貼り付けましょう。`〇〇` はあなたの商品名に書き換えてください。

```
あなたはマーケティングの専門家です。
「〇〇」という商品・サービスについて、想定される顧客の特徴を次の観点で整理してください。

1. 年代・性別・職業
2. その人が抱える悩み（3つ）
3. その人が商品を知るきっかけ（2つ）

表形式で出力してください。
```

> **ポイント**: 「マーケティングの専門家として」という一文を入れると、AI がその分野の知識を使って回答してくれます。これを「役割指定」と呼びます。

AI の回答が出たら、メモ帳や Google ドキュメントなどにコピーしておきます。

**良い例**: 「30 代・会社員・子育て中の母親。毎日の夕食準備に悩んでいる。Instagram 広告で知る」

**悪い例**: 「女性」——これだけでは、どんな悩みがあるどんな人か分かりません。もっと具体的に絞り込みましょう。

### もし顧客像がぼんやりしたら

追加でこう聞いてみてください。

```
その中で最も典型的な 1 人のユーザー像（ペルソナ）を作ってください。
名前・年齢・職業・1日のスケジュール・悩みを含めてください。
```

こうすると「30 代・山田花子さん・共働きの会社員・朝 7 時に出勤して…」のように、リアルな 1 人の姿が浮かびます。

## ステップ 2: 商品の強みを書き出す

次に「何を売るのか」を整理します。同じ AI チャット画面で、続けて次のように入力しましょう。

```
同じ商品「〇〇」について、他社にはない強みを 3 つ挙げてください。
それぞれ 30 字以内で、数値や具体例を含めて説明してください。
```

**良い例**: 「冷凍なのにレンチン 3 分で完成——忙しい人でも手軽」（具体的で数字がある）

**悪い例**: 「便利でいい感じ」（抽象的すぎて、何がどう良いのか伝わりません）

> **なぜ「30 字以内で、数値を含めて」と指示するのか？** AI は指示がゆるいと抽象的に書きがちです。文字数や「数値を入れて」という条件を付けると、具体的な回答が返ってきます。これは AI を上手に使うコツのひとつです。

## ステップ 3: 競合を調べる

最後に「誰と比べられるか」を整理します。

```
「〇〇」と同じような商品を提供している競合を 3 社挙げてください。
各社について次の 3 点を 1 文ずつ書いてください。
- 強み
- 弱み
- 「〇〇」との違い
ただし確実に存在する企業名のみを扱ってください。不確かな場合は「代表的なカテゴリの例」として扱ってください。
```

**良い例**: 「A 社——品揃え豊富だが配達に 3 日かかる。B 社——当日配達可能だが送料が高い。〇〇との違い: 当社は送料無料で翌日届く」

**悪い例**: 「競合は特にいません」——たいていの場合、何らかの代替手段（スーパーで買う、自分で作る、など）があります。AI にそう言われたら「直接の競合がいない場合、顧客が代わりに使っている手段は？」と聞き返しましょう。

> **注意**: AI はときどき実在しない企業名を出すことがあります。「確実に実在する企業のみ」という一文を入れておくことで架空企業を減らせます。出てきた企業名が本当に存在するか、Google で軽く検索して確認するとさらに安心です。

## ステップ 4: まとめて 1 枚にする

ここまでの 3 つの結果（顧客・商品・競合）を 1 つのドキュメントにまとめましょう。AI に頼むと一瞬です。

```
これまでの回答をまとめて、「〇〇」の市場整理シートを 1 枚の Markdown 文書にしてください。
見出しは「顧客」「商品の強み」「競合比較」の 3 つに分けてください。
最後に「まとめ: 〇〇の市場でのポジション」を 2-3 文で書いてください。
```

> **Markdown（マークダウン）とは？** 見出しや箇条書きをシンプルな記号（`#` や `-`）で書ける書き方です。メモ帳や Notion にそのまま貼り付けると、きれいに表示されます。

出力された文書をコピーして、あなたの PC に保存します。ファイル名は `market-context-〇〇.md` がおすすめです（`〇〇` は商品名に置き換えてください）。

## 成果物を確認する

次のチェックリストをひとつずつ確認してください。すべて「はい」になれば完成です。

- [ ] 顧客の特徴が具体的に書かれている（「30 代・会社員・子育て中」のように、3 つ以上の特徴がある）
- [ ] 商品の強みが 3 つ以上あり、それぞれ数値や具体例を含んでいる
- [ ] 競合が 1 社以上挙がっていて、強み・弱みが書かれている
- [ ] 3 つのセクション（顧客・商品・競合）が 1 つのファイルにまとまっている
- [ ] ファイル名に商品名が入っていて、後で見返せる

完成したら、画面のスクリーンショット（＝画面の写真）を撮っておきましょう。これがこのレッスンの成果物（＝「やりました」の証拠）です。

> **スクリーンショットの撮り方**: Windows は `Win + Shift + S`、Mac は `Cmd + Shift + 4` を押すと、画面の一部を写真として保存できます。

## つまずきポイントに対処する

| よくあるつまずき | こうすれば解決できます |
|---|---|
| AI が架空の企業名を挙げる | プロンプトに「確実に実在する企業のみ」を追加し、Google で存在確認する |
| 顧客像がぼんやりする | 「具体的な 1 人のペルソナを作って」と追加質問する |
| 強みが抽象的になる | 「30 字以内で、数値や具体例を含めて」と条件を付ける |
| 商品が決まっていない | 「自分が最近買って良かったもの」で練習する |
| AI の回答が長すぎて整理できない | 「箇条書き 3 つ以内で」と文字数や個数を制限する |
| 同じ AI チャットで続きを打つのを忘れた | 新しいチャットを開いた場合は、最初に「〇〇という商品のマーケティング分析をしています」と前提を伝え直す |
