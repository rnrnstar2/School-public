# 行動変容の追跡方法を設計する

「研修を受けたのに、何も変わらなかった」——そんな声を聞いたことはありませんか？
知識を詰め込むだけの研修と、本当に人の行動を変える研修の違いは、**「あとでどう変わったかを追跡しているか」**にあります。

たとえば料理教室を想像してみてください。「塩を少なめにしましょう」と教えたあと、生徒さんが自宅で実際にどう料理しているかを確認しなければ、教えた効果があったかわかりませんよね。行動変容の追跡とは、これと同じことを研修の世界で行うことです。

この Atom では、あなたが設計した研修について「受講者の行動がどう変わったか」を測る仕組みを、AI ツールの力を借りて **15分** で作ります。

---

## 前提を確認する

この Atom に取り組む前に、以下の状態であることを確認してください：

- あなたが設計した研修のゴール（受講者にどうなってほしいか）が決まっている
- ChatGPT・Claude・Cursor のいずれかをブラウザまたはアプリで開ける状態にある

プログラミングの知識は一切不要です。AI に日本語で話しかけるだけで進められます。

---

## ステップ1：追跡したい行動を「見える化」する

まず、追跡したい行動を具体的に書き出します。「意識が変わった」のような曖昧な表現ではなく、**誰が見てもわかる動作**にするのがポイントです。

判断のコツは **「カメラで撮れるかどうか」** で考えることです。カメラに映る行動なら、十分に具体的です。

**良い例：**
- 「会議で発言する回数が週1回以上になる」
- 「報告書を提出する際、必ずチェックリストを使う」
- 「顧客からの問い合わせに24時間以内に返信する」

**悪い例：**
- 「マインドセットが変わる」→ どう変わったか目に見えない
- 「理解度が上がる」→ 行動ではなく状態の話になっている

### AIに聞いてみる

ChatGPT や Claude を開いて、次のプロンプト（＝AIへの指示文）をコピー＆ペーストしてください。`【 】` の中だけ、あなたの研修内容に書き換えます。

```
私は【〇〇（例：新入社員向けビジネスマナー）】という研修を設計しています。
ゴールは「受講者が【〇〇（例：名刺交換を正しい手順で行える）】こと」です。

受講者の行動変容を追跡するために、
以下の条件を満たす測定可能な行動指標を5つ提案してください。

条件：
- 第三者が観察できる具体的な動作であること
- 「〜する」「〜を行う」のように動詞で終わること
- 定量的に数えられる、または「やった／やらなかった」で判定できること
```

AI の回答が出たら、**その中から特に重要な3〜5個を選んで**メモしておきます。全部を使う必要はありません。

> **ヒント:** AI の提案がぴんとこない場合は、「もっと現場で観察しやすい指標にしてください」と追加で伝えると、より実用的な案が出てきます。

---

## ステップ2：測定タイミングを決める

行動が変わったかどうかは、一度だけの測定ではわかりません。料理の上達を確認するのに「一度だけ味見する」のではなく、「1週間後」「1ヶ月後」と何度か味見するのと同じです。

一般的なタイミングは次の3つです：

| タイミング | 確認する内容 |
|:---|:---|
| **直後**（研修終了後すぐ） | やろうとする意志があるか |
| **1〜2週間後** | 実際にやり始めたか |
| **1ヶ月後** | 習慣として定着したか |

**良い例：** 「直後」と「1ヶ月後」の2回に絞る → シンプルで回収しやすい

**悪い例：** 「3日後、1週間後、2週間後、1ヶ月後、3ヶ月後」→ 対象者の負担が大きく回答が集まらない

### AIに相談する

先ほどの行動指標を AI に伝えて、測定タイミングの提案をもらいましょう。

```
以下の行動指標それぞれについて、
「直後」「1〜2週間後」「1ヶ月後」のうち、
どのタイミングで測るのが最も効果的か理由つきで教えてください。

行動指標：
1. 【ステップ1で選んだ指標をここに貼り付け】
2. ...
```

AI の回答を参考にしつつ、**最終判断はあなた自身**で行います。研修の現場をいちばん知っているのはあなたです。

---

## ステップ3：追跡シートをAIに作らせる

行動指標と測定タイミングが決まったら、実際に追跡するためのシート（表）を作ります。

AI に次のプロンプトを送ってください：

```
以下の行動指標と測定タイミングを使って、
行動変容を追跡するシートのひな形を作ってください。

行動指標：
【ここにステップ1で選んだ指標を貼り付け】

測定タイミング：
【ここにステップ2で決めたタイミングを貼り付け】

出力形式の条件：
- マークダウン形式の表にする
- 行に「行動指標」、列に「測定タイミング」を配置する
- 各セルには「達成 / 未達成 / 未回答」の選択肢を入れる
- 表の下に「記入上の注意点」を3つ箇条書きで添える
```

![AIで追跡シートを生成する画面](/lesson-assets/atom.training-designer.behavior-change-tracking/screen_capture.png)

出力された表をコピーして、Google スプレッドシートや Excel に貼り付ければ、すぐに使える追跡シートの完成です。

> **ヒント:** 表がうまくスプレッドシートに貼れない場合は、AI に「CSV形式（＝表計算ソフトで読み込める形式）で出力してください」と頼むとスムーズです。

## 成果物を確認する

この Atom の成果物は、あなたが作った **行動変容追跡プラン** です。以下の4つすべてにチェックがつけば完成です：

- [ ] 測定可能な行動指標が **3〜5個** ある（「カメラで撮れる」レベルの具体性）
- [ ] 各指標に測定タイミング（いつ確認するか）が紐づいている
- [ ] 追跡シートのひな形が表として完成している
- [ ] 追跡シートを自分のスプレッドシートまたはメモにコピーできた

すべてチェックがついたら、**完成した追跡シートのスクリーンショットを撮って保存**します。これがこの Atom の完了証拠（＝できたことの記録）になります。

## つまずきポイントを知っておく

| よくあるつまずき | こうすれば解決できます |
|:---|:---|
| 行動指標が抽象的になってしまう | 「他の人が見てもわかるか？」と自問する。わからなければ AI に「もっと具体的にして」と依頼 |
| 測定タイミングが多すぎて迷う | 最初は **「直後」と「1ヶ月後」の2回だけ** に絞る |
| AI の出力がイメージと違う | 研修の対象者・業種・規模を追加で伝えると精度が上がる（例：「IT企業の新卒30名向け」） |
| 指標が多すぎて追えない気がする | 重要な **3つ** に絞ってスタート。慣れたら増やせばOK |
| AI が英語で返答してしまう | プロンプトの先頭に「日本語で回答してください。」を追加する |
