# AIにFAQ初稿を生成させる

「お客様からよく聞かれる質問」をまとめたFAQ（よくある質問集）は、カスタマーサポートの強い味方です。でも、ゼロから書き始めると意外と時間がかかりますよね。

このレッスンでは、AIチャットツール（ChatGPTやClaudeなど）に「たたき台」を作ってもらい、それをあなたが手直しして完成させる方法を学びます。料理にたとえると、AIが下ごしらえ済みの食材を用意してくれて、あなたが味付けと盛り付けをする——そんなイメージです。

## このレッスンのゴールを確認する

- **やること**: AIにFAQの初稿（＝最初の下書き）を作ってもらう
- **完成物**: 5〜10問のQ&Aが並んだMarkdown形式のFAQ文書
- **所要時間**: 約15分

## 準備するものを確認する

- AIチャットツール（以下のどれかひとつでOK）
  - ChatGPT（無料版で十分です）
  - Claude（claude.ai）
  - その他お好みのAIチャット
- FAQを作りたいサービスや製品の情報（ざっくりでOK）

## ステップ1: AIに渡す「お題」を整理する

AIに良いFAQを作ってもらうには、まず「何についてのFAQか」を伝える必要があります。

以下の3つを箇条書きでメモしてください（完璧でなくて大丈夫です）。

1. **サービス／製品の名前と概要**: 何を提供しているか、ひと言で
2. **想定する読者**: 誰がこのFAQを読むか（例: 初めてのお客様、既存ユーザーなど）
3. **よく聞かれる話題**: 思いつく範囲で3〜5個（例: 料金、使い方、解約方法など）

**良い例:**
> - サービス名: オンライン英会話「ABC English」
> - 読者: 無料体験を検討している方
> - よく聞かれること: 料金プラン、レッスンの予約方法、キャンセル規定、講師の選び方、必要な機材

**悪い例:**
> - うちのサービスについて
> - お客様向け
> - いろいろ

具体的に書くほど、AIの出力の品質が上がります。

## ステップ2: AIにプロンプトを送る

AIチャットを開いて、以下のようなプロンプト（＝AIへの指示文）を送ります。コピーして、カギ括弧の中をあなたの情報に書き換えてください。

```
あなたはカスタマーサポートの専門家です。
以下のサービスについて、FAQ（よくある質問）の初稿を作ってください。

【サービス概要】
「ここにサービス名と概要を書く」

【想定読者】
「ここに読者像を書く」

【含めてほしい話題】
- 「話題1」
- 「話題2」
- 「話題3」

【出力フォーマット】
- Q&A形式で5〜10問
- 各回答は3行以内で簡潔に
- Markdown形式（見出しに ## を使う）
```

![プロンプト送信の流れ](/lesson-assets/atom.cs-automator.faq-draft-generation/diagram.png)

### プロンプトのコツ

| やるといいこと | 理由 |
|---|---|
| AIに「役割」を与える（例: 専門家） | 回答の質が安定します |
| 出力形式を指定する | 後から編集しやすくなります |
| 話題を箇条書きで渡す | 漏れを防げます |

## ステップ3: AIの出力を受け取って確認する

AIが生成したFAQをそのまま使うのではなく、以下の観点でチェックしましょう。

- **事実は正しいか？**: 料金や手順など、実際のサービス内容と合っているか
- **言葉づかいは適切か？**: お客様に読んでもらうのにふさわしいトーンか
- **抜け漏れはないか？**: 重要なのに含まれていない質問はないか
- **余計な質問がないか？**: あまり聞かれない質問が混ざっていないか

![AIの出力を確認する画面](/lesson-assets/atom.cs-automator.faq-draft-generation/screen_capture.png)

## ステップ4: 手直しして完成させる

確認で見つかった問題を修正します。全部書き直す必要はありません。AIの出力をベースに、以下を行ってください。

1. **間違っている回答を修正する**: 正しい情報に書き換える
2. **足りない質問を追加する**: 実際によく聞かれるのに抜けているQ&Aを足す
3. **不要な質問を削除する**: 的外れなQ&Aを削る
4. **言い回しを調整する**: あなたのサービスのトーンに合わせる

### 修正が多いときのコツ

もしAIの初稿が的外れだった場合は、プロンプトを修正して再度AIに送り直すのが効率的です。最初のプロンプトに「前回の出力では〇〇が不足していたので、△△を重点的に」と追加するだけでOKです。

## 完成を確認する

以下をすべて満たしていれば、このレッスンは完了です。

- [ ] 5〜10問のQ&Aが含まれたFAQ文書ができている
- [ ] 各回答が事実と一致している（あなた自身で確認済み）
- [ ] AIに送ったプロンプトのスクリーンショットを保存した

おめでとうございます！ AIに下書きを任せて、自分は「判断と仕上げ」に集中する——この流れが身につくと、FAQ以外の文書（マニュアル、メールテンプレートなど）にも応用できます。

## つまずいたときは

| 困りごと | 対処法 |
|---|---|
| AIの回答が短すぎる／長すぎる | プロンプトに「各回答は〇行程度で」と文量を指定する |
| 的外れな質問ばかり出てくる | 「含めてほしい話題」をもっと具体的に書く |
| 専門用語が多すぎる | プロンプトに「専門用語を使わず、中学生でもわかる言葉で」と追記する |
| Markdown形式にならない | 「出力フォーマット」の指定を再度伝える。具体例を1問分見せるとより確実 |
