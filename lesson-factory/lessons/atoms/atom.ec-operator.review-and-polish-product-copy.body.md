# 説明文を校正して整える

たとえば、あなたがレストランを経営しているとします。おいしい料理を作っても、メニューの説明が「肉と野菜の料理」だけだったら、お客さんはなかなか注文してくれませんよね。「北海道産黒毛和牛を、季節の有機野菜と特製赤ワインソースで仕上げました」と書いてあれば、思わず頼みたくなります。

ネットショップの商品説明文も同じです。商品の魅力が伝わる言葉を選ぶことで、お客さんの「買いたい！」という気持ちを引き出せます。

このレッスンでは、AIツール（＝人工知能を使った便利なWebサービス。例：ChatGPT、Claude）を活用して、商品の説明文をぐっと魅力的にする方法を学びます。所要時間はおよそ15分です。

![校正フロー図](/lesson-assets/atom.ec-operator.review-and-polish-product-copy/diagram.png)

## 前提を確認する

このレッスンを始める前に、次の準備が整っていることを確認してください。

- ChatGPTやClaudeなどのAIチャットツール（＝AIと対話できるWebサービス）にブラウザからアクセスできる
- 校正したい商品の説明文を手元に用意している（ネットショップの管理画面からコピーしてください）

まだAIツールを使ったことがない方は、ChatGPT（chatgpt.com）またはClaude（claude.ai）に無料登録しておきましょう。

## 元の説明文を用意する

まずは、あなたのネットショップにある商品の説明文をそのままコピーします。どんな文章でも構いません。

**良い例：**

> 「このTシャツは綿100％で作られています。肌触りがよく、夏でも涼しく着られます。サイズはS/M/Lの3種類です。」

→ 具体的な素材・季節・サイズが書かれています。校正の出発点として申し分ないです。

**悪い例：**

> 「いいTシャツです。おすすめです。」

→ 情報が少なすぎて、お客さんは判断できません。でも大丈夫、これから良くしていきます！

## AIに校正を頼む

AIツールに、次のような指示文（プロンプト＝AIへのお願い文）を送ります。以下の文章をコピーして、「ここにあなたの商品説明文を貼る」の部分だけ自分の説明文に書き換えて送ってください。

```
次のネットショップの商品説明文を、お客さんが買いやすくなるように校正してください。
条件は以下の通りです。
- 敬語で丁寧な口調にする
- 商品の魅力が伝わる具体的な表現を使う
- 読みやすく段落を分ける
- 200〜300文字程度にまとめる
- 事実にない特徴は追加しない

商品説明文：
ここにあなたの商品説明文を貼る
```

ここで何をしているかというと、AIに「どう直してほしいか」の条件をまとめて伝えているんです。レストランで「味は薄めで、アレルギーがあるから卵は抜いて」と注文の希望を伝えるのと同じです。条件を具体的に書くほど、あなたの望む結果が返ってきます。

![AIに校正を依頼した画面](/lesson-assets/atom.ec-operator.review-and-polish-product-copy/screen_capture.png)

## AIの結果を確認する

AIから返ってきた文章を読んで、次の3つのポイントをチェックします。これが一番大事なステップです。

1. **事実は正しいか** — AIが勝手に商品にない機能や特徴を追加していないか確認します。たとえば「防水機能付き」と書いてあるのに実際は防水でなければ、クレームの原因になります
2. **口調は合っているか** — あなたのショップの雰囲気に合う言葉遣いかチェックします。高級ブランドなのにカジュアルすぎたり、カジュアルなショップなのに堅すぎたりしないか
3. **読みやすいか** — 文が長すぎず、段落が適切に分かれているかを見ます。スマートフォンで読むお客さんが多いので、1段落は2〜3文程度が目安です

## 必要に応じて追加の指示を出す

AIの結果がイマイチだったら、続けて追加の指示を出しましょう。何度でもやり直せるので安心してください。

**良い例：**

> 「もう少し親しみやすい口調にしてください。また、送料が無料であることも追記してください。」

→ 「どう変えたいか」と「何を追加したいか」が具体的に伝わるので、AIも的確に直してくれます。

**悪い例：**

> 「もっといい感じにして」

→ 「いい感じ」が何を意味するか分からないので、AIも迷ってしまいます。思った結果が出なかったときは「何が違うのか」を言葉にしてみましょう。

### よく使う追加指示のテンプレート

状況に応じて以下をコピーして送ると便利です。

- 「文章をもう少し短くして、150文字以内にまとめてください」
- 「ターゲットは30代女性です。その層に刺さる表現に変えてください」
- 「箇条書きを使って、商品のポイントを3つにまとめてください」
- 「季節感を出して、夏向けの表現に変えてください」

## 最終版を決めてショップに反映する

AIとのやり取りで納得できる文章ができたら、以下の手順で反映しましょう。

1. 完成した文章をコピーする
2. ネットショップの管理画面を開いて、商品ページの編集画面に移動する
3. 元の説明文を新しい文章に貼り替える
4. **プレビュー画面で確認する** — 実際のお客さんが見る画面で、文字化けや改行崩れがないかチェック
5. 保存（公開）する

元の説明文と新しい文章を並べて見比べて、「こっちのほうが買いたくなるな」と感じられたら成功です。

## つまずきポイントを知っておく

- **AIが事実と違うことを書くことがある** — 商品にない機能や価格を勝手に追加することがあります。AIの文章は「たたき台」と考えて、必ず事実チェックをしてください。これを「ハルシネーション（＝AIがもっともらしいウソをつくこと）」と呼びます
- **指示があいまいだと望む結果にならない** — 「丁寧に」「短く」だけでなく、「敬語で」「150文字以内で」のようにできるだけ具体的な条件を伝えるのがコツです
- **一度で完璧にしようとしない** — 何回かAIとやり取りしながら少しずつ良くしていくのがおすすめです。プロのライターも何度も書き直して仕上げます
- **元の文章を消さずに残しておく** — 万が一新しい文章がしっくりこなかった場合に戻せるよう、元の文章はメモ帳などにコピーしておきましょう

## 完了を確認する

以下の3つがすべてできていれば、このレッスンは完了です。

- [ ] AIに校正を依頼して、改善された商品説明文を受け取った
- [ ] 事実チェック（嘘の機能や価格が入っていないか）を行った
- [ ] ネットショップの管理画面に新しい説明文を反映した（またはすぐ反映できる状態にした）
