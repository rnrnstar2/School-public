# AIでアイコン・アバターを制作する

SNSのプロフィール画像やブログのロゴなど、「ちょっとしたアイコンが欲しいけど、絵は描けない…」と思ったことはありませんか？ 実は今、AI画像生成ツールを使えば、絵を描くスキルがなくても、自分だけのオリジナルアイコンやアバターを数分で作れます。

このレッスンでは、AI画像生成ツールに「こういう画像がほしい」と文章で伝えるだけで、プロフィールアイコンやキャラクターアバターを制作する方法を学びます。料理にたとえると、レシピ（＝プロンプト）を書くだけでシェフ（＝AI）が自動で料理を仕上げてくれるようなイメージです。

## このレッスンで得られること

- AIに「どんなアイコンがほしいか」を伝える文章（プロンプト）の書き方がわかる
- 実際にアイコン・アバターを1点生成して、ダウンロードできる
- 生成した画像をSNSやプロフィールに設定できる状態にする

## 前提として必要なこと

- AI画像生成の基本操作ができること（`atom.ai-freelancer.image-gen-basics` を完了済み）
- ブラウザでAI画像生成ツール（ChatGPT の DALL·E、Canva AI、Leonardo AI など）にアクセスできること

## 全体の流れを確認する

![制作フロー](/lesson-assets/atom.ai-freelancer.icon-avatar-generation/diagram.png)

制作は大きく4ステップで進みます。

1. **どんなアイコンにしたいか決める** — 用途・雰囲気・色味を整理する
2. **AIへの指示文（プロンプト）を書く** — 具体的な言葉で伝える
3. **生成して調整する** — 気に入るまで微調整する
4. **保存して使う** — 正方形にトリミングして完成

## ステップ1: どんなアイコンにしたいか決める

いきなりAIに指示を出す前に、まず「どんな見た目にしたいか」を3つのポイントで整理しましょう。ノートやメモアプリに書き出すのがおすすめです。

| 項目 | 考えること | 例 |
|------|-----------|----|
| 用途 | どこで使う？ | SNSプロフィール、ブログアイコン、名刺 |
| 雰囲気 | どんな印象にしたい？ | やさしい、プロフェッショナル、かわいい |
| モチーフ | 何を描く？ | 猫キャラクター、抽象的な幾何学模様、自分の似顔絵風 |

**良い例**: 「フリーランスの名刺用に、やわらかい水彩風の猫キャラクターを作りたい」 — 用途・雰囲気・モチーフが揃っている

**悪い例**: 「なんかいい感じのアイコン」 — AIに伝える情報が少なすぎて、期待と違う画像が出やすい

## ステップ2: AIへの指示文（プロンプト）を書く

AI画像生成ツールでは、「こういう画像を作って」という指示文のことを**プロンプト**と呼びます。良いプロンプトを書くコツは、**具体的に、短く、要素を分けて書く**ことです。

### プロンプトの基本テンプレート

以下のテンプレートに当てはめて書いてみましょう。

```
[モチーフ]、[スタイル]、[色味]、[背景]、[用途に合ったサイズ指定]
```

### 具体的なプロンプト例

**例1: かわいい猫アイコン**
```
かわいい猫のキャラクター、フラットイラスト風、パステルカラー、白背景、正方形、SNSアイコン向け、シンプル
```

**例2: プロフェッショナルな抽象アバター**
```
抽象的な人物シルエット、ミニマルデザイン、青とグレーの配色、グラデーション背景、ビジネス用プロフィール画像
```

**例3: 手描き風の似顔絵アバター**
```
20代女性の似顔絵、やわらかい水彩イラスト風、茶色い髪、笑顔、ベージュ背景、丸型アイコン向け
```

### やりがちな失敗と対処法

| 失敗パターン | なぜダメ？ | 改善方法 |
|-------------|-----------|----------|
| 「アイコンを作って」だけ | AIが何を描くか判断できない | モチーフ・スタイル・色を追加する |
| 要素を詰め込みすぎる | 情報が多すぎて破綻する | 主要な要素を5〜7個に絞る |
| 英語で書かないといけないと思い込む | 日本語対応ツールなら日本語でOK | 使うツールの対応言語を確認する |

## ステップ3: 生成して調整する

![生成画面](/lesson-assets/atom.ai-freelancer.icon-avatar-generation/screen_capture.png)

プロンプトを入力して画像を生成したら、以下のポイントをチェックしてください。

### チェックリスト

- [ ] モチーフは意図したものになっているか？
- [ ] 色味やスタイルはイメージ通りか？
- [ ] アイコンとして使えるシンプルさがあるか？（細かすぎると縮小時に潰れる）
- [ ] 不自然な部分（指が多い、文字が崩れているなど）がないか？

### 気に入らないときの調整テクニック

1. **同じプロンプトで再生成する** — AIは毎回違う結果を出すので、数回試すだけで好みの画像が出ることがあります
2. **プロンプトの一部を変える** — 「パステルカラー」を「ビビッドカラー」に変えるなど、一度に1要素だけ変更するのがコツです
3. **「〜は含めないで」と書き足す** — 不要な要素が出たら、除外指示を追加します（例:「テキストは含めないで」）

## ステップ4: 保存して使えるようにする

生成した画像をダウンロードしたら、次のことを確認しましょう。

1. **ファイル形式**: PNG形式が最も汎用的です。背景を透明にしたい場合もPNGを選びましょう
2. **サイズ**: SNSプロフィール用なら400×400ピクセル以上あれば十分です
3. **トリミング**: 正方形でない場合は、画像編集ツール（Canvaの無料版でもOK）で正方形に切り取ります

## よくあるつまずきと解決策

### 「思ったのと全然違う画像が出てくる」
→ プロンプトの要素が足りない可能性が高いです。ステップ2のテンプレートに戻って、モチーフ・スタイル・色味・背景の4点が入っているか確認しましょう。

### 「文字やロゴを入れたいのにうまくいかない」
→ 現時点のAI画像生成は文字の描画が苦手です。文字を入れたい場合は、Canvaなどの別ツールで後から重ねるのが確実です。

### 「商用利用していいの？」
→ ツールによって利用規約が異なります。ChatGPT（DALL·E）は有料プランで商用利用可、Leonardo AIも条件付きで可など、必ず使うツールの規約を確認してください。詳しくは `atom.ai-freelancer.image-copyright-commercial` を参照してください。

## 完成を確認する

以下の3つが揃ったら、このレッスンは完了です。

1. AIで生成したアイコンまたはアバター画像が1点以上、手元に保存されている
2. その画像が正方形（または用途に合ったサイズ）にトリミングされている
3. 実際にSNSのプロフィールやブログのアイコンとして設定できる状態になっている

最終的な成果物として、**生成したアイコン画像のスクリーンショット**を保存しておきましょう。これがあなたの「AIでアイコンを作れるようになった」証明になります。
