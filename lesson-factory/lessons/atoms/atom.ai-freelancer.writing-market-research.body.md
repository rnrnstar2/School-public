# ライティング案件の市場と単価を調査する

「副業でライティングをやってみたいけど、実際いくらぐらい稼げるの？」——これは多くの人が最初に持つ疑問です。スーパーで買い物をするとき、まず値札を見て「これなら買える」「これは高い」と判断しますよね。副業のライティングも同じで、まず「どんな案件がいくらで出ているか」を知ることが、最初の一歩になります。

このレッスンでは、AIチャット（ChatGPTやClaude）を使って、クラウドソーシングサイトに出ているライティング案件の種類と相場をすばやく整理する方法を学びます。最終的に、あなた自身の手で「ライティング市場リサーチまとめ」というドキュメントを1枚作り上げます。

## 前提を確認する

このレッスンを始める前に、以下が揃っていることを確認してください。

- インターネットに接続できるパソコンまたはスマートフォン
- ChatGPT または Claude のアカウント（無料プランでOK）
- クラウドワークスまたはランサーズのアカウント（閲覧だけなら登録不要ですが、案件の詳細を見るには無料登録が必要です）

## ライティング案件の種類を知る

まず、「ライティング案件」と一口に言っても、実はいろいろな種類があります。料理に例えると、「料理」の中にも和食・洋食・中華があるように、ライティングにも複数のジャンルがあります。

主なライティング案件のジャンル:

| ジャンル | 内容の例 | 文字単価の相場 |
|---------|---------|-------------|
| SEO記事 | 検索で上位に出るためのブログ記事 | 0.5〜3円/文字 |
| 商品紹介・レビュー | ECサイトの商品説明文 | 0.5〜2円/文字 |
| メルマガ・SNS投稿 | メールニュースレターやSNS文章 | 1本500〜3,000円 |
| インタビュー記事 | 取材をもとにした記事 | 1本5,000〜30,000円 |
| コピーライティング | 広告のキャッチコピーやLP文章 | 1本10,000〜50,000円 |

![ライティング案件ジャンル分類図](/lesson-assets/atom.ai-freelancer.writing-market-research/diagram.png)

## AIに市場調査を手伝ってもらう

ここからが本題です。クラウドソーシングサイトで案件を1つ1つ見ていくのは大変ですが、AIに「どんな観点で調べればいいか」を聞くことで、効率的にリサーチができます。

### ステップ1: AIに調査の切り口を聞く

ChatGPTまたはClaudeを開いて、以下のプロンプト（＝AIへの指示文）をコピー＆ペーストしてください。

```
クラウドワークスやランサーズでライティング案件を探す初心者です。
以下の観点で、ライティング案件の市場調査チェックリストを作ってください。

1. 案件のジャンル別の単価相場（文字単価・記事単価）
2. 初心者が受注しやすいジャンルのトップ3
3. 避けたほうがいい案件の特徴
4. 月5万円を目指す場合の案件数の目安

表形式でまとめてください。
```

AIが返してくれたチェックリストを、そのままコピーしてメモ帳やGoogleドキュメントに貼り付けておきましょう。

### ステップ2: 実際の案件をチェックする

次に、クラウドワークスまたはランサーズを開いて、以下の手順で案件を確認します。

1. サイトのトップページで「ライティング」と検索する
2. 表示された案件を5〜10件ざっと見る
3. 各案件について以下をメモする:
   - 案件タイトル
   - 報酬（文字単価または記事単価）
   - 必要なスキルや条件
   - 納期

![クラウドソーシングサイトの検索画面](/lesson-assets/atom.ai-freelancer.writing-market-research/screen_capture.png)

### ステップ3: AIに分析してもらう

メモした案件情報をAIに渡して、分析してもらいましょう。以下のプロンプトをコピーして、メモした内容を「ここに案件情報を貼り付け」の部分に差し替えてください。

```
以下はクラウドソーシングサイトで見つけたライティング案件の情報です。
これらを分析して、以下の形式でまとめてください。

【分析してほしいこと】
- 単価の高い順に並べ替え
- 初心者向け度を★1〜★3で評価
- 各案件の「狙い目ポイント」と「注意点」を一言ずつ

【案件情報】
ここに案件情報を貼り付け
```

## 良い例と悪い例で判断力をつける

リサーチをまとめるとき、こんな違いが出ます。

**良い例:**
> 「SEO記事は文字単価0.5〜3円。初心者は1円未満の案件から始めて実績を作り、3ヶ月後に1円以上を狙う。月5万円には1円×5,000文字×10本が目安。」

**悪い例:**
> 「ライティングは稼げる。単価は案件による。頑張れば月5万円いける。」

良い例では具体的な数字と時間軸が入っています。悪い例はあいまいで、次に何をすればいいかわかりません。あなたのまとめも、できるだけ具体的な数字を入れることを意識してください。

## つまずきやすいポイントと対策

| つまずき | 対策 |
|---------|------|
| 案件が多すぎてどれを見ればいいかわからない | まず「ライティング」で検索して、最初の10件だけに絞る |
| 文字単価と記事単価が混在してて比較しにくい | AIに「すべて文字単価に換算して」と頼む |
| 低単価の案件ばかりで不安になる | 最初は実績づくり。3〜5件こなしたら単価交渉できるようになる |
| AIの回答が一般的すぎて役に立たない | 実際の案件情報を貼り付けて、具体的なデータを渡す |
