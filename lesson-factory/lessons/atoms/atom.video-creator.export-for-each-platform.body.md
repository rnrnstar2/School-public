# 媒体別に書き出しを最適化する

## なぜ媒体ごとに書き出しを変えるのかを知る

一つの動画を YouTube、TikTok、Instagram に投稿するとき、それぞれに合った「サイズ」や「画質」に合わせて書き出す必要があります。

料理に例えると分かりやすいです。同じカレーを作っても、レストランならお皿に美しく盛り付け、お弁当なら小分け容器に入れて、屋台なら手軽なカップに入れる――このように「提供先」に合わせて形を変えますよね。動画の書き出しもまったく同じです。

この Atom では、あなたが編集した動画を各プラットフォーム（＝投稿先のサービス）に最適な設定で書き出せるようになることを目指します。かかる時間は約 15 分です。

![プラットフォーム別書き出しフロー](/lesson-assets/atom.video-creator.export-for-each-platform/diagram.png)

## 前提を確認する

始める前に、次の条件を満たしているか確認してください。

- 動画の編集が終わっていて、書き出すファイルが手元にある
- 投稿先のプラットフォームのアカウントを持っている
- 動画編集ソフト（CapCut、DaVinci Resolve、Premiere Pro など）がインストール済みである

## プラットフォーム別の最適な設定を押さえる

まずは主要なプラットフォームごとの「適正」を一覧で押さえましょう。

| プラットフォーム | 画面の形（縦横比） | 解像度の目安 | 動画の長さ |
|---|---|---|---|
| YouTube | 横長 16:9 | 1920×1080 | 制限なし（ショートは 60 秒以内） |
| TikTok | 縦長 9:16 | 1080×1920 | 10 分以内 |
| Instagram リール | 縦長 9:16 | 1080×1920 | 90 秒以内 |
| Instagram フィード | 正方形 1:1 または縦長 4:5 | 1080×1080 / 1080×1350 | 60 秒以内 |
| X（旧 Twitter） | 横長 16:9 または正方形 1:1 | 1920×1080 / 1080×1080 | 2 分 20 秒以内 |

### 良い例・悪い例

**良い例**: TikTok 用に 9:16（縦長）で書き出した動画をそのまま投稿する

**悪い例**: YouTube 用の 16:9（横長）動画を TikTok にそのまま投稿する → 上下に黒い帯が入って見栄えが悪くなります

**良い例**: 投稿先ごとに解像度（＝画面の細かさ）を 1080p に揃える

**悪い例**: 4K で書き出した超大容量ファイルを TikTok にアップする → アップロードに時間がかかり、画質も自動で圧縮されてしまいます

## AI に最適な書き出し設定を聞く

各プラットフォームの推奨設定を毎回覚える必要はありません。AI ツールに聞けば、あなたの環境に合った設定をすぐに教えてもらえます。

### ステップ 1: AI に聞くプロンプトを作る

次の文章をコピーして、あなたの情報に書き換えてから ChatGPT や Claude に貼り付けてください。

```
私は [あなたのソフト名] を使っています。
[投稿先のプラットフォーム名] に投稿する動画の書き出し設定を、
初心者向けに手順つきで教えてください。
推奨するフォーマット、解像度、ビットレートも含めてお願いします。
```

**書き換え例**: 「私は CapCut を使っています。TikTok に投稿する動画の書き出し設定を、初心者向けに手順つきで教えてください。推奨するフォーマット、解像度、ビットレートも含めてお願いします。」

### ステップ 2: 複数プラットフォーム分をまとめて聞く

もし YouTube と TikTok の両方に投稿したいなら、次のように聞くと一度で済みます。

```
CapCut で編集した動画を YouTube（横長 16:9）と TikTok（縦長 9:16）の
両方に投稿したいです。それぞれの書き出し設定と、
1本の動画から両方を効率よく作るコツを教えてください。
```

### ステップ 3: 困ったときの追加質問

AI の回答が専門的すぎて分からない場合は、遠慮なく聞き返しましょう。

```
もっと簡単に説明してください。CapCut の画面のどこをクリックすればいいか、
スクリーンショットで示すように1ステップずつお願いします。
```

### 良い例・悪い例

**良い例**: 「CapCut で Instagram リール用に書き出す設定は？」と具体的に聞く

**悪い例**: 「いい感じに書き出して」とだけ聞く → 自分の環境に合わない回答が返ってきます

## 書き出しを実行する

AI から設定を教えてもらったら、実際に書き出してみましょう。

### 手順

1. 編集ソフトでプロジェクト（＝作業中の動画）を開く
2. 書き出し（エクスポート）メニューを開く
3. プラットフォームに合った「縦横比」と「解像度」を選ぶ
4. フォーマット（＝ファイルの種類）を **MP4** にする
5. ビットレート（＝1 秒あたりのデータ量）を AI が教えてくれた値に設定する
6. 書き出しボタンを押す

![書き出し設定画面の例](/lesson-assets/atom.video-creator.export-for-each-platform/screen_capture.png)

> **ポイント**: 最初に「横長 16:9」（YouTube 用）で書き出し、次に「縦長 9:16」（TikTok・Instagram リール用）で書き出すと効率的です。

## つまずきポイントを解決する

書き出し作業でよくある困りごとと、その対処法をまとめました。

| よくある困りごと | 原因 | 解決方法 |
|---|---|---|
| 縦横比の変え方が分からない | 編集ソフトごとに場所が異なる | 「プロジェクト設定」や「シーケンス設定」にあります。新しく動画を作る前に設定するのがコツです |
| 書き出したら画質が粗い | ビットレートが低すぎる | AI に「高画質にしたい」と伝えて推奨値を聞きましょう |
| ファイルサイズが大きすぎてアップロードできない | 解像度やビットレートが高すぎる | 解像度を一段下げる（4K → 1080p）か、ビットレートを少し下げてください |
| 書き出しにとても時間がかかる | 元の動画が長い、またはパソコンの性能が足りない | 短い動画（30 秒程度）で先に試して設定を確認してから、本番を書き出しましょう |

どうしても解決できない場合は、AI にエラーメッセージや困っている状況をそのまま伝えてください。

```
[ソフト名] で書き出そうとしたら [起きた問題] になりました。
原因と解決方法を教えてください。
```

## 確認する

書き出しが終わったら、投稿前に次のチェックリストを必ず確認してください。

- [ ] ファイルをスマホや PC で再生して、映像が見切れていないか確認した
- [ ] 音声が正常に聞こえるか確認した
- [ ] ファイルサイズが異常に大きくないか確認した（目安: 1 分あたり 50〜200 MB）
- [ ] プラットフォームの投稿画面でプレビュー（＝投稿前の確認表示）を見て、違和感がないか確かめた
- [ ] 投稿先ごとに正しい縦横比で書き出せている

すべてチェックできたら、あなたの動画は各プラットフォームに最適な形で書き出し完了です！

### 成果物の目安

今回の成果物は、**少なくとも 2 つのプラットフォーム向けに書き出した動画ファイル**（またはその書き出し設定のスクリーンショット）です。書き出した画面のスクリーンショットを撮っておくと、次回以降に同じ設定をすぐ再現できます。
