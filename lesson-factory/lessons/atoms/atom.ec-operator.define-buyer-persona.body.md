# 購入者像を具体化する

## はじめに

あなたが大切な友人にプレゼントを選ぶとき、その人の趣味や生活スタイルを思い浮かべませんか？ ネットショップで商品を売るときも同じです。「誰に届けたいか」がぼんやりしたままだと、商品説明も写真も「なんとなく」になってしまいます。

このレッスンでは、AI（＝あなたの質問に文章で答えてくれるツール。ChatGPTやClaudeなどが代表的です）を使って、**あなたの商品を買ってくれそうな架空の人物像（＝ペルソナ）**を1人分作ります。料理でいえば「レシピを決める前に、誰に食べてもらうかを決める」ステップです。15分あれば完成しますよ！

![ペルソナ作成フロー](/lesson-assets/atom.ec-operator.define-buyer-persona/diagram.png)

## 使うAIツールを選ぶ

以下のどれか1つを開いてください。どれでも同じ結果が得られます。

| ツール名 | 開き方 |
|---------|--------|
| **ChatGPT** | ブラウザで chat.openai.com を開いてログイン |
| **Claude** | ブラウザで claude.ai を開いてログイン |
| **Cursor** | アプリを起動し、右側のチャット欄を使う |

まだどれも使ったことがなければ、**ChatGPT**が始めやすいです。無料プランで十分使えます。

## 商品情報をAIに伝える

AIはあなたの商品のことを何も知らない「新しく来た助手」のようなものです。まずは商品のことを教えてあげましょう。

下のプロンプト（＝AIへの指示文）をコピーして、`【 】`の部分だけあなたの商品に書き換えてから送信してください。

```text
私は【商品ジャンル（例：ハンドメイドアクセサリー）】をネットショップで販売しています。
この商品の一番の魅力は【主な特徴（例：天然石を使った一点もの）】です。
価格帯は【例：3,000〜5,000円】です。

この商品を買ってくれそうな人の、具体的な架空プロフィールを1人分作ってください。
以下の項目をすべて含めてください：
- 名前（架空でOK）
- 年齢
- 職業
- 住んでいる地域
- 休日の過ごし方
- よく使うSNS
- 買い物するときのこだわり
- この商品を「欲しい！」と思うきっかけ
```

**ポイント**: 価格帯を入れると、AIが収入や生活水準をイメージしやすくなり、より現実的な人物像が出てきます。

## AIの出力を確認する

AIが作ってくれたプロフィールを読んでみましょう。以下のチェックで「良い出力」かどうかを判断できます。

**良い例：**
- 「佐藤花子、29歳、IT企業の事務職、横浜市在住。休日はカフェ巡りとInstagram投稿。ネットで買い物するときは口コミを必ず3件以上読む。友人の誕生日プレゼントを探しているときにInstagramの広告で見かけて興味を持った」
- → 一人の人間として想像でき、商品との接点（きっかけ）が具体的

**悪い例：**
- 「20〜30代の女性。おしゃれが好き」
- → ぼんやりしていて、商品説明を書くヒントにならない

悪い例のようになったら、次のように追加で指示してみてください：

```text
もう少し具体的にしてください。特に「この商品を欲しいと思ったきっかけ」と「普段どんなSNSを見ているか」を詳しくお願いします。
```

## ペルソナを深掘りする（応用）

基本のプロフィールができたら、もう一歩踏み込んでみましょう。以下のプロンプトを続けて送ると、商品ページ作りに直接役立つ情報が得られます。

```text
この人物が商品を購入するとき、一番の不安は何ですか？
また、どんな言葉が商品説明に書いてあったら安心して購入ボタンを押せますか？
3つずつ挙げてください。
```

この回答は、商品説明文の「お客様の声」や「よくある質問」を作るときにそのまま使えます。

## 成果物を保存する

AIの出力が納得できるものになったら、保存しましょう。

1. AIの回答をすべて選択してコピーする（Ctrl+C / Cmd+C）
2. メモ帳やGoogleドキュメントなど、普段使っている文書ツールに貼り付ける
3. ファイル名を「ペルソナ_（商品名）_（今日の日付）」にして保存する

これがあなたの**購入者像（ペルソナ）ドキュメント**の完成です！

![AIでのやり取り画面](/lesson-assets/atom.ec-operator.define-buyer-persona/screen_capture.png)

## 完成を確認する

保存したペルソナドキュメントを見直して、以下の3つに「はい」と言えるか確認してください。

- [ ] その人の顔や生活が具体的にイメージできるか？
- [ ] その人があなたの商品を「どうやって見つけるか」を想像できるか？
- [ ] その人が「なぜ買うか」の理由が書かれているか？

3つすべて「はい」なら大成功です！ もし1つでも「いいえ」があれば、AIに「〇〇の部分をもっと具体的にしてください」とお願いしてみましょう。

## つまずいたときは

| 困りごと | 対処法 |
|---------|--------|
| AIの回答が英語で返ってきた | 「日本語で回答してください」と追加で送る |
| 自分の商品と合わない人物像が出た | プロンプトの商品説明をもっと具体的に書き直す（価格帯・素材・使う場面を追加） |
| 何度やっても「ぼんやりした人物像」しか出ない | 「名前・年齢・職業・住所・趣味・SNS・購入きっかけを全て含めて」と項目を指定し直す |
| AIが長すぎる文章を返してきた | 「箇条書きで簡潔にまとめ直してください」と送る |
