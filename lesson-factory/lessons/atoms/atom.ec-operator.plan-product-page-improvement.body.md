# 商品ページ改善の実践計画を立てる

あなたのお店の商品ページ、お客様にとって「わかりやすい」と自信を持って言えますか？

商品ページは、実店舗でいう「店員さんの接客」と「商品のディスプレイ」が合わさったものです。ネットショップでは、このページがお客様との唯一の接点になります。だからこそ、少しずつ改善していくことが売上アップにつながります。

このレッスンでは、**AIチャットツール（＝人工知能を使った対話型の便利ツール）を相棒にして、15分で商品ページの改善計画を立てます**。「何を」「なぜ」「どうやって」直すかを書き出した計画書が、あなたの成果物です。

![改善計画の全体フロー](/lesson-assets/atom.ec-operator.plan-product-page-improvement/diagram.png)

---

## 準備する

始める前に、次の2つを手元に用意してください。

- **改善したい商品ページのURL**（＝インターネット上の住所。ブラウザの上部に表示されている文字列です）を1つ
- **AIチャットツール**を1つ（以下のどれでもOK。無料版で十分です）

| ツール名 | 特徴 | 開き方 |
|---|---|---|
| **ChatGPT** | もっとも有名。会話形式で答えてくれる | chat.openai.com にアクセス |
| **Claude** | 長い文章の分析が得意 | claude.ai にアクセス |
| **Gemini** | Google製。検索結果も参考にしてくれる | gemini.google.com にアクセス |

特別なソフトのダウンロードは不要です。スマホでもパソコンでも、ブラウザ（＝インターネットを見るためのアプリ。Safari や Chrome など）があれば始められます。

---

## 商品ページの「今」を診断する

まずは改善したい商品ページを開いて、5つのチェックポイントを確認します。料理をする前に「冷蔵庫の中身を確認する」のと同じで、今の状態を知らないと何を直すか決められません。

次のリストを見ながら、メモ帳やスマホのメモアプリに「○」「×」を書いてください。

| # | チェック項目 | ○ or × |
|---|---|---|
| 1 | 商品写真は3枚以上ある（正面・アップ・使用イメージ） | |
| 2 | 商品説明の冒頭に「この商品は○○な人におすすめ」が書いてある | |
| 3 | 価格の近くに「この価格の理由」や「お得な理由」が書いてある | |
| 4 | お客様の声（レビュー）が表示されている | |
| 5 | 「カートに入れる」ボタンがすぐ見つかる位置にある | |

「×」が多いほど伸びしろがある、ということ。落ち込む必要はありません。

## AIに改善アイデアを出してもらう

ここからが本番です。AIチャットツールを開いて、以下の質問文をコピーして貼り付けてください。`【 】` の部分だけ、あなたの情報に書き換えます。

```
私はネットショップを運営しています。以下の商品ページを改善したいです。
ECサイト初心者にもわかりやすく、具体的な改善案を5つ教えてください。
それぞれ「何をするか」「なぜ効果があるか」「作業時間の目安」を添えてください。

商品ページURL: 【ここにURLを貼る】
商品ジャンル: 【例: アパレル、食品、雑貨など】
現状の悩み: 【例: アクセスはあるのに購入されない】
```

**何をしているか**: AIに「あなたのお店専属のアドバイザー」になってもらい、商品ページの改善点を洗い出してもらっています。

![AIチャットでの質問と回答の例](/lesson-assets/atom.ec-operator.plan-product-page-improvement/screen_capture.png)

### AIの回答が期待通りでないときの追加プロンプト

AIの最初の回答がピンとこない場合は、以下のように追加で聞いてみましょう。

| 困りごと | 追加で送る文 |
|---|---|
| 回答が難しい言葉ばかり | 「専門用語を使わず、小学生にもわかる言葉で説明し直してください」 |
| 改善案が抽象的すぎる | 「それぞれの改善案について、実際にページのどこをどう変えるか、具体例を1つずつ見せてください」 |
| 自分の商品に合わない提案 | 「私の商品は【商品の特徴】です。この商品に特化した改善案に絞ってください」 |

### 良い改善案と悪い改善案の見分け方

- ✅ **良い例**: 「商品写真の1枚目に、手に持っているサイズ感が伝わる写真を追加する（10分）」
- ✅ **良い例**: 「商品説明の冒頭に『30代の乾燥肌に悩む方へ』とターゲットを明記する（5分）」
- ❌ **悪い例**: 「ページをもっとおしゃれにする」（何をどう変えるか不明）
- ❌ **悪い例**: 「プロのカメラマンに撮影を依頼する」（今すぐ一人ではできない）

## 改善案を3つ選ぶ

AIが出してくれた5つの改善案から、**今すぐ自分でできるものを3つ**選びます。旅行の荷造りと同じで、全部詰め込むと重すぎます。「本当に必要なもの」だけ選びましょう。

選ぶ基準はこの3つです。

| 基準 | 質問 |
|---|---|
| ⏱ 時間 | 15分以内で終わるか？ |
| 💰 コスト | お金をかけずにできるか？ |
| 👤 自力 | 自分ひとりで今日できるか？ |

3つとも「はい」と答えられる改善案だけを選んでください。

迷ったときは、AIにこう聞きましょう。

```
この5つの改善案のうち、15分以内・無料・一人でできるものだけ上位3つに絞ってください。
```

---

## 計画書を書き出す

選んだ3つの改善案を、以下のテンプレートに沿って書き出します。AIチャットに次のように頼むと、テンプレートを埋めた形で出力してくれます。

```
以下のテンプレートに沿って、選んだ3つの改善案を計画書としてまとめてください。

【商品ページ改善計画】
作成日: ○月○日
対象商品: （商品名）
対象ページURL: （URL）

■ 改善1
・何をするか:
・なぜやるか（期待される効果）:
・作業時間の目安:
・完了の目安（どうなったらOKか）:

■ 改善2
（同じ形式）

■ 改善3
（同じ形式）
```

AIが出力した計画書をメモ帳やドキュメントにコピーして保存し、**スクリーンショット（＝画面の写真）を撮ってください**。これがあなたの成果物です。

> **スクリーンショットの撮り方**
> - Windows: `Windowsキー + Shift + S` を同時に押す
> - Mac: `Command + Shift + 4` を同時に押す
> - スマホ: 電源ボタン + 音量上ボタンを同時に押す

---

## 確認する

計画書が完成したら、以下の5つをチェックしましょう。すべて「はい」なら合格です。

- [ ] 改善が3つ書かれていて、それぞれ「何をするか」が一読でわかる
- [ ] 3つとも「15分以内・無料・一人で可能」な内容になっている
- [ ] AIの提案をそのままコピペせず、自分のお店に合わせた内容にカスタマイズした
- [ ] 「完了の目安」が具体的で、作業が終わったかどうか判断できる
- [ ] 計画書のスクリーンショットが保存できている

---

## つまずいたときの対処法

| つまずき | 原因 | 対処法 |
|---|---|---|
| AIの回答が難しい | 専門用語で返される | 「小学生にもわかる言葉で説明して」と送る |
| 改善案が多すぎて選べない | 優先度がわからない | 「今すぐ・無料・一人で」の3条件で絞る |
| 商品ページURLがわからない | ページの開き方がわからない | ブラウザで自分のお店を検索→商品をクリック→上部のアドレスバーをコピー |
| AIがURLを開けないと言われる | AIがWebページを読めないモデル | 商品説明文と写真の枚数を文章でコピーして質問文に貼り付ける |
| そもそも商品ページがない | まだ出品していない | まず商品を1つ登録してから、このレッスンに戻る |

## まとめ

おつかれさまでした！ あなたは今、AIを使って商品ページの改善計画を立てるスキルを身につけました。

この計画書は「やることリスト」です。一度に全部やる必要はありません。今日1つ、明日もう1つ、と少しずつ進めていきましょう。小さな改善の積み重ねが、お客様から見たページの印象を大きく変えます。
