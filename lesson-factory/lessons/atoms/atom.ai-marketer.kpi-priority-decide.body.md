# 優先KPIを決める

旅行の荷造りを思い出してみてください。スーツケースには全部入りません。「今回は何を一番大事にする？」と決めないと、荷物が溢れてしまいます。

ビジネスも同じです。売上・訪問者数・いいね数……見たい数字（＝KPI）はたくさんあります。でも、すべてを同時に追うと迷子になります。このレッスンでは、**AIを相談相手にしながら、今のあなたが一番集中すべきKPIを1つだけ決める**力を身につけます。

## KPIとは何かを理解する

KPI（Key Performance Indicator ＝ 重要業績評価指標）は、「目標に近づいているか」を測るための数字のことです。家計簿でいうと「毎月の貯金額」のようなもの——ゴールに向かって進んでいるかを一目で確認できます。

- **良い例**: 「今月の無料会員登録数」→ 具体的で、数えられて、毎日確認できる
- **悪い例**: 「ビジネスを成長させる」→ あいまいで測れないのでKPIとして使えない

## 今のビジネス状況を整理する

まずは、あなたのビジネス（または担当しているプロジェクト）が今どの段階にあるかを整理します。ビジネスの成長には大きく3つの段階があります。

| 段階 | やること | 例 |
|------|----------|----|
| 1. 知ってもらう | まだ存在を知らない人に届ける | SNS投稿、広告、口コミ |
| 2. 使ってもらう | 興味を持った人に試してもらう | 無料体験、初回割引 |
| 3. 続けてもらう | 一度使った人にまた来てもらう | メルマガ、リピート特典 |

**AIに聞いてみましょう。** ChatGPTやClaudeを開いて、次のプロンプト（＝AIへの指示文）をコピペしてください。

```
私のビジネスは「[あなたのビジネス内容を書く]」です。
今は以下の3段階のうち、どの段階にいると思いますか？
1. サービスを知ってもらう段階
2. 使ってもらう段階
3. 使い続けてもらう段階

判断の理由と、この段階でよく使われるKPIの候補を5つ教えてください。
```

> **ポイント**: `[あなたのビジネス内容を書く]` の部分を自分の状況に書き換えてから送信してください。「お弁当の宅配サービス」「ハンドメイドアクセサリーのネットショップ」など、具体的に書くほどAIの回答が的確になります。

![AIにKPI候補を聞いた画面例](/lesson-assets/atom.ai-marketer.kpi-priority-decide/screen_capture.png)

## 候補KPIを3〜5個書き出す

AIの回答をヒントに、あなたの状況に合いそうなKPIを3〜5個書き出します。メモ帳やノートアプリに箇条書きで十分です。

- **良い例**: 「ウェブサイト訪問者数」「無料トライアル登録数」「メルマガ登録数」「Instagramフォロワー数」
- **悪い例**: 「売上」「利益」→ まだ「知ってもらう段階」だと測りにくく、自分の行動で直接動かしにくい

迷ったら、AIにこう聞いてみましょう。

```
さっき挙げてくれた5つのKPIについて、
「毎日確認しやすい順」に並べ替えてください。
それぞれ、どこで数字を見られるかも教えてください。
```

## 優先するKPIを1つに絞る

書き出した候補の中から、次の3つの質問でふるいにかけます。

| チェック項目 | 質問 | 例 |
|-------------|------|----|
| 測れるか？ | 毎日（または毎週）確認できる数字か | Google Analyticsで訪問者数を見られる → ○ |
| 動かせるか？ | あなたの行動で増やせる数字か | SNS投稿を増やせば訪問者が増える → ○ |
| 今大事か？ | 今の段階で一番意味がある数字か | まだ知ってもらう段階なので訪問者数 → ○ |

**AIに最終確認してもらいましょう。**

```
以下のKPI候補から、今の私の状況に最も適した1つを選んで、
理由を教えてください。

私の状況：[ビジネス内容と段階を書く]
候補：
1. [候補1]
2. [候補2]
3. [候補3]

選ぶ基準は「毎日測れる」「自分の行動で動かせる」「今の段階に合っている」の3つです。
```

AIの意見を参考にしつつ、**最終的にはあなた自身で決めてください**。AIはあくまで相談相手です。あなたの現場の感覚のほうが正しいことも多いです。

## 選んだKPIを書き残す

決めたKPIと理由をテキスト（メモ帳やNotionなど）に書き残してください。これがこのレッスンの成果物です。

書き方の例：

```
■ 優先KPI：毎日のウェブサイト訪問者数
■ 理由：今は「サービスを知ってもらう段階」なので、
  一番見るべき数字は訪問者数だから。
■ 確認方法：Google Analyticsの「リアルタイム」画面で毎朝チェック
■ 目標値：1日100人（現在は約30人）
```

> **よくある失敗**: 理由を書かずにKPIだけ決めてしまうと、1週間後に「なぜこの数字を追っていたんだっけ？」となりがちです。理由と確認方法もセットで書いておきましょう。

## 確認する

次のチェックリストで完成度を確認しましょう。

- [ ] KPIを**1つだけ**選べているか
- [ ] 選んだKPIは具体的な「数」で測れるか（「○○数」「○○率」など）
- [ ] 毎日（または毎週）確認できる場所を書いたか
- [ ] 「なぜこのKPIか」の理由を書いたか
- [ ] 目標値（大まかでOK）を書いたか

すべて「はい」なら完成です。スクリーンショットを撮って保存しましょう。

## つまずきやすいポイント

| つまずき | 対処法 |
|----------|--------|
| 「全部大事」で絞れない | まずは1つだけ決めてください。残りの候補はメモしておいて、次の段階に進んだときに使います。スーツケースの荷造りと同じで、「今回の旅で一番必要なもの」だけ入れましょう |
| AIの提案をそのまま採用してしまった | AIはあなたのビジネスを100%は知りません。AIの提案を「たたき台」にして、あなたの現場感覚で最終判断してください |
| 売上をKPIにしてしまう | 売上は「結果」の数字です。それより手前にある数字（訪問数、登録数、問い合わせ数など）を選ぶと、毎日の行動で動かしやすくなります |
| KPIを決めたが確認方法がわからない | AIに「○○を毎日確認するには、どのツールのどの画面を見ればいいですか？」と聞いてみましょう。無料ツールの使い方まで教えてくれます |
