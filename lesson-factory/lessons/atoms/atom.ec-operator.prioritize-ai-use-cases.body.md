# AI導入の優先順位を整理する

このレッスンでは、あなたの日常業務のなかで「どこにAI（＝人工知能。ここでは ChatGPT や Claude のような対話型ツールを指します）を導入するといちばん効果的か」を見つけ、優先順位を整理する方法を学びます。

レッスンが終わるころには、「最初にAIに任せるべきタスク」が1つ決まり、その理由を自分の言葉で説明できるようになっています。

## たとえ話で理解する

忙しい平日の夜、夕食を作る場面を想像してみてください。時間と体力は限られています。冷蔵庫にある食材のなかから、「一番手軽で、しかもおいしいもの」を選びますよね。AIの導入もこれと同じです。限られた時間のなかで、一番効果が高くて導入がラクなものから始めるのがコツです。

## 業務のタスクを洗い出す

まずは、あなたが日々の業務で行っている「繰り返しの作業」を5〜10個書き出しましょう。紙でもメモアプリでもかまいません。

思いつかないときは、ChatGPT や Claude に次のように聞いてみましょう。

> **プロンプト例**:
> 「私はECサイトを運営しています。日々の業務で繰り返しやっている作業を洗い出したいです。ECサイト運営でよくある繰り返し作業を10個リストアップしてください。それぞれ1行で簡潔にお願いします。」

AIが出してくれたリストを見て、「あ、これは自分もやっている」と思うものにチェックを入れてください。自分だけの作業リストが短時間でできあがります。

- **良い例**：商品説明文の作成、メルマガの下書き、よくある質問への返答テンプレート作成、SNS投稿文の作成
- **悪い例**：お客様からのクレーム対応（その場その場で臨機応変な判断が必要なもの）、取引先との価格交渉（人間関係や状況判断が中心のもの）

## 優先順位マトリクスで整理する

洗い出したタスクを、次の2つの軸でグループ分けします。

1. **効果（＝AIに任せるとどれくらい時短になるか）**：毎回30分かかる作業なら効果「高」、5分で終わるなら効果「低」
2. **手軽さ（＝AIに指示を出すのがどれくらい簡単か）**：「この文章を要約して」のように1回の指示で済むなら手軽さ「高」、何度もやり取りが必要なら手軽さ「低」

![優先順位マトリクス](/lesson-assets/atom.ec-operator.prioritize-ai-use-cases/diagram.png)

| | 手軽さ「高」 | 手軽さ「低」 |
|---|---|---|
| **効果「高」** | すぐ始める！ | 次のステップで取り組む |
| **効果「低」** | 余裕があれば試す | 今は後回しでOK |

### AIを使ってマトリクスを作る

タスクの分類に迷ったら、AIに手伝ってもらいましょう。

> **プロンプト例**:
> 「以下の業務タスクを『効果（時短になる度合い）』と『手軽さ（AIへの指示の簡単さ）』の2軸で分類してください。それぞれ高/低で4グループに分けて、理由も一言添えてください。
> - 商品説明文の作成
> - メルマガの下書き
> - 在庫数の確認
> - SNS投稿文の作成
> - 顧客レビューへの返信テンプレート作成」

![AIでマトリクスを作成した画面](/lesson-assets/atom.ec-operator.prioritize-ai-use-cases/screen_capture.png)

AIの回答をそのまま使うのではなく、「自分の業務ではどうだろう？」と照らし合わせて調整するのがポイントです。

## 成果物をまとめる

マトリクスで「効果が高く、手軽なもの」に分類されたタスクのなかから、最初にAIに任せるタスクを1つ選んでください。

以下のテンプレートをコピーして、あなたの内容で埋めてみましょう。ChatGPTやClaudeに貼り付けて「この内容をきれいに整えて」と頼んでもOKです。

```
# 最初にAIに任せるタスク

## 選んだタスク
（例：商品説明文の作成）

## 選んだ理由
- 効果：（例：毎回30分かかっている → AIなら5分で下書きができそう）
- 手軽さ：（例：商品の特徴を箇条書きにして渡すだけで指示できる）

## 最初に試すプロンプト
（例：「以下の商品特徴をもとに、ECサイト用の商品説明文を200文字で書いてください。」）
```

## 確認する

成果物ができたら、以下の3つをチェックしてください。すべて「はい」なら合格です。

- [ ] 選んだタスクは「週に2回以上」繰り返し発生するものになっているか？
- [ ] AIへの指示を1〜2文で伝えきれるか？（長い説明が必要なタスクは「手軽さ」が低い可能性あり）
- [ ] 「最初に試すプロンプト」を実際にAIに入力して、期待に近い結果が返ってくるか？

## つまずきポイントと対処法

| つまずき | 対処法 |
|---|---|
| 「AIは何でもできる」と思って大きすぎるタスクを選んでしまう | 最初は「文章をまとめる」「アイデアを出す」など、AIが得意なことだけに絞りましょう |
| タスクを洗い出しても、どれが効果的かわからない | 「1週間のなかで一番時間がかかっている作業は何か？」と自分に問いかけてください |
| マトリクスの分類で迷って手が止まる | 迷ったら「手軽さ高」に仮置きして先に進みましょう。あとから移動しても大丈夫です |
| AIに聞いたら的外れな回答が返ってきた | 「私はECサイトの〇〇を担当しています」と自分の状況を1文追加すると精度が上がります |
