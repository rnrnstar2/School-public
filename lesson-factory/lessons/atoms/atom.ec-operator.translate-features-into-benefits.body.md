# 特徴を訴求価値に変換する

あなたのネットショップの商品ページを見に来たお客さんは、「これが欲しい！」と一瞬で判断しています。

料理で考えてみましょう。メニューに「国産小麦粉、北海道バター使用」とだけ書いてあったら、お客さんは「ふーん」で終わってしまいます。でも「外はサクサク、中はしっとり。朝食のテーブルが笑顔になるトースト」と言われたら、食べたくなりますよね。

これと同じことがネットショップでも起きています。このレッスンでは、商品の「特徴」（＝材料や仕様などの事実）を、お客さんが思わず欲しくなる「訴求価値」（＝ベネフィット。お客さんがどうなるか、どう感じるか）に変換する練習をします。AIチャットツールに手伝ってもらうので、文章が苦手でも大丈夫です。

![特徴から訴求価値への変換フロー](/lesson-assets/atom.ec-operator.translate-features-into-benefits/diagram.png)

## 前提を確認する

このレッスンに必要なものは次のとおりです。

- 手元に商品の「特徴」が3〜5個書き出してあること（材料、機能、サイズ、重さ、素材などなんでもOK）
- AIチャットツール（＝テキストを入力すると文章を作ってくれるツール）をブラウザで開いておくこと。以下のどれでも構いません：
  - **ChatGPT**（chat.openai.com）— 無料プランでOK
  - **Claude**（claude.ai）— 無料プランでOK
  - **Gemini**（gemini.google.com）— Googleアカウントがあれば無料

特別なプログラミング知識は一切不要です。15分あれば終わります。

## 特徴と訴求価値の違いを理解する

まずは基本を押さえましょう。この2つの違いがわかると、商品ページの書き方がガラリと変わります。

| 特徴（フィーチャー） | 訴求価値（ベネフィット） |
|---|---|
| 商品が「何であるか」「何を持っているか」 | お客さんが「どうなるか」「どう感じるか」 |
| 防水コーティング | 雨の日でも安心。荷物が濡れない |
| 100%オーガニックコットン | 肌が敏感な人でも安心して着られる |
| バッテリー20時間 | 充電を忘れても1日中使える |

**良い例**：
- 特徴「軽量200g」→ 訴求価値「肩がこらない。長時間持ち歩いても疲れません」
- 特徴「防水IPX7」→ 訴求価値「突然の雨でも中身は無傷。安心して持ち歩けます」

**悪い例**：
- 特徴「軽量200g」→「とても軽いです」（＝特徴の言い換えで終わっている。お客さんの生活がどう変わるかが書かれていない）
- 特徴「防水IPX7」→「IPX7規格対応の防水性能があります」（＝専門用語のままでお客さんに伝わらない）

ポイントは「だから、お客さんはどうなるの？」を常に自分に問いかけることです。

## AIに手伝ってもらって変換する

では、AIチャットツールを使って実際に変換してみましょう。次のプロンプト（＝AIへの指示文）をコピーして、お使いのAIチャットに貼り付けてください。

「〇〇〇〇」の部分をあなたの商品の特徴に書き換えるのを忘れないでください。

```
次の商品特徴を、ネットショップのお客さんに伝える「訴求価値」に変換してください。

商品特徴：〇〇〇〇

条件：
- 特徴そのものを言い換えるのではなく、「お客さんがどうなるか・どう感じるか」を書く
- 具体的な場面や感情を含める
- 3パターンの訴求価値を提案する
- それぞれの訴求価値に「どんなお客さんに刺さるか」も1行で添える
```

AIが3パターンの提案を返してくれます。まだ選ばなくて大丈夫です。次のステップでもう少し磨きます。

### AIの結果がいまいちだったら追加で聞く

最初の結果がピンとこなかった場合は、次のように追加で指示してみてください（これを「追加プロンプト」と呼びます）。

**ターゲットを絞りたい場合：**
```
30代の働くママ向けに書き直してください。朝の忙しい時間帯に使う場面を想定して。
```

**もっと感情に訴えたい場合：**
```
もっとお客さんの感情（安心感、ワクワク感、誇らしさなど）を前面に出してください。
```

**競合との差を出したい場合：**
```
他の似た商品と比べて「ここが違う」と感じてもらえる表現にしてください。
```

## 自分の言葉で磨き上げる

AIの提案をそのまま使うのではなく、あなたの商品を知っているからこそ書ける「リアルな一言」を加えましょう。これが他のショップとの差になります。

**良い例**：
- AI提案「長時間使っても疲れません」→ 磨き上げ後「1日中使っても夕方にはまだ元気。仕事帰りのカフェでもサッと取り出せる頼れる相棒です」
- AI提案「お子さんの肌にも安心」→ 磨き上げ後「うちの2歳の娘が気に入って毎日着ています。洗濯しても柔らかさが変わらないのが嬉しいポイント」

**悪い例**：
- AI提案を一切修正せずそのままコピペする（＝あなたの商品らしさがゼロになる）
- 専門用語を追加して難しくする（＝お客さんが離れてしまう）

磨き上げのコツは「自分が使ったときの体験」や「お客さんの声」を一言だけ足すことです。これだけでグッとリアルになります。

## 結果を確認する

次の4つのチェックポイントで仕上がりを確認しましょう。

- ✅ 「お客さんがどうなるか・どう感じるか」が書かれているか
- ✅ 特徴の単なる言い換えになっていないか
- ✅ 具体的な場面や感情が含まれているか
- ✅ あなたの商品らしさ（体験談やお客さんの声など）が反映されているか

すべて「はい」なら完成です！

変換前の特徴と変換後の訴求価値を並べた状態でスクリーンショット（＝画面の写真）を撮って保存しましょう。これがこのレッスンの成果物です。

**スクリーンショットの撮り方**：
- Mac：`Command + Shift + 4` を押して範囲を選択
- Windows：`Windows + Shift + S` を押して範囲を選択
- スマホ：電源ボタン＋音量ボタンを同時に押す

## つまずきやすいポイントに対処する

**「AIの提案が全部ピンとこない」**
→ 特徴をもう少し具体的に書き直してから再度AIに聞いてみましょう。「軽量」だけではなく「重さ200gの軽量ナイロン製」と数値や素材名を添えると、AIの回答の精度がグンと上がります。

**「結局どう書けばいいか迷う」**
→ 身近な人に「この文章だと欲しいと思う？」と聞いてみましょう。第三者の反応が一番の目安になります。家族や友人にLINEで送るだけでもOKです。

**「全部同じような表現になってしまう」**
→ お客さんのライフスタイルごとに切り分けてみましょう。「忙しい朝に」「リラックスした週末に」「大切な人への贈り物に」など場面を変えると表現が広がります。AIへの追加プロンプトで「別の場面を想定して」とお願いするのも効果的です。

**「AIに何を指示したらいいかわからない」**
→ このレッスンのプロンプト例をそのままコピペして、商品特徴だけ書き換えてください。最初はそれで十分です。慣れてきたら自分なりのアレンジを加えてみましょう。
