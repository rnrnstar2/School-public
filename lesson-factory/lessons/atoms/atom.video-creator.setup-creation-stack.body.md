# AI制作ツールの環境を整える

料理をする前に「包丁・まな板・フライパン」を揃えるように、動画づくりでもまずは道具を揃えます。このレッスンでは、AIの力を借りて動画コンテンツを作るための **3つのツール** を15分で準備します。

完了すると「AIに相談しながら動画の素材を自分で作り始められる状態」になります。

![ツール構成図](/lesson-assets/atom.video-creator.setup-creation-stack/diagram.png)

## 前提を確認する

このレッスンを始める前に、次の2つを用意してください。

- インターネットに接続できるブラウザ（ChromeやSafariなど）
- 受信できるメールアドレス

プログラミングの知識は一切不要です。すべてブラウザ上で完結します。

## AIチャットツールを登録する（目安：5分）

AIチャットツール（＝AIと対話しながら文章を作れるサービス）は、動画の構成案や台本を書くときの「相談相手」になります。

### 選ぶ

次のどれか **一つだけ** を無料プランで登録してください。迷ったらChatGPTがおすすめです。

| ツール名 | 特徴 | こんな人向け |
|---------|------|------------|
| **ChatGPT**（OpenAIが提供） | 利用者が最も多く、情報も見つけやすい | 初めてAIを使う人 |
| **Claude**（Anthropicが提供） | 日本語が丁寧で、長い文章の整理が得意 | 台本を丁寧に作りたい人 |
| **Gemini**（Googleが提供） | Googleアカウントですぐ始められる | すでにGoogleをよく使う人 |

**良い例**: 「ChatGPTに登録して、試しに話しかけてみる」→ 1つに集中する

**悪い例**: 「3つ全部一気に登録する」→ どれを使っているか分からなくなる

### 動作を確認する

登録できたら、次のメッセージをコピーしてAIに送ってみてください。

```
YouTubeの1分動画の構成案を3つ考えて。テーマは「朝のルーティン」です。
```

AIから構成案が返ってくれば成功です。もし英語で返事が返ってきたら、「日本語で答えてください」と追加で送ればOKです。

## AI画像生成ツールを登録する（目安：5分）

AI画像生成ツール（＝言葉で指示すると画像を作ってくれるサービス）は、サムネイル画像や動画に差し込む素材を作るときに使います。

### 選ぶ

次のどちらか **一つだけ** を無料プランで登録してください。

- **Canva** — デザインのひな形（テンプレート）が豊富で初心者向け。AI画像生成機能も内蔵されています
- **Copilot**（Microsoftが提供） — ブラウザ上ですぐに画像生成を試せる

**良い例**: 「Canvaに登録して、『夕暮れのカフェの風景、写真風』と入力し、画像が1枚できるか試す」

**悪い例**: 「登録だけして、何も作らずに終わる」→ 使えるかどうか分からないまま

### 動作を確認する

登録できたら、次の指示文をコピーして画像を1枚作ってみてください。

```
明るいカフェで読書をしている人、イラスト風、温かい色合い
```

画像が1枚表示されれば成功です。最初は思い通りにならなくて当たり前です。指示文を少し変えるだけで結果が大きく変わるので、何度か試してみてください。

## 画面録画の方法を確認する（目安：3分）

あなたのパソコン画面を録画する（＝動いている画面を動画として保存する）機能は、操作手順を説明する動画を作るときに便利です。新しいソフトのインストールは不要で、パソコンに最初から入っている機能を使います。

### 録画メニューを開く

- **Windows の場合**: キーボードの `Windowsキー＋G` を同時に押す → 録画メニューが開きます
- **Mac の場合**: キーボードの `Command＋Shift＋5` を同時に押す → 録画メニューが開きます

![画面録画の操作例](/lesson-assets/atom.video-creator.setup-creation-stack/screen_capture.png)

### 動作を確認する

1. 上の方法で録画を開始する
2. 5秒ほど待ってから録画を停止する
3. 動画ファイルが保存されていることを確認する

保存先がわからないときは:
- **Windows**: 「ビデオ」フォルダ → 「キャプチャ」フォルダの中
- **Mac**: デスクトップ上に `.mov` ファイルとして保存されます

## 3つのツールが揃ったことを確認する

次のチェックリストをすべて満たせば、あなたの「AI制作キッチン」の完成です。

- [ ] AIチャットツールにメッセージを送り、返事が返ってきた
- [ ] AI画像生成ツールで画像を1枚作れた
- [ ] パソコンの画面を5秒間録画し、ファイルが保存された

すべて確認できたら、**3つのツールの画面をまとめてスクリーンショット（＝画面の写真）で撮る**か、チェック済みの画面を1枚撮って保存してください。それがこのレッスンの成果物です。

## つまずきやすいポイントに対処する

| こんなとき | 対処法 |
|-----------|-------|
| 登録画面で有料プランの案内が出た | 「後で」「スキップ」「Free plan」を選んでください。まずは無料で十分です |
| AIの返事が英語になる | 最初のメッセージに「日本語で答えてください」と書けば切り替わります |
| 生成した画像がイメージと違う | 「〇〇風、△△な色合い、□□な雰囲気」のように形容詞を増やすと精度が上がります |
| 録画ファイルが見つからない | Windows:「ビデオ」→「キャプチャ」、Mac: デスクトップを確認してください |
| 録画メニューが開かない（Windows） | 「設定」→「ゲーム」→「Game Bar」がオンになっているか確認してください |

> **AIに聞いてみよう**: もし途中でわからないことがあったら、AIチャットツールに「〇〇の登録方法を初心者向けに教えて」と聞いてみてください。AIが手順を案内してくれます。これも立派なAI活用です！
