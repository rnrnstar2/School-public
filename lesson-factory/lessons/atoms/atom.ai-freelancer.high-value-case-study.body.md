# 高単価案件の受注事例を分析する

旅行に行く前に「この季節はこのホテルが安い」「このルートだと効率がいい」と口コミを読みますよね。フリーランス（＝会社に属さず自分の力で仕事を受ける人）の世界でも、うまくいっている人のやり方を調べることが一番の近道です。

このレッスンでは、AIチャットツール（＝ChatGPTやClaudeなど、質問すると答えてくれるWebサービス）を使って、高単価案件（＝1件あたりの報酬が高い仕事）の受注事例をたった15分で分析し、あなただけの「勝ちパターン」を作ります。

![分析フロー図](/lesson-assets/atom.ai-freelancer.high-value-case-study/diagram.png)

## 目標を確認する

このレッスンを終えると、あなたは次のことができるようになります。

- 高単価案件の受注事例を3〜5個見つけて、共通点をまとめられる
- AIに分析を頼んで、自分だけの「勝ちパターン」を1つ書き出せる
- 次に自分が狙うべき案件の方向性がわかる

## 前提をそろえる

このレッスンに取り組む前に、次のものを用意してください。

- AIチャットツール（ChatGPT、Claude など）のアカウントを無料で作成済みであること
- Webブラウザー（Chrome や Safari など）が使えること
- メモ帳アプリ（パソコンに最初から入っているものでOK）

プログラミングの経験は一切不要です。すべてAIと会話しながら進めます。

## ステップ1: 事例を集める（5分）

まずは「高単価で受注した人」の生の声を探します。次の3つの場所から、合計3〜5個の事例を集めてください。

### 探す場所

1. **クラウドワークスやランサーズのブログ**：実際に受注した人が手順を書いています
2. **note（＝日本のブログサービス）**で「フリーランス 高単価 受注」と検索
3. **X（旧Twitter）**で「#フリーランス 高単価 受注」と検索

### 良い事例と悪い事例の見分け方

| 判断基準 | ✅ 良い事例 | ❌ 悪い事例 |
|---|---|---|
| 金額 | 「Webサイト制作で50万円」と具体的 | 「たくさん稼げた」だけで金額不明 |
| 手順 | 受注までの流れが書いてある | 「フリーランスは稼げる」という感想だけ |
| 再現性 | 初心者でも真似できそう | 10年のキャリアが前提 |

見つけた事例のURLをメモ帳にコピーしておいてください。

## ステップ2: AIに事例を分析してもらう（5分）

AIチャットツール（ChatGPTやClaude）を開いて、次のメッセージをそのままコピーして送ってください。

```
これからフリーランスの高単価案件の事例をいくつか共有します。
それぞれの事例から、受注できた理由や共通パターンを分析してください。
特に「初心者でも再現できるポイント」に注目してまとめてください。

事例1: [ここにURLまたは記事の要約を貼る]
事例2: [ここにURLまたは記事の要約を貼る]
事例3: [ここにURLまたは記事の要約を貼る]
```

> **ポイント**: URLだけだとAIが読めないことがあります。その場合は記事の本文をコピーして貼り付けてください。ChatGPTの有料版やClaudeならURLを読める場合もあります。

AIからの回答が出たら、さらに次のように聞きます。

```
その共通パターンを、フリーランス初心者が明日から始められる
具体的な行動リストに変換してください。
各行動は「〇〇する」の形で、1つずつ短く書いてください。
```

![AI分析画面の例](/lesson-assets/atom.ai-freelancer.high-value-case-study/screen_capture.png)

### AIへの質問のコツ

- ✅ 良い例：「初心者にはどれが一番始めやすいですか？理由も教えてください」と深掘りする
- ✅ 良い例：「回答を箇条書き5つ以内にまとめてください」と短く頼む
- ❌ 悪い例：AIが長文で答えてきたのに全部読もうとして時間が足りなくなる
- ❌ 悪い例：1回の質問で終わりにして、追加の質問をしない

## ステップ3: 勝ちパターンをまとめる（3分）

AIの回答をもとに、あなただけの「勝ちパターン」を1つ書き出します。次のテンプレート（＝ひな形）をAIチャットにコピーして送ってください。

```
以下のテンプレートを、先ほどの分析結果をもとに埋めてください。
私はフリーランス初心者で、AIツールが使えます。

【私の勝ちパターン】
- 狙う案件ジャンル：_____
- ターゲット（誰に提供するか）：_____
- 提供するもの（何を作るか）：_____
- AIの活用ポイント（どこでAIを使うか）：_____
- 目標単価（1件あたり）：_____
- 最初の1歩（今週中にやること）：_____
```

AIが埋めてくれた内容を読んで、「これなら自分にもできそうだ」と思える内容になっているか確認してください。しっくりこなければ、「もっと簡単にできるものに変えてください」とAIに頼んでOKです。

## ステップ4: 成果物を保存する（2分）

次の内容をメモ帳にまとめて保存してください。これがこのレッスンの成果物（＝完成した作品）です。

1. **集めた事例の一覧**：URL 3〜5個と、各事例の一行まとめ
2. **AIの分析結果**：共通パターンと行動リスト
3. **あなたの勝ちパターン**：テンプレートを埋めたもの

保存したら、画面のスクリーンショット（＝画面の写真）を撮ってください。

- **Macの場合**: `Command + Shift + 3` で画面全体を撮影
- **Windowsの場合**: `Windows + PrintScreen` で画面全体を撮影

## 確認する

次のチェックリストで、レッスンがうまくいったか確認しましょう。すべてにチェックが入れば完了です。

- [ ] 事例を3個以上集めた
- [ ] AIに分析を頼んで、共通パターンを3つ以上出せた
- [ ] 共通パターンを「初心者向けの行動リスト」に変換できた
- [ ] 勝ちパターンのテンプレートを埋められた（空欄なし）
- [ ] 勝ちパターンの「最初の1歩」が今週中にできるレベルの具体性がある
- [ ] スクリーンショットを撮った

## つまずきポイントを知る

| つまずきポイント | 原因 | 対策 |
|---|---|---|
| 事例が見つからない | 検索ワードが合っていない | 「フリーランス 月○万円 事例」と金額を変えて検索する。noteとXの両方で探す |
| AIがURLを読めない | 無料版はURL読み込み非対応の場合がある | 記事の本文をコピーして直接貼り付ける |
| AIの回答が的外れ | 指示があいまい | 「初心者向けに」「具体的な行動で」と条件を追加して聞き直す |
| 勝ちパターンが抽象的すぎる | AIに「具体的に」を伝えていない | 「初心者が明日できる行動」と指定してAIに書き直させる |
| 15分で終わらない | 事例を読み込みすぎている | 事例は要約だけ読み、詳細分析はAIに任せる |
