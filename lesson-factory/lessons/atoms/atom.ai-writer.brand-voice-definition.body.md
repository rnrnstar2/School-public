# ブランドボイスガイドを作成する

あなたがSNSやブログで情報を発信するとき、「この人っぽいな」と読者に感じてもらえる文章の個性――それが**ブランドボイス**です。たとえば料理でいえば、同じカレーでも家庭ごとに「うちの味」がありますよね。文章にも同じように「自分の味」を決めておくと、AIに文章を書いてもらうときにもブレない仕上がりになります。

このレッスンでは、AIとの対話を通じて **ブランドボイスガイド**（＝自分の文章スタイルをまとめた1枚のドキュメント）を15分で完成させます。

## 前提として必要なもの

- ChatGPT・Claude など、AIチャットツールをひとつ使える状態にしておく
- 自分が過去に書いた文章（SNS投稿・ブログ・メールなど）を2〜3個コピーできる状態にしておく

## ブランドボイスとは何かを理解する

ブランドボイスは次の4つの要素で成り立っています。

![ブランドボイス4要素の図](/lesson-assets/atom.ai-writer.brand-voice-definition/diagram.png)

1. **トーン（声のトーン）**: フォーマル？カジュアル？友達に話すような感じ？
2. **語彙（使う言葉の選び方）**: 専門用語を使う？平易な日本語中心？
3. **リズム（文章の長さやテンポ）**: 短い文を連ねる？じっくり説明する？
4. **価値観（大事にしていること）**: 正確さ重視？親しみやすさ重視？

旅行ガイドブックにたとえると、「地球の歩き方」と「ことりっぷ」では同じ観光地を紹介していても、読んだときの印象がまったく違いますよね。それがブランドボイスの違いです。

## 自分の文章をAIに分析してもらう

まず、あなたの過去の文章をAIに読ませて、今の文章スタイルを客観的に把握します。

### ステップ1: 過去の文章を集める

自分が書いた文章を2〜3個コピーしてください。SNS投稿、ブログ記事、仕事のメールなど、なんでもOKです。「自分らしいな」と思えるものを選ぶのがコツです。

### ステップ2: AIに分析を依頼する

以下のプロンプト（＝AIへの指示文）をコピーして、チャット画面に貼り付けてください。`【ここに文章を貼る】` の部分を自分の文章に置き換えます。

```
以下は私が書いた文章です。この文章のスタイルを分析して、
次の4つの観点でまとめてください。

1. トーン（フォーマル〜カジュアルの度合い）
2. 語彙の特徴（よく使う言い回し、避けている表現）
3. リズム（文の長さ、段落の切り方）
4. 価値観（何を大事にして書いているか）

---
【ここに文章を貼る】
```

AIが返してくれた分析結果を読んで、「たしかにそうだな」と思える部分にマーカーを引く気持ちで確認してください。

**良い例**: 過去のブログ記事と、友人へのメッセージの両方を貼って「場面による使い分け」も把握する

**悪い例**: 他人が書いた文章や、テンプレートをそのままコピーした文章を貼る（自分のスタイルが見えなくなります）

## ブランドボイスガイドを生成する

分析結果をもとに、AIにガイドを一気に作ってもらいます。以下のプロンプトを使ってください。

```
先ほどの分析結果をもとに、私のブランドボイスガイドを
以下のフォーマットで作成してください。

## ブランドボイスガイド

### 基本トーン
（1文で表現。例：「友人にコーヒーを飲みながら話すような、温かくカジュアルなトーン」）

### 語彙ルール
- 使う言葉の例: ○○、○○
- 避ける言葉の例: ○○、○○

### 文章のリズム
- 1文の目安: ○○文字以内
- 段落の目安: ○○文ごと

### 大事にする価値観
- ○○
- ○○

### AIへの指示テンプレート
（このガイドに沿って文章を書いてもらうときにAIに渡す定型文）
```

AIが生成したガイドの中で、しっくりこない部分があれば遠慮なく修正を指示してください。たとえば「トーンはもう少しカジュアル寄りにして」「語彙ルールに絵文字の方針も追加して」のように伝えれば、AIが調整してくれます。

![AIとの対話画面例](/lesson-assets/atom.ai-writer.brand-voice-definition/screen_capture.png)

## 完成したガイドを検証する

作ったガイドが本当に使えるかどうか、その場でテストしましょう。

### テスト方法

完成したブランドボイスガイドをAIに渡して、短い文章を1本書いてもらいます。

```
以下のブランドボイスガイドに従って、
「今日のランチにおすすめのお店」について
3〜4文の短い紹介文を書いてください。

---
【ここにブランドボイスガイドを貼る】
```

出てきた文章を読んで、「自分が書きそうな文章になっているか？」をチェックしてください。

- **OK のサイン**: 「自分っぽい！」と思える。友人に見せても違和感がなさそう
- **NG のサイン**: 「なんか堅いな」「自分はこんな言い方しないな」と感じる → ガイドのトーンや語彙ルールを修正して再テスト

## つまずきやすいポイント

| よくある困りごと | 対処法 |
|---|---|
| 自分の文章が見つからない | LINEの長めのメッセージやSNSのコメントでもOKです |
| AIの分析がピンとこない | 「もっと具体的に、私の文章から実際のフレーズを引用して説明して」と追加で聞いてみてください |
| ガイドが長くなりすぎた | 「このガイドを箇条書き中心で半分の長さに要約して」と頼んでください |
| テストの文章が自分っぽくない | ガイドのどの部分がズレているか指摘して「基本トーンをもっと〇〇に修正して」と伝えてください |

## まとめ

このレッスンでは、AIとの3回のやりとりで **自分専用のブランドボイスガイド** を作りました。

1. 過去の文章をAIに分析してもらう
2. 分析結果をもとにガイドを生成する
3. テスト文を書かせて検証する

完成したガイドは、今後AIに文章を書いてもらうときに毎回渡すことで、「自分らしい」文章を安定して生み出せるようになります。ガイドはいつでも修正・更新できるので、使いながら育てていってください。
