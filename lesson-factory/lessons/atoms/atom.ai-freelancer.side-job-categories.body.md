# AI副業の案件カテゴリを把握する

レストランのメニューを見ずに注文するのは不安ですよね。AI副業でも同じです。まずは「どんな仕事があるか」を知ることで、あなたに合う仕事を見つけやすくなります。

このレッスンでは、AIを使った副業の主な6つのカテゴリを整理し、あなたがどれから始めるのがよいかを見極めます。最後に、自分だけのカテゴリ整理メモを作って完了です。

![AI副業カテゴリマップ](/lesson-assets/atom.ai-freelancer.side-job-categories/diagram.png)

## 前提を確認する

このレッスンに特別なツールのインストールは必要ありません。以下の準備だけしてください。

- インターネットに接続されたパソコンまたはスマートフォン
- ChatGPT または Claude が使える状態（無料プランでOKです）
- クラウドワークスやランサーズなどの「クラウドソーシング（＝インターネット上で仕事を依頼・受注できるサービス）」のサイトを開けること（アカウントがなくても検索だけなら見られます）

## AI副業の主なカテゴリを知る

AI副業の案件は、大きく分けると次の6つのカテゴリに分類できます。

### 1. 文章作成・ライティング

ChatGPTやClaudeなどのAIに指示を出して、ブログ記事、商品説明文、メール文面などを作る仕事です。最も案件数が多く、初心者が始めやすいカテゴリです。

- **良い例**: 「初夏の挨拶メールを5パターン作成して」とAIに指示し、できた文章を自分で確認・修正してから納品する
- **悪い例**: AIが書いた文章をそのままコピペして納品する（事実と違う内容や不自然な表現が含まれるリスクがあります）

**AIへのプロンプト例**:
> あなたはフリーランスのライターです。「夏のキャンプ初心者ガイド」というタイトルで、500文字程度のブログ記事を書いてください。読者は20代〜30代の社会人です。

### 2. 画像生成

MidjourneyやDALL-Eなどの「画像生成AI（＝言葉で指示すると自動的に画像を作ってくれるツール）」を使って、SNS用のバナーやイラスト、商品画像を作る仕事です。

- **良い例**: クライアントの要望を聞き取り、何度かAIで生成した画像から最適なものを選び、細部を調整して納品する
- **悪い例**: AIに1回だけ指示を出して、出てきた最初の画像をそのまま納品する

### 3. データ入力・アノテーション（＝データにラベル付けする作業）

AIを賢くするための学習データを作る仕事です。たとえば画像に「これは猫」「これは犬」とタグを付けたり、文章のニュアンスを判定したりします。特別なスキルがなくても始められるので、最初の一歩として人気があります。

### 4. チャットボット構築・運用

ChatGPTの技術を使って、企業の問い合わせ窓口を自動で返答する仕組みを作る仕事です。最近はプログラミング不要で設定できるツール（Dify、Botpressなど）も増えているため、非エンジニアでもチャレンジできます。

### 5. AI導入の提案・相談

「この業務にAIをどう活かせるか」を企業に提案するコンサルティング的な仕事です。AIツールを複数使いこなせるようになってから挑戦しやすいカテゴリで、単価も高めです。

### 6. 翻訳・語学サポート

AI翻訳ツール（DeepLやChatGPTなど）を使いながら、高品質な翻訳成果物を作る仕事です。AIだけでは不十分な表現を人間が校正（＝内容を確認して直すこと）するのがポイントです。語学力がある方には特におすすめです。

## AIに相談してカテゴリを絞り込む

すべてのカテゴリを一度に始める必要はありません。ここでは、AIを使って自分に合うカテゴリを見つけてみましょう。

ChatGPTやClaudeに、次のようなプロンプトを送ってみてください。

**AIへのプロンプト例**:
> 私はプログラミング経験がない会社員で、AIを使った副業を始めたいです。
> 以下の情報をもとに、AI副業の6つのカテゴリ（文章作成、画像生成、データ入力、チャットボット構築、AI導入提案、翻訳）の中から、私に合いそうなものを2つ提案してください。理由も教えてください。
>
> - 普段の仕事: （例：事務職でメールや報告書をよく書く）
> - 得意なこと: （例：文章を書くのが好き）
> - 使える時間: （例：平日の夜に1〜2時間）
> - 気になるカテゴリ: （例：文章作成が気になる）

AIの返答を読んで、「なるほど」と思えたカテゴリをメモしておきましょう。

![カテゴリ選びの流れ](/lesson-assets/atom.ai-freelancer.side-job-categories/diagram.2.png)

### 選び方のポイント

1. **得意なことを思い出す** — 文章を書くのが好きなら「文章作成」、デザインに興味があるなら「画像生成」から
2. **初期費用を確認する** — 有料ツールが必要なカテゴリは、まず無料プランで試す
3. **案件数をチェックする** — クラウドソーシングサイトでどのカテゴリの案件が多いかを実際に見る

- **良い例**: 「普段SNSの投稿をよく書いているから、まずは文章作成カテゴリの案件を3件探してみよう」
- **悪い例**: 「全部やらないと稼げない気がするから、全部同時に始めよう」（一度に広げすぎると続かないことが多いです）

## 案件を実際に探す

クラウドソーシングサイトで検索して、どんな案件が実際にあるかを確認しましょう。

![案件検索画面の例](/lesson-assets/atom.ai-freelancer.side-job-categories/screen_capture.png)

1. クラウドソーシングサイト（クラウドワークスやランサーズなど）を開く
2. 検索窓に「AI」または「ChatGPT」と入力して検索する
3. 出てきた案件を6つのカテゴリに振り分けてみる
4. 自分がやってみたくなる案件を3つピックアップする

見つけた案件について、AIに「この案件は初心者でもできそうですか？」と聞いてみるのもおすすめです。

**AIへのプロンプト例**:
> 以下のクラウドソーシング案件を見つけました。AI副業初心者の私でも対応できそうか、難易度を3段階（やさしい・ふつう・むずかしい）で教えてください。
>
> 案件タイトル: （ここに見つけた案件のタイトルを貼る）
> 案件内容: （ここに案件の説明を貼る）

## 成果物を作る

このレッスンの成果物は、次の内容を含むメモです。メモ帳やGoogleドキュメントなど、使い慣れたツールで構いません。

1. **6つのカテゴリの名前と、それぞれ1文の説明**
2. **自分が興味を持ったカテゴリ（1〜2つ）とその理由**（AIに相談した結果も含めてOK）
3. **実際に見つけた案件のタイトルを3つ**（見つからなかった場合は、検索で使ったキーワードを書く）

書けたら保存して、完成したメモのスクリーンショット（＝画面の写真）を撮っておきましょう。

## つまずきポイントを押さえる

| つまずき | 解決策 |
|---|---|
| 「どのカテゴリも難しそう」 | まずは文章作成カテゴリが一番始めやすいです。ChatGPTに「〇〇についてのブログ記事を書いて」と伝えるだけで試せます |
| 「案件が見つからない」 | 検索キーワードを「AI」以外にも「ライティング」「画像生成」「アノテーション」「ChatGPT」などに変えてみましょう |
| 「カテゴリとツールの対応がわからない」 | AIに「〇〇カテゴリの副業で使う代表的なツールを3つ教えて」と聞いてみましょう |
| 「AIの返答が期待と違う」 | プロンプトに自分の状況（経験、使える時間、興味）を具体的に書き足すと、精度が上がります |

## 確認する

以下の3つがすべてYesになっていれば、このレッスンは完了です。

- [ ] 6つのカテゴリの名前と説明をメモに書けた
- [ ] 自分が興味を持つカテゴリを1〜2つ選び、理由も書けた
- [ ] クラウドソーシングサイトで案件を検索し、結果をメモに記録した

## 次のステップへ進む

- カテゴリを選べたあなたは、次のステップで実際にAIツールの準備を進めます
- 迷ったら「文章作成」から始めてみてください。案件も多く、成功体験を積みやすいカテゴリです
