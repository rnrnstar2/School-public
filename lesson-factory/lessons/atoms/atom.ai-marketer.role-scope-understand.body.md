# AIマーケターの役割を理解する

レストランを想像してみてください。
シェフがいて、ウェイターがいて、メニューを作る人がいます。
でも、どんな料理を作ればお客さんが喜ぶかを考えて、お店の宣伝を計画する人も必要ですよね。
それが「マーケター」という役割です。

AIマーケターは、この宣伝の計画を **AIツール（＝人工知能を使った便利なソフト）の力を借りて進める人** です。
プログラミングは書きません。AIと一緒に考えるのが仕事です。

このレッスンでは、AIマーケターが **何をして、何をしないか** をはっきりさせ、最後にあなた自身の言葉で役割をまとめてもらいます。

![AIマーケターの役割範囲](/lesson-assets/atom.ai-marketer.role-scope-understand/diagram.png)

## AIマーケターが「やること」を知る

AIマーケターは、主に次の3つを行います。

### 1. 市場調査（＝どんな人に、何が売れそうかを調べること）

AIに質問して、ざっくりした傾向をつかみます。

**具体的なプロンプト例（ChatGPTやClaudeにそのまま貼ってOK）：**

```
あなたはマーケティングリサーチャーです。
20代女性が2024年に注目している美容トレンドを5つ挙げて、
それぞれ「なぜ人気か」を1文で教えてください。
```

このように聞くと、AIが調査レポートのたたき台を作ってくれます。あなたはその中から「自分の商品に関係ありそうなもの」を選ぶだけです。

### 2. コンテンツ作り（＝ブログやSNSの投稿文を考えること）

AIにキャッチコピーや記事のたたき台を作ってもらい、それを自分で整えます。

**具体的なプロンプト例：**

```
以下の商品について、Instagram投稿用のキャッチコピーを3パターン作ってください。
- 商品名: ○○ハンドクリーム
- ターゲット: 20代の乾燥肌に悩む女性
- トーン: 親しみやすく、押し売り感なし
```

### 3. 効果を振り返る（＝やってみた結果を数字で確認すること）

アクセス数やクリック数をAIにまとめてもらい、次の手を考えます。

**具体的なプロンプト例：**

```
以下のSNS投稿の結果データを分析して、次に何を改善すべきか3つ提案してください。
- 投稿A: いいね120、クリック15
- 投稿B: いいね45、クリック32
- 投稿C: いいね200、クリック8
```

### 良い例
- 「AIに3パターンのキャッチコピーを出してもらって、一番しっくりくるものを選んで手直しする」
- 「AIの調査結果を読んで、自分のビジネスに関係あるポイントだけ抜き出す」

### 悪い例
- 「AIが出した文章をそのままコピペして終わらせる」（自分の考えが入っていない）
- 「AIに丸投げして結果も確認しない」（数字を見ないと改善できない）

---

## AIマーケターが「やらないこと」を確認する

役割を理解するうえで、「やらないこと」も大切です。

- **コードを書かない** → プログラミングは別の役割の人が担当します。AIマーケターはAIに日本語で指示を出します
- **デザインを直接いじらない** → 画像の編集やレイアウト調整は、テンプレート（＝ひな形）を使うか、デザイナーに頼みます
- **データベース（＝顧客情報の保管庫）を操作しない** → お客さんの情報は、すでに用意された管理画面から見ます

「料理でいうと、AIマーケターはメニューを考える人。フライパンを持つのは別の人」と覚えておくと分かりやすいです。

## AIツールで自分の役割を整理する

それでは、AIツールを使って「AIマーケターの役割」を自分の言葉でまとめてみましょう。

### 手順

**ステップ1: AIに壁打ちする**

ChatGPT（https://chat.openai.com）またはClaude（https://claude.ai）を開いて、次のプロンプトを貼り付けてください。

```
私はこれからAIマーケターとして働きたいと思っています。
AIマーケターの主な仕事内容を3つと、やらなくていいことを2つ教えてください。
初心者にもわかるように、それぞれ1〜2文で説明してください。
```

**ステップ2: AIの回答を元に自分の言葉で書き直す**

スマホやパソコンでメモアプリ（Googleドキュメント、Notion、メモ帳など何でもOK）を開いて、次の3つについてそれぞれ2〜3文で書いてください。

1. AIマーケターが **やること**（AIの回答を参考に、自分の理解で書き直す）
2. AIマーケターが **やらないこと**
3. 自分が **一番やってみたいこと**

![メモアプリでまとめを書いている画面例](/lesson-assets/atom.ai-marketer.role-scope-understand/screen_capture.png)

**ステップ3: スクリーンショットを撮る**

書き終えたら、画面の写真（スクリーンショット）を撮っておきましょう。これがこのレッスンの成果物になります。

- iPhone: 電源ボタン＋音量上ボタンを同時押し
- Android: 電源ボタン＋音量下ボタンを同時押し
- パソコン（Mac）: Command＋Shift＋3
- パソコン（Windows）: Windowsキー＋PrintScreen

### 良い例
> AIマーケターは市場調査やコンテンツ作りをAIと一緒に進める。コードは書かないし、デザインの細かい作業もしない。私はキャッチコピーをAIと一緒に考える作業をやってみたい。

### 悪い例
> AIマーケターはAIを使って仕事をする。（←「何をするか」が具体的に書かれていない）

---

## 結果を確認する

書き終えたら、次の3つをチェックしましょう。すべてクリアすればこのレッスンは完了です。

- [ ] 「やること」に3つの具体的な仕事内容が書かれているか（市場調査・コンテンツ作り・効果振り返りなど）
- [ ] 「やらないこと」が1つ以上あり、プログラミングやデザイン作業が含まれていないか
- [ ] 「やってみたいこと」がAIマーケターの役割の範囲内か（コードを書く、のような範囲外のものになっていないか）

## つまずきやすいポイント

| つまずき | 対処法 |
|---|---|
| 「AIマーケターと普通のマーケターの違いが分からない」 | 普通のマーケターは手作業で調べたり書いたりすることが多いですが、AIマーケターはAIに下書きや分析をお願いして時短します。「自転車と電動自転車」のような関係です |
| 「やらないことが多すぎて不安になる」 | 大丈夫です。やらないことは他の役割の人が担当します。あなたは「考えて指示を出す」ことに集中してください |
| 「何を書けばいいか分からない」 | まずは手順の「ステップ1」でAIに聞いてみてください。AIの回答をヒントに、自分の言葉で書き換えるだけでOKです |
| 「AIの回答が難しい言葉ばかりで理解できない」 | 「もっと簡単な言葉で、中学生にもわかるように説明し直してください」とAIに追加で聞いてみてください。何度聞き直しても大丈夫です |
