# 1商品の集客設計をまとめる

あなたがお店を開いたと想像してみてください。
どんなに美味しいケーキを作っても、のれんの外に「ここはケーキ屋です」と書かなければ、お客さんは来てくれませんよね。

商品やサービスを売るときも同じです。
**「誰に」「どう伝えて」「どう興味を持ってもらうか」をまとめたもの**が集客設計です。

この Atom では、AI チャットツール（ChatGPT や Claude など）を使って **1つの商品の集客設計書** を15分で仕上げます。
完成すると「自分の商品を、誰に・どうやって届けるか」が1枚の書類にまとまった状態になります。

---

## 商品を1つ決める

まずは集客設計の対象となる商品を **1つだけ** 選びます。

> なぜ1つなのか？ 商品を1つに絞ると伝えるメッセージがブレず、初めてでも迷わないからです。旅行の計画も「まず行き先を1か所決める」と話が進みやすいのと同じです。

- **良い例**: 「自家製レモンケーキ（1個400円）」── 名前と価格がはっきりしている
- **良い例**: 「SNS運用代行プラン（月額5万円）」── サービス名と金額が具体的
- **悪い例**: 「ケーキ全般」── 範囲が広すぎて誰に何を伝えるか決められない
- **悪い例**: 「コンサル」── どんな相談に乗るのかわからない

決めたら、商品名・価格・特徴の3点をメモ帳に書き出しておきます。この3点が後のステップすべての土台になります。

---

## ターゲット客層を言葉にする

次に「誰に買ってほしいか」を具体化します。
AI に考える材料を渡して、候補を出してもらいましょう。

### AI への聞き方

ChatGPT や Claude を開いて、以下のプロンプト（＝AI への指示文）をコピペしてください。`【 】` の部分だけ、あなたの商品に書き換えます。

```
次の商品について、最も買ってくれそうな客層を3パターン教えてください。
それぞれ「年齢層・性別・悩みや欲求」を含めてください。

商品名：【自家製レモンケーキ】
価格：【400円】
特徴：【地元の有機レモンを使用、毎朝手作り】
```

### AI の回答を読んで1つ選ぶ

AI は3パターンほど候補を出してくれます。読んで「うちのお客さんはこれだ！」と一番しっくりくるものを1つ選びましょう。

- **良い例**: 「近隣に住む30代の母親。子どものおやつに安心な素材のものを選びたい」── 年齢・性別・悩みが入っている
- **悪い例**: 「女性」── 広すぎて、何を伝えればいいかわからない

> **迷ったら？** 「この人が目の前にいたら、どう話しかけるか想像できる」かどうかで判断してください。想像できるなら十分具体的です。

## 集客チャネルを選ぶ

集客チャネル（＝お客さんに情報を届ける手段）を選びます。
「その人たちが普段どこを見ているか」を基準に考えましょう。

| チャネル | 向いている客層の例 | 始めやすさ |
|---|---|---|
| Instagram | 若い世代、写真で判断する人 | ★★★ すぐ始められる |
| LINE公式アカウント | 地域のリピーター、常連候補 | ★★☆ 初期設定が少し必要 |
| Googleビジネスプロフィール | 「近くの○○」と検索する人 | ★★☆ 登録に数日かかる場合あり |

**まずは1つだけ選びましょう。** 3つ同時に始めると手が回らなくなります。

選んだら、AI にそのチャネルでの集客アイデアを出してもらいます。

```
【近隣の30代母親】をターゲットに、【Instagram】で【自家製レモンケーキ】を
集客するアイデアを5つ教えてください。
条件：
- 初心者でもすぐ始められるもの
- 無料でできるもの優先
- 各アイデアに「具体的な最初の一歩」を添えて
```

> **ポイント**: AI への指示に「条件」をつけると、より実用的な回答が返ってきます。「初心者向け」「無料」「具体的な一歩」のような条件が効果的です。

## AI に設計書をまとめてもらう

ここまでの情報をすべて AI に渡して、集客設計書（＝「誰に・何を・どうやって届けるか」をまとめた企画書）を仕上げてもらいます。

```
以下の情報をまとめて、1商品の集客設計書を作ってください。

- 商品名：【自家製レモンケーキ】
- 価格：【400円】
- ターゲット：【近隣の30代母親。子どものおやつに安心なものを選びたい】
- メインチャネル：【Instagram】
- 集客アイデア：
  【ここにAIの前の回答を貼り付け】

以下の見出し構成で出力してください：
1. 商品概要（商品名・価格・特徴を3行で）
2. ターゲット客層（誰に届けるか）
3. 集客チャネルと施策（何をするか、リスト形式で）
4. 最初の1週間でやること（3つまでに絞る）
```

### 出てきた設計書を確認する

AI が返してきた文章は「たたき台」です。以下の視点でチェックしてください。

- **事実と違う部分はないか？**（AI は存在しない機能やサービスを書くことがあります）
- **自分の言葉で説明できるか？**（わからない表現があれば、AI に「もっとわかりやすく言い換えて」と頼む）
- **実際にやれそうか？**（「毎日3投稿」など現実的でないものは修正する）

問題があった場合は、気になる箇所をコピーして AI に以下のように聞きましょう。

```
以下の部分を修正してください。
【修正したい部分を貼り付け】
理由：【なぜ変えたいか。例：毎日3投稿は現実的でないので週2回にしたい】
```

## 成果物を保存して確認する

完成した設計書をテキストファイルに保存します。

1. AI の回答をすべて選択してコピーする（`Ctrl+C` / `Cmd+C`）
2. メモ帳やGoogleドキュメントに貼り付ける
3. ファイル名を「集客設計_商品名_日付.txt」にして保存する（例：`集客設計_レモンケーキ_20260413.txt`）
4. 保存したファイルを開き、スクリーンショット（＝画面の写真）を撮る

### 完成チェックリスト

以下の4項目すべてに ✓ がつけば完成です。

- [ ] **商品名** が1つに絞られている（「ケーキ全般」のような曖昧な書き方になっていない）
- [ ] **ターゲット** に「年齢層・性別・悩みや欲求」のうち少なくとも2つが書かれている
- [ ] **集客チャネル** が1つ以上選ばれ、具体的な施策が書かれている
- [ ] **最初にやること** が3つ以内で明記されている（何から手をつけるかわかる状態）

## つまずきやすいポイントと対策

| つまずき | なぜ起きるか | 対策 |
|---|---|---|
| ターゲットが「20〜60代の男女」のように広すぎる | 絞ると機会を逃す気がして怖い | まず1人の「理想のお客さん」を想像する。その人に刺さればOK |
| AI の回答をそのままコピーして終わる | AI が正しいと思い込む | 必ず「自分の商品で本当にこれができるか？」を考える。事実と違えば修正する |
| チャネルを3つ同時に始めようとする | 「やらないと不安」という気持ち | まず1つだけ。1週間続いたら2つ目を検討する |
| AI にうまく伝わらず的外れな回答が返ってくる | 指示（プロンプト）に具体情報が少ない | 「商品名・価格・特徴・ターゲット」の4つを必ず含めて指示する |
| 15分で終わらない | 各ステップで悩みすぎる | 完璧を目指さない。「70点のたたき台」をまず作り、後から直す |
