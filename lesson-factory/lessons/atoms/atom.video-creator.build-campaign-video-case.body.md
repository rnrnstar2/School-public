# プロモ動画案件を完成させる

料理でいうと、プロモ動画を作るのは「お弁当を完成させる」のに似ています。主菜（メインメッセージ）が決まったら、副菜（テロップやBGM）を組み合わせて盛り付けるだけ。このレッスンでは、AIツールをフル活用して、プロモ動画案件を15分以内に仕上げる一連の流れを体験します。

最終ゴールは「完成した動画のスクリーンショットを撮って保存する」ことです。レッスンが終わるころには、次の案件が来ても同じ手順で自力で仕上げられるようになります。

![案件全体フロー](/lesson-assets/atom.video-creator.build-campaign-video-case/diagram.png)

## 前提を確認する

このレッスンを始める前に、次の準備が整っていることを確認してください。

- **AIチャットツール**を使える状態にしておく
  - ChatGPT（無料版でOK）、Claude、Gemini など、どれか1つ
  - 構成案やテロップの文案を考えてもらうために使います
- **AI動画生成ツール**（＝AIを使って動画や画像を自動で作るサービス）のアカウントを持っている
  - 例: Runway、Pika、Vrew など
  - どれを選べばいいか迷ったら、まずは無料プランがある Vrew がおすすめです
- インターネットにつながったパソコンがある
- マイク（ナレーションを録音する場合のみ。なくてもOK）

> 特別なソフトのインストールは不要です。ブラウザだけで進められます。

## 案件の要件を整理する

まずは「何を・誰に・どう伝えるか」を明確にします。料理のレシピを決める工程です。ここを曖昧にすると、あとで「思ってたのと違う」と作り直しになります。

やること:

1. クライアントやチームからもらった要件メモを手元に置く
2. 次の4項目を書き出す
   - **目的**: この動画で何を伝えたいか（例: 新商品の魅力を伝える）
   - **対象**: 誰に見てほしいか（例: 20〜30代の女性）
   - **長さ**: 動画の秒数（例: 30秒）
   - **雰囲気**: どんなトーンにしたいか（例: 明るくポップ、落ち着いて高級感）

> **AIに手伝ってもらうコツ**: 要件が曖昧なときは、AIチャットに「〇〇の商品プロモ動画を作ります。ターゲットや雰囲気を整理するために質問してください」と聞くと、考えるべきポイントを教えてくれます。

### 良い例・悪い例

- ✅ **良い例**: 「新発売のフルーツティーを、10代後半〜20代の学生に向けて、15秒で爽やかさを伝える。明るくポップな雰囲気で」
- ❌ **悪い例**: 「いい感じの動画をお願いします」（何を・誰に・どう、が全部抜けている）

## AIで構成案（シナリオ）を作る

要件が決まったら、AIに構成案を書いてもらいます。AIは「レシピを提案してくれる料理アシスタント」のような存在です。ゼロから考えるのではなく、AIが出してくれた案をたたき台にして選ぶのがポイントです。

### ステップ1: AIチャットに構成案を依頼する

以下のプロンプト（＝AIへの指示文）をコピーして、ChatGPTやClaudeに貼り付けてください。太字の部分をあなたの案件に合わせて書き換えます。

```
以下の条件で、プロモ動画の構成案を3案作ってください。
各案は、シーンごとに「秒数・映像の説明・テロップ文」を表にしてください。

- 目的: **新商品のフルーツティーの爽やかさを伝える**
- 対象: **10代後半〜20代の学生**
- 動画の長さ: **15秒**
- 雰囲気: **明るくポップ**
```

### ステップ2: 構成案を選ぶ

出てきた3案から一番しっくりくるものを1つ選びます。選ぶときのポイント:

- 各シーンに具体的な秒数が割り当てられているか
- 合計秒数が要件の長さと一致しているか
- テロップの文言がターゲットに合っているか

### 良い例・悪い例

- ✅ **良い例**: 「シーン1: 0-3秒 フルーツが弾ける映像 テロップ『新登場！』」のように秒数・映像・テロップが揃った構成を選ぶ
- ❌ **悪い例**: 「なんとなく良さそう」で選んで、秒数が合計25秒になっていることに気づかない

## 動画素材をAIで生成する

構成案に沿って、シーンごとに画像や動画の素材を作ります。

### ステップ1: シーンの説明をAIに伝える

選んだ構成案の各シーンについて、AI動画生成ツールに説明を入力します。以下のテンプレートを使うと、AIが正確に素材を作りやすくなります。

```
以下のシーンの動画素材を生成してください。
- 場面: 明るいカフェのテラス席
- 登場するもの: フルーツティーのグラス、氷、レモンスライス
- 動き: グラスに炭酸水を注ぐ、泡が立つ
- 画面の雰囲気: 自然光、暖かい色合い
- アスペクト比: 16:9（横長）
```

### ステップ2: 生成結果を確認する

生成された素材を見て、イメージに合っているか確認します。合わなければ説明を修正してもう一度生成します。

![ツール画面の確認](/lesson-assets/atom.video-creator.build-campaign-video-case/screen_capture.png)

> **コツ**: 「いつ・どこで・誰が（何が）・何をしている」まで具体的に書くと、思い通りの素材が出やすくなります。「きれいな映像」のような抽象的な指示はNGです。

### 良い例・悪い例

- ✅ **良い例**: 「白い背景のスタジオで、赤いフルーツティーのグラスにストローが刺さっている。真上からのアングル」
- ❌ **悪い例**: 「フルーツティーの映像」（どんな画角・背景・雰囲気か伝わらない）

## 動画を組み立てる

素材が揃ったら、動画編集ツールで組み立てます。ここは「お弁当箱に料理を詰める」イメージです。

1. 素材をタイムライン（＝動画の時間軸に沿って素材を並べる場所）に、構成案の順番通りに配置する
2. 各シーンの長さを構成案の秒数に合わせてトリミング（＝不要な部分を切り取ること）する
3. シーンの切り替わりにテロップ（＝画面に表示される文字）を追加する
4. BGM（＝背景音楽）を乗せる

> **AIにテロップ文を仕上げてもらう**: テロップの言い回しに迷ったら、AIチャットに「以下のテロップ文を、〇〇向けにキャッチーに言い換えてください」と頼むと、複数の候補を出してくれます。

> AI動画生成ツールに編集機能があれば、そのまま使えます。なければ、CapCut（無料の動画編集アプリ）などで組み立ててください。

## 完成した動画を確認する

最後に、次の5つのポイントをチェックします。すべて「はい」になるまで修正してください。

- [ ] 動画の長さは要件通りか（±1秒以内）
- [ ] メインメッセージが最初の3秒で伝わるか
- [ ] テロップに誤字・脱字がないか
- [ ] 音声と映像のタイミングが合っているか
- [ ] 対象（ターゲット）が見て違和感のない雰囲気か

すべてクリアしたら完成です！動画の書き出し画面、または再生画面のスクリーンショット（＝画面の写真）を撮って、成果物として保存しましょう。

## つまずきポイントと対策

| つまずき | よくある原因 | 対策 |
|---|---|---|
| AIの生成結果がイメージと違う | 指示が「きれいに」「いい感じに」など抽象的 | 「いつ・どこで・何が・何をしている」を入れる |
| 動画の合計秒数が合わない | 構成案で各シーンの秒数を決めていない | ステップ2で秒数入りの構成を選ぶ |
| テロップが読みにくい | 文字が小さい・表示時間が短い | 1テロップ3秒以上表示、文字サイズを大きめに |
| 素材の画角がバラバラ | シーンごとにアスペクト比を指定していない | プロンプトに「16:9」などアスペクト比を必ず入れる |
| 何回やってもAIが良い素材を出さない | 1回で完璧を求めている | まず70点の素材を集めて組み立て、気になる箇所だけ再生成する |
