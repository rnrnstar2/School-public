# 危ない表現を検出して修正する

## このAtomでできるようになること

マーケティングの文章にひそむ「危ない表現」を、AIに見つけてもらい、安全な書き方に直せるようになります。料理のレシピを味見してもらうように、AIに「法律のチェック係」をお願いする方法を身につけましょう。所要時間は約15分です。

## たとえ話で理解する

レストランのメニューを想像してください。「このラーメンを食べると必ず痩せます」と書いてあったら、ちょっと怪しいですよね。でも「カロリー控えめで、健康的な食事の一部にお使いいただけます」なら納得しやすいはず。

マーケティングの文章でも同じことが起きています。勢いよく書いた言葉が、法律やルールに引っかかる――最悪の場合、行政から指導や処分を受ける可能性もあります。このAtomでは、AIに「メニューのチェック係」をしてもらう方法を学びます。

## 前提を確認する

始める前に、次の2つを用意してください。

- **AIチャットツール**: 次のどれか1つがブラウザで開ける状態であること
  - ChatGPT（chat.openai.com）
  - Claude（claude.ai）
  - Cursor のチャット機能
- **チェックしたい文章**: あなたが書いた、またはこれから公開予定のマーケティング文章を1つ以上

> まだ文章がない場合は、下の「練習用サンプル」をそのまま使ってください。

### 練習用サンプル

```
新発売のサプリメント「ヘルスブースト」は、飲むだけで体脂肪が30%減少！
医師も推薦する最強の健康食品です。
他社製品より確実に効果があり、リピート率No.1を誇ります。
今なら初回限定で80%OFF！この機会を逃すと二度と手に入りません。
```

## 「危ない表現」とは何か知る

日本の広告に関係する法律は、主に2つあります。

- **景品表示法**（けいひんひょうじほう）＝お客さんを誤解させるような表示を禁止する法律です。「一番」「No.1」「他社より優れている」といった根拠のない比較表現が対象になります
- **薬機法**（やっきほう）＝健康食品や化粧品について、許可なく医療的な効果をうたうことを禁止する法律です。「治る」「痩せる」「若返る」などの断言が対象になります

これらに触れる表現をまとめて「危ない表現」と呼びます。違反すると、会社名が公表されたり、課徴金（＝罰金のようなもの）を科されたりします。

### 良い例と悪い例を見比べる

| パターン | 悪い例（危ない） | 良い例（安全） | どの法律？ |
|---|---|---|---|
| 効果の断言 | 「必ず美白になります」 | 「美白成分※を配合し、透明感をサポートします」 | 薬機法 |
| 比較表現 | 「他社より一番安い」 | 「当社調べで類似製品と同等の価格帯です」 | 景品表示法 |
| 医療的な表現 | 「肩こりが治ります」 | 「肩の凝りを和らげる設計です（※個人の感想）」 | 薬機法 |
| 限定煽り | 「二度と手に入りません」 | 「初回限定キャンペーン中です」 | 景品表示法 |

## AIに文章をチェックしてもらう

AIチャットツールを開き、次の**プロンプト**（＝AIへの指示文）をコピーして貼り付けます。`---` の下に、あなたのマーケティング文章を貼りましょう。

### ステップ1: プロンプトをコピーする

```
あなたは日本の広告法務の専門家です。
以下のマーケティング文章を、次の観点からチェックしてください。

1. 景品表示法（優良誤認・有利誤認）に該当しうる表現
2. 薬機法に該当しうる表現（健康・美容・医療系の効果効能の断言）
3. その他、消費者に誤解を与えうる表現

各項目について以下の形式で回答してください：
- 該当箇所: 「原文のまま引用」
- リスクレベル: 高 / 中 / 低
- 理由: なぜ危ないのか1-2文で
- 修正案: 安全な言い換え

---
（ここにあなたの文章を貼り付ける）
```

このプロンプトが先ほどの「1行版」より優れている理由は、AIに**役割**（広告法務の専門家）と**出力フォーマット**（該当箇所・リスクレベル・理由・修正案）を指定しているからです。AIは具体的な指示があるほど正確に答えてくれます。

### ステップ2: 文章を貼って送信する

`---` の下に、練習用サンプルまたはあなた自身の文章を貼り付けて、送信ボタンを押します。

![AIチェック画面](/lesson-assets/atom.ai-marketer.claim-risks-correct/screen_capture.png)

### ステップ3: もっと深掘りする（オプション）

AIの回答を読んで、気になる箇所があれば追加で質問しましょう。便利な追加プロンプトを3つ紹介します。

**グレーゾーンを洗い出す**:
```
上記に加えて、グレーゾーン（違法ではないが行政指導のリスクがある）の表現も指摘してください。
```

**業界別ルールを確認する**:
```
この文章が化粧品（または健康食品、不動産、金融商品）の広告の場合、業界固有のガイドラインに照らして追加のリスクはありますか？
```

**修正案を自然な日本語に仕上げる**:
```
修正案をもう少し自然で魅力的な表現に書き直してください。安全性は保ったまま、読み手が「使ってみたい」と思える文章にしてください。
```

## 結果を読み取って修正する

AIからの回答は、リスクレベルで優先度をつけて対応しましょう。

| リスクレベル | 意味 | 対応 |
|---|---|---|
| **高** | 法令違反の可能性が高い | **必ず修正する**。AIの修正案をベースに書き換える |
| **中** | グレーゾーン。指導を受ける可能性あり | **できる限り修正する**。判断に迷ったら法務担当や専門家に相談 |
| **低** | 直ちに問題はないが改善の余地あり | 余裕があれば修正する |

### 良い修正の進め方

1. まず「高リスク」を全て修正する
2. 次に「中リスク」に取り組む
3. 修正案が不自然なら、AIの案をヒントにあなた自身の言葉で書き直す
4. 修正後の文章をもう一度AIに送って再チェックする

### 良い修正の例

- **修正前**: 「効果が証明されています」
- **修正後**: 「ご愛用者の声をご紹介します（※個人の感想であり、効果を保証するものではありません）」
- **修正の理由**: 「証明」は科学的根拠が必要。個人の感想として紹介し、注意書きを添える

### 悪い修正の例

- **修正前**: 「効果が証明されています」
- **悪い修正**: 「効果が科学的に証明されています」（→ さらに危険度が上がっている！根拠なく「科学的」を足しただけ）
- **悪い修正**: 「一番おすすめです」→「当店が一番おすすめです」（→「一番」がそのまま残っていて景品表示法リスクあり）

## 成果物をまとめる

元の文章と修正後の文章を並べて、1つのドキュメント（＝メモやファイル）にまとめます。これがこのAtomの**成果物**（＝あなたが作るもの）です。

以下のテンプレートをコピーして使ってください。

```
# 広告表現チェック結果

- チェック日: 2026年○月○日
- 対象: （文章のタイトルや用途を書く）
- 使用AIツール: （ChatGPT / Claude / Cursor）

## チェック結果

### 修正1
- 修正前: 「（元の表現）」
- 修正後: 「（安全な表現）」
- 理由: （なぜ危ないのか）
- リスクレベル: 高 / 中 / 低

### 修正2
（同じ形式で繰り返す）

## 修正後の全文
（修正を反映した完成版をここに貼る）
```

## 検証する

成果物が次の条件を満たしているか、1つずつチェックしましょう。

- [ ] 元の文章と修正後の文章が並べて書かれている
- [ ] 修正箇所ごとに理由が添えられている
- [ ] 「必ず〜」「〜が治る」「No.1」などの断言・最上級表現が修正後に残っていない
- [ ] 修正後の文章を再度AIに送り、「高リスク」の指摘が0件になっている
- [ ] 第三者が読んでも不自然でない、自然な日本語になっている

> 全てにチェックが入ったら、このAtomは完了です！

## つまずきポイントに対処する

| つまずき | 対処法 |
|---|---|
| AIが「問題なし」と答えるが不安 | 「グレーゾーンの表現も含めて指摘してください」と追加で依頼する |
| AIの修正案が固くて不自然 | 修正案はヒントとして使い、あなた自身の言葉で書き直す。その後再度AIにチェックを依頼する |
| どの法律に引っかかるのか分からない | 「どの法律のどの条項が関係するか、初心者向けに教えてください」とAIに聞く |
| 修正後もまだ心配 | 別のAIツール（ChatGPTで書いたならClaudeで）でもう一度チェックし、2つの結果を比較する |
| 自社の業界特有のルールが気になる | 「この文章が○○業界の広告の場合、追加で気をつけるべきルールはありますか？」とAIに聞く |
