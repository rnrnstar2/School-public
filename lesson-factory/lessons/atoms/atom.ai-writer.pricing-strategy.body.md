# AI時代の単価設定戦略を立てる

あなたが書いた文章には、どれくらいの値段をつけていますか？

「1文字〇円」と決めるのは、レストランのメニューに「食材費だけで値段を決める」ようなものです。お客さんが払うのは、美味しい料理そのもの——つまり「あなたが届ける価値」に対してお金を出しています。

AIを使うと文章づくりのスピードは劇的に上がります。でも「早く書けたから安くする」のではなく、「AIのおかげでより高い価値を届けられるから、単価を上げる」という方向で考えてみましょう。

このレッスンでは、AI時代にふさわしい単価の考え方を**1枚の戦略シート**にまとめます。15分で終わるように手順を絞っています。

![単価設定の全体図](/lesson-assets/atom.ai-writer.pricing-strategy/diagram.png)

## 前提を確認する

このレッスンを進める前に、次の条件を満たしているか確認してください。

- 何かしらの文章を人に提供した経験がある（ブログ、業務文書、SNS投稿など）
- ChatGPTやClaudeなどのAIチャットツールを一度でも使ったことがある

どちらも当てはまらなくても読めますが、実際の単価設定イメージが湧きやすくなります。

## 単価の「3つの柱」を知る

単価を決めるときに意識するのは、次の3つです。

1. **かかった時間** → 「1時間〇〇円」で計算する方法
2. **届ける価値** → 「相手がどれだけ得をするか」で計算する方法
3. **市場の相場** → 「他の人がいくらで提供しているか」を参考にする方法

### 良い例と悪い例を見る

- **良い例**: 「SEO記事（＝検索結果で上位に表示される記事）を1本書いて、企業の問い合わせを月50件増やす。その価値は月50万円相当だから、記事1本を15万円で提供する」
- **悪い例**: 「AIで1時間で書けたから、時給3,000円×1時間＝3,000円にする」

AIを使うと短時間で書けるようになります。でも、単価を時間だけで決めると、上手になるほど安くなってしまうという不思議な現象が起きます。

## 自分の「価値」を書き出す

ここからは手を動かします。メモ帳、紙、またはAIチャットを開いてください。

### 手順1：AIに「得意ジャンル」を壁打ちする

いきなり自分で考えるのが難しければ、AIに手伝ってもらいましょう。ChatGPTやClaudeを開いて、次のプロンプト（＝AIへの指示文）をコピペしてください。

> 私はライティングの仕事をしています。過去に書いたことがあるジャンルは「ブログ記事」「商品の紹介文」「SNS投稿」です。この経験をもとに、私が提供できそうな文章ジャンルを5つ提案してください。それぞれ「頼む側にとってのメリット」を1文で添えてください。

💡 **ポイント**: 「ブログ記事」「商品の紹介文」「SNS投稿」の部分は、あなた自身の経験に書き換えてください。

AIの提案を読んで、しっくりくるジャンルを**3つ**選びます。

### 手順2：「相手が得すること」をお金に変換する

選んだ3ジャンルについて、頼んだ人がどう嬉しいかを考えます。自分で考えてもいいですし、もう一度AIに聞いてもOKです。

> 「Web記事」を依頼する企業にとって、良い記事1本でどれくらいの売上効果がありますか？具体的な金額の目安を教えてください。

ざっくりで構いません。例えば——

- 「お客さんが月に10人増えそう。1人あたりの売上が5,000円なら、月5万円の売上増」
- 「その5万円のきっかけが私の記事1本なら、5〜10万円の価値はある」

この計算を「バリューベース（＝価値をもとにした）単価設定」と呼びます。

### 手順3：市場の相場をAIでリサーチする

次のプロンプトで、いまの市場感を確認しましょう。

> 日本のフリーランスライター市場で、「SEO記事」「商品紹介文」「プレスリリース」の一般的な単価相場を教えてください。初心者向け・中級者向け・上級者向けの3段階で整理してください。

💡 **ポイント**: ジャンル名は手順1で選んだものに書き換えてください。AIが出した数字はあくまで目安ですが、自分の単価が「安すぎないか」を判断する材料になります。

## AIを単価戦略に組み込む

AIは「時間を節約する道具」ではなく「価値を増幅する道具」として考えましょう。

| AIの使い方 | あなたの価値がどう上がるか |
|---|---|
| リサーチを深める | 記事の説得力が上がる → 単価アップの根拠になる |
| バリエーションを出す | 選択肢が広がり、クライアントの満足度が上がる |
| 品質チェックに使う | 修正の往復が減る → 納期短縮で信頼度アップ |
| 構成案を整える | 企画提案の質が上がる → 高単価案件を獲りやすくなる |

### 良い例と悪い例

- **良い例**: 「AIで10パターンの見出し案を出して、ベストな1つを選ぶ。この選択眼こそが私の価値だから、AIを使っていることは単価に影響しない」
- **悪い例**: 「AIで書いたから手間がかかっていないし、半額にしよう」

## 戦略シートをまとめる

ここまで書き出した内容を、1枚のシートにまとめます。以下のテンプレートを使ってください。メモ帳に直接書いても、AIに整形してもらってもOKです。

AIに整形を頼む場合はこのプロンプトを使います。

> 以下の情報をもとに、「単価設定戦略シート」をMarkdown形式の表で作ってください。
> - 得意ジャンル：（あなたの3ジャンルを入力）
> - 届ける価値：（手順2で書いた内容を入力）
> - 今の単価：（現在の金額を入力）
> - 目標単価：（目指したい金額を入力）
> - AIの活用方法：（表から選んだ内容を入力）
> - 最初に試す期間：3か月

完成イメージはこのような表です。

| 項目 | あなたの回答 |
|------|-------------|
| 得意ジャンル | （例：Web記事、商品紹介文、メルマガ） |
| 届ける価値 | （例：月5万円の売上増につながる記事） |
| 今の単価 | （例：1文字3円 / 記事1本3,000円） |
| 目標単価 | （例：1文字10円 / 記事1本15万円） |
| AIをどう使うか | （例：リサーチと品質チェック） |
| 最初に試す期間 | （例：3か月） |

このシートをスクリーンショットで撮って、このレッスンの成果物とします。

## 結果を確認する

シートが完成したら、次の3つをチェックしてください。すべてにチェックがつけばこのレッスンは完了です。

- [ ] 「届ける価値」に具体的な金額イメージが書いてある
- [ ] AIを使うことで「価値が増える」方向で書けている（「安くする」方向になっていない）
- [ ] 目標単価が今の単価より高い数字になっている

## つまずきやすいポイントと対策

| つまずき | 対策 |
|---|---|
| 「自分の価値がわからない」 | まずは「頼んだ人が喜ぶこと」を1つだけ書いてみましょう。金額は後から考えて大丈夫です。AIに「〇〇という仕事を依頼する人は何を期待していますか？」と聞くのも手です |
| 「目標単価が高すぎる気がする」 | 相場より高くても問題ありません。まずは1人にその価格で提案して反応を見てみましょう。断られても「高すぎた」のではなく「伝え方を工夫する余地がある」と考えます |
| 「AIを使っていることを伝えるべき？」 | 伝えるかどうかは自由です。重要なのは「成果の質」であって「作り方」ではありません |
| 「ジャンルが思いつかない」 | 手順1のプロンプトをそのままAIに投げてみてください。思いもよらなかった選択肢が出てくることがあります |
