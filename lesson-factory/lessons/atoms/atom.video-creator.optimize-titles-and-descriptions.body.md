# タイトルと概要欄を最適化する

動画を作ったあと、「タイトルが思いつかない」「概要欄に何を書けばいいかわからない」と悩んだことはありませんか？

料理に例えると、動画本体は「丁寧に作った料理」、タイトルと概要欄は「お店の看板とメニュー」です。どんなに美味しい料理でも、看板が地味だったりメニューが何も書かれていなかったりすれば、お客さんは立ち止まってくれません。逆に、中身がひと目で伝わる看板があれば、お客さんは安心して入ってきてくれます。

このレッスンでは、AIツール（＝人工知能を使った便利なWebサービスのこと）を活用して、あなたの動画の「看板」を15分以内に魅力的に仕上げる方法を学びます。

![タイトル最適化の全体フロー](/lesson-assets/atom.video-creator.optimize-titles-and-descriptions/diagram.png)

## 前提を確認する

- 動画がすでにアップロード済み、または近日中にアップロードする予定であること
- ChatGPT（chat.openai.com）やClaude（claude.ai）などのAIチャットツール（＝テキストで質問すると答えてくれるサービス）をブラウザで開けること

どちらも準備できていれば、このレッスンは15分以内で終わります。

## 動画の情報を整理する

まず、AIに「何の動画か」を正確に伝えるために、以下の3点をメモ帳やスマホのメモアプリに書き出します。ここが雑だとAIの回答も的外れになるため、できるだけ具体的にしてください。

| 項目 | 書く内容 | 良い例 | 悪い例 |
|------|----------|--------|--------|
| テーマ | 動画の主題 | 初心者向け・スマホだけでできる料理動画の撮り方 | 撮影について |
| ターゲット | 見てほしい人 | 動画制作を始めたばかりの20〜30代 | みんな |
| 特徴 | 長さや形式 | 10分・実演つき・3ステップ構成 | 普通の動画 |

## タイトル案をAIに出してもらう

### AIにプロンプト（＝お願い文）を送る

ChatGPTやClaudeを開いて、次のテンプレートをコピーし、`[ ]` の中だけを自分の動画に書き換えて送信します。

```
以下の動画情報をもとに、YouTubeのタイトル案を5つ作ってください。
条件：
- 30文字以内に収める（スマホで途切れないようにする）
- 数字を1つ以上入れる
- 視聴者が「自分のことだ」と感じる言葉を入れる

- テーマ：[あなたの動画のテーマ]
- ターゲット：[ターゲット層]
- 特徴：[動画の長さや特徴]
```

> **ポイント**: 条件を具体的に伝えるほど、AIは的確な案を出してくれます。「いい感じに」のような曖昧な指示は避けましょう。

### 追加のプロンプト例：もっと良い案がほしいとき

最初の5案がしっくりこない場合は、続けて次のように送ります。

```
もう5つ別のパターンでお願いします。今度は「疑問形」や「〜しませんか？」で始まるタイトルも混ぜてください。
```

### タイトル案を選ぶ基準

AIから返ってきた案を、次の3つの基準で選びます。

1. **具体性**: 何について・誰向けかがひと目でわかるか
2. **数字の力**: 「3つのコツ」「10分で」など数字が入っているか
3. **長さ**: スマホの検索結果で途切れない30文字以内か

**良い例：**
- 「スマホ撮影が変わる！初心者の3つのコツ」— 具体的で数字入り、28文字
- 「10分で解決！動画初心者の照明の悩み」— ターゲットとベネフィットが明確、20文字

**悪い例：**
- 「撮影について話しました」— 何の動画かわからず、クリックする動機がない
- 「【完全保存版】超初心者でも絶対に失敗しない最強のスマホ撮影テクニック大公開！」— 長すぎて途切れる（42文字）

## 概要欄をAIに書いてもらう

### 概要欄のテンプレートを作ってもらう

同じAIチャットに続けて、次のように送信します。

```
次の動画タイトルに合うYouTube概要欄を作ってください。
構成は次の3ブロックにしてください：
1. 冒頭の紹介（2〜3文。この動画で何がわかるかを簡潔に）
2. 動画の目次（タイムスタンプ付き。例：0:00 はじめに）
3. 関連リンクの枠（SNS・次に見てほしい動画のURLを入れる場所）

タイトル：[選んだタイトル]
```

### 自分の言葉で整える

AIが書いた文章はそのまま使えますが、次の2か所だけ自分の言葉に直すと「あなたらしさ」がぐっと出ます。

- **冒頭のあいさつ**: 「こんにちは！〇〇です。」とチャンネル名を入れる
- **目次のタイムスタンプ**: 実際の動画に合わせて時間を調整する（AIは動画の中身を見られないため、時間は必ずズレます）

**良い例：** 「こんにちは！〇〇です。この動画では、スマホだけで動画のクオリティを上げる3つのコツを紹介します」

**悪い例：** AIの出力を一切読まずにそのまま貼り付ける — タイムスタンプがデタラメになったり、誤字が残ったりします

## タイトルと概要欄を登録する

![YouTube Studioでの登録手順](/lesson-assets/atom.video-creator.optimize-titles-and-descriptions/screen_capture.png)

1. YouTube Studio（＝YouTubeの管理画面。studio.youtube.com）をブラウザで開く
2. 左メニューの「コンテンツ」をクリック
3. 該当動画のサムネイルまたはタイトルをクリックして「詳細」画面を開く
4. 「タイトル」欄に選んだタイトルを貼り付ける
5. 「説明」欄にAI作成＋自分で整えた概要文を貼り付ける
6. 右上の「保存」ボタンを押す

## 結果を確認する

保存後、スマホまたはブラウザのシークレットモード（＝ログインしていない状態で見る機能）で動画ページを開いて、次の3点をチェックします。

- [ ] **タイトル**: 検索結果やおすすめ欄で途切れずに表示されているか（スマホで32文字以内が目安）
- [ ] **概要欄の冒頭**: 「もっと見る」を押さなくても、最初の2〜3行で動画の内容が伝わるか
- [ ] **リンク**: 概要欄に貼ったURLが正しくクリックできるか

3つすべてにチェックが入れば、このレッスンは完了です。

## つまずきやすいポイントと対処法

| つまずき | 原因 | 対処法 |
|----------|------|--------|
| AIの回答が的外れ | 動画情報（テーマ・ターゲット・特徴）が曖昧 | 3つの項目を表の「良い例」レベルまで具体的に書き直してから再送信する |
| タイトルが長すぎる | AIは情報を詰め込みがち | 「30文字以内で」と条件に明記する。それでも長ければ「もっと短くして」と追加で伝える |
| 概要欄のリンクが動かない | URLの前後にスペースがない | URLの前後に半角スペースか改行を入れる |
| 概要欄のタイムスタンプがズレている | AIは動画の中身を見ていない | 自分で動画を再生しながら、各セクションの開始時間を手動で修正する |
| 何度やってもしっくりこない | 1回で完璧を目指している | まず公開して、1週間後にアナリティクス（＝視聴データ）を見てから改善すればOK |
