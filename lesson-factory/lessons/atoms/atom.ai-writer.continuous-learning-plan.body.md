# AI進化に追従する学習計画を立てる

AI（＝人工知能のこと。ここでは ChatGPT や Claude のような文章生成ツールを指します）は、毎月のように新機能が追加されます。「何を学べばいいか分からない」と感じているあなたは一人ではありません。

このレッスンでは、**15分で完成する「マイ学習計画」** を作ります。季節の献立（＝その時どきのメニュー表）を考えるように、AI の新しい情報をチェックして自分の学びを更新する仕組みを作りましょう。

完成すると、こんなドキュメントができあがります。

![学習計画サイクル図](/lesson-assets/atom.ai-writer.continuous-learning-plan/diagram.png)

## 目指すものを決める

まずは「AI を何に使いたいか」を一つだけ決めます。旅行の目的地を一つ決めるのと同じです。あれもこれもと欲張ると迷子になります。

**良い例**：
- ブログ記事の執筆時間を半分にする
- 毎週のSNS投稿をAIに手伝ってもらう

**悪い例**：
- AI をすべて理解する（範囲が広すぎて、どこから手をつければよいか分からなくなります）
- プログラミングもデザインも全部できるようになる（一度に複数の目標を立てると続きません）

### AIに相談して目的を絞る

目的が思いつかないときは、ChatGPT や Claude に聞いてみましょう。以下のプロンプト（＝AIへの指示文）をそのままコピーして貼り付けてください。

```
私は普段の仕事で[ここにあなたの仕事内容を入れる]をしています。
AIツールを使って効率化したいのですが、
最初に取り組むべき目標を1つだけ提案してください。
条件：具体的で、1ヶ月以内に成果が見えるものにしてください。
```

AIが提案してくれた目標を参考に、メモ用紙かスマホのメモアプリに、あなたの目的を一文で書いてください。

## 情報源を3つだけ選ぶ

AI のニュースは世界中にあふれています。料理のレシピサイトが多すぎて迷うのと同じです。情報源（＝ニュースや更新情報を発信している場所）は**3つまで**に絞りましょう。

おすすめの組み合わせ：

1. **ChatGPT の公式ブログ**（openai.com/blog）— 新機能の発表が一番早い
2. **Claude の公式ブログ**（anthropic.com/news）— Claude の新機能や活用事例が分かる
3. **あなたがよく使っているAIツールの更新ページ** — 使っているものの新機能が一番役立つ

**良い例**：使っているツールの公式ブログ + ニュースサイト1つ + YouTube チャンネル1つ

**悪い例**：X（旧Twitter）で「AI」と検索して毎日スクロールする

> 悪い例は情報が断片的で、何が重要か判断しにくくなります。

### AIに情報源を提案してもらう

自分に合った情報源が分からないときは、AIに聞いてみましょう。

```
私の目的は「[あなたの目的]」です。
この目的に合ったAI関連の情報源を3つ提案してください。
条件：日本語で読めるもの、週1回チェックすれば十分なもの、
初心者にも分かりやすいものを選んでください。
```

## 週1回のチェック習慣を作る

毎日チェックする必要はありません。週に1回、15分だけ新しい情報を確認する時間を決めましょう。週末の朝のコーヒータイムにするなど、既存の習慣にくっつけるのがコツです。

手順：

1. カレンダーに「AIチェック」という予定を週1回入れる（＝毎週決まった曜日・時間に予定として登録すること）
2. 時間が来たら、選んだ3つの情報源を5分ずつ眺める
3. 自分の目的に関係ありそうなものがあれば、メモに一言まとめる
4. メモを見返して「今週試したいこと」を一つだけ書く

**良い例**：毎週日曜10:00に「ChatGPTブログ → メモ1行 → 今週試すこと1つ」を15分で終える

**悪い例**：思い出したときに30分以上ブラウザをさまよう

> 時間を区切らないと、情報を追いかけること自体が目的になってしまいます。

## 学習計画を書き出す

ここまで決めたことを一枚のドキュメント（＝文書）にまとめます。以下のテンプレートをコピーして、あなたの内容を埋めてください。メモアプリでも Google ドキュメントでも、使い慣れたものでOKです。

```markdown
# わたしのAI学習計画

## 目的
（例：ブログ記事の執筆時間を半分にする）

## 情報源（3つまで）
1. 
2. 
3. 

## チェック日時
（例：毎週日曜 10:00〜10:15）

## 今週のメモ
- （例：ChatGPTにカスタム指示機能が追加された）

## 今週試すこと（1つだけ）
- （例：カスタム指示を設定してブログ下書きを作ってみる）
```

### AIにテンプレートを埋めてもらう

ここまでのメモ（目的・情報源・チェック日時）をまとめてAIに渡すと、整った学習計画に仕上げてくれます。

```
以下の情報をもとに「マイAI学習計画」を
Markdown形式で整えてください。

目的：[あなたの目的]
情報源：[選んだ3つ]
チェック日時：[決めた曜日と時間]
```

出来上がったら、その画面のスクリーンショット（＝画面の写真を撮る機能。スマホなら電源ボタン＋音量下ボタン、パソコンなら Windows は「PrintScreen」キー、Mac は「Command + Shift + 3」で撮れます）を撮ってください。その画像がこのレッスンの成果物（＝完成したもの）です。

## 成果物を確認する

あなたの学習計画が以下の条件を満たしているか、一つずつ確認しましょう。

| チェック項目 | 確認のポイント | ✅ / ❌ |
|---|---|---|
| 目的が一文で書かれている | 「〜を半分にする」「〜を毎週やる」のように具体的か | |
| 情報源が3つ以内 | 4つ以上になっていないか。多すぎると続かない | |
| チェック曜日・時間が決まっている | 「毎週◯曜日 ◯時」と具体的な日時があるか | |
| 今週試すことが1つだけ書かれている | 2つ以上書いていないか。欲張ると何も試せない | |

すべて ✅ がつけば完成です！

## つまずきやすいポイント

| つまずき | よくある状況 | 対策 |
|---|---|---|
| 「目的が思いつかない」 | AIで何ができるかイメージが湧かない | 上の「AIに相談して目的を絞る」のプロンプトをそのまま使う |
| 「情報源が多すぎて絞れない」 | あれもこれも気になる | 今使っているAIツールの公式ページ1つだけでスタートしてOK |
| 「習慣が続かない」 | 最初の2週間で忘れてしまう | カレンダーの予定に通知（＝リマインダー）を設定する。完了したらチェックする習慣にする |
| 「何を試せばいいか分からない」 | 新機能を見ても使い方が分からない | 情報源で見つけた新機能の名前をそのままAIに「◯◯の使い方を教えて」と聞く |
| 「計画を作ったが実行できない」 | 15分の中で何をすればいいか迷う | 最初の5分＝情報源を開く、次の5分＝メモする、最後の5分＝試すことを1つ書く、と分割する |

これであなた専用のAI学習計画が完成しました。来週のチェック日時に、早速この計画を使ってみましょう。1回やってみて「ここを変えたい」と思ったら、いつでも計画を更新してOKです。計画は完璧に作ることより、続けることが大事です。
