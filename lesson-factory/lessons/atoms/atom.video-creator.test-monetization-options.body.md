# 収益化の選択肢を比較する

あなたが動画を2〜3本アップしたとしましょう。少しずつ再生数が伸びてきて、「これ、何かお金になるのかな？」と思ったら、もう収益化を考えるタイミングです。

でも、お金をもらう方法は1つじゃありません。ここを料理に例えてみましょう。

小さな食堂を開いたとします。**弁当を1個ずつ売る**方法もあれば、**月額の定期便**にする方法もあります。**スポンサーにお店を宣伝してもらう**道もある。どれも正解ですが、あなたの食堂の規模や客層によって「いま一番合う方法」が変わってきます。

このレッスンでは、動画の収益化にはどんな選択肢があるかを一通り知り、AIと一緒に「いまの自分に一番合いそうな方法」を比較表にまとめます。15分で終わるよう、手順をシンプルに進めます。

![収益化の6つの選択肢](/lesson-assets/atom.video-creator.test-monetization-options/diagram.png)

## 収益化の6つの方法を知る

動画からお金を得る主な方法は、大きく6つあります。ここでは「どういう仕組みか」だけをざっと確認します。

1. **広告収入（＝プラットフォームが広告を表示し、その一部があなたに入る仕組み）**
   - YouTubeの「パートナープログラム」などが代表例です。一定の再生数を超えると申請できます。

2. **スポンサーシップ（＝企業があなたに依頼して、動画内で商品やサービスを紹介する契約）**
   - 企業から直接お金をもらう方法です。あなたの視聴者層と合う企業を見つけることが鍵です。

3. **ファン支援（＝見てくれた人が「ありがとう」の気持ちでお金を送れる仕組み）**
   - YouTubeの「スーパーチャット」や「メンバーシップ」、他にも「fanbox」や「OFuse」などがあります。

4. **サブスクリプション（＝月額で限定コンテンツを見られるようにする仕組み）**
   - 毎月決まった収入が見込めるのが魅力です。ただし、継続して限定コンテンツを作り続ける必要があります。

5. **グッズ販売（＝オリジナルのTシャツやステッカーなどの商品を売る方法）**
   - ファンとの一体感が生まれますが、在庫や発送の管理が少し大変です。

6. **デジタル教材販売（＝あなたの知識をまとめた動画やテキストを有料で売る方法）**
   - 「このジャンルなら人に教えられる」という得意分野がある人に向いています。

## 自分の状況に合う方法を選ぶ

6つの方法を知った上で、「いまの自分」に合いそうなものを見つけましょう。

判断のポイントは3つです。

- **視聴者数**: 今どれくらい見られているか
- **更新頻度**: どれくらいのペースで動画を出せそうか
- **得意分野**: 何か人に教えられるジャンルがあるか

### 良い例と悪い例

- ✅ **良い例**: 「いまは月に2本のペースで、再生数は平均200回。まずはファン支援から始めてみよう」
- ✅ **良い例**: 「料理動画で反響があるから、デジタル教材としてレシピ集を売ってみよう」
- ❌ **悪い例**: 「まだ動画を1本も出していないけど、サブスクリプションを始めよう」（限定コンテンツを提供し続けるのが難しい）
- ❌ **悪い例**: 「まずは全部同時に始めよう」（やることが多すぎてどれも中途半端になる）

## AIで比較表を作る

ここからAIを使って、あなたに合いそうな方法を並べた「比較表」を作ります。

AI（ChatGPTやClaudeなど）に、次のようなプロンプト（＝AIへの指示文）をそのままコピーして貼り付けてください。

```
私は動画クリエイターです。以下の条件で、収益化の選択肢を比較する表をマークダウン形式で作ってください。

- 現在の視聴者数: まだ少ない（月平均200〜500再生）
- 更新頻度: 月に2〜3本
- 得意分野: （あなたの得意分野をここに書く）
- 優先したいこと: まずは小さく始められること

比較する項目:
1. 広告収入
2. スポンサーシップ
3. ファン支援
4. サブスクリプション
5. グッズ販売
6. デジタル教材販売

表の列は: 方法名 / はじめやすさ(高・中・低) / 想定収入の目安 / 必要な準備 / あなたへのおすすめ度(★1〜5) / 理由
```

上のテキストの「あなたの得意分野をここに書く」の部分だけ、ご自身の内容に書き換えてから送信してください。

### AIの回答を整えるコツ

AIから返ってきた比較表が少しイマイチだと感じたら、追加でこう聞いてみてください。

```
この表に「3ヶ月後の目標アクション」という列を追加してください。それぞれの方法について、3ヶ月以内にやるべき最初の一歩を具体的に書いてください。
```

こうすると、「選んだ後に何をすればいいか」まで見えるので、次のアクションに迷いにくくなります。

![AIで作った比較表の画面例](/lesson-assets/atom.video-creator.test-monetization-options/screen_capture.png)

## 結果を確認する

AIが作った表を見て、次の3点をチェックしましょう。

1. **おすすめ度が最も高い方法はどれか** — その方法をまず試す候補にします
2. **「はじめやすさ」が「高」のものがいくつあるか** — すぐ始められる方法が複数あれば迷わず1つ選びます
3. **「必要な準備」にできないことがないか** — もし「1000人以上のチャンネル登録者が必要」といった条件があって今は足りない場合は、別の方法を優先します

確認できたら、表全体のスクリーンショット（＝画面の表示内容を画像として保存したもの）を撮ってください。それがこのレッスンの成果物です。

### スクリーンショットの撮り方

- **Windows**: `Windows`キー + `Shift` + `S` を同時に押すと、画面の一部を選んで保存できます
- **Mac**: `Command` + `Shift` + `4` を同時に押すと、画面の一部を選んで保存できます
- **スマホ**: 電源ボタン＋音量ボタンを同時に押すと撮れます（機種によって少し異なります）

## つまずきポイントを押さえる

| つまずきポイント | 対策 |
|---|---|
| AIの回答が一般的すぎる | プロンプトの「得意分野」をもう少し具体的に書き直す（例: 「料理」→「時短で作れるおつまみのレシピ」） |
| どれから始めればいいか迷う | 「はじめやすさ」が「高」のものを1つだけ選び、まずは1ヶ月試す |
| 収益化に罪悪感を感じる | あなたが提供する価値（＝動画を見て得られる楽しさや知識）に対して、対価をもらうのは自然なことです |
| 表が表示されない・壊れている | AIに「表をマークダウン形式で出力してください」と再度伝える |
| プロンプトを変えたのに結果が変わらない | 「前の回答は忘れて、以下の条件だけで新しく表を作ってください」と最初に一文を追加する |

## 次のアクション

比較表ができたら、いちばんおすすめ度が高かった方法を1つ選んでください。次のステップでは、その方法を実際に始める手順を学びます。

無理に全部やらなくて大丈夫。「まずは1つ、小さく始める」が一番の近道です。
