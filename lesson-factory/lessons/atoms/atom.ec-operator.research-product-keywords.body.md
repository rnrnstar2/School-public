# 商品検索キーワードを調べる

ネットショップで商品を売るとき、一番大切なのは「お客さんに見つけてもらうこと」です。どんなに良い商品でも、検索結果に出なければ、お店の奥に隠れた商品棚と同じです。

たとえば、駅前の実店舗を思い浮かべてください。通行人に気づいてもらうために「看板」や「のぼり旗」を出しますよね。ネットショップでは、**検索キーワード**（＝お客さんが検索窓に入力する言葉）がその看板の役割を果たします。

このレッスンでは、AIツール（＝人工知能を使った便利ツール。ChatGPTやClaudeなど）に手伝ってもらいながら、あなたの商品にぴったりの検索キーワードをリストアップします。所要時間は約15分です。

![キーワード調査の全体フロー](/lesson-assets/atom.ec-operator.research-product-keywords/diagram.png)

## 前提を確認する

始める前に、以下の2つだけ準備してください。

- **AIツールが使える状態であること** — ChatGPT（chat.openai.com）またはClaude（claude.ai）をブラウザで開けばOKです。無料プランで十分です
- **調べたい商品が1つ決まっていること** — 例：「手作り大豆キャンドル」「オーガニックコットンのベビー服」など

特別なソフトのインストールや有料プランへの登録は不要です。

## ステップ1：キーワードの「種」を書き出す

まず、あなたの商品を **3〜5個の言葉** で説明してみましょう。この説明がキーワードの「種」になります。難しく考えなくて大丈夫です。友人に「どんな商品？」と聞かれたときに答える言葉をそのまま書いてみてください。

**良い例：**
- 「無添加 手作り 大豆キャンドル」 → 商品の特徴が具体的に伝わる
- 「プレゼント用 誕生日 アロマキャンドル」 → 用途＋シーンが入っている

**悪い例：**
- 「キャンドル」 → 広すぎて大手ショップに埋もれてしまう
- 「いい感じのやつ」 → お客さんはこの言葉で検索しない

## ステップ2：AIにキーワード案を出してもらう

AIツール（ChatGPTやClaude）の入力欄に、次の文章をそのままコピーして貼り付けてください。「●●」の部分だけ、あなたの商品名に置き換えます。

```
「●●」という商品をネットショップで販売します。
この商品を探している人が検索しそうなキーワードを20個挙げてください。
以下の条件でお願いします：
- 一般のお客さん目線で、日常的な言葉を使う
- 「商品名そのもの」「用途・シーン」「悩み・目的」の3カテゴリに分けて出す
- 2語〜3語の組み合わせを中心にする
```

**何をしているか：** AIに「お客さんが実際に検索窓に入力しそうな言葉」をたくさん出してもらっています。カテゴリ分けを指示することで、あとで整理しやすくなります。

![AIにキーワードを出してもらう画面](/lesson-assets/atom.ec-operator.research-product-keywords/screen_capture.png)

### うまくいかないときの追加プロンプト

AIの回答が期待と違った場合、以下をそのまま追加で送ってみてください。

**専門用語が多すぎる場合：**
```
専門用語を避けて、普通のお客さんが普段使う言葉に置き換えてください。
```

**似たようなキーワードばかり出る場合：**
```
もっと視点を変えて、「悩みから検索する人」「プレゼントとして探す人」「季節のイベントで探す人」のパターンも追加してください。
```

**商品の説明がうまくできない場合：**
```
この商品の写真を見てください（画像をアップロード）。この商品の特徴を5つ挙げて、それぞれの特徴から検索キーワードを3つずつ考えてください。
```

## ステップ3：キーワードを3グループに分ける

AIから出てきたキーワードを、次の3つのグループに仕分けします。迷ったときは「自分がお客さんだったら、この言葉で検索するか？」と考えてみてください。

| グループ | どんなキーワード？ | 例 |
|---|---|---|
| **すぐ使う** | 商品にぴったり合っていて、お客さんが実際に使いそう | 「大豆キャンドル 無添加」「手作りキャンドル ギフト」 |
| **様子を見る** | 関連はあるけど少し広い。試してみる価値あり | 「アロマグッズ おすすめ」「リラックス アイテム」 |
| **今は使わない** | 関連が薄い、または大手が強すぎて勝てない | 「キャンドル」「プレゼント」 |

**判断のコツ：**
- 1語だけのキーワード → ほぼ「今は使わない」に入る（競合が多すぎる）
- 2〜3語の組み合わせ → 「すぐ使う」か「様子を見る」の候補
- お客さんの悩みが入っている → 「すぐ使う」の有力候補（例：「癒し キャンドル 寝室」）

## ステップ4：成果物としてまとめる

分類が終わったら、AIに表の形に整えてもらいましょう。以下をコピーして、AIの入力欄に貼り付けてください。

```
以下のキーワードを「すぐ使う」「様子を見る」「今は使わない」の3グループに分けた表を作ってください。マークダウン形式でお願いします。

【すぐ使う】
（あなたが「すぐ使う」に分けたキーワードをここに貼る）

【様子を見る】
（あなたが「様子を見る」に分けたキーワードをここに貼る）

【今は使わない】
（あなたが「今は使わない」に分けたキーワードをここに貼る）
```

**何をしているか：** あなたが分類した結果をAIにきれいな表に整形してもらっています。

出てきた表をコピーして、テキストファイルやメモアプリに保存してください。これがこのレッスンの成果物です。

## 結果を確かめる

最後に、以下のチェックリストで成果物を確認しましょう。すべてにチェックが入れば完了です。

- [ ] 「すぐ使う」グループに **5個以上** のキーワードが入っている
- [ ] すべてのキーワードが **2〜3語の組み合わせ** になっている（1語だけのものは「今は使わない」に移す）
- [ ] **お客さん目線の言葉** になっている（業界用語や専門用語が混じっていないか）
- [ ] 実際にそのキーワードで **検索してみて** 、似た商品が表示されるか確認した
- [ ] 成果物（キーワード分類表）を **テキストファイルに保存** した

## つまずきやすいポイントと対処法

| つまずき | 原因 | 対処法 |
|---|---|---|
| AIが専門用語ばかり出す | プロンプトに「お客さん目線で」と入れていない | 「一般のお客さんが普段使う言葉に直してください」と追加で伝える |
| キーワードが20個も出てこない | 商品の説明が漠然としている | 商品の写真をAIにアップロードして「特徴を5つ挙げて」から始める |
| どのグループに入れるか迷う | 判断基準があいまい | 「自分がお客さんなら、この言葉で検索するか？」を基準にする |
| 「すぐ使う」が5個に満たない | 商品の切り口が少ない | AIに「用途別」「季節別」「贈り物シーン別」で追加キーワードを出してもらう |
| 似たキーワードばかり並ぶ | 最初の「種」が1方向だけ | ステップ2の追加プロンプト「視点を変えて」を使う |
