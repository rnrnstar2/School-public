# 改善ポイントを抽出する

あなたのネットショップ、なんとなく「売上が伸びないな」と感じていませんか？

たとえば、あなたがレストランを経営しているとします。毎日料理を作って配膳していると、自分では気づかない課題があります。「スープの味が薄い」「メニューの文字が小さすぎる」「入口が分かりにくい」。外部の人に食べてもらったり見てもらったりすると、こうした改善ポイントが次々と見つかります。

このレッスンでは、AI（＝人工知能で、ここでは ChatGPT や Claude のような対話型ツールのこと）を「外部の目利き」として使い、あなたのネットショップの改善ポイントをリストアップします。最終的に「すぐできる改善」と「じっくり取り組む改善」に分類した改善リストが手元に残ります。

## 前提を確認する

このレッスンを進める前に、次のものを用意してください。

- ネットショップの「商品説明文」「トップページの文章」「お客様のレビュー」などのテキスト（コピーして使える状態）
- 以下のAIツールのいずれか（無料プランでOK）
  - **ChatGPT**（https://chat.openai.com）— 最も広く使われている対話AI
  - **Claude**（https://claude.ai）— 長い文章の分析が得意な対話AI
- 15分の作業時間

> どのツールを使っても手順はほぼ同じです。このレッスンでは ChatGPT の画面で説明しますが、Claude でも同じプロンプト（＝AIへの指示文）をそのまま使えます。

## 全体の流れを把握する

![改善ポイント抽出フロー](/lesson-assets/atom.ec-operator.extract-improvement-points/diagram.png)

全体の流れは次の3ステップです。

1. **素材を集める** — あなたのショップのテキストを用意する（3分）
2. **AIに分析してもらう** — テキストをAIに渡し、改善点を指摘してもらう（5分）
3. **結果を整理する** — 改善リストをまとめ、優先順位をつける（7分）

## ステップ1: ショップのテキストを集める

まず、あなたのショップから次のテキストをコピーしてメモ帳などに貼ります。

- 商品ページの商品説明文（1〜3商品でOK）
- トップページのキャッチコピーや紹介文
- 最近のお客様レビュー（3〜5件）

**良い例**:
「商品Aの説明文（全文）、商品Bの説明文（全文）、レビュー3件」というように、複数種類のテキストをまるごとコピーする

**悪い例**:
商品名だけを羅列する（「ワンピースA、バッグB、靴C」だけでは情報が少なすぎてAIが具体的な改善点を出せません）

> **ヒント**: テキストは多いほうが的確な分析になります。迷ったら「売れていない商品」の説明文を優先して集めましょう。

## ステップ2: AIに分析を依頼する

ChatGPT または Claude を開き、次のプロンプトをコピーして貼り付けます。「ここにテキストを貼る」の部分を、先ほど集めたテキストに置き換えてください。

```
以下は私のネットショップのテキストです。
EC運営のプロの視点で、改善できるポイントを箇条書きで教えてください。
それぞれの改善ポイントについて、「なぜ改善が必要か」「どう直せばいいか」もセットで書いてください。

---ここから---
ここにテキストを貼る
---ここで終わり---
```

このプロンプトは「私のショップの文章を渡すから、プロの目線で改善点をリストアップして」とAIに依頼する指示文です。

![AI分析の画面イメージ](/lesson-assets/atom.ec-operator.extract-improvement-points/screen_capture.png)

### AIの回答が物足りないときの追加プロンプト

AIの最初の回答が「もう少し具体的にほしいな」と思ったら、続けて以下のメッセージを送ってみましょう。

**回答が一般的すぎるとき**:
```
もっと具体的に教えてください。たとえば、どの文章のどの部分をどう変えればいいか、修正例を示してください。
```

**改善点が多すぎて困るとき**:
```
売上への影響が大きい順に、上位5つだけに絞ってください。
```

**特定の観点で深掘りしたいとき**:
```
商品説明文だけに注目して、「購入ボタンを押したくなる文章」にするための改善点を教えてください。
```

> **ポイント**: AIとの会話は1回で終わらなくてOKです。「もっと詳しく」「ここを深掘りして」と何度もやり取りすることで、より使える改善リストになります。

## ステップ3: 結果を整理する

AIからの回答を受け取ったら、次のように整理します。

1. AIの回答をすべてコピーする
2. メモ帳やGoogleドキュメントに貼り付ける
3. それぞれの改善ポイントを次の2グループに分ける

| グループ | 意味 | 例 |
|---|---|---|
| **すぐできる**（30分以内） | テキストの修正だけで対応できるもの | 「商品説明にサイズ感の記載を追加する」 |
| **じっくり取り組む**（1日以上） | 写真の撮り直しやページ構成の変更が必要なもの | 「商品写真を使用シーンが伝わるものに差し替える」 |

**良い例**:
「商品説明にサイズ感が足りない → すぐできる → 今日中にサイズ表を追記する」のように、改善内容・難易度・次のアクションをセットで書く

**悪い例**:
AIの回答をそのまま保存して終わる（次に何をすればいいかわからなくなります）

> **ヒント**: 「すぐできる」グループから1つ選んで、このレッスンの直後に実行してみましょう。小さな改善を1つ実行する経験が、次の改善につながります。

## 完了を確認する

次のチェックリストで、改善リストが使える状態か確認しましょう。

- [ ] 改善ポイントが3つ以上リストアップされている
- [ ] それぞれの改善ポイントに「なぜ良くないか」の理由が書かれている
- [ ] それぞれの改善ポイントに「どう直せばいいか」の具体的な修正案が書かれている
- [ ] 「すぐできる」と「じっくり取り組む」に分類されている
- [ ] 「すぐできる」から最初に着手する1つを選んでいる

すべてチェックできたら、このレッスンは完了です。あなたのショップの改善リストが手元にある状態になりました。

## つまずき対策を知る

| つまずきポイント | 原因 | 対策 |
|---|---|---|
| AIの回答が一般的すぎる | 渡したテキストが少ない、または短い | ショップのテキストを3種類以上・各100文字以上渡す。「具体的な修正例も示してください」と追加で聞く |
| 日本語が不自然になる | AIが英語ベースで考えている | プロンプトの最後に「日本語で、自然な表現で回答してください」と追記する |
| 改善点が多すぎて混乱する | 絞り込み指示をしていない | 「売上への影響が大きい順に5つだけ教えてください」と追加で聞く |
| 何をコピーすればいいかわからない | 対象テキストが曖昧 | まずは「一番売れていない商品」の商品ページを丸ごとコピーするところから始める |
| AIが的外れなことを言う | ショップのジャンルや状況を伝えていない | 「私は○○（ジャンル）のネットショップを運営しています」と冒頭に追記する |
