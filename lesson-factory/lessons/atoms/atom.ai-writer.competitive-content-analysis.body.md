# 競合コンテンツをAIで分析する

あなたがブログ記事やSNS投稿を書くとき、「同じテーマで他の人はどんなことを書いているんだろう？」と気になったことはありませんか？

競合コンテンツの分析とは、同じジャンルで人気のある記事やSNS投稿を調べて、**何が読者に刺さっているのか**を見つける作業のことです。料理にたとえると、人気レストランのメニューを研究して「どんな味付けがウケているか」を把握するようなもの。自分の料理（＝コンテンツ）をもっと美味しくするためのヒントを集める工程です。

このレッスンでは、AIチャット（ChatGPTやClaude）を使って、競合の記事を効率よく分析し、**自分のコンテンツに活かせるポイントをまとめたレポート**を15分以内で作ります。

![分析フロー全体図](/lesson-assets/atom.ai-writer.competitive-content-analysis/diagram.png)

---

## 前提を確認する

- AIチャットツール（ChatGPT、Claude、Geminiなど）をブラウザで開ける状態であること
- 分析したいテーマやジャンルが1つ決まっていること（例：「副業 始め方」「AIライティング コツ」など）
- 競合として参考にしたい記事やSNS投稿を2〜3本、URLまたはコピーで手元に用意しておくこと

---

## ステップ1：分析する競合コンテンツを集める

まずは「どの記事を分析するか」を決めます。Google検索やSNSで、あなたが書きたいテーマのキーワードを入力し、上位に出てくる記事を2〜3本ピックアップしましょう。

### 良い例
- 検索上位の記事で、はてなブックマークやSNSシェアが多いもの
- 自分と同じターゲット読者（例：初心者向け）に書かれている記事

### 悪い例
- 企業の公式ヘルプページ（分析対象としては構造が違いすぎる）
- 3年以上前の古い記事（情報が古く参考になりにくい）

集めた記事のURLをメモ帳などに貼り付けておきます。記事本文をコピーしてテキストファイルに保存しておくとこの後の作業がスムーズです。

---

## ステップ2：AIに分析を依頼するプロンプトを作る

AIに「この記事を分析して」と丸投げするだけでは、ぼんやりした回答しか返ってきません。**何を知りたいのか**を具体的に伝えることがコツです。

以下のプロンプト（＝AIへの指示文）をコピーして、AIチャットに貼り付けてください。`【 】`の中はあなた自身の情報に書き換えます。

```
以下の記事を「競合コンテンツ分析」の観点で分析してください。

■ 私のテーマ：【副業 始め方】
■ 私のターゲット読者：【会社員で副業未経験の30代】

■ 分析してほしい記事：
---
【ここに記事の本文をコピー＆ペーストする】
---

以下の5項目に分けて、箇条書きで回答してください。
1. この記事の主なメッセージ（1〜2文で要約）
2. 読者を引きつけている工夫（タイトル・導入・構成など）
3. カバーしているトピック一覧
4. 足りていないと思われる情報や視点
5. 私が自分の記事を書くときに活かせるポイント
```

### 良い例
- 「私のターゲット読者は〇〇」と明記する → AIが読者目線で分析してくれる
- 分析観点を5項目に絞る → 回答がブレずに整理される

### 悪い例
- 「この記事を分析して」だけ → 何を知りたいか不明で、表面的な要約が返ってくる
- 10項目以上を一度に聞く → 回答が長くなりすぎて読みきれない

---

## ステップ3：2〜3本分の分析結果を並べて比較する

ステップ2のプロンプトを、集めた記事ごとに繰り返します。2〜3本分の分析が出揃ったら、次はAIに**横断比較**をしてもらいます。

以下のプロンプトを使ってください。

```
ここまでに分析した3本の記事を横断的に比較して、以下をまとめてください。

1. 3本に共通しているトピック（読者が「必須」と感じている内容）
2. 1本だけがカバーしていた独自の切り口
3. どの記事にも足りていない情報や視点
4. 私が差別化するために狙うべきポイント（3つ以内）
```

この横断比較の結果が、あなたのコンテンツ戦略の核になります。

## ステップ4：分析レポートを仕上げる

最後に、ここまでの分析をひとつのレポートにまとめます。AIに以下のように頼みましょう。

```
ここまでの競合分析の結果を、以下のフォーマットで1枚のレポートにまとめてください。

# 競合コンテンツ分析レポート
## テーマ：【あなたのテーマ】
## 分析日：【今日の日付】

### 分析した記事
- 記事1：タイトルとURL
- 記事2：タイトルとURL
- 記事3：タイトルとURL

### 共通トピック（必須カバー項目）
- ...

### 差別化ポイント（自分の記事で狙う切り口）
- ...

### 注意点・避けるべきこと
- ...

### 次のアクション
- ...
```

AIが出力したレポートをコピーして、Googleドキュメントやメモアプリに保存すれば完成です。

---

## 完成を確認する

以下の3つが揃っていれば、このレッスンは完了です。

- [ ] 2〜3本の競合記事をAIで個別に分析できた
- [ ] 横断比較で「共通点」「差別化ポイント」を整理できた
- [ ] 分析レポートを1枚のドキュメントとして保存できた

---

## よくあるつまずきと対処法

### 「記事の本文が長すぎてAIに貼り付けられない」
記事全文をコピーする必要はありません。**導入部・見出し一覧・まとめ部分**の3箇所だけを抜き出して貼り付ければ、AIは構造を十分に把握できます。

### 「AIの分析が浅い・ありきたりに感じる」
プロンプトに**あなた自身の状況**を追加してみてください。例えば「私はライター歴3ヶ月の初心者です」「SEO（＝検索で上位に表示されるための工夫）を意識した記事にしたい」など、背景情報を足すとAIの回答が具体的になります。

### 「どの記事を選べばいいかわからない」
迷ったら、Google検索の1ページ目（上位10件）から、**個人ブログ2本＋企業メディア1本**の組み合わせで選ぶと、異なる視点をバランスよくカバーできます。
