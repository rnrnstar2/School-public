# リピート受注を生む関係構築術

お気に入りのレストランを思い出してみてください。料理がおいしいのはもちろんですが、店員さんが「前回お好きだったメニュー、今日もありますよ」と声をかけてくれたら、きっとまた行きたくなりますよね。

フリーランス（＝個人で仕事を受ける働き方）の世界も同じです。最初の仕事を無事に終えたあと、クライアント（＝仕事を依頼してくれる人）に「またお願いしたい」と思ってもらうための小さな習慣を、このレッスンで身につけます。AI チャットツールを使って、関係構築に必要なメッセージや提案を効率よく準備する方法を学びましょう。

## 前提を確認する

このレッスンに進む前に、次の条件を満たしているか確認してください。

- これまでに少なくとも 1 回、クライアントへ仕事を納品した経験がある（架空の設定でも OK）
- ChatGPT や Claude などの AI チャットツール（＝質問すると文章で答えてくれるツール）を使える状態にある

まだ納品経験がない方は、まず「初案件を獲得する」レッスンから始めるのがおすすめです。

## ステップ 1：フォローアップメッセージを作る

納品が終わったら、そのまま静かに去るのではなく「ありがとうございます」と感謝を伝えるメッセージを送りましょう。これがリピート受注の第一歩です。

AI チャットツール（ChatGPT や Claude）を開いて、次のプロンプト（＝AI への指示文）をそのままコピーして貼り付けてください。

```
あなたはフリーランスの仕事術に詳しいメンターです。
以下の条件で、クライアントへの納品後フォローアップメッセージを作成してください。

【条件】
- 感謝の気持ちを自然に伝える
- 成果物に不明点があれば対応する姿勢を示す
- 今後も力になりたいという意欲を控えめに伝える
- 200文字以内
- 丁寧だけど堅すぎないトーン（ビジネスメールではなくチャットを想定）

【背景情報（あなた自身の状況に書き換えてください）】
- クライアント名: 田中さん
- 納品した仕事: 会社紹介スライド5枚
- 仕事期間: 1週間
```

**ポイント**: 「【背景情報】」の部分は、あなた自身の実際の（または想定する）クライアント情報に書き換えてから送信してください。AI が状況に合ったメッセージを作ってくれます。

### 良い例と悪い例を比べる

**良い例**：「田中様、この度はスライド制作をお任せいただきありがとうございました。もし内容の修正やご不明点があれば、いつでもお気軽にご連絡ください。またお力になれることがあれば嬉しいです。」

**悪い例**：「納品しました。よろしくお願いします。」

良い例には「感謝」「サポート姿勢」「今後への意欲」の 3 要素が入っています。悪い例は事務連絡だけで、関係性を深める要素がありません。

## ステップ 2：3 段階の関係構築プランを立てる

1 回のメッセージだけでなく、時間をおいて連絡を重ねることで「この人は信頼できる」という印象を育てます。

![関係構築タイミング図](/lesson-assets/atom.ai-freelancer.repeat-client-strategy/diagram.png)

上の図のように、納品後の関係構築には 3 つのベストタイミングがあります。

| タイミング | 目的 | 具体的なアクション |
|---|---|---|
| **納品直後（0〜3 日）** | 感謝を伝える | お礼メッセージ ＋ 不明点がないか確認 |
| **1〜2 週間後** | 活用状況を気遣う | 成果物が役に立っているか軽くたずねる |
| **1 ヶ月後** | 次の仕事につなげる | 関連する提案や役立ち情報を送る |

次のプロンプトを AI に送って、3 タイミング分のメッセージ案をまとめて作りましょう。

```
フリーランスがクライアントとの関係を深めるための
3段階フォローアップメッセージを作成してください。

【条件】
- タイミング: 納品直後（感謝+確認）、1〜2週間後（活用状況の気遣い）、1ヶ月後（新しい提案）
- 各メッセージ200文字以内
- 各メッセージの冒頭に「目的：◯◯」と1行添える
- 押し付けがましくない、自然なトーン

【背景情報】
- クライアント名: 田中さん
- 納品した仕事: 会社紹介スライド5枚
- クライアントの業界: IT企業
```

**ポイント**: 1 ヶ月後のメッセージで「新しい提案」を入れるのが難しいと感じたら、AI に追加で聞いてみましょう。

```
田中さん（IT企業）に喜ばれそうな追加提案のアイデアを3つ出してください。
前回の仕事は会社紹介スライド5枚でした。
関連する仕事で、フリーランスが提案できるものを教えてください。
```

AI が「採用向けスライドの更新」「サービス紹介資料」「社内向け説明資料」など、具体的な提案ネタを出してくれます。

## ステップ 3：成果物をまとめる

作成した 3 つのメッセージ案を、1 つのドキュメント（＝まとめ文書）に整理しましょう。

![完成ドキュメント例](/lesson-assets/atom.ai-freelancer.repeat-client-strategy/screen_capture.png)

次の構成でまとめてください。

```
# リピート受注プラン：[クライアント名]様

## 基本情報
- クライアント名: [名前]
- 納品した仕事: [内容]
- 納品日: [日付]

## フォローアップメッセージ

### 1. 納品直後（0〜3日後）
目的: 感謝を伝え、不明点を解消する
メッセージ: [ここに書く]
送信予定日: [日付]

### 2. 1〜2週間後
目的: 成果物の活用状況を気遣う
メッセージ: [ここに書く]
送信予定日: [日付]

### 3. 1ヶ月後
目的: 次の仕事につながる提案をする
メッセージ: [ここに書く]
送信予定日: [日付]

## 自分なりの工夫ポイント
- [1〜2行で、自分が意識した工夫を書く]
```

**良い例**：各タイミングの目的が明確で、メッセージがクライアントの状況に合った自然な日本語になっている

**悪い例**：AI の出力をそのままコピーして、自分のクライアントの文脈と合わない一般的な表現のままになっている

AI の出力はあくまで「たたき台」です。必ず自分の言葉に直して、クライアントとの実際のやり取りに合うように調整しましょう。

## 検証する

成果物ができたら、次のチェックリストで最終確認をしましょう。すべてにチェックが入れば、このレッスンは完了です。

- [ ] 3 つのタイミング（納品直後・1〜2 週間後・1 ヶ月後）のメッセージがすべて揃っている
- [ ] 納品直後のメッセージに「感謝」と「サポート姿勢」が含まれている
- [ ] 1〜2 週間後のメッセージが押し付けがましくなく、相手を気遣う内容になっている
- [ ] 1 ヶ月後のメッセージにクライアントの業界に関連した具体的な提案が含まれている
- [ ] AI の出力をそのまま使わず、自分の言葉で書き直している
- [ ] 各メッセージの送信予定日を記入している

## つまずき対策

| つまずきポイント | 解決策 |
|---|---|
| 「しつこく思われないか不安」 | メッセージの冒頭に「お忙しいところ失礼します」など相手を気遣う一言を入れる。間隔を空けて送ることで、しつこさは感じさせません |
| AI が作った文章が堅すぎる・不自然 | プロンプトに「友人に話しかけるような柔らかい口調で」と追加する。それでも堅い場合は「この文章をもっとカジュアルにしてください」と追加で指示する |
| 1 ヶ月後に何を提案すればいいか分からない | AI に「[クライアントの業界]で最近よくある課題を3つ教えて」と聞いて、そこから提案のヒントを得る |
| クライアントから返信がなくて不安 | 返信がないことは「迷惑」ではなく「忙しい」場合がほとんど。2 回目以降のメッセージは予定通り送って OK。ただし 3 回連続で返信がなければ間隔を延ばす |
| 背景情報の書き方が分からない | まずは架空のクライアントで練習する。「IT企業の田中さん、スライド5枚を納品」のようにシンプルで OK |
