# AIで事業計画書・企画書を作成する

あなたは「事業計画書を書かないといけないけど、何から手をつければいいかわからない…」と思ったことはありませんか？ 白紙のドキュメントに向かうのは、料理でいえば「冷蔵庫の中身だけでフルコースを作って」と言われるようなものです。

このレッスンでは、AI（人工知能）を「優秀な下書き担当」として使い、事業計画書や企画書のたたき台を **15分** で組み立てます。AIが骨組みを作ってくれるので、あなたは内容のブラッシュアップに専念できます。

![企画書作成フロー](/lesson-assets/atom.ai-freelancer.business-doc-ai/diagram.png)

## このレッスンで身につくこと

- AIに事業計画書の「たたき台」を作らせるプロンプト（＝AIへの指示文）が書ける
- AIの出力を自分の言葉で修正し、説得力のある計画書に仕上げられる

## 準備する

このレッスンを始める前に、次の2つを確認してください。

1. **AIチャットツールにログインしている** — ChatGPT・Claude・Gemini など、どれでもOKです。まだアカウントがない方は「AIツールをセットアップする」レッスンを先に済ませてください
2. **作りたい企画のアイデアがぼんやりある** — 完璧でなくて大丈夫です。「こんなサービスがあったらいいな」程度で十分です

## ステップ1：AIに渡す情報を整理する

料理にたとえると、この段階は「どんな料理を作りたいか、材料は何があるか」を書き出すステップです。AIは渡された情報が多いほど的確な下書きを返してくれます。

次の4項目をメモ帳やノートに書き出してみましょう（1〜2行ずつで十分です）。

| 項目 | 書く内容 | 記入例 |
|------|----------|--------|
| 事業名 | サービスや商品の名前 | ペットの悩み相談AIアプリ |
| ターゲット | 誰のどんな悩みを解決するか | 犬のしつけに悩む20〜30代会社員 |
| 収益モデル | どうやってお金をもらうか | 月額980円のサブスク（＝毎月定額払いの仕組み） |
| 含めたい項目 | 計画書に入れたいセクション | 事業概要・課題・解決策・競合分析・収益見込み |

> **ポイント**: 「ターゲット」は広すぎると刺さらない計画書になります。「20代女性」ではなく「仕事が忙しくて犬のしつけに悩んでいる20代会社員」のように絞りましょう。

## ステップ2：AIにプロンプトを入力する

整理した情報を使って、AIにプロンプト（指示文）を送ります。ここが一番大事なステップです。

### 良いプロンプトの例

```
あなたはプロの事業計画コンサルタントです。
以下の条件で、新しい事業の事業計画書のたたき台をMarkdown形式で作成してください。

・事業名：ペットの悩み相談AIアプリ
・ターゲット：仕事が忙しくて犬のしつけに悩んでいる20代〜30代の会社員
・収益モデル：月額980円のサブスク（毎月定額払いの仕組み）
・記載項目：事業概要、ターゲット層の課題、解決策、競合の強みと弱み、収益モデル、今後のスケジュール
・文量：A4で3〜5ページ程度
```

### 悪いプロンプトの例

```
事業計画書を書いて
```

なぜ悪いかというと、情報が足りなさすぎて、AIがどの業界のどんなサービスかわからず、ありきたりな内容しか返せないからです。レストランで「何かおいしいもの作って」と注文するようなものです。

![AIへの入力と出力](/lesson-assets/atom.ai-freelancer.business-doc-ai/screen_capture.png)

### プロンプトをさらに良くするコツ

- **役割を与える**: 冒頭に「あなたはプロの〇〇です」と書くと、AIがその専門家の視点で回答します
- **形式を指定する**: 「Markdown形式で」「箇条書きで」「表形式で」など出力形式を伝えると整った結果が返ります
- **文量の目安を伝える**: 「A4で3〜5ページ」のように書くと、短すぎたり長すぎたりしません

## ステップ3：AIの出力を読み、修正する

AIが返してきた文章は、あくまで「たたき台（下書き）」です。そのまま使うのではなく、あなた自身の言葉と事実に基づいて修正しましょう。

### 修正チェックリスト

- [ ] **数字は正しいか？** — AIが出した売上予測や市場規模は推定値です。自分で調べた数字に置き換えましょう
- [ ] **自分の強みが入っているか？** — 他の人にはない、あなただけの経験・スキル・ネットワークを書き加えましょう
- [ ] **違和感のある表現はないか？** — 固すぎる言い回しや、実態と違う記述は自然な言葉に直しましょう
- [ ] **ターゲットの悩みがリアルか？** — 実際に周りの人から聞いた声があれば、それを盛り込むと説得力が増します

### 追加で AI に頼めること

たたき台を受け取った後、さらにAIにお願いできることがあります。

```
この事業計画書について、投資家やクライアントが突っ込みそうなポイントを
3つ挙げて、それぞれへの回答案も作ってください。
```

こうすると、計画書の弱点を事前にカバーできます。

## ステップ4：成果物を保存する

仕上がった計画書をコピーして、お好きなツールに保存しましょう。

- **Googleドキュメント** — 共有や共同編集がしやすい
- **Notion** — 見出しの構造をそのまま活かせる
- **Wordファイル** — 印刷して渡す場面に便利

ファイル名には日付を入れておくと、後から見返すときに便利です。
例：`事業計画書_ペット相談AI_20260412.md`

## 確認する

以下のチェックが全てOKなら、このレッスンは完了です。

- [ ] AIに具体的なプロンプトを送り、事業計画書のたたき台を受け取った
- [ ] AIの出力を自分の言葉・数字で修正した
- [ ] 完成した計画書をファイルとして保存した

## つまずきやすいポイント

| よくある問題 | 解決策 |
|-------------|--------|
| AIの出力が一般的すぎる | プロンプトにターゲットや収益モデルなどの具体情報を追加してやり直す |
| 文章が長すぎて読みにくい | 「箇条書きで簡潔にまとめ直して」とAIに追加指示する |
| 数字（市場規模など）が嘘っぽい | AIの数字はあくまで推定。自分で調べた数字に置き換える |
| 途中でAIの回答が切れた | 「続きを書いて」と送るだけで、途中から再開してくれます |
