# 改善仮説を立ててLPを改善する

このレッスンでは、LP（ランディングページ＝商品やサービスを紹介するための専用のWebページ）の改善を、AIと一緒に行います。

LPの改善は、お店のショーウィンドウをより魅力的に改装するようなものです。ウィンドウの前でお客さんがどこに目を留めているか（ヒートマップ）を観察し、「次はこう変えたらもっと見てもらえるはず」という仮説を立てるのが今回のゴールです。

最終的に、あなた自身の言葉で書いた「LP改善計画メモ」を1枚仕上げます。所要時間は約15分です。

![LP改善の全体フロー](/lesson-assets/atom.ai-marketer.heatmap-hypothesis-improve/diagram.png)

## 前提を確認する

次の2つを手元に用意してください。

1. **ヒートマップのデータまたはスクリーンショット**  
   ヒートマップ（＝Webページのどこがよくクリック・閲覧されているかを色で示す図）がまだ無い場合は、Microsoft Clarity などの無料ツールで取得できます。もし手元に無ければ、「LP改善の練習用にサンプルのヒートマップデータを作ってください」とAIに頼んで、練習用データを生成してもらいましょう。
2. **AIツール**  
   ChatGPT、Claude、Gemini など、どれでもOKです。ブラウザで開いておきましょう。

## ステップ1: ヒートマップの「気になるポイント」を3つ書き出す

ヒートマップを眺めて、気になる部分を3つだけ箇条書きにしてみてください。難しく考えず、「ここが赤いのに成果につながっていない」「ここは全然見られていない」など、気づいたことで大丈夫です。

**良い例:**
- ファーストビュー（＝ページを開いたとき最初に見える範囲）の画像は赤い（よく見られている）が、すぐ下のキャッチコピーはほとんど青い（見られていない）
- 「料金表」のセクションで離脱が多い
- CTAボタン（＝「申し込む」「資料請求」などの行動を促すボタン）の周辺のクリックが少ない

**悪い例:**
- 全体的にイマイチ（→ 具体的にどこが？が無いと次に進めません）

## ステップ2: AIにヒートマップの状況を伝えて改善案をもらう

ステップ1で書き出した内容を、AIに伝えます。コツは「誰向けのLPか」「何が課題か」「いくつアイデアがほしいか」を明記することです。

### プロンプト例（そのままコピーして使えます）

```
私はLP（ランディングページ）の改善をしています。

【ターゲット】30代の会社員で、副業に興味がある人
【現状のヒートマップの課題】
- ファーストビューの画像はよく見られているが、直下のキャッチコピーは読まれていない
- 料金表セクションで離脱が多い
- CTAボタン周辺のクリック率が低い

この3つの課題それぞれについて、改善のアイデアを1つずつ、合計3つ出してください。
それぞれ「何を」「どう変えるか」「なぜ効果がありそうか」の3点で教えてください。
```

> **ヒント:** ヒートマップのスクリーンショットをAIに画像として貼り付けると、さらに的確な回答が返ってきます。ChatGPTやClaudeは画像を読み取れるので、「この画像はLPのヒートマップです。気になる点と改善案を教えてください」と一言添えるだけでOKです。

**うまくいかないとき:**

AIの回答がぼんやりしていたら、次のように追加で聞いてみましょう。

```
もう少し具体的に教えてください。
例えば、キャッチコピーの文言をどう変えるか、具体的な文案を1つ示してもらえますか？
```

## ステップ3: 一番よさそうな改善案を選ぶ

AIが出してくれた案をすべて採用する必要はありません。次の2つの基準で、まず1つだけ選びましょう。

| 基準 | 質問 |
|------|------|
| 課題との結びつき | ヒートマップで見つけた具体的な課題を解決できそうか？ |
| 実行のしやすさ | 自分（またはチーム）で今すぐ着手できるか？ |

迷ったときは、AIに「この3つの案のうち、最も手軽に試せて効果が出やすいのはどれですか？理由も教えてください」と聞いてみてください。

## ステップ4: 改善計画メモを仕上げる

選んだ改善案を、以下のテンプレートに沿って短いメモにまとめます。これが今回の成果物です。

```
# LP改善計画メモ

## 対象ページ
（LPの名前やURLなど）

## ヒートマップで見つけた課題
- （ステップ1で書いた気になるポイント）

## 採用する改善案
- 何を: （どの要素を変えるか）
- どう変えるか: （具体的な変更内容）
- 期待する効果: （なぜ効くと思うか）

## 次にやること
- （例: Canvaでバナーを作り直す、コピーを差し替える、など）
```

> **ヒント:** このテンプレートもAIに「LPの改善計画書のテンプレートを作ってください」と頼めば、あなたの状況に合わせたものを作ってもらえます。

## 完了チェックリスト

次の3つすべてに「はい」と言えたら、このレッスンは完了です。

- [ ] ヒートマップの課題を3つ以上具体的に書き出せた
- [ ] AIを使って課題ごとの改善案を取得できた
- [ ] 改善案を1つ選び、改善計画メモを完成させた

## つまずきポイントに対処する

| こんなとき | こうしてみよう |
|------------|----------------|
| ヒートマップがまだ無い | AIに「練習用のヒートマップデータを作って」と頼んで、架空データで練習する |
| AIが的外れな回答をする | 「誰向けのLPか」「何を売っているか」「現在の課題は何か」をもっと詳しく書き直して再度質問する |
| 改善案が多すぎて選べない | AIに「最も手軽で効果が高い案を1つ選んで、理由を教えて」と聞く |
| プロンプトの書き方がわからない | AIに「LP改善のプロンプトのひな形を作ってください」と聞くと、テンプレートをもらえる |
| 改善計画メモをどこまで書けばいいかわからない | 上のテンプレートの空欄を全部埋められたらOK。完璧でなくて大丈夫 |
