# 指示に必要な素材を整理する

料理をする前に、食材と調理器具をすべて揃えますよね。
それと同じで、AIツールに仕事を頼む前にも「材料」をきちんと揃えておくことが大切です。
材料が足りないと、AIはあなたの意図を推測してしまい、思った通りの結果が出ないことが多くなります。

このレッスンでは、AIに指示を出す前に「何を」「どういう形で」まとめておくべきかを、実際にAIツールを使いながら学びます。
所要時間は約15分です。

## 素材の5つの分類を知る

AIに渡す素材は、大きく5つのグループに分けられます。

| 分類 | 内容の例 | あるとどうなるか |
|------|----------|------------------|
| 商品情報 | 商品名、価格、素材、サイズ、機能 | 商品の特徴を正確に反映できる |
| ブランドの世界観 | トーン＆マナー（＝文章の雰囲気やルール）、禁止事項 | 店舗の雰囲気に合う文章になる |
| ターゲット像 | 顧客の年齢、性別、関心事 | 読み手に刺さる言葉を選べる |
| 参考事例 | 競合の良い文章、自社の過去の成功例 | イメージとのズレが減る |
| 目的・ゴール | 「商品説明文を300字で作りたい」など | 完成物の形が明確になる |

すべて揃っている必要はありません。**あるものでOK**です。ただ、「どれが足りないか」を知っておくだけで、結果の質が大きく変わります。

## AIに足りない情報を聞いてみる

「5分類のうち、自分がどこまで整理できているか分からない」というときは、AIに手伝ってもらいましょう。
ChatGPT や Claude（クロード）を開いて、以下のようなプロンプト（＝AIへの指示文）を送ってみてください。

**プロンプト例：**
```
私はECサイトで新しい商品の説明文を書こうとしています。
以下の5つの分類について、まだ足りていない情報があれば
質問形式で教えてください。

1. 商品情報：オーガニックコットンタオル、3,980円
2. ブランドの世界観：未定
3. ターゲット像：未定
4. 参考事例：なし
5. 目的・ゴール：商品ページ用の説明文（200〜300字）
```

AIが「ターゲットの年齢層は？」「どんな雰囲気の文章にしたいですか？」などと質問してくれるので、答えられる範囲で返します。答えられない質問は「まだ決まっていない」と伝えてOKです。

## 素材をひとつのドキュメントにまとめる

素材がバラバラの場所にあると、AIに渡すときに書き漏れが起きます。
料理で言えば「冷蔵庫、棚、戸棚に分散した食材を、一度まな板の上に並べる」イメージです。

**良い例：**
```
【商品名】 オーガニックコットンタオル
【価格】 3,980円（税込）
【素材】 100%オーガニックコットン
【ターゲット】 環境に関心のある30代女性
【トーン】 丁寧で落ち着いた文体、「エコ」を前面に出す
【目的】 商品ページ用の説明文（200〜300字）
【禁止事項】 「最安値」「No.1」などの根拠のない表現は使わない
【参考】 https://example.com/competing-product （この商品ページの雰囲気が近い）
```

**悪い例：**
```
タオルの商品ページを書いて。
```

良い例のように情報が並んでいると、AIは文脈を正確に読み取り、的外れな回答を減らせます。
悪い例のようにヒントが少ないと、AIが勝手に推測してしまい、ブランドの雰囲気と合わない文章が出力されます。

### まとめ方の手順

1. **テンプレート（＝ひな形）を用意する** ― 上の「良い例」の構成をコピーして使ってください
2. **今ある情報を埋める** ― 商品ページや資料から該当箇所をコピー＆ペーストします
3. **足りない項目に「未定」と書く** ― 無理にでっち上げず、「ここはまだ決まっていない」と明示します
4. **AIに不足を指摘してもらう** ― 上の「AIに足りない情報を聞いてみる」のプロンプト例を使います
5. **保存する** ― メモアプリやドキュメントソフトに保存します

## AIに渡す前に見直す

素材がまとまったら、次の3点を確認してください。
この見直しも、AIに手伝ってもらえます。

- **矛盾はないか** ― 「高級感」と書いているのに価格が「980円」だとイメージがぶれます
- **曖昧さはないか** ― 「いい感じに」ではなく「丁寧な文体で」のように具体的にしましょう
- **目的は明確か** ― 「説明文を書いて」より「商品ページ用の説明文を200〜300字で書いて」のほうが確実です

**AIで見直すプロンプト例：**
```
以下の素材整理ドキュメントに矛盾や曖昧な箇所がないか
チェックしてください。問題があれば具体的に指摘してください。

（ここに自分の素材整理ドキュメントを貼り付ける）
```

## よくあるつまずきと対策

| つまずき | 原因 | 対策 |
|----------|------|------|
| AIが全然違う文章を出す | 素材が少なく推測に頼っている | 5分類のうち最低3つは埋める |
| 毎回同じような文章になる | 参考事例がなく方向性が不明 | 競合1サイトのURLを参考事例として添える |
| ブランドの雰囲気と合わない | トーンの指示がない | 「丁寧」「カジュアル」など形容詞を1つでも入れる |
| 文字数が多すぎる/少なすぎる | ゴールの文字数を書いていない | 「〇字程度」と具体的に書く |
| 何を書けばいいか分からない | テンプレートなしで白紙から始めている | 上の「良い例」をコピーして、自分の商品に置き換える |
| AIの質問に全部答えられない | 社内で決まっていない情報がある | 「未定」と書いてOK。AIは未定の部分を推測してくれるので、結果を見て調整する |

## 成果物を確認する

このレッスンの成果物は、上記「良い例」のような**素材整理ドキュメント**1点です。
以下のチェックリストで確認してください。

- [ ] 5分類（商品情報・ブランドの世界観・ターゲット像・参考事例・目的）のうち**3つ以上**が埋まっている
- [ ] 矛盾する内容が含まれていない（AIの見直しプロンプトでチェック済み）
- [ ] 目的・ゴールに「何を」「何文字で」が具体的に書かれている
- [ ] 足りない項目は空欄ではなく「未定」と明示されている

すべてチェックできたら、完成したドキュメントのスクリーンショットを撮って保存してください。
この素材整理ドキュメントは、次のレッスンでAIに指示を出すときの「材料リスト」として使います。
