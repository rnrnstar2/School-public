# 商品情報を整理する

商品情報の整理は、**引っ越し前の荷造り**に似ています。

引っ越しのとき、あちこちに散らばっている荷物を「服」「本」「食器」と分類して箱にまとめますよね。商品情報も同じです。名前、値段、写真の説明、おすすめポイント――そうした情報がバラバラにあると、お客さんに伝わりません。整理してひとつの「商品カード」にまとめることで、ネットショップにそのまま掲載できるようになります。

このレッスンでは、AIツール（＝人工知能を使った便利ツール。ChatGPTやClaudeなど）を活用して、商品情報をきれいに整理する方法を15分で学びます。

![商品情報整理の全体フロー](/lesson-assets/atom.ec-operator.organize-product-information/diagram.png)

## 前提を確認する

このレッスンを始める前に、以下を用意してください。

- **AIチャットツール**のアカウント（ChatGPT、Claude など。無料版でOKです）
- **整理したい商品**を1つ以上決めておく（手元に商品がある、または写真と説明メモがある状態）

特別なソフトのインストールは不要です。ブラウザでAIチャットツールを開ければすぐ始められます。

## 商品情報を集める

まずは、整理したい商品の情報を書き出します。

引っ越しのたとえで言うと、**リビングに荷物を全部集める**ステップです。紙のメモ、スマホのメモアプリ、商品パッケージの写真――情報源はどこでも大丈夫です。

次の表の項目を意識して、思いつくままにメモしましょう。

| 項目 | 書く内容の例 |
|------|---------------|
| 商品名 | 手作り柚子胡椒 50g |
| 値段 | 580円（税込） |
| 特徴・ウリ | 無添加、地元の柚子使用 |
| どんな人におすすめか | 辛いもの好き、贈り物を探している人 |
| 写真のメモ | 黄色いパッケージ、中身のクローズアップ |
| サイズ・内容量 | 50g（約20回分） |
| 素材・原材料 | 柚子（国産）、青唐辛子（国産）、塩 |

すべて埋まらなくても大丈夫です。ある程度まとまったら次に進みます。

### 良い例と悪い例を見比べる

**良い例**：「手作り柚子胡椒 50g。580円（税込）。地元の柚子と青唐辛子だけを使った無添加。辛さは中辛で、お味噌汁やうどんにひと振りで香りが立つ。50gで約20回分。写真は黄色いパッケージと中身のクローズアップを撮影済み」

**悪い例**：「柚子胡椒。唐辛子入り。安い。」（← 情報が少なすぎて、AIもお客さんも何が良いか分かりません）

> **ワンポイント**: 迷ったら、商品のパッケージや仕入れ伝票を手元に置いて、目に見える情報をそのまま書き写すところから始めましょう。

## AIツールに整理を頼む

集めた情報をAIチャットツールに渡して、見やすい商品カードに整理してもらいましょう。

次のメッセージをコピーして、AIツールの入力欄に貼り付けてください。「かぎ括弧」で囲まれた部分だけ、あなたの商品情報に書き換えてくださいね。

```text
以下の商品情報を、ネットショップに掲載しやすい形式に整理してください。

商品情報：
「ここに集めたメモを貼り付ける」

出力形式：
- 商品名
- キャッチコピー（15文字以内）
- 商品説明（100〜200文字・お客さんに語りかける丁寧な文体）
- おすすめポイント（箇条書き3つ）
- 注意事項（アレルギーや保存方法があれば）

条件：
- 商品情報にない事実は絶対に追加しないでください
- 難しい言葉を使わず、誰でも分かる日本語にしてください
```

これは「プロンプト（＝AIへの指示文）」と呼ばれるものです。何をしてほしいかを具体的に書くほど、AIは期待に近い結果を返してくれます。

![AIツールでの入力・出力例](/lesson-assets/atom.ec-operator.organize-product-information/screen_capture.png)

### もし結果がイマイチだったら？ 追加で指示を出す

AIの最初の回答が100点でなくても大丈夫です。会話の続きとして追加の指示を送れば、AIが修正してくれます。

よく使う追加指示の例を紹介します。

| やりたいこと | AIに送るメッセージ例 |
|---|---|
| キャッチコピーを変えたい | 「キャッチコピーを、贈り物向けのニュアンスに変えてください」 |
| 文章が長すぎる | 「商品説明を150文字以内に短くしてください」 |
| もっとカジュアルにしたい | 「もう少しカジュアルで親しみやすい口調にしてください」 |
| 別の商品もまとめて整理したい | 「同じ形式で、次の商品も整理してください。商品情報：（ここに貼る）」 |

> **コツ**: 「ぜんぶ書き直して」より「キャッチコピーだけ変えて」のように、**変えてほしい箇所をピンポイントで伝える**と、残りの部分が崩れずに修正できます。

## 出力結果を確認する

AIが返してきた文章を読んで、次のポイントを確認しましょう。

### 確認チェックリスト

1. **事実は合っているか** — 値段、原材料、サイズなど、元のメモと照らし合わせて間違いがないか確認します。AIは「それっぽい嘘」をつくことがあるので、ここが一番大切です
2. **語気は丁寧か** — お客さんに届く文章として自然か。「です・ます」調になっているか
3. **見やすさは十分か** — 箇条書きや段落が整っていて、スマホで見ても読みやすいか
4. **キャッチコピーが魅力的か** — 15文字以内に収まっていて、商品の一番の魅力が伝わるか

### 良い確認の例と悪い確認の例

**良い確認**：「元のメモに『580円（税込）』と書いたのに、AIの出力では『500円台』になっている → 修正を指示する」

**悪い確認**：「だいたい合ってるからOK」（← 値段や原材料のズレを見逃すと、お客さんとのトラブルにつながります）

## 商品カードを保存する

最終的に良いものができたら、保存しましょう。

1. AIの回答をコピーする（回答欄の右上にコピーボタンがあるツールもあります）
2. メモアプリやドキュメント（Googleドキュメントなど）に貼り付ける
3. ファイル名を「**商品名_商品カード**」にする（例：「柚子胡椒_商品カード」）

> **ヒント**: 複数の商品を扱う場合は、1つのファイルに全商品をまとめるより、**商品ごとにファイルを分ける**と後から探しやすくなります。

## できたか確認する

次のチェックリストで確認しましょう。すべてチェックがついたら、このレッスンは完了です！

- [ ] 商品名、値段、説明、おすすめポイントが揃っている
- [ ] キャッチコピーが15文字以内で、商品の魅力が伝わる
- [ ] お客さん目線で読みやすい「です・ます」調の文章になっている
- [ ] 元の商品情報と照らし合わせて、事実と異なる内容が含まれていない
- [ ] 完成した商品カードを保存できた

## つまずきやすいポイント

| つまずき | なぜ起こる？ | 対処法 |
|----------|-------------|--------|
| AIの回答に事実と違う内容がある | AIは入力情報を「補って」しまうことがあります | プロンプトに「商品情報にない事実は追加しないで」と書く。上の例では既に入っています |
| AIが長すぎる文章を返す | 文字数を指定しないと、AIは詳しく書こうとしがち | プロンプトに「100〜200字で」と文字数を指定する |
| 何を書けばいいか分からない | 商品を日頃扱っていると「当たり前」に感じて言語化しにくい | 商品パッケージや仕入れ伝票をそのまま見て、目に入った文字を書き写す |
| キャッチコピーがしっくり来ない | 1回目の出力が最適とは限りません | 「もう3案出してください」とAIに頼んで、一番しっくり来るものを選ぶ |
| 2商品目以降で形式がバラバラになる | AIは会話が長くなると前の指示を忘れることがある | 「最初の商品カードと同じ形式で」と一言添える。または新しいチャットで同じプロンプトを使う |
