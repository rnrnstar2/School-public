# 表示義務の基本を確認する

あなたがネットショップをオープンするとき、お店の「看板」や「メニュー」に書くべきことが法律で決まっています。

レストランを想像してみてください。入口に店名があったり、メニューに価格が書いてあったりするのは、お客さんが安心して注文できるようにするためです。ネットショップも同じで、法律（＝国が決めたルール）によって「必ず表示しなさい」と決められている項目があります。これを**表示義務（＝見せるべき情報を法律で義務付けたもの）**と呼びます。

このレッスンでは、AIツール（＝人工知能を使った便利な道具。ChatGPTやClaudeなど）を活用して、あなたのお店に必要な表示項目のチェックリストを15分で作ります。

![表示義務チェックの流れ](/lesson-assets/atom.ec-operator.check-labeling-requirements/diagram.png)

---

## 知る — 8つの表示項目を把握する

ネットショップで法律上の表示義務の対象となる主な項目は、次の8つです。「特定商取引法（＝消費者を守るために通信販売などのルールを定めた法律）」で決められています。

| No. | 項目 | ざっくり説明 |
|-----|------|---------------|
| 1 | 販売業者の氏名または名称 | お店をやっている人・会社の名前 |
| 2 | 住所 | 本店や事務所の場所 |
| 3 | 電話番号 | お客さんが問い合わせできる番号 |
| 4 | 販売価格 | 商品の値段（税込みか税抜きかも明記） |
| 5 | 送料 | 配達にかかる費用 |
| 6 | 引渡時期 | 注文してから何日で届くか |
| 7 | 返品・交換の条件 | 間違って買ったときにどうするか |
| 8 | 支払方法・支払時期 | どうやって、いつお金を払うか |

> **良い例**：ショップページの「特定商取引法に基づく表記」というリンク先に、8項目すべてがそろっている。
>
> **悪い例**：商品ページに価格だけ書いてあり、送料や返品条件がどこにも見つからない。

---

## 調べる — AIでチェックリストを作る

では、AIツールを使ってチェックリストを作りましょう。ここでは ChatGPT または Claude（クロード）を使います。どちらでもやり方は同じです。

### ステップ1：AIツールを開く

- **ChatGPT** を使う場合 → ブラウザで ChatGPT のサイトを開く
- **Claude** を使う場合 → ブラウザで Claude のサイトを開く

どちらも無料プランで大丈夫です。

### ステップ2：プロンプト（＝AIへの指示文）を送る

次のメッセージをそのままコピーして、AIに送信してください。

```
私はECサイト（ネットショップ）の運営者です。
特定商取引法に基づく表示義務のチェックリストを、表形式で作成してください。

以下の3列でお願いします：
1. 項目名
2. 記載すべき内容の説明
3. 当ショップの状況（空欄にしておく）

また、各項目について「書き方の良い例」を1つずつ添えてください。
```

（↑ これはAIに「チェックリストを表で作って、良い例も付けて」とお願いしている文です。そのままコピーして貼り付けてください。）

### ステップ3：結果を保存する

AIが返してくれた表をコピーして、メモ帳・Googleドキュメント・Notionなど、あなたが普段使っているツールに貼り付けます。

![AIの回答をコピーする画面](/lesson-assets/atom.ec-operator.check-labeling-requirements/screen_capture.png)

> **ポイント**：AIの回答をそのまま使うのではなく、必ず次の「入力する」ステップであなた自身の情報に書き換えてください。AIが作るのは「ひな形（テンプレート）」であって、完成品ではありません。

---

## 入力する — あなたのお店の情報を書き込む

貼り付けた表の「当ショップの状況」列に、あなたのお店の情報を1つずつ書き込んでいきます。

### 迷ったらAIに相談する

書き方がわからない項目があれば、AIに追加で聞くことができます。たとえば：

```
個人でネットショップを運営しています。
「返品・交換の条件」はどう書くのが一般的ですか？
初心者向けに、よくあるパターンを3つ教えてください。
```

> **良い例**：住所は「東京都渋谷区神南1-2-3 ABCビル4F」のように番地・建物名まで正確に書く。
>
> **悪い例**：住所を「東京都」とだけ書き、詳細を省略する。

> **良い例**：送料を「全国一律500円（税込）、沖縄・離島は別途800円」のように条件を分けて明記する。
>
> **悪い例**：送料を「別途」とだけ書き、金額がわからない。

---

## 確認する — 完成度をチェックする

すべての項目を埋めたら、以下の3点を見直します。

- [ ] 8項目がすべて埋まっているか → 1つでも空白があるとNG
- [ ] 価格に「税込」か「税抜」のどちらかが書いてあるか
- [ ] 電話番号が実際につながる番号か（テスト発信して確認）

### AIで最終チェックする

作成したチェックリストをAIに貼り付けて、漏れがないか確認してもらいましょう。

```
以下は私のネットショップの「特定商取引法に基づく表記」です。
法律上の必須項目に漏れがないかチェックして、問題があれば指摘してください。

（↓ここにあなたが作った表を貼り付ける）
```

AIが「問題ありません」と返してくれたら、チェックリストの完成です。指摘があった場合は修正しましょう。

## つまずきやすいポイント

| つまずき | 対策 |
|----------|------|
| 「表示はどこに書けばいい？」 | ショップ作成サービス（BASE、STORES、Shopifyなど）の「設定」→「特定商取引法に基づく表記」ページにまとめるのが一般的です。設定メニューの中を探してみましょう。 |
| 「送料が配送先によって変わる」 | 「送料は配送先により異なります。詳しくは送料一覧をご確認ください」と書き、別ページに一覧表を用意します。AIに「地域別送料の表を作って」と頼むと便利です。 |
| 「返品条件が決まっていない」 | まずは「商品到着後7日以内・未開封に限り返品可。送料はお客様負担」という基本パターンで始めましょう。AIに「ネットショップの返品ポリシーのテンプレート」と聞けば、業種別の例を出してくれます。 |
| 「個人の住所を載せたくない」 | バーチャルオフィス（＝住所だけ借りられるサービス）を利用する方法があります。AIに「個人のネットショップ 住所公開 対策」と聞いてみましょう。 |

## このレッスンのゴール

ここまでの手順を終えると、あなたの手元には次のものが残っています：

- **表示義務チェックリスト**（8項目すべてにあなたのお店の情報が埋まった表）
- **AIによる最終チェック結果**（漏れがないことの確認）

これが、このレッスンの成果物です。実際にショップの「特定商取引法に基づく表記」ページへ反映する準備が整いました。
