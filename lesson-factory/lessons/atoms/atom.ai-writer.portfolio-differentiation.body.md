# AI時代に選ばれるポートフォリオ戦略を立てる

あなたがフリーランスや副業で「この人に頼みたい」と思ってもらうには、ポートフォリオ（＝自分の作品集・実績まとめ）が欠かせません。でも、AIが誰でも使える今、「きれいな文章が書ける」だけでは差がつきにくくなっています。

このレッスンでは、AIを活用しながらも「あなたらしさ」が伝わるポートフォリオ戦略を15分で組み立てます。料理にたとえると、AIは優秀な調理器具——同じフライパンを使っても、レシピの組み合わせや盛り付けで「あなたの店の味」になるのと同じです。

## 理解する：なぜ今ポートフォリオ戦略が必要なのか

AI時代のポートフォリオで大切なのは3つです。

1. **プロセスを見せる** — 完成品だけでなく「どうやってAIと一緒に作ったか」を示す
2. **判断力を見せる** — AIの出力をそのまま使わず、どこを直したか・なぜ直したかを伝える
3. **得意分野を絞る** — 「なんでもできます」より「この領域ならお任せ」が選ばれる

## 準備する：ポートフォリオの土台を決める

まず、あなたの強みと対象読者を整理します。ChatGPTやClaudeに次のように聞いてみてください。

**プロンプト例：**

```
私は[あなたの経歴や得意なこと]です。
フリーランスとして[ターゲット業界/クライアント層]向けに
仕事を受注したいと考えています。

以下を提案してください：
1. 私の強みを活かせるポートフォリオのテーマ3案
2. 各テーマで載せるべき作品の種類
3. 競合と差別化できるポイント
```

### 良い例と悪い例

**良い例：** 「元・飲食店勤務で、飲食業界向けのSNS投稿文とメニュー説明文に特化したポートフォリオを作る」

**悪い例：** 「ライティング全般なんでもやります。ブログもSNSもメールも対応可能」

## 組み立てる：ポートフォリオの構成を設計する

次に、ポートフォリオに載せる作品とその見せ方を決めます。以下のフレームワークを使います。

### 1作品あたりの構成

| 要素 | 内容 | 目的 |
|------|------|------|
| タイトル | 何を作ったか一言で | 興味を引く |
| 背景 | なぜこれを作ったか | 課題理解力を示す |
| プロセス | AIをどう使い、何を自分で判断したか | 判断力を示す |
| 成果物 | 完成した作品そのもの | 品質を示す |
| 振り返り | うまくいった点・改善点 | 成長姿勢を示す |

**プロンプト例：**

```
以下の作品をポートフォリオに載せたいです。
「プロセス」セクションで、AIとの協働をどう説明すれば
クライアントに安心感を与えられるか、文例を3パターン提案してください。

作品：[あなたの作品の簡単な説明]
AIの使い方：[どのツールをどう使ったか]
自分が判断した部分：[何を直した・加えたか]
```

## 差別化する：「あなたに頼む理由」を言語化する

ポートフォリオの冒頭には、あなたを選ぶ理由を端的に示す「ひとこと紹介文」を置きます。

**プロンプト例：**

```
以下の情報をもとに、ポートフォリオの冒頭に置く
自己紹介文（100文字以内）を5パターン作ってください。
「AIを使いこなす」だけでなく「この人の判断力に価値がある」
と感じてもらえるトーンにしてください。

得意分野：[あなたの専門領域]
実績：[主な成果]
強み：[判断力を発揮した具体例]
```

## 検証する：完成度をチェックする

作ったポートフォリオ戦略を以下の観点で確認してください。

- [ ] 得意分野が1〜2つに絞れているか
- [ ] 「AIをどう使ったか」のプロセスが最低1作品で書けているか
- [ ] 「なぜ自分に頼むべきか」がひとことで言えるか
- [ ] ターゲット（誰に見せたいか）が明確か

## つまずきやすいポイント

| つまずき | 対処法 |
|----------|--------|
| 「載せられる作品がない」と感じる | 架空のクライアントを想定して練習作品を1つ作る。それ自体がプロセスの実演になる |
| AIの出力をそのまま載せてしまう | 「AIの初稿→自分の修正→完成版」の3段階を並べて見せる |
| 分野を絞れない | まず1つに仮決めして3作品作る。合わなければ変えればいい |

## まとめ

このレッスンで、あなたは以下ができるようになりました。

1. AI時代に選ばれるポートフォリオの3要素（プロセス・判断力・専門性）を理解した
2. AIを使ってポートフォリオの方向性と構成を設計した
3. 「あなたに頼む理由」を言語化した

次のステップとして、実際にポートフォリオの作品を1つ完成させてみましょう。
