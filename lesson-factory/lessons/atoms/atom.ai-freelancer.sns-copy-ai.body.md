# SNS投稿コピーをAIで作成する

料理でいうと、AIは「腕の良いアシスタントシェフ」です。あなたが「こんな味がいいな」と伝えれば、仕込みから盛り付けまで手伝ってくれます。このレッスンでは、そのアシスタントに「SNSで目を引くコピー（＝投稿文）」を作ってもらう方法を15分で身につけます。

完成すると、あなたは「AIに頼めば、いつでも投稿文のたたき台が手に入る」という感覚をつかめます。

![SNSコピー作成フロー](/lesson-assets/atom.ai-freelancer.sns-copy-ai/diagram.png)

## 前提を確認する

このレッスンを始める前に、次の2つを用意してください。

- **AIチャットツールのアカウント**：ChatGPT（＝OpenAIが提供する対話型AIサービス）やClaude（＝Anthropicが提供するAIアシスタント）など、無料プランで始められるもので大丈夫です
- **投稿先のイメージ**：X（旧Twitter）、Instagram、Facebookなど、「どこに投稿したいか」をざっくり決めておきます

どちらもまだの方は、先に「AIツールをセットアップする」レッスンを済ませておくとスムーズです。

## ステップ1：プロンプト（＝AIへの指示文）を組み立てる

AIに美味しい料理を作ってもらうには、注文をはっきり伝えることが大切です。レストランで「おまかせで」と頼むより、「辛さ控えめの和風パスタをお願いします」と言ったほうが、自分好みの一皿が出てきますよね。その注文書にあたるのが「プロンプト（＝AIに渡す指示文）」です。

次の **4つの要素** を入れると、AIの的中率がぐんと上がります。

| 要素 | 説明 | 記入例 |
|------|------|--------|
| **目的** | 何のために書くか | キャンペーンの告知 |
| **投稿先** | どのSNSか（文字数が変わります） | Instagramのキャプション |
| **ターゲット** | 誰に読んでほしいか | 30代・子育て中の女性 |
| **トーン** | どんな話し方にするか | 親しみやすく、絵文字あり |

### 良い例

> 「あなたはSNSマーケティングのプロです。Instagramのキャプション用に、30代の子育て中の女性向けに、オーガニックスキンケア商品の新発売を告知する文章を3パターン作ってください。トーンは親しみやすく、絵文字を適度に使ってください。各パターン150文字以内でお願いします。」

→ 4つの要素がすべて入っていて、パターン数と文字数まで指定しています。

### 悪い例

> 「いい感じのSNS投稿を作って」

→ どのSNSか、誰に向けてか、何の目的かが不明です。AIは当てずっぽうで答えるしかなく、的外れな内容になりがちです。

**あなた用のプロンプトを書いてみましょう。** まずはメモ帳やテキストエディタに、上の4要素を埋める形で1つ書いてみてください。

## ステップ2：AIにコピーを作ってもらう

プロンプトが書けたら、AIに送ります。ここではChatGPTを例に説明しますが、Claudeなど他のツールでも手順はほぼ同じです。

1. AIチャットツール（ChatGPTなど）をブラウザで開く
2. 入力欄にステップ1で作ったプロンプトを貼り付ける
3. 送信ボタン（紙飛行機のアイコン）を押す
4. 返ってきた文章を読む

![AIツール画面でプロンプトを送信する様子](/lesson-assets/atom.ai-freelancer.sns-copy-ai/screen_capture.png)

### 返ってきた結果を育てる

初回の結果が完璧でなくても大丈夫です。料理の味見をして「もう少し塩を足して」と伝えるように、AIにも追加の指示（＝フィードバック）を送りましょう。

使えるフィードバックの例：

- 「もう少し短くして。80文字以内に収めて」
- 「2番目のパターンが良いので、そのトーンを活かしてあと2案出して」
- 「もっとカジュアルな口調に変えて。友だちに話しかけるように」
- 「ハッシュタグを3つ追加して」

このように何度かやり取りすることで、少しずつ理想のコピーに近づけられます。1回で完璧を目指す必要はありません。

## ステップ3：成果物を選んでまとめる

AIとの会話で出てきたコピーの中から、投稿に使う1本を選びます。選ぶときの **3つのチェックポイント** はこちらです。

- **文字数は投稿先の制限に収まっているか？** X（旧Twitter）は140文字、Instagramキャプションは2,200文字が目安です
- **ターゲット読者が「自分のことだ」と感じる内容か？** 読んだ人が「これ私のことだ」と思えれば成功です
- **次の行動が明確か？** 「詳しくはプロフィールのリンクから」「コメントで教えてください」など、読者が次に何をすればよいかが書かれていますか？

選んだコピーをテキストファイルにコピー＆ペーストして保存します。**これがこのレッスンの成果物**です。

## つまずきポイントに対処する

| つまずき | 原因 | 対処法 |
|----------|------|--------|
| AIが英語で返してくる | プロンプトに言語指定がない | プロンプトの先頭に「日本語で答えてください。」を追加する |
| 返答が長すぎる | 文字数の制約を伝えていない | 「〇〇文字以内で」と具体的な数字を指定する |
| どのパターンも似た内容になる | AIが安全な方向に寄せている | 「全く違う切り口で3パターン出して。1つはユーモア、1つは共感、1つは驚きを入れて」と具体的に指示する |
| AIが事実と違うことを書く | AIは「もっともらしい文章」を作るのが仕事で、事実確認はしない | 具体的な数字・商品名・日付は必ず自分で確認してから投稿する |
| 投稿先の雰囲気と合わない | SNSごとの文化の違いを指定していない | 「Xでバズるような短文で」「Instagramのフィードに合う丁寧な文章で」とSNSの特徴を伝える |

## 完了を確認する

次の3つすべてにチェックが付けば、このレッスンは完了です。

- [ ] 4つの要素（目的・投稿先・ターゲット・トーン）を含むプロンプトを自分で書けた
- [ ] AIとやり取りして、少なくとも1回はフィードバックで結果を改善できた
- [ ] 投稿に使えるコピー1本をテキストファイルに保存した
