# 既存クライアントへの追加提案で売上を伸ばす

いつも行っているお店で「今日は〇〇もおすすめですよ」と声をかけられると、つい追加で注文してしまう……そんな経験はありませんか？
実は、これと同じことがあなたの仕事でもできます。新しいクライアントを探すのは大変ですが、すでに信頼関係があるクライアントへの追加提案なら、ぐっとハードルが下がります。

この Atom では、AI を使って「クライアントに喜ばれる追加提案」を15分で作り上げます。

追加提案には大きく2つのパターンがあります。

- **アップセル（＝今よりワンランク上のサービスへの提案）**：たとえばバーガーのLサイズへの変更
- **クロスセル（＝関連する別のサービスの提案）**：たとえばバーガーへのポテトの追加

どちらも「クライアントにとってメリットがあるから提案する」という点が大切です。押し売りではなく、相手の課題解決の延長線上にあるのが良い提案です。

![アップセル・クロスセルのイメージ図](/lesson-assets/atom.ai-freelancer.upsell-cross-sell/diagram.png)

## 前提を確認する

この Atom に取り組む前に、次の条件を満たしているか確認してください。

- すでに1人以上のクライアントがいる（過去のクライアントでもOK）
- そのクライアントとの仕事内容を思い出せる
- ChatGPT や Claude など、AIチャットツールを使える状態にある

まだクライアントがいない場合は、まずは案件獲得系の Atom を先に進めましょう。

## ステップ1: クライアントの状況を整理する

まずは、あなたが今のクライアントに「どんな仕事をしているか」「その仕事の周りにどんな課題がありそうか」を AI と一緒に整理します。

ChatGPT や Claude を開いて、次のプロンプト（＝AIへの指示文）をコピペして送ってみてください。〇〇の部分はあなたの状況に書き換えます。

```
あなたはフリーランスのビジネスコンサルタントです。
以下の状況をもとに、このクライアントに追加で提案できそうなサービスを5つ考えてください。

【現在の仕事内容】
・クライアント名（仮名でOK）：〇〇
・業種：〇〇
・依頼されている仕事：〇〇
・その仕事の頻度：月〇回

【条件】
・今の仕事と関連性が高いものを優先する
・クライアントの業種にとって実際に役立つものにする
・各アイデアに「なぜこのクライアントに合うか」の理由を1行添える
```

AIから5つのアイデアが返ってきたら、次の基準で **1〜2個に絞り込み** ます。

| 基準 | 確認すること |
|------|------------|
| あなたにできるか | 今のスキルで対応可能？少し勉強すればできる？ |
| クライアントに必要か | 実際に困っていそう？過去に話題に出た？ |
| 金額をつけやすいか | 「〇〇円でやります」と言える内容か？ |

**良い例**：「SNS運用を依頼されているが、投稿画像の質に課題がありそう。画像の選定・加工をセットで月額提案しよう」

**悪い例**：「とりあえずホームページも作れますと言ってみよう」（クライアントのニーズと関係ない）

## ステップ2: AIで提案文を下書きする

絞り込んだ提案について、AIに下書きを作ってもらいます。次のプロンプトをコピペして使いましょう。

```
以下の条件で、既存クライアントへの追加提案メッセージを作ってください。

【提案内容】
・提案するサービス：〇〇
・なぜこのクライアントに合うか：〇〇
・想定価格帯：月〇〇円 or 1回〇〇円

【メッセージの条件】
・これまでの関係への感謝を冒頭に入れる
・提案の理由を「あなたの〇〇を見ていて」のように具体的に書く
・押し付けない表現にする（「もしご興味があれば」等）
・長さは200〜300文字
・チャットツール（LINE・Slack等）で送る想定
```

![AIで提案文を生成している画面](/lesson-assets/atom.ai-freelancer.upsell-cross-sell/screen_capture.png)

### 下書きを自分の言葉に整える

AIが出した文章をそのまま送るのは NGです。次のチェックリストで調整しましょう。

- [ ] クライアントとの実際のやり取りを思い出して、口調を合わせたか
- [ ] 「AI っぽい」硬い表現を自分の言い方に直したか
- [ ] 具体的な金額や期間を入れたか（入れない場合は「詳細をお送りします」にする）

**良い例**：
> 「先日の〇〇の件、おかげさまで好評だったと伺い嬉しいです！実は〇〇を拝見していて、△△もお手伝いできそうだなと感じていました。もしご興味があれば、簡単な内容と費用感をまとめてお送りしますが、いかがでしょうか？」

**悪い例**：
> 「追加の仕事はいかがですか？お見積もりします。」（唐突・抽象的・関係性が見えない）

## ステップ3: 送るタイミングを決めて送る

提案文ができたら、クライアントに送ります。普段使っている連絡手段（LINE、Slack、メールなど）で OK です。

**おすすめの送信タイミング**：

| タイミング | なぜ良いか |
|-----------|----------|
| 今の仕事が一段落した直後 | 成果を実感しているタイミングで受け入れやすい |
| 「ありがとう」と言われた直後 | 信頼度が高まっている瞬間 |
| 新しいシーズンやイベント前 | 「ちょうど考えていた」となりやすい |

**避けたほうがいいタイミング**：
- クライアントが忙しそうな時期
- 納品物に修正依頼が入っている最中
- 深夜・早朝（非常識な印象を与える）

## 完成した成果物を確認する

以下の4つがすべてクリアできたら、この Atom は完了です。

1. **提案アイデアリスト** ー AIで出した5つのアイデアと、絞り込んだ1〜2個のメモがある
2. **提案メッセージ** ー 自分の言葉に整えた200〜300文字の提案文が完成している
3. **送信タイミング** ー いつ・どの手段で送るかを決めている
4. **スクリーンショット** ー AIとのやり取り画面、または完成した提案文のスクリーンショットを1枚撮っている

すべてのチェックが済んだら、提案文のスクリーンショットを保存して完了です。

## つまずきやすいポイントと対処法

| つまずき | 原因 | 対処法 |
|---------|------|--------|
| 何を提案すればいいか分からない | クライアントの課題を把握できていない | ステップ1のプロンプトに業種と現在の仕事を具体的に入力し直す |
| AIの提案がピンとこない | プロンプトの情報が少ない | クライアントとの過去のやり取り（困っていたこと・要望）を追記して再度聞く |
| 提案文がAIっぽくて不自然 | そのまま使おうとしている | 「自分ならこう言う」を意識して、3箇所以上書き換える |
| 断られるのが怖い | 「押し売り」と思われることへの不安 | 「もしご興味があれば」を入れるだけで印象が変わる。断られても関係は壊れない |
| 金額をどうすればいいか分からない | 相場感がない | AIに「〇〇のサービスのフリーランス相場を教えて」と聞いてみる |
