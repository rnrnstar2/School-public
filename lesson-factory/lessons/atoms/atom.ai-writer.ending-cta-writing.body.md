# 結論と行動喚起を仕上げる

あなたがブログ記事やSNS投稿を書いているとき、「最後をどう締めればいいかわからない」と感じたことはありませんか？

たとえば料理レシピを紹介する記事なら、最後に「ぜひ今夜試してみてください！」と一言あるだけで、読んだ人が実際にキッチンに立つ確率がぐっと上がります。これが**行動喚起（CTA＝Call To Action）**です。

このレッスンでは、AIチャットツール（ChatGPTやClaude）を使って、記事の「結論」と「行動喚起」をセットで仕上げる方法を学びます。15分以内に、あなたの記事にぴったりの締めくくりが完成します。

## 理解する — 結論と行動喚起の違い

まず、「結論」と「行動喚起」は別のものです。

- **結論**: 記事全体の要点を短くまとめるパート。「つまり、こういうことです」
- **行動喚起（CTA）**: 読者に次に何をしてほしいかを伝えるパート。「さあ、やってみましょう」

レストランにたとえると、結論は「お会計」、行動喚起は「またのお越しをお待ちしています」にあたります。両方そろって初めて、読者の体験がきれいに締まります。

![結論とCTAの構造図](/lesson-assets/atom.ai-writer.ending-cta-writing/diagram.png)

### 良い例と悪い例

**良い例（結論 + CTA）:**
> この記事では、朝の10分ストレッチが集中力を高める3つの理由を紹介しました。明日の朝、目覚まし時計を10分早めてセットして、まずは1週間だけ試してみてください。

**悪い例（尻切れトンボ）:**
> 以上が朝のストレッチについてです。参考にしてください。

悪い例は結論があいまいで、読者が「で、何をすればいいの？」と迷ってしまいます。

## 準備する — AIに渡す情報を整理する

AIに結論と行動喚起を書いてもらうには、次の3つの情報を手元に用意します。

1. **記事の本文**（下書きの全文、またはメインの主張を3行にまとめたもの）
2. **読者がどんな人か**（例：「副業を始めたい30代会社員」「健康を気にするシニア」）
3. **読んだ後にしてほしいこと**（例：「メルマガ登録」「レシピを今夜試す」「無料体験を申し込む」）

この3つがそろっていれば、AIはかなり的を射た締めくくりを提案してくれます。

## 実践する — AIに結論と行動喚起を生成してもらう

### ステップ1: プロンプトを入力する

ChatGPTやClaudeを開いて、以下のプロンプトをコピペし、【】の部分だけ自分の記事の内容に書き換えてください。

```
以下の記事に合った「結論」と「行動喚起（CTA）」をセットで書いてください。

■ 記事の要約:
【ここに記事の主旨を3行で書く】

■ 読者像:
【例: 副業でライティングを始めたい20〜30代】

■ 読者にしてほしいこと:
【例: 次の記事も読んでもらう / メルマガに登録してもらう】

条件:
- 結論は3文以内でまとめる
- 行動喚起は具体的な動詞で始める（「登録する」「試す」「シェアする」など）
- 押しつけがましくない、親しみやすいトーンで
```

### ステップ2: 出力を確認する

AIが返してくれた文章を読み、以下の2点をチェックします。

- **結論が記事の内容とずれていないか？** — ずれている場合は「記事の主旨は〇〇なので、そちらに合わせてください」と追加で指示します
- **行動喚起が具体的か？** — 「ぜひ参考にしてください」のようにあいまいなら、「もっと具体的に、読者が今すぐできるアクションを入れてください」と指示します

![AIの出力を確認する画面](/lesson-assets/atom.ai-writer.ending-cta-writing/screen_capture.png)

### ステップ3: 微調整する

そのまま使えそうなら、自分の記事の末尾に貼り付けて完了です。もう少し調整したいときは、次のようなフォローアップを送ります。

```
もう少しカジュアルなトーンにしてください。
読者が「やってみよう！」と感じるような一言を最後に追加してください。
```

**良い微調整の例:**
> 「トーンをもう少し柔らかくして」「行動喚起の動詞を『試す』から『始める』に変えて」

**悪い微調整の例:**
> 「もっといい感じにして」（何が「いい感じ」か伝わらず、AIが迷います）

## つまずきポイントを知る

| よくあるつまずき | 解決のヒント |
|---|---|
| AIの結論が記事と関係ない内容になる | 記事の本文か要約をプロンプトに含めましたか？情報が足りないとAIは推測で書きます |
| 行動喚起が「ぜひ参考にしてください」で終わる | 「読者が今すぐできる具体的なアクション」を指定し直してください |
| 何パターンも生成されて選べない | 「1パターンだけ出してください」と指示するか、最初に気に入ったものを選んでOKです |
| 長すぎる結論が出力される | 「結論は3文以内にしてください」と文数を明示してください |

## 確認する — 完了チェックリスト

以下の項目をすべて満たしていれば、このレッスンは完了です。

- [ ] AIに結論と行動喚起を生成するプロンプトを送った
- [ ] 結論が記事の内容を正しくまとめているか確認した
- [ ] 行動喚起に具体的な動詞（「試す」「登録する」「シェアする」など）が含まれている
- [ ] 完成した結論 + CTAを自分の記事の末尾に貼り付けた
- [ ] 声に出して読んで、押しつけがましくないトーンであることを確認した

完了したら、仕上がった記事の末尾部分のスクリーンショットを撮って提出してください。
