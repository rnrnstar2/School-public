# SEO向けの商品タイトルを作る

あなたがネットショップで商品を出品するとき、商品ページの「タイトル」はお店の看板のようなものです。看板に何も書いていなければ、お客さんは通り過ぎてしまいますよね。

でも、看板に「すごいTシャツ！！今すぐ買って！！」と叫ぶように書いても、お客さんは「怪しい…」と感じて離れてしまいます。

良い商品タイトルとは、「**どんな商品か**」がひと目でわかり、しかも**検索エンジン（Googleや楽天の検索窓）で見つかりやすい**ものです。このような見つかりやすさを高める工夫をSEO（＝検索エンジン最適化、サーチエンジンから見つかりやすくするための調整）と呼びます。

このレッスンでは、AIチャットツール（ChatGPTやClaude など）を使って、SEOに強い商品タイトルを15分で作る方法を学びます。

## 準備する — 商品の情報を集める

まず、タイトルを作りたい商品の情報を手元に揃えましょう。料理のレシピを作るときに材料を揃えるのと同じです。材料が足りなければ、どんなに腕の良い料理人でもおいしい料理は作れません。AIも同じで、情報が少ないと良いタイトルを作れません。

集める情報は次の4つです。

| 情報 | 具体的にどんなこと？ | 例 |
|------|---------------------|-----|
| **商品名** | メーカー名や製品名 | オーガニックコットンキッズTシャツ |
| **主な特徴** | 素材、色、サイズなど | 天然オーガニックコットン、ホワイト、110cm |
| **誰向けか** | 年齢、性別、用途など | 3〜5歳の子ども向け |
| **売りポイント** | 他にはない強み | 肌に優しい、洗濯しても型くずれしにくい |

> **良い例**: 「商品名：オーガニックコットンキッズTシャツ。素材は天然オーガニックコットン。色はホワイト。サイズは110cm。3〜5歳向け。肌に優しいのが売り」
>
> **悪い例**: 「白いTシャツです」（情報が少なすぎて、AIも良いタイトルを作れません）

メモ帳やスマホのメモ機能に箇条書きでまとめておきましょう。

## AIに指示を出す — プロンプトをコピペして送る

手元の情報が揃ったら、AIチャットツールを開きます。ここでは **ChatGPT** または **Claude**（Anthropic社が提供するAIチャット）のどちらでも構いません。ブラウザで開けるので、特別なインストールは不要です。

次のメッセージ（プロンプト＝AIへの指示文）をコピーして、AIチャットに貼り付けてください。（）の中だけを、あなたの商品情報に書き換えます。

```
ネットショップ用のSEOに強い商品タイトルを5つ提案してください。

商品情報：
- 商品名：（ここに商品名を入れる）
- 特徴：（ここに特徴を入れる）
- 対象：（ここに誰向けかを入れる）
- 売りポイント：（ここに強みを入れる）

条件：
- タイトルは30〜60文字
- 検索されそうなキーワードを前半に含める
- 記号や感嘆符は控えめにする
- 各タイトル案の末尾に文字数を表示する
```

### プロンプトをもっと良くするコツ

AIの提案がいまいちだった場合、以下のように**追加の指示**を送ることで改善できます。これをAIとの「会話のキャッチボール」と考えてください。

- 「楽天市場で検索されやすいキーワードを重視してください」
- 「もっとカジュアルな言い回しにしてください」
- 「別の切り口で5つ追加してください」
- 「1番と3番を組み合わせて新しい案を作ってください」

> **良い追加指示**: 「ターゲットは30代の主婦で、ギフト用途を想定しています。その観点を加えて再提案してください」
>
> **悪い追加指示**: 「もっと良くして」（何を改善してほしいかAIに伝わりません）

## タイトル案を選ぶ・整える

AIから5つの提案が返ってきたら、次の3つのポイントで選びます。

1. **商品の内容がひと目で伝わるか** — 知らない人が読んで「何の商品か」すぐわかりますか？
2. **30〜60文字に収まっているか** — 長すぎると検索結果で途切れて表示されます
3. **不自然な表現がないか** — AIは時々ぎこちない日本語を出します。声に出して読んでみましょう

選んだタイトルがそのまま使えることもありますし、少し言葉を入れ替えて使うこともあります。AIの出力は「たたき台（＝下書き）」として活用しましょう。

> **良い例**: 「【天然素材】オーガニックコットン キッズ半袖Tシャツ ホワイト 110cm」（42文字、特徴と商品名が前半に入っている）
>
> **悪い例**: 「超激安！！最強Tシャツ 白キッズ用 すぐ買って」（誇張表現が多く、商品の詳細が伝わらない。検索にもひっかかりにくい）

## 確認する — チェックリストで仕上げる

タイトルを決めたら、次のチェックリストで最終確認しましょう。

- [ ] 商品の「種類」がわかる（Tシャツ、バッグなど）
- [ ] 商品の「特徴」が少なくとも1つ入っている（素材、色、サイズなど）
- [ ] 30〜60文字に収まっている（AIに文字数を数えてもらってもOK）
- [ ] 不自然な日本語になっていない（声に出して読んで確認）
- [ ] 感嘆符（！）や記号が多すぎない（【】は1セットまで）
- [ ] 自分の出品するサイト（楽天、Amazon、メルカリなど）のタイトル文字数制限を超えていない

すべてチェックできたら完成です。**完成したタイトルをメモ帳などにコピーして保存し、そのスクリーンショットを撮りましょう**。これがこのレッスンの成果物です。

## つまずいたときの対処法

| 困りごと | 原因 | やってみること |
|----------|------|---------------|
| AIが長すぎるタイトルを出す | 文字数条件が伝わっていない | 「30〜60文字厳守で再提案してください」と追加で送る |
| 商品名だけのタイトルになる | 特徴情報が足りない | プロンプトの「特徴」「対象」欄をもっと詳しく書き直して再送する |
| どれも似たような提案になる | AIが同じ方向で考えている | 「まったく別の切り口で5つ提案してください」と送る |
| AIが英語で返答してきた | 日本語の指示が足りない | プロンプトの最初に「日本語で回答してください。」を追加する |
| 変な日本語が混ざっている | AIの翻訳ミス | 自分で自然な言い回しに直す。AIは下書き、最終判断はあなた |
