# 人間とAIの役割分担モデルを設計する

あなたがレストランのオーナーシェフだとしましょう。料理のレシピを考えるのはあなたですが、野菜を洗ったり皮をむいたりする下ごしらえは、優秀なアシスタントに任せることができます。AIとの仕事もこれと同じです。「何を作るか」「どんな味にするか」を決めるのは人間の役割で、「素材を整える」「たたき台を作る」といった作業をAIに任せるのです。

このレッスンでは、あなた自身の仕事について「ここは人間がやるべき」「ここはAIに頼める」という分担表を1枚作ります。15分ほどで完成します。

## このレッスンで得られること

- 自分の仕事を「判断・創造」と「作業・整理」に分けられるようになる
- AIに任せてよい範囲を自信をもって決められるようになる
- 実際に使える「役割分担シート」が1枚できあがる

## 前提：用意するもの

- ChatGPT、Claude、またはお好みのAIチャットツール（無料版でOK）
- メモ帳やGoogleドキュメントなど、文章を書けるもの

## 役割分担の考え方を知る

仕事には大きく2つのタイプがあります。

- **判断・創造タイプ**: 方針を決める、アイデアを出す、品質をチェックする、お客様と交渉する
- **作業・整理タイプ**: 情報を集める、下書きを作る、フォーマットを整える、データをまとめる

AIが得意なのは「作業・整理タイプ」です。一方、最終的な判断や、人間関係に関わる仕事は人間がやるべきです。

![役割分担の考え方](/lesson-assets/atom.ai-writer.human-ai-collaboration-model/diagram.png)

### 良い分担の例

| 仕事の内容 | 担当 | 理由 |
|---|---|---|
| ブログ記事の構成案を作る | AI | たたき台を素早く出せる |
| 記事のトーンや方向性を決める | 人間 | ブランドの判断が必要 |
| 誤字脱字をチェックする | AI | 見落としが少ない |
| 最終的に公開してよいか判断する | 人間 | 責任を持つのは人間 |

### 悪い分担の例

| やりがちなこと | なぜ良くないか |
|---|---|
| AIが書いた文章をそのまま公開する | 事実誤認やトーンのズレを人間がチェックしていない |
| すべて自分で書いてAIを使わない | 時間がかかる作業まで抱え込んでしまう |

## 自分の仕事を棚卸しする

まず、あなたが普段やっている仕事を5〜10個リストアップしましょう。大きな仕事でも小さな仕事でもかまいません。

**AIへのプロンプト例：**

```
私は[あなたの職種]です。日常業務を「判断・創造が必要な仕事」と「情報整理・下書きなどの作業」に分けたいです。まず、[職種]がよく行う業務を10個挙げてください。
```

AIが出してくれたリストを見て、「自分の仕事に当てはまるもの」「当てはまらないもの」を振り分けましょう。AIのリストはあくまで出発点です。自分の実感に合わせて自由に修正してください。

## 役割分担シートを作る

棚卸しができたら、次は分担シートを作ります。以下のプロンプトをAIに送ってみましょう。

**AIへのプロンプト例：**

```
以下の業務リストについて、「人間がやるべき」「AIに任せられる」「人間とAIで協力する」の3つに分類して、表形式でまとめてください。分類の理由も一言添えてください。

業務リスト：
1. [あなたの業務1]
2. [あなたの業務2]
3. [あなたの業務3]
...
```

AIが作った表をもとに、あなた自身の判断で調整します。ここが一番大切なステップです。

![分担シート作成画面](/lesson-assets/atom.ai-writer.human-ai-collaboration-model/screen_capture.png)

### 調整のポイント

- **「AIに任せられる」に分類されても、あなたがやりたい仕事は人間の欄に移してOK**です。楽しさや成長の機会を手放す必要はありません
- **機密情報を扱う仕事**は、AIに渡す情報の範囲を慎重に考えましょう。社外秘のデータをそのままAIに貼り付けるのはNGです
- **「協力する」が一番多くなるのが自然**です。完全にAIだけ・人間だけという仕事は意外と少ないものです

## 完成した分担シートを確認する

最終的に、以下の3つが揃っていれば完成です。

1. **業務リスト**: あなたの主な仕事が5個以上リストされている
2. **分担の分類**: 各業務に「人間」「AI」「協力」のいずれかが割り当てられている
3. **判断の理由**: なぜその分担にしたのか、一言メモがある

完成したシートのスクリーンショットを撮っておきましょう。これがこのレッスンの成果物になります。

## よくあるつまずきと対処法

| つまずき | 対処法 |
|---|---|
| 業務が思い浮かばない | 先週1週間のカレンダーやメールを振り返って、やったことを書き出す |
| 全部「人間がやるべき」に感じる | まず「情報を集める」「下書きを作る」だけAIに試してみる。1つでもOK |
| AIの分類が自分の感覚と合わない | それが正常です。AIの提案はたたき台。あなたの判断で自由に変更してください |
| 機密情報をAIに渡していいか不安 | 会社のAI利用ポリシーを確認する。不明なら、固有名詞を伏せて一般化した情報で試す |

## まとめ

このレッスンでは、自分の仕事を棚卸しして「人間・AI・協力」の3分類で役割分担シートを作りました。大切なのは、**最終判断は常に人間がする**ということです。AIは優秀なアシスタントですが、あなたの仕事の方針を決めるのはあなた自身です。

次のステップとして、分担シートの「AIに任せられる」欄から1つ選んで、実際にAIにやらせてみましょう。「思ったより使える」「ここは微調整が必要だな」という実感が、あなたのAI活用力を一段上げてくれます。
