# 編集フローを高速化する

料理をするとき、食材をあらかじめ切って並べておくと作業がぐんと速くなりますよね。動画編集も同じです。素材を整理し、AIツールに任せられる部分を先に決めておくと、「あの素材どこだっけ？」「この間（ま）、いるのかな？」と悩む時間を大きく減らせます。

この Atom では、あなたが撮り終えた動画素材を使い、AIの力を借りて15分以内にサクサク進む編集フローを1つ組み立てます。最終的に「自分の手順」をメモに残すところまでがゴールです。

![編集フロー全体図](/lesson-assets/atom.video-creator.build-fast-edit-workflow/diagram.png)

## 前提を確認する

始める前に、次の2つが揃っているかチェックしてください。

- 動画素材（スマホやカメラで撮った映像ファイル）が手元にある
- パソコンでブラウザを開ける状態である

特別なソフトのインストールは不要です。今回はブラウザ上で動くAI動画編集ツールを使います。

> **まだ素材がない場合**: スマホで30秒ほど自分の部屋を撮影して練習素材にしてもOKです。

## ステップ1：素材をフォルダに整理する

撮った動画ファイルをパソコンのフォルダにまとめましょう。ここが雑だと、あとから「あのカットどこ？」と探す時間が編集時間を食いつぶします。

- **良い例**：`2024-06-trip-day1` のように「日付＋テーマ」でフォルダ名を付ける
- **悪い例**：`新しいフォルダ (3)` のまま放置する

料理でいう「食材をざるに分ける」作業です。最初に1分かけるだけで、あとの編集がぐっと楽になります。

### AIに整理の相談をする

素材が多くて分け方に迷ったら、ChatGPTやClaudeに聞いてみましょう。

**プロンプト例**:
```
旅行の動画素材が20本あります。
以下のファイル名を見て、編集しやすいフォルダ分けを提案してください。

MOV_0012.mp4（空港到着）
MOV_0013.mp4（ホテル外観）
…
```

AIが「到着編」「観光編」「食事編」のようにグループ分けを提案してくれます。

## ステップ2：AIに文字起こしとカット候補を出させる

Descript（＝AIで動画の字幕と編集ができるブラウザツール）などに動画をアップロードし、AIに文字起こし（＝動画の音声を文字に変換すること）をさせます。

![文字起こし画面の例](/lesson-assets/atom.video-creator.build-fast-edit-workflow/screen_capture.png)

文字起こしが終わったら、AIが「不要そうな箇所（えーっと、長い無言の間など）」を自動でハイライトしてくれます。その候補を確認して、削除してよい部分を決めましょう。

- **良い例**：「えーっと」「あのー」などの埋め言葉だけを丁寧に確認して削除する
- **悪い例**：AIの提案をすべてそのまま削除して、話が途切れてしまう

> **判断に迷ったら**: 「この部分を消すと話がつながらなくなりそうかな？」と思ったら、消さずに残しましょう。あとから消すのは簡単ですが、戻すのは面倒です。

### AIにカット判断を手伝ってもらう

どこを残してどこを削るか迷ったら、文字起こしテキストをコピーしてAIに相談できます。

**プロンプト例**:
```
以下は動画の文字起こしです。
3分以内の動画にしたいので、削除してよい部分を「★削除候補」と
マークしてください。話の流れが途切れないよう注意してください。

---
（ここに文字起こしテキストを貼る）
```

## ステップ3：テンプレートで仕上げる

カットが終わったら、タイトルやエンディングのテンプレート（＝ひな形）を当てはめて完成させます。AIツールには最初からテンプレートが用意されていることが多いので、好みのデザインを選ぶだけでOKです。

### タイトルテキストをAIに考えてもらう

テンプレートに入れる文言もAIに提案してもらえます。

**プロンプト例**:
```
旅行Vlogのオープニングタイトルを5案考えてください。
雰囲気はカジュアルで、10文字以内にしてください。
```

## 結果を確認する

仕上がった動画を1回通して再生し、次の3点をチェックしてください。

- [ ] 動画の最初と最後にタイトル／エンディングが入っているか
- [ ] 「えーっと」などの不要な間（ま）が削れているか
- [ ] 全体の長さが狙いの時間（3〜5分など）に収まっているか

この3点をクリアできたら、あなたの編集フローは完成です。

## 自分の編集手順をメモに残す

せっかく組み立てたフローを忘れないように、簡単なメモを作りましょう。次回の編集がさらに速くなります。

**メモのテンプレート（コピーして使ってください）**:
```
## 私の編集フロー
1. 素材を「___」ルールでフォルダ分けする
2. ___（ツール名）にアップして文字起こし
3. 埋め言葉と___秒以上の無言をカット
4. テンプレート「___」を適用
5. 通し再生で3点チェック
```

このメモが今回のAtomの成果物（＝できあがったもの）です。

## つまずきやすいポイント

| よくある困りごと | 対処法 |
|---|---|
| 動画のアップロードに時間がかかる | ファイルサイズが大きい場合は5〜10分かかります。Wi-Fi環境で始めましょう |
| 文字起こしの精度がいまいち | AIは完璧ではありません。必ず自分の目で確認して直してください |
| カットしすぎて話が途切れる | 削除する前後を再生して、つながりが自然か確認する癖をつけましょう |
| どのAIツールを使えばいいかわからない | まずはDescript（無料プランあり）を試してみてください |
