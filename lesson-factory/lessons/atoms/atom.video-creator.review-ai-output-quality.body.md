# AI出力の品質を見極める

料理に例えるなら、AIは「食材を切って鍋に入れてくれる優秀な助手」です。
味見をして「もう少し塩が足りない」「火が強すぎる」と判断するのは、あなたの役目です。
このレッスンでは、AIが作った文章や画像を「味見」するコツを15分で学びます。

## このレッスンで手に入るもの

- AI出力を見極める **3つのチェック観点**
- チェック結果を元に **AIへ修正指示を出す具体的なプロンプト例**
- 毎回使える **品質チェックリスト**（コピーして手元に置けます）

## 3つのチェック観点を知る

AIが出力（＝AIが生成した文章や画像などの結果）を出したら、次の3つを確認します。

![AI出力チェックの3観点](/lesson-assets/atom.video-creator.review-ai-output-quality/diagram.png)

### 1. 事実が正しいか確認する

AIはもっともらしい文章を書くのが得意ですが、事実と違うことを書くことがあります。
これを「ハルシネーション（＝AIがもっともらしいウソをつくこと）」と呼びます。
数字や固有名詞は特に注意して見ましょう。

- **良い例**: 「東京タワーの高さは333m」という記述を検索して確認してから動画に使う
- **悪い例**: AIが書いた数字をそのまま信じて動画を公開してしまう

> **AI活用のコツ**: 事実確認をAI自身に手伝ってもらうこともできます。ChatGPTやClaudeに次のように聞いてみましょう。
>
> ```
> 以下の文章に事実と違う部分がないかチェックしてください。
> 間違いがあれば正しい情報と出典を教えてください。
>
> （チェックしたい文章をここに貼る）
> ```
>
> ただし、AIのチェック結果もさらに検索で裏取りするのがベストです。

### 2. あなたの意図に合っているか確かめる

AIに出した指示（＝プロンプト）と実際の結果を比べましょう。
指示と違っていたら、具体的にどう変えたいかを伝えて書き直してもらいます。

- **良い例**: 「初心者向けに説明して」と頼んだのに専門用語が多いことに気づき、「もっとやさしい言葉で書き直して。専門用語は使わないで」と追加指示を出す
- **悪い例**: 指示と違っても「まあいいか」でそのまま使ってしまう

> **修正指示のプロンプト例**:
>
> ```
> この文章を次の条件で修正してください。
> - 対象読者: 動画編集をはじめたばかりの人
> - トーン: 友達に教えるような親しみやすい口調
> - 変えてほしい点: 3段落目の説明が長すぎるので、3行以内に短くする
> ```

### 3. 違和感がないか読み返す

最後は人間の感覚です。読んでいて引っかかる部分がないか確かめましょう。
声に出して読むと不自然な表現に気づきやすいです。

- **良い例**: 「この言い回し、なんか変だな」と感じた部分にメモをして直す
- **悪い例**: 違和感があっても「AIが書いたから大丈夫だろう」とスルーする

> **AIを使った磨き上げ**: 違和感がある箇所を見つけたら、次のように頼んでみましょう。
>
> ```
> 以下の文章の「〇〇」という部分に違和感があります。
> もっと自然な言い回しに書き換えてください。候補を3つ出してください。
>
> （文章をここに貼る）
> ```

## 実際にチェックしてみる

ここからは手を動かしてみましょう。次の手順で進めます。

### ステップ1: AIに文章を作ってもらう

ChatGPT、Claude、またはお好みのAIツールを開いて、次のプロンプトを送ってください。

```
YouTubeの動画説明欄に載せる文章を書いてください。
テーマ: 初心者向けスマホ動画撮影のコツ
長さ: 200文字程度
```

### ステップ2: 3つの観点でチェックする

AIが返してきた文章を、先ほどの3つの観点で見ていきましょう。

1. **事実チェック** — 数字や固有名詞が出てきたら、検索して正しいか確かめる
2. **意図チェック** — 「初心者向け」「200文字程度」の条件を満たしているか見比べる
3. **違和感チェック** — 声に出して読んでみて、引っかかる箇所をメモする

![チェック作業の画面例](/lesson-assets/atom.video-creator.review-ai-output-quality/screen_capture.png)

### ステップ3: 修正指示を出す

問題が見つかったら、AIに修正を頼みましょう。ポイントは**具体的に伝える**ことです。

- **良い修正指示**: 「2行目の『超高画質』という表現を、初心者に分かりやすい言葉に変えてください」
- **悪い修正指示**: 「もっとよくして」（何をどう直すのかAIに伝わりません）

## チェックリストをまとめる

毎回同じ基準で見極められるように、次のリストを手元に置いておきましょう。
このリストをコピーして、メモアプリやノートに保存しておくと便利です。

- [ ] 事実に誤りがないか検索で確認した
- [ ] 頼んだ内容（対象読者・長さ・トーンなど）と一致している
- [ ] 不自然な表現や違和感がない
- [ ] 動画を見る人に伝わる書き方になっている
- [ ] 修正が必要な箇所にはAIへ具体的な修正指示を出した

## つまずき対策を知る

| よくあるつまずき | 対処法 |
|---|---|
| 「AIが書いたから正しいはず」と思い込む | AIも間違えます。レストランの料理でも味見は必要です。必ず人間が確認しましょう |
| 直すべきところが多すぎて混乱する | 一度に全部直そうとせず、一番重要なものから1つずつ。「事実の間違い → 意図のズレ → 細かい違和感」の順がおすすめです |
| 何が正しいか自分でも分からない | まずは「しっくりくるかどうか」の感覚を大切に。分からない部分だけ検索するのがコツです |
| AIに修正を頼んでも思い通りにならない | 「ここをこう変えて」と具体的に伝えましょう。一度に複数の修正を頼むより、1つずつ頼む方が精度が上がります |

## 完了を確認する

次の2つができたら、このレッスンは完了です。

1. **AIが生成した文章を3つの観点（事実・意図・違和感）でチェックした**
2. **問題を見つけてAIに具体的な修正指示を出し、改善された結果を得た**

チェック済みの文章と修正前後の比較をスクリーンショットで残しておくと、自分の成長記録にもなります。
