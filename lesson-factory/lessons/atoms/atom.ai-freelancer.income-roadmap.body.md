# 月5万→10万→20万のロードマップを描く

あなたは「AIを使って自分の稼ぎ方を見つけたい」と思っていませんか？
いきなり「月20万円稼ぐ！」と大きな目標を立てるのは、地図も持たずに知らない土地へ旅に出るようなものです。まずは「月5万円」という最初の目的地をクリアし、そこから「10万」「20万」と階段を一段ずつ登るように計画を立てましょう。

このレッスンでは、AIツール（＝ChatGPTやClaudeのような対話型AI）を相談相手にして、**あなただけの無理のないロードマップ（＝収入アップの道順表）**を15分で作ります。完成したロードマップは、メモ帳やGoogleドキュメントに保存して、いつでも見返せるようにします。

![ロードマップ全体像](/lesson-assets/atom.ai-freelancer.income-roadmap/diagram.png)

---

## 用意するもの

- **AIツール**（次のどれか1つでOK）
  - ChatGPT（無料版で十分です）— [https://chat.openai.com](https://chat.openai.com)
  - Claude（無料版あり）— [https://claude.ai](https://claude.ai)
- **メモ帳アプリ**（パソコンのメモ帳、Googleドキュメント、Notionなど何でもOK）

---

## ステップ1: AIに「自分の現在地」を伝える

AIに道順を聞くときは、旅行と同じで「今どこにいるか」と「どこに行きたいか」を伝えることが大切です。この指示文のことを**プロンプト（＝AIへのお願い文）**と呼びます。

### 良い例と悪い例

**良い例:**
```
私は30代の会社員で、プログラミングの経験はありません。
普段使っているスキルは、ExcelとPowerPointです。
AIツール（ChatGPTやClaude）を活用して、
副業で最初の3ヶ月で月5万円を稼ぐための
具体的なステップを教えてください。
各ステップには「何をするか」「どのくらい時間がかかるか」
「必要な道具」を含めてください。
```

**悪い例:**
```
稼ぐ方法を教えて
```
→ これでは情報が少なすぎて、AIも「ブログを書きましょう」のような一般的な回答しかできません。

**ポイント**: 自分の「年代・職業・持っているスキル・使えるツール」を具体的に書くと、AIの回答がグッと具体的になります。

## ステップ2: 3段階のロードマップをAIと一緒に作る

1つのチャットの中で、順番に3回質問します。AIは前の会話を覚えているので、毎回ゼロから説明し直す必要はありません。

### 質問1: 月5万円のステップを聞く

ステップ1で作ったプロンプトをAIに送ります。AIが回答したら、内容をメモ帳にコピーしてください。

### 質問2: 月10万円のステップを聞く

続けて、同じチャットの中で次のように聞きます。

```
ありがとうございます。月5万円を達成できた前提で、
次の3ヶ月で月10万円にステップアップするには
どうすればいいですか？
単価を上げる方法と、案件数を増やす方法の
両方を教えてください。
```

### 質問3: 月20万円のステップを聞く

最後に、同じチャットで聞きます。

```
月10万円を達成できた前提で、
さらに月20万円を目指すには何が必要ですか？
「自分の得意パッケージ（＝セットメニュー）を作る」
「リピーターを増やす」などの観点で教えてください。
```

## ステップ3: ロードマップを1枚にまとめる

AIの回答をそのまま保存するだけだと、あとで見返しにくくなります。次のプロンプトで、AIに「見やすくまとめて」とお願いしましょう。

```
ここまでの会話をもとに、
月5万→10万→20万のロードマップを
1枚の表にまとめてください。
各ステップに「やること」「期間の目安」「使うAIツール」
「達成のサイン」の4列を入れてください。
```

AIが表形式で出力してくれるので、それをメモ帳やGoogleドキュメントにコピーして保存します。これがあなたの**完成ロードマップ**です。

## 確認する

完成したロードマップを見ながら、次の5つをチェックしてみましょう。全部にチェックがつけば合格です。

- [ ] 「月5万」「月10万」「月20万」の3段階が含まれている
- [ ] 最初のステップは、今日からでも始められる内容になっている
- [ ] 各ステップに「具体的にやること」が書かれている（「頑張る」のような曖昧な表現ではない）
- [ ] 各ステップに「期間の目安」が入っている
- [ ] ロードマップがメモ帳やドキュメントに保存されている

## つまずきポイントと対処法

### AIの回答が難しくて読めない

専門用語（＝カタカナの難しい言葉）が出てきたら、次のように聞き返しましょう。

```
すみません、「○○」という言葉の意味がわかりません。
中学生でもわかる言葉で言い換えてください。
```

AIは何度聞き直しても怒りません。遠慮せずに聞いてください。

### AIの提案が自分に合っていない気がする

「プログラミングで稼ごう」など、自分のスキルと合わない提案が来たら、次のように伝えます。

```
プログラミングは未経験なので、それ以外の方法で
提案し直してください。
私が得意なのは○○と○○です。
```

### ロードマップが漠然としている

「SNSを頑張る」のような曖昧なステップが出てきたら、こう聞きます。

```
「SNSを頑張る」をもっと具体的にしてください。
「週に何回、何を、どこに投稿するか」まで
落とし込んでください。
```

### やる気が出ない・不安になる

ロードマップ全体を見て圧倒されたら、**月5万円の最初のステップだけ**に集中してください。「今日やる1つ」だけ決めれば十分です。残りのステップは、最初の目標を達成してから見返しましょう。

---

## まとめ

このレッスンでは、AIを相談相手にして「月5万→10万→20万」の3段階ロードマップを作りました。大切なのは、完璧なロードマップを作ることではなく、**最初の一歩を明確にすること**です。ロードマップは、進みながらいつでもAIに相談して修正できます。まずは最初のステップに取りかかってみましょう。
