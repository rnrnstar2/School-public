# ChatGPTとClaudeを案件タイプ別に使い分ける

たとえば、あなたが料理をするとします。フライパンは「炒める・焼く」が得意で、鍋は「煮る・茹でる」が得意です。どちらも調理器具ですが、作りたい料理に合わせて道具を選ぶのは当たり前ですよね。

AIツールも同じです。ChatGPTとClaudeはどちらも優秀なAI（＝人工知能、人間の代わりに考えたり文章を作ったりするプログラム）ですが、得意なことが違います。このレッスンでは、フリーランスの案件（＝お客さんから依頼された仕事）ごとに「どちらのAIを使えば最も良い結果が出るか」を自分で判断できるようになります。

最終的に、あなたは **使い分けの判断メモ** を1つ作って、今後の仕事で迷わず選べるようになります。

## 前提を確認する

このレッスンを始める前に、次の2つが済んでいることを確かめましょう。

- ChatGPTのアカウントを持っていて、チャット画面（＝AIとメッセージをやり取りする画面）を開ける
- Claudeのアカウントを持っていて、チャット画面を開ける

どちらかまだの場合は、先にアカウントを作ってから戻ってきてください。無料プランで大丈夫です。

## それぞれの得意分野を知る

まず、2つのAIの「性格の違い」をざっくりつかみましょう。

![ChatGPTとClaudeの得意分野比較図](/lesson-assets/atom.ai-freelancer.chatgpt-claude-comparison/diagram.png)

### ChatGPTが得意なこと

ChatGPT（チャットGPT）はOpenAI社が作ったAIです。こんな場面で力を発揮します。

- **アイデアをたくさん出す** — キャッチコピー、ブログのネタ、商品名などを一気に出したいとき
- **短い文章をサクッと作る** — SNSの投稿文、メールの返事、商品説明など
- **画像を作る** — DALL-E（ダリー）という画像生成機能が組み込まれていて、イラストや写真風の画像を作れる
- **ネット検索と組み合わせて答える** — 「最新の○○を調べて」のように、情報収集しながら回答できる

### Claudeが得意なこと

Claude（クロード）はAnthropic社が作ったAIです。こんな場面で力を発揮します。

- **長い文章を丁寧に書く** — レポート、企画書、マニュアルなど、まとまった文書を構成から考えて書く
- **文章の推敲（すいこう＝読み直して手直しすること）** — 日本語の表現をより自然で読みやすく整える
- **長い資料を読み込んで要約する** — PDFやドキュメントを渡して「大事なところだけ教えて」と頼む
- **複雑な指示に従う** — 「○○の条件で、△△の形式で、□□に注意して書いて」のように条件が多い依頼

### ひとことで覚えるなら

> ChatGPT＝**広く・速く・量を出す**タイプ、Claude＝**深く・丁寧に・質を高める**タイプ

## 案件タイプ別の選び方を知る

具体的な案件の例で見てみましょう。

### 良い例：こう選ぶとうまくいく

| 案件のタイプ | おすすめのAI | 理由 |
|---|---|---|
| SNS投稿を10本まとめて書く | ChatGPT | 短い文章をたくさん出すのが得意 |
| 企画書（3〜5ページ）を書く | Claude | 長めの文章を論理的にまとめるのが得意 |
| 商品のキャッチコピーを20個出す | ChatGPT | バリエーション豊かなアイデア出しが得意 |
| 長い資料を要約して報告する | Claude | 文書を正確に読み取り要点を抽出するのが得意 |
| ブログ記事のネタリストを作る | ChatGPT | 幅広いアイデアをすぐ並べられる |
| お客さんへの提案書を仕上げる | Claude | フォーマルな文書を丁寧に作るのが得意 |

### 悪い例：こうすると失敗しやすい

- ❌ **いつも同じAIだけ使う** → 得意分野を活かせず、品質がまちまちになります
- ❌ **ChatGPTに10ページの企画書を一度に書かせる** → 長い文章では途中で話がブレやすくなります
- ❌ **Claudeにキャッチコピーを50個出させる** → アイデアの幅が狭くなりがちです

## 実際に使い分けてみる（ハンズオン）

ここからは、実際にあなたのチャット画面を開いて練習します。全体で10〜15分で終わります。

### 手順1：ChatGPTでアイデアを出す

ChatGPTのチャット画面を開いて、次の文章をそのままコピー＆貼り付けして送信してください。

```
フリーランスとして受注できる仕事のキャッチコピーを5つ考えてください。
テーマは「初心者向けのオンライン講座を作る手伝い」です。
親しみやすい雰囲気でお願いします。
```

**何をしているか：** ChatGPTに「アイデアをたくさん出す」という得意な仕事をお願いしています。

出てきた5つのキャッチコピーをメモ帳やテキストファイルにコピーして保存しておきましょう。

![ChatGPTでアイデアを出している画面](/lesson-assets/atom.ai-freelancer.chatgpt-claude-comparison/screen_capture.png)

### 手順2：Claudeで文章を整える

次にClaudeのチャット画面を開きます。手順1で出たキャッチコピーの中から一番良さそうなものを1つ選んで、次のように入力してください。

```
以下のキャッチコピーを使って、Instagramに投稿する紹介文（150字程度）を
丁寧な日本語で書いてください。
ターゲットは「副業を始めたい30代の会社員」です。

キャッチコピー：（ここに選んだキャッチコピーを貼る）
```

**何をしているか：** Claudeに「条件を守りながら丁寧に文章を仕上げる」という得意な仕事をお願いしています。

### 手順3：使い分け判断メモを作る

最後に、今の体験をもとにメモ帳で次の表を埋めてください。これが今回の**成果物**です。

```
■ 私のAI使い分け判断メモ

【ChatGPTを使う場面】
- （例：キャッチコピーを出すとき）
- 
- 

【Claudeを使う場面】
- （例：お客さんに見せる文章を仕上げるとき）
- 
- 

【両方組み合わせるパターン】
- （例：ChatGPTでネタ出し → Claudeで清書）
```

自分が受けそうな仕事を想像しながら、最低でも各2つずつ書いてみましょう。

## 結果を確認する

練習が終わったら、次の3つを自分でチェックしてください。

- [ ] ChatGPTから出たキャッチコピー5つは、テーマに合った内容だったか？
- [ ] Claudeが整えた投稿文は、そのままSNSに載せられそうな自然な日本語か？
- [ ] 「使い分け判断メモ」に、自分の言葉で各2つ以上書けたか？

3つともチェックが付いたら、このレッスンは完了です。

## つまずきやすいポイントと対策

| つまずき | 対策 |
|---|---|
| AIが返事をしてくれない | インターネット接続を確認して、ページを再読み込みしてください |
| 出てきた日本語が不自然 | 「もっと自然な日本語にしてください」と追加でお願いしましょう |
| どちらのAIを選ぶか迷う | 迷ったらまず「ChatGPTでアイデア出し → Claudeで仕上げ」の流れを試してみましょう |
| ChatGPTとClaudeで同じ結果が出た | 短い文章では差が出にくいことがあります。500字以上の長い文章で比べると違いがわかりやすくなります |
| 無料プランで制限にかかった | 1日の利用回数に上限があります。時間を空けてから続きをやりましょう |

## まとめ

このレッスンで、あなたはChatGPTとClaudeの使い分けを体験し、自分の判断基準をメモに残しました。覚えておくポイントは3つだけです。

- **アイデアを広く・たくさん出したい** → ChatGPT
- **文章を丁寧に・正確にまとめたい** → Claude
- **両方を組み合わせる** と一番よい結果が出やすい

この判断ができると、フリーランスの案件を受けるとき「この仕事はどっちのAIで進めよう？」と迷わず決められます。次のレッスンでは、この使い分けスキルを実際の案件で活かす方法を学びます。
