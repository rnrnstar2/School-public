# CTAとオファー文を改善する

あなたが作ったランディングページやSNS投稿に、「今すぐ申し込む」「無料で試す」といったボタンや誘い文句がありますよね。これを**CTA（コール・トゥ・アクション＝読者に次の行動を促す言葉）**と呼びます。そして、「初月無料」「先着10名限定」のように読者の背中を押す条件提示が**オファー文**です。

この2つは、たとえるなら料理の「最後の味付け」。どんなに良い素材（商品やサービス）を使っていても、味付けがぼんやりしていると「おいしそう！食べたい！」と思ってもらえません。このレッスンでは、AIを使ってCTAとオファー文を「思わずクリックしたくなる味付け」に仕上げる方法を学びます。

![CTA改善フロー](/lesson-assets/atom.ai-marketer.cta-offer-improve/diagram.png)

## 前提を確認する

- AIチャットツール（ChatGPTまたはClaude）が使える状態であること
- 改善したいCTAやオファー文が1つ以上あること（なければ、このレッスン内でサンプルを使います）

## CTAの「良い例」と「悪い例」を知る

まず、改善の方向性をつかむために、よくある例を見てみましょう。

**悪い例（ぼんやりしていてクリックしたくならない）：**
- 「こちらをクリック」
- 「詳細はこちら」
- 「お問い合わせください」

**良い例（具体的で、読者のメリットが伝わる）：**
- 「無料テンプレートを今すぐダウンロードする」
- 「3分で診断結果を見る」
- 「あなた専用のプランを受け取る」

違いがわかりますか？良いCTAには「何が手に入るか」と「どれくらい簡単か」が入っています。

## AIにCTAの改善案を出してもらう

ここからが本番です。AIに今のCTAを見せて、改善案を出してもらいましょう。

### ステップ1：現状のCTAをAIに伝える

以下のプロンプト（＝AIへの指示文）をコピーして、ChatGPTやClaudeに貼り付けてください。`【　】`の中をあなたの情報に書き換えます。

```
私は【業種・サービス内容を簡単に】を提供しています。
現在のCTAボタンの文言は「【現在のCTA文】」です。

このCTAを改善したいです。以下の条件で5つの改善案を出してください：
- 読者が得られるメリットを具体的に入れる
- 15文字以内に収める
- 緊急性または限定感を1つ入れる
```

### ステップ2：AIの回答から「一番しっくりくる案」を選ぶ

AIは5つほど案を出してくれます。全部が完璧である必要はありません。「これが一番自分のサービスに合っている」と感じるものを1つ選んでください。

**選ぶときのコツ：**
- 声に出して読んでみて、自然に聞こえるか？
- 自分がお客さんだったら、クリックしたくなるか？
- サービスの雰囲気（カジュアル？フォーマル？）に合っているか？

## オファー文を強化する

次はオファー文です。CTAの近くに置く「背中を押すひと言」を改善します。

以下のプロンプトを使ってください：

```
先ほど選んだCTAは「【選んだCTA文】」です。
このCTAの近くに置くオファー文（読者の背中を押す条件提示）を3パターン作ってください。

条件：
- 「無料」「限定」「期間」のいずれかの要素を含める
- 嘘や誇大表現は使わない
- 1〜2文で収める
```

**良いオファー文の例：**
- 「今だけ初月無料。いつでも解約できます」
- 「先着30名限定の特別価格でご案内中」
- 「7日間のお試し期間つき。クレジットカード不要」

**悪いオファー文の例：**
- 「お得です」（何がお得かわからない）
- 「今すぐ！急いで！！」（煽りすぎで信頼を損なう）

## 改善前と改善後を並べて確認する

最後に、改善の効果を目で見て確認しましょう。以下のプロンプトでAIにまとめてもらいます：

```
以下の改善前と改善後を表形式で見やすくまとめてください：

改善前CTA：「【元のCTA】」
改善後CTA：「【選んだCTA】」

改善前オファー文：「【元のオファー文、なければ「なし」】」
改善後オファー文：「【選んだオファー文】」

それぞれ、なぜ改善されたのかを1行で説明を添えてください。
```

## つまずきやすいポイントと対策

| よくあるつまずき | 対策 |
|---|---|
| AIの提案がどれもピンとこない | プロンプトに「ターゲットは【具体的な読者像】」を追加してみてください |
| 文字数が長すぎる案ばかり出る | 「10文字以内で」と文字数制限を明記する |
| 自分のサービスに合わない表現が出る | 「トーンは【カジュアル/フォーマル/親しみやすい】で」と指定する |
| オファー文が嘘っぽくなる | 「実際に提供できる条件だけを使って」と明記する |

## 完了を確認する

以下の2つが手元にあれば、このレッスンは完了です：

1. **改善後のCTA文**（1つ以上）
2. **改善後のオファー文**（1つ以上）

できれば、改善前後の比較表のスクリーンショットも保存しておきましょう。次回、別のページやキャンペーンでCTAを書くときに、今回のパターンをそのまま応用できます。
