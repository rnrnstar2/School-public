# 企画指示のプロンプトを作る

「動画を作りたいけど、何から手をつければ…」と思ったことはありませんか？ 実は、AIチャットツールに**上手にお願いする文章（＝プロンプト）**を書けるようになるだけで、企画書のたたき台が15分で手に入ります。このレッスンでは、あなた自身の動画テーマを使って、実際にAIに企画書を作ってもらうところまで体験します。

## このレッスンで手に入るもの

- AIが作ってくれた「動画の企画書」1枚（あなたのテーマに合わせたもの）
- 次回から自分で企画プロンプトを書けるスキル

所要時間は約15分です。

## 前提を確認する

始める前に、次の2つだけ用意してください。

1. **AIチャットツールを開ける状態にする** — ChatGPT（chat.openai.com）またはClaude（claude.ai）をブラウザで開き、ログインしておきます。アカウントがまだなら、どちらも無料で作れます
2. **作りたい動画のテーマをぼんやり決めておく** — 「新商品の紹介」「旅行Vlog」「料理レシピ」など、一言でOKです。決まっていなければ「自己紹介動画」をテーマにしましょう

## 企画指示プロンプトの3要素を知る

レストランで料理を注文するとき、「なんかおいしいもの」と言うより「和風で、しょっぱすぎない、野菜多めの定食」と伝えた方が、期待に近い料理が出てきますよね。

AIへのお願いも同じです。ただ「企画書を作って」と言うだけでは、ぼんやりした回答しか返ってきません。次の**3つの要素**を具体的に伝えることで、AIの回答がグッと良くなります。

![プロンプト3要素の図](/lesson-assets/atom.video-creator.prompt-for-creative-briefs/diagram.png)

| 要素 | 何を書くか | 一言でいうと |
|------|-----------|-------------|
| **目的** | この動画で何を伝えたいか | 「なぜ作る？」 |
| **対象** | 誰に見てほしいか | 「誰に届ける？」 |
| **構成の希望** | 長さ・流れ・雰囲気 | 「どう見せる？」 |

## 「目的」を一文で決める

まずは「この動画を作る理由」を一文で書き出します。

**良い例**:
- 「新商品の使い方を、初めて聞く人に分かりやすく伝える」
- 「自分のチャンネルに興味を持ってもらうための自己紹介をする」

**悪い例**:
- 「面白い動画を作りたい」→ 何が面白いかはAIには分かりません
- 「バズりたい」→ 具体的な中身がないのでAIも困ってしまいます

コツは「〇〇を、△△に伝える」の形にすることです。

## 「対象」を具体的にイメージする

次に「誰に見てほしいか」を、年齢・関心・状況まで想像して書きます。

**良い例**:
- 「30代の忙しい主婦で、時短家電に興味があるが調べる時間がない人」
- 「就活中の大学3年生で、自己PRの作り方に悩んでいる人」

**悪い例**:
- 「みんな」→ 全員に届く動画は、結局誰にも刺さりません
- 「若い人」→ もう少し絞ると、言葉遣いや映像のテイストが自然と決まります

## 「構成の希望」を付け足す

最後に、動画の長さ・流れ・雰囲気のヒントを追加します。数字を入れるとAIが判断しやすくなります。

**良い例**:
- 「60秒以内で、最初の3秒で視線を引く導入を入れる。最後に商品サイトのリンクを案内する」
- 「3分程度、ステップごとに字幕を入れる構成で、落ち着いたトーン」

**悪い例**:
- 「よしなにお願いします」→ AIには「よしなに」が通じません
- 「とにかくカッコよく」→ 何がカッコいいかは人それぞれなので、「テンポよく」「シンプルに」など具体的な言葉に置き換えましょう

## プロンプトを組み立ててAIに送る

3つの要素が揃ったら、下のテンプレートに当てはめて一つの文章にまとめます。

```
あなたは映像企画のプロフェッショナルです。
以下の条件で、動画の企画書を作ってください。

- 目的：（ここに一文で書いた目的を入れる）
- 対象：（ここに具体的な対象者を入れる）
- 構成：（ここに長さ・流れ・雰囲気の希望を入れる）
- トーン：親しみやすく、押しつけがましくない
```

**やってみましょう！** 上のテンプレートをコピーして、カッコ内をあなたのテーマに書き換えてください。書き換えたら、ChatGPTまたはClaudeの入力欄に貼り付けて送信します。

**入力例**（自己紹介動画の場合）:

```
あなたは映像企画のプロフェッショナルです。
以下の条件で、動画の企画書を作ってください。

- 目的：自分のYouTubeチャンネルに興味を持ってもらうための自己紹介をする
- 対象：20〜30代で、副業や新しいスキルに興味がある社会人
- 構成：90秒以内、最初の5秒で「こんな悩みありませんか？」と問いかけ、最後にチャンネル登録を促す
- トーン：親しみやすく、押しつけがましくない
```

![AIチャット画面でのプロンプト入力例](/lesson-assets/atom.video-creator.prompt-for-creative-briefs/screen_capture.png)

## AIの回答をチェックする

AIから企画書の案が返ってきたら、次の3つの観点で確認します。

| チェック項目 | 確認する内容 | OKなら |
|-------------|-------------|--------|
| ✅ 目的 | 最初に決めた「一文の目的」と合っているか | そのまま次へ |
| ✅ 対象 | 具体的な対象者に届く内容になっているか | そのまま次へ |
| ✅ 構成 | 指定した長さや導入が反映されているか | そのまま次へ |

どれかが足りないと感じたら、AIに追加でお願いします。たとえば：

```
導入部分をもっとインパクトのある問いかけに変えてください。
対象者の「時間がない」という悩みにもっと寄り添う内容にしてください。
```

**一度で完璧な回答が来なくて大丈夫です。** 2〜3回やりとりするうちに、どんどん良くなっていきます。

## つまずきやすいポイントを知る

- **プロンプトが長くなりすぎる** — 最初のプロンプトは150文字程度を目安にしましょう。細かい修正は後から追加指示で伝えた方がうまくいきます
- **「〇〇風にして」と書いたのに伝わらない** — 「〇〇風」はAIが知らない場合があります。「明るく」「テンポよく」「落ち着いた」など、誰にでも通じる言葉に置き換えましょう
- **AIの回答が的外れに感じる** — 3要素のうちどれが足りなかったかを振り返り、その部分だけ「〇〇を追加して」と伝えてください
- **どのAIツールを使えばいいか迷う** — ChatGPTもClaudeも、企画書づくりにはどちらも十分使えます。両方試してみて、好みで選んでOKです

## 完了を確認する

次の2つができていれば、このレッスンは完了です。

1. **AIに企画指示プロンプトを送って、企画書の回答を受け取った**
2. **3つのチェック項目（目的・対象・構成）で回答を確認し、必要なら追加指示を1回以上送った**

回答のスクリーンショットを保存しておくと、あとで振り返るときに便利です。

## まとめ

企画指示のプロンプトは、**目的・対象・構成**の3つを具体的に書いてAIに渡すだけ。この型を覚えれば、商品紹介でもVlogでもチュートリアルでも、どんな動画の企画書でもAIと一緒にサクッと作れるようになります。次に動画のアイデアが浮かんだら、ぜひこのテンプレートを使ってみてください。
