# 撮影と編集に強い台本を書く

## はじめに — 台本は動画の「レシピ」

料理で考えてみましょう。レシピなしでいきなり調理を始めると、途中で「あ、玉ねぎ買い忘れた」「火が強すぎた」と慌てることになりますよね。動画制作もまったく同じです。**台本（＝撮る前に書く「設計図」）**をしっかり用意しておくと、撮影も編集もぐんとスムーズになります。

このレッスンでは、AI ツールを使って「撮影しやすく、編集しやすい」台本を1本完成させます。15分で終わるよう、手順を絞って進めます。

## 台本に何を書くかを知る

良い台本には、次の **3つの要素** が揃っています。

1. **セリフやナレーション** — 実際に話す言葉
2. **映像メモ** — どんな画面を映すか（カメラの向きや、Bロール＝本編を補う素材映像のこと）
3. **編集メモ** — カットのつなぎ方、テロップ（＝画面に出す文字）、音楽のタイミング

### 良い例と悪い例を比べる

**良い例：**

```
[00:00-00:10]
映像：手元でコーヒーを注ぐクローズアップ
セリフ：「朝のコーヒー、淹れるだけで1日のリズムが決まります」
編集：ゆっくりフェードイン（＝画面がじんわり明るくなる演出）、BGM静かめで開始
```

→ 映像・セリフ・編集の指示が1セットになっているので、撮影しながら迷いません。

**悪い例：**

```
コーヒーの話をする
```

→ 何を映すか、どう編集するかが分からず、撮影で手戻りが増えます。

## テーマと構成を決める

台本を書き始める前に、まず **テーマ** と **動画の長さ** を決めましょう。決まっていないと、AI に指示を出しても的外れな結果が返ってきます。

### 決めること一覧

| 項目 | 例 |
|---|---|
| テーマ | 「在宅デスク周りの整理術」 |
| 動画の長さ | 30秒のショート動画 |
| 誰に向けて？ | リモートワーク初心者 |
| 伝えたいこと1つ | 「3ステップで机が片付く」 |

> **ポイント**：最初はショート動画（30秒〜1分）がおすすめです。短い方が台本も撮影もコンパクトに済みます。

---

## AI にたたき台を作ってもらう

まずは AI に「たたき台（＝最初の草案）」を出してもらいましょう。次のプロンプト（＝AI への指示文）をコピーして、ChatGPT や Claude などの AI ツールに貼り付けます。

### 基本のプロンプト

```
あなたは映像台本のプロです。
以下の条件で台本を書いてください。

【テーマ】[あなたのテーマを入力]
【動画の長さ】30秒
【ターゲット視聴者】[対象を入力]
【伝えたいこと】[メッセージを1つ入力]
【構成】フック（最初の3秒で興味を引く）→ 本文 → 行動喚起

各シーンに必ず以下の3要素を書いてください：
1. セリフ（話す言葉）
2. 映像メモ（カメラの向き・被写体・Bロール指示）
3. 編集メモ（テロップ・BGM・カット指示）
```

> **何をしているか**：AI に「映像メモと編集メモ付きの台本」を作らせています。`[ ]` の部分だけ、あなた自身の内容に書き換えて使ってください。

### プロンプトの良い書き方・悪い書き方

**良い例：**
```
【テーマ】在宅デスク周りの整理術
【ターゲット視聴者】リモートワーク初心者
【伝えたいこと】3ステップで机が片付く
```
→ 条件が具体的なので、AI が的確な台本を出してくれます。

**悪い例：**
```
台本を書いて
```
→ 条件が足りず、AI が何を書けばいいか分からず汎用的な出力になります。

## たたき台を撮影・編集向けに磨く

AI の出力をそのまま使うと、実際に撮影する場面で困ることがあります。次の **4つのチェック** を順番に確認して手直ししましょう。

### チェック1：セリフが自然か口に出して読む

声に出して読んで、息継ぎの場所や言いにくい言い回しを直します。

- **良い例**：「たった3つのステップで、机がスッキリします」（リズムよく読める）
- **悪い例**：「3つのステップを実施することにより、デスク環境の改善が見込めます」（硬くて息が続かない）

> **コツ**：迷ったら、友達に話しかけるつもりで書き直してみてください。

### チェック2：映像メモが具体的か確認する

「何を・どの角度で・どのくらいの長さ」が書かれているかを見ます。

- **良い例**：「パソコンの画面を上から俯瞰で撮す（3秒）」
- **悪い例**：「パソコンの映像」

### チェック3：編集メモにタイミングと演出を書く

- **良い例**：「テロップ『ステップ1：引き出しを空にする』を画面下に表示。BGMはアップテンポに切り替え」
- **悪い例**：「適当に編集する」

### チェック4：時間配分を確認する

全シーンの秒数を合計して、目標の動画の長さに収まるかチェックします。

| シーン | 秒数 |
|---|---|
| フック | 3秒 |
| ステップ1 | 8秒 |
| ステップ2 | 8秒 |
| ステップ3 | 8秒 |
| 行動喚起 | 3秒 |
| **合計** | **30秒** |

> AI に「合計秒数が30秒を超えていたら削ってください」と追加で指示すると、自動で調整してくれます。

---

## AI にさらに修正を頼む

チェックして気づいた改善点は、AI に修正を頼みましょう。以下は修正依頼のプロンプト例です。

```
さっきの台本を以下の観点で修正してください。
1. セリフをもっとカジュアルな口調にする
2. 映像メモに「カメラの向き」と「秒数」を追加する
3. 合計が30秒以内に収まるように調整する
```

> **何をしているか**：最初の出力をベースに、具体的な改善ポイントを指定して AI に手直しさせています。一度に全部直そうとせず、3つ程度に絞るのがコツです。

---

## 成果物を保存して確認する

完成した台本をテキストファイルとして保存し、次のチェックリストで最終確認します。

### 完成チェックリスト

- [ ] 各シーンに **セリフ・映像メモ・編集メモ** の3要素が揃っている
- [ ] セリフを声に出して自然に読める
- [ ] 映像メモに「何を・どの角度で」が書かれている
- [ ] 編集メモにテロップ・BGM・カット指示のうち少なくとも1つがある
- [ ] 全シーンの秒数合計が目標の動画の長さ以内に収まっている
- [ ] 撮影当日に、この台本だけ見れば迷わず進められる

すべてチェックがついたら、**台本全体のスクリーンショット**（＝画面の写真）を撮って保存します。これがこのレッスンの成果物です。

---

## つまずき対策

| つまずきポイント | 対策 |
|---|---|
| AI の出力が長すぎる | プロンプトで「30秒」「3シーン以内」など具体的な数字で制限する |
| 映像メモが抽象的すぎる | 「誰の・何を・どの角度で・何秒」の4点セットで書き直す |
| セリフが硬くて不自然 | 敬語を外し、友人に話す口調に書き換える。AI に「カジュアルに」と指示してもOK |
| 編集メモが思いつかない | **テロップ・BGM・カット** の3種類から最低1つ選んで書く |
| テーマが決まらない | まず「自分が最近やって良かったこと」を1つ選ぶ。AI に「ショート動画のテーマ案を5つ出して」と聞くのも手 |
| 時間配分がオーバーする | 合計秒数を表にして可視化する。AI に秒数付きで修正を依頼する |

---

## まとめ

このレッスンでは、AI ツールを使って「撮影しやすく、編集しやすい台本」を1本作りました。ポイントをおさらいします。

1. **台本の3要素**（セリフ・映像メモ・編集メモ）を揃える
2. **AI にたたき台を作らせる** — プロンプトは条件を具体的に書く
3. **4つのチェック** で撮影現場で困らないレベルに磨く
4. **修正もAIに頼める** — 改善ポイントを3つ程度に絞って伝える

次は実際にこの台本を使って撮影してみましょう。台本がしっかりしていれば、撮影は驚くほどスムーズに進みますよ。
