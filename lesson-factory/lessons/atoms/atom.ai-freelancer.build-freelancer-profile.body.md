# 選ばれるプロフィールを作成する

フリーランスの仕事を受注するとき、クライアント（＝仕事を依頼する人）が最初に見るのが「プロフィール」です。お店でいえば看板やメニュー表のようなもの。どんなに腕が良くても、看板がなければお客さんは入ってきてくれません。

この Atom では、AIチャット（ChatGPT や Claude など）を相談相手にしながら、クラウドソーシングサイトに掲載できるプロフィール文を1本仕上げます。所要時間は約15分です。

## 確認する — 始める前に必要なもの

- AIチャットツールがすぐ使える状態であること（ChatGPT、Claude、Gemini のいずれか）
- 登録予定のクラウドソーシングサイトを1つ決めていること（ランサーズ、クラウドワークス、ココナラ など）
- 自分がやりたい仕事のジャンルをざっくり決めていること（例：資料作成、記事ライティング、データ入力 など）

## 理解する — 選ばれるプロフィールの3要素

クライアントがプロフィールで知りたいのは、大きく3つだけです。

1. **何ができるか**（スキル・対応範囲）
2. **どれくらい信頼できるか**（実績・姿勢）
3. **頼みやすいか**（対応スピード・コミュニケーション）

この3つがはっきり伝わるプロフィールは、実績ゼロでも仕事の相談が届きやすくなります。

## 作成する — AIにプロフィールの叩き台を作ってもらう

### ステップ1：AIに自分の情報を伝える

AIチャットを開いて、以下のようなプロンプト（＝AIへの指示文）を送ります。コピーしてそのまま貼り付け、`【 】`の中だけ自分の情報に書き換えてください。

```
私はクラウドソーシングで副業を始めたいと思っています。
以下の情報をもとに、クラウドワークスに掲載するプロフィール文を作ってください。
300〜400文字程度で、です・ます調でお願いします。

- 名前（表示名）：【ニックネームでOK】
- やりたい仕事：【例：PowerPointでの資料作成】
- 経験・得意なこと：【例：会社で5年間プレゼン資料を作ってきた】
- 使えるツール：【例：PowerPoint、Canva、ChatGPT】
- 対応可能な時間帯：【例：平日夜と土日】
- アピールしたいこと：【例：納期厳守、丁寧なやり取り】
```

### ステップ2：AIの出力を確認する

AIが出してきた文章を読んで、次のチェックポイントを確認しましょう。

- [ ] 「何ができるか」が書かれている
- [ ] 「信頼できそう」と感じるポイントがある
- [ ] 「頼みやすそう」と思える言い回しがある
- [ ] 嘘や誇張が入っていない

### ステップ3：AIに修正を頼む

気になる部分があれば、追加でAIに修正を頼みます。

**良い修正指示の例：**

```
「5年間の実務経験」という部分を、もう少し具体的にしてください。
例えば「社内向け提案書を年間30本以上作成」のような数字を入れたいです。
```

```
全体的にもう少しカジュアルで親しみやすい文体にしてください。
硬すぎると初めてのクライアントが声をかけづらいので。
```

**悪い修正指示の例：**

```
もっと良くして。
```
→ 何を良くしたいのかAIに伝わりません。「どこを」「どう変えたいか」を具体的に書きましょう。

```
実績10件以上と書いてください。
```
→ 実績がないのに嘘を書くのはNGです。信頼を失う原因になります。

## 仕上げる — プロフィールを完成させる

### ステップ4：最終チェックをAIに頼む

修正が終わったら、最終チェックとしてAIにこう聞きましょう。

```
このプロフィール文を、クラウドソーシングで仕事を発注するクライアントの目線で読んでください。
「この人に頼みたい」と思えるかどうか、改善点があれば教えてください。
```

AIが指摘してくれた点を必要に応じて修正すれば、プロフィールの完成です。

### ステップ5：サイトに登録する

完成したプロフィール文を、クラウドソーシングサイトの自己紹介欄にコピー＆ペーストします。

![プロフィール登録画面のイメージ](/lesson-assets/atom.ai-freelancer.build-freelancer-profile/screen_capture.png)

## 検証する — 本当に「選ばれる」プロフィールになっているか

以下の観点でセルフチェックしましょう。

| チェック項目 | OK? |
|---|---|
| 「何ができるか」が1文目から30秒以内にわかる | |
| 嘘や誇張がない | |
| 対応可能時間が明記されている | |
| 文章が300文字以上ある（短すぎるとスキップされやすい） | |
| 誤字脱字がない | |

## つまずきやすいポイント

- **「書くことがない」と感じる場合**: 仕事の経験がなくても、趣味や日常でやっていることがスキルになります。AIに「私は普段〇〇をしているのですが、これをアピールできるように書けますか？」と聞いてみましょう。
- **長すぎるプロフィールになった場合**: クライアントは何十人ものプロフィールを見ます。400文字を超えたらAIに「300文字以内に要約してください」と頼みましょう。
- **AIの出力がそのまますぎる場合**: AIの文章をそのまま使うと、他の人と似た文面になりがちです。最後に「自分らしさが出るエピソードを1つ追加して」と頼むと差別化できます。
