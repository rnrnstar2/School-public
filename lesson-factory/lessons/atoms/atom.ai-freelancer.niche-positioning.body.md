# 自分だけのニッチポジションを見つける

「副業で稼ぎたいけど、何をすればいいか分からない」——これは多くの人が最初にぶつかる壁です。料理にたとえると、「何か作りたいけどレシピが決まらない」状態ですね。このレッスンでは、AIの力を借りて「あなたならではの得意分野」を15分で見つけます。

このレッスンを終えると、**自分の経験・スキル・興味を掛け合わせた「ニッチポジション」を1つ言語化した文書**が手元に残ります。

## 前提を確認する

- 特別なツールのインストールは不要です
- ChatGPT（無料版でOK）またはClaude（Anthropic社のAIチャット）のどちらかが使える状態にしておいてください
- 「自分の強みが分からない」という状態でも大丈夫です。AIが質問しながら一緒に掘り下げます

## ステップ1: 自分の「持ちネタ」を棚卸しする

まず、あなたの経験や知識を3つの視点で書き出します。紙でもメモアプリでもOKです。

1. **仕事で得たスキル**: 今の仕事や過去の仕事で身につけたこと（例：接客、Excel、プレゼン資料作成）
2. **趣味・好きなこと**: 休日にやっていること、詳しい分野（例：キャンプ、料理、ゲーム、子育て）
3. **よく相談されること**: 周囲から頼られるテーマ（例：パソコンの使い方、旅行プランニング）

各視点で最低2つずつ、合計6つ以上を目指しましょう。「そんなにない」と思っても、AIが後から引き出してくれるので安心してください。

## ステップ2: AIに「ニッチ候補」を出してもらう

ここがこのレッスンの核心です。AIに以下のプロンプト（＝AIへの指示文）をコピペして送ります。

**良い例（プロンプト）:**

```
あなたはフリーランス副業のポジショニング専門家です。
以下は私のスキル・趣味・強みの棚卸しです。

【仕事スキル】
- （ステップ1で書いた内容をここに貼る）

【趣味・好きなこと】
- （同上）

【よく相談されること】
- （同上）

これらを掛け合わせて、AI副業で月5万円を目指せる
「ニッチポジション」の候補を5つ提案してください。
各候補には以下を含めてください：
1. ポジション名（10文字以内）
2. ターゲット顧客（誰に売るか）
3. 提供する価値（何を届けるか）
4. AIの活用方法（どうAIを使うか）
5. 競合が少ない理由
```

**悪い例（プロンプト）:**

```
副業のアイデアを教えて
```

→ これだと一般的なアイデアしか返ってきません。あなた固有の情報を入れることで、AIがパーソナライズされた提案をしてくれます。

## ステップ3: 候補を絞り込む

AIが5つの候補を出してくれたら、次の3つの基準で1つに絞ります。

| 基準 | 質問 | 合格ライン |
|------|------|------------|
| **楽しさ** | 週末にこれをやっても苦にならない？ | ワクワクする、または少なくとも嫌じゃない |
| **需要** | お金を払ってでも欲しい人がいる？ | 「いそう」と思えればOK |
| **AI活用度** | AIを使えば自分一人でも回せる？ | 作業の50%以上をAIに任せられる |

迷ったら、AIにこう聞いてみましょう：

```
上の5つの候補のうち、以下の条件で最もおすすめの1つを
理由つきで教えてください。
- 私が楽しく続けられそうなもの
- 月5万円に最短で到達できそうなもの
- AIの活用比率が最も高いもの
```

## ステップ4: ニッチポジション宣言文を書く

選んだポジションを、以下のテンプレートに当てはめて1つの文にまとめます。これが今日の成果物です。

**テンプレート:**

> 私は「**（ターゲット顧客）**」に対して、「**（自分の強み）**」と「**AI**」を組み合わせて、「**（提供する価値）**」を届ける**（ポジション名）**です。

**良い例:**

> 私は「個人経営の飲食店オーナー」に対して、「飲食業10年の接客経験」と「AI」を組み合わせて、「集客用SNS投稿の企画・作成」を届ける「飲食店AI広報サポーター」です。

**悪い例:**

> 私はAIを使って副業します。

→ 誰に・何を・どうやって、が入っていないと行動に移せません。

## 検証する：できあがりを確認する

以下のチェックリストで、あなたの宣言文を確認しましょう。

- [ ] ターゲット顧客が具体的に書かれている（「誰でも」はNG）
- [ ] 自分の経験・強みが含まれている
- [ ] AIの活用ポイントが明確
- [ ] 10秒で読んで「何をする人か」が伝わる

全部チェックできたら、その宣言文のスクリーンショットを撮っておきましょう。これがこのレッスンの完了証拠になります。

## つまずきやすいポイント

### 「自分に強みなんてない」と思ってしまう

これは一番多い悩みです。でも安心してください。AIに「私は普通の会社員で特技がないのですが」と正直に伝えてみてください。AIは会話を通じて、あなた自身が気づいていない強みを引き出してくれます。

### 候補が多すぎて決められない

完璧な選択をしようとしなくて大丈夫です。副業のポジションは後から変えられます。まずは「一番気になるもの」を選んでください。試してみて合わなければ、いつでも方向転換できます。

### AIの回答が一般的すぎる

プロンプトに具体的な情報が足りない可能性があります。ステップ1の棚卸しをもう少し詳しく書いて、もう一度AIに送ってみてください。「前職は何年経験したか」「趣味はどのくらいの頻度か」などの数字を入れると精度が上がります。
