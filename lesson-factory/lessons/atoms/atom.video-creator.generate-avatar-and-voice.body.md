# アバターと音声を生成する

動画を作りたいけれど「顔出しは恥ずかしい」「自分の声に自信がない」——そんなあなたでも大丈夫です。AIツールを使えば、あなたの代わりになるアバター（＝デジタルの分身キャラクター）と、自然に聞こえるAI音声を15分以内で作れます。このレッスンが終わるころには、動画に使えるアバター画像と音声ファイルが手元に揃っている状態になります。

## たとえ話でイメージする

料理に例えてみましょう。アバターは「お皿の盛り付け」、音声は「味付け」です。どんなにいい内容の動画でも、見た目がパッとしなければ目を引けません。逆に見た目だけ整えても声が不自然だと、視聴者は離れてしまいます。この2つをセットで整えるのが、このレッスンのゴールです。

## 用意するものを確認する

- インターネットに接続されたパソコン
- ウェブブラウザ（＝インターネットを見るためのアプリ。Google Chromeがおすすめ）
- メールアドレス（各サービスの登録に使います）

特別なソフトのインストールは不要です。すべてブラウザ上で完結します。

![アバター作成から音声生成までの流れ](/lesson-assets/atom.video-creator.generate-avatar-and-voice/diagram.png)

## アバター作成ツールを選ぶ

無料で使える代表的なツールを2つ紹介します。

| ツール名 | 特徴 | 向いている人 |
|---|---|---|
| Ready Player Me | 写真から3Dアバターを作れる。ブラウザだけで完結 | 初めてアバターを作る人 |
| VRoid Studio | イラスト風のアバターを細かく作り込める。アプリのダウンロードが必要 | キャラクターの見た目にこだわりたい人 |

**良い例**: 「まずはReady Player Meで、写真1枚からサクッとアバターを作ってみよう」とシンプルなツールから始める

**悪い例**: いきなり有料の高機能ツールに登録して、設定項目の多さに戸惑ってしまう

このレッスンでは、一番かんたんな **Ready Player Me** を使って進めます。

## アバターを生成する

### 手順

1. ブラウザで「Ready Player Me」を検索し、公式サイトを開く
2. 「Create Avatar」ボタンを押す
3. メールアドレスを入力して無料登録する
4. 自分の顔写真をアップロードする（またはパソコンのカメラで撮影する）
5. AIが写真からアバターを自動で生成してくれる
6. 髪型・服装・アクセサリーなど、好みに合わせてカスタマイズする
7. 完成したら「Download」を押して、PNG画像（＝よく使われる画像保存形式）としてダウンロードする

> **コツ**: 正面を向いて、明るい場所で撮った写真を使うと、自然なアバターが生成されやすいです。暗い写真や横顔の写真だと、思った通りの仕上がりにならないことがあります。

![アバター作成画面の例](/lesson-assets/atom.video-creator.generate-avatar-and-voice/screen_capture.png)

### AIに相談してアバターの方向性を決める

どんなアバターにするか迷ったら、ChatGPTやClaudeに相談してみましょう。以下のプロンプト（＝AIへの指示文）をコピーして貼り付けるだけでOKです。

```
私はYouTubeで料理動画を作りたいと思っています。
顔出しはしたくないので、AIアバターを使おうと考えています。
以下の条件で、どんなアバターの見た目がいいかアドバイスをください。

- チャンネルのジャンル: 料理
- ターゲット視聴者: 20〜30代の一人暮らし
- 雰囲気: 親しみやすく、清潔感がある
```

「料理」の部分をあなたの動画のジャンルに変えれば、そのまま使えます。AIが「明るい色の服がおすすめ」「笑顔のアバターがいい」など具体的なアドバイスをくれるので、カスタマイズの参考にしてください。

## 音声生成ツールを選ぶ

日本語に対応した無料の音声生成ツールを紹介します。

| ツール名 | 特徴 | 向いている人 |
|---|---|---|
| VOICEVOX | 完全無料で商用利用も可能。多数のキャラクター音声を選べる | 日本語で本格的に動画を作る人 |
| CoeFont | ブラウザだけで完結。登録後すぐに試せる | まずは手軽に音声生成を体験したい人 |

**良い例**: 「まずはCoeFontの無料版で短い1文を音声にしてみる」と、登録だけですぐ試す

**悪い例**: 英語専用のツールを選んでしまい、日本語が読み上げられないことに後から気づく

このレッスンでは、ダウンロード不要でブラウザだけで使える **CoeFont** を使って進めます。

## 音声を生成する

### 手順

1. ブラウザで「CoeFont」を検索し、公式サイトを開く
2. 無料登録をする（GoogleアカウントでもOK）
3. 音声キャラクター（＝声の種類）を選ぶ。いくつか試聴して、自分の動画に合う声を見つける
4. テキストボックスに読み上げたい文章を入力する
5. 「音声合成」ボタン（＝文字から音声を自動作成する機能）を押す
6. プレビューで音声を確認し、OKならダウンロードする

### AIに読み上げ原稿を作ってもらう

読み上げる文章もAIに手伝ってもらえます。ChatGPTやClaudeに次のように聞いてみましょう。

```
以下の条件で、YouTube動画の冒頭15秒分のナレーション原稿を書いてください。

- 動画のテーマ: 一人暮らしの簡単レシピ
- 口調: やさしく親しみやすい
- 聞き手: 料理初心者
- 長さ: 50〜80文字
- 句読点を適切に入れて、音声合成で自然な間が出るようにしてください
```

> **コツ**: 句読点の「。」「、」の位置で自然に間が入ります。一文は短くすると聞き取りやすくなります。

**良い例**: 「こんにちは。今日は、かんたんパスタの作り方を紹介します。」（一文が短く、間が自然）

**悪い例**: 「こんにちは今日はかんたんパスタの作り方を紹介しますよろしくお願いします」（一文が長く、間がない）

## 成果物を確認する

次の2つが手元に揃ったら、このレッスンは完了です。

- ✅ アバターの画像ファイル（PNG形式）がパソコンに保存されている
- ✅ 音声ファイル（WAVまたはMP3形式）がパソコンに保存されている

### 確認チェックリスト

| 確認ポイント | OKの基準 |
|---|---|
| アバターの画像がぼやけていないか？ | 拡大しても顔のパーツがはっきり見える |
| アバターの雰囲気が動画の内容に合っているか？ | 自分の動画に出てきても違和感がない |
| 音声が不自然に途切れていないか？ | 通して聴いて引っかかる箇所がない |
| 音声の長さが想定に合っているか？ | 15秒〜1分程度になっている |
| 音声ファイルを再生アプリで開けるか？ | ダブルクリックで音が再生される |

## つまずきやすいポイントを知っておく

| つまずき | 原因 | 解決策 |
|---|---|---|
| アバターの顔が似ていない | 写真の明るさ・角度が悪い | 正面向き＋明るい場所で撮り直してリトライする |
| 音声が機械的に聞こえる | 句読点が少なく一文が長い | 文を短く区切り、句読点を増やす。読み上げスピードを少し下げる |
| ダウンロードしたファイルが見つからない | ブラウザの保存先がわからない | 「ダウンロード」フォルダを開く。ブラウザ下部の通知バーからも開ける |
| ファイルが開けない | 対応アプリがない | PNG→ブラウザにドラッグで表示可。音声→VLCメディアプレーヤー（無料）で再生 |
| 登録メールが届かない | 迷惑メールフォルダに入っている | 迷惑メールフォルダを確認。それでもなければ別のメールアドレスで再登録 |
