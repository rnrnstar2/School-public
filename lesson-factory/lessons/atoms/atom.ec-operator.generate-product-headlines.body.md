# 商品名と見出しを生成する

レストランのメニューを想像してみてください。同じポテトフライでも、「カリカリ香ばしい 手切りポテトフライ」と書いてあれば、思わず注文したくなりますよね。ECサイト（＝ネットショップ）の商品名や見出しも同じです。いい名前をつけるだけで、お客さんの目を引き、売上が変わってきます。

このレッスンでは、AIツール（＝人工知能を使った便利なサービス。ChatGPTやClaudeなど）を活用して、あなたの商品にぴったりの商品名とキャッチコピーを作る方法を学びます。15分ほどで「自分でAIに商品名を作ってもらえる」状態になります。

![商品名生成フロー](/lesson-assets/atom.ec-operator.generate-product-headlines/diagram.png)

## 準備する

始める前に、次の2つを用意してください。

- **AIツールのアカウント**: ChatGPT（https://chat.openai.com）またはClaude（https://claude.ai）のどちらか1つでOKです。無料プランで十分です
- **商品の基本情報**: カテゴリー、特徴（3〜5個）、対象のお客さん、価格帯をメモしておく

プログラミングの知識は一切不要です。ブラウザ（＝インターネットを見るためのアプリ。SafariやChromeなど）だけで完結します。

## ステップ1: 商品の「売り」を整理する

AIに頼む前の一番大事な準備です。あなたの商品が「何が魅力的か」を3〜5個リストアップしましょう。これがレシピの材料です。材料がはっきりしていれば、料理（＝商品名）も美味しく仕上がります。

以下のテンプレートを埋めてみてください。

```
■ 商品カテゴリー: （例: シャンプー）
■ 特徴1: （例: アミノ酸配合）
■ 特徴2: （例: 無添加）
■ 特徴3: （例: 泡立ちが良い）
■ 対象のお客さん: （例: 乾燥髪で悩んでいる方）
■ 価格・容量: （例: 700ml 1,980円）
■ 競合との違い: （例: ドラッグストア品より成分がやさしい）
```

### 良い例

- 「アミノ酸配合」「無添加」「泡立ちが良い」という**成分の特徴**
- 「乾燥髪でもまとまる」という**使った後の効果**
- 「700mlで1,980円」という**価格の安さ**
- 「ドラッグストア品より刺激が少ない」という**競合との違い**

### 悪い例

- 「いいシャンプーです」→ 感想だけではAIが具体情報を持てず、ありきたりな名前しか出ません
- 特徴を何も書かない → AIが「高品質シャンプー」のような一般的すぎる名前を出してしまいます

## ステップ2: AIに商品名と見出しを依頼する

AIツール（ChatGPTやClaude）を開き、次のプロンプト（＝AIへの指示文）をコピーして貼り付けてください。`【 】` の部分だけ、あなたの商品情報に書き換えます。

```
あなたはECサイトの商品名コピーライターです。
以下の商品情報をもとに、商品名を5パターン作ってください。
それぞれに合うキャッチコピー（見出し）も1つずつ添えてください。

# 商品情報
- カテゴリー: 【シャンプー】
- 特徴: 【アミノ酸配合、無添加、泡立ちが良い】
- 対象のお客さん: 【乾燥髪の方】
- 価格・容量: 【700ml 1,980円】
- 競合との違い: 【ドラッグストア品より成分がやさしい】

# ルール
- 商品名は20文字以内
- 見出しは30文字以内
- トーン: 親しみやすく、信頼感のある日本語
- 「最強」「絶対」などの誇張表現は使わない
- 商品名の形式: 「表」で出力する
```

上のテキストをAIの入力欄に貼り付けて送信するだけです。数秒で5パターンが出力されます。

![AIの出力画面](/lesson-assets/atom.ec-operator.generate-product-headlines/screen_capture.png)

### AIの出力例

| # | 商品名 | 見出し |
|---|--------|--------|
| 1 | 潤うアミノ酸シャンプー 無添加処方 | 洗うたびツヤが生まれる。髪に優しい1本 |
| 2 | 乾燥髪まとまるアミノシャンプー | 泡立ち抜群で指通りスルスル |
| 3 | やさしさ処方 アミノ酸シャンプー | 成分にこだわる人のための無添加ケア |
| 4 | もっちり泡のアミノシャンプー 700ml | パサつく髪もしっとりまとまる |
| 5 | 無添加アミノ酸ヘアシャンプー | 乾燥知らずの新習慣。1,980円から |

### 良い出力のポイント

- 特徴（アミノ酸・無添加）がちゃんと商品名に入っている
- 対象のお客さん（乾燥髪）に刺さる言葉がある
- 嘘や誇張がない

### 悪い出力の見分け方

- 「最高級」「奇跡の」など、根拠のない強い表現がある → 使わない
- あなたの商品にない特徴が書かれている → 嘘になるので却下

## ステップ3: 結果をふるいにかける

AIが出力した5パターンをそのまま全部採用せず、次の3つの基準で絞り込みましょう。

| 基準 | チェック内容 | 例 |
|------|-------------|----|
| 検索しやすいか | お客さんが検索しそうな言葉が入っているか | ○「アミノ酸 シャンプー」→ 検索されやすい |
| 具体性があるか | 「すごい」ではなく「どうすごい」が伝わるか | ○「泡立ち抜群」 ×「すごいシャンプー」 |
| 嘘がないか | 実際の性能・成分と一致しているか | ×「オーガニック」（実際は違うのに使う） |

自分のショップの雰囲気に合うものを**2〜3個**残してください。

## ステップ4: AIに微調整を頼む

絞り込んだ商品名をさらに磨きます。追加の指示を出すだけでOKです。

### 微調整プロンプトの例

**文字数を変えたいとき:**
```
3番目の商品名をもっと短く、10文字以内にしてください。
```

**トーンを変えたいとき:**
```
見出しを少し丁寧な言い回しに変えてください。
高級感のあるトーンでお願いします。
```

**別の切り口がほしいとき:**
```
「価格のお得感」を前面に出したバージョンを
追加で3パターン作ってください。
```

このように何度でも修正を依頼できます。3〜4回のやりとりで、あなた好みの文章に仕上がります。

## ステップ5: 成果物を保存する

最終的に決まった商品名と見出しを保存しましょう。

1. メモ帳やスプレッドシート（＝オンラインで使える表計算ソフト。Googleスプレッドシートなど）に、決定した商品名と見出しを貼り付ける
2. AIとのやりとり画面のスクリーンショット（＝画面の写真を撮る機能）を撮る
   - **Windowsの場合**: `Windows + Shift + S` を同時に押す
   - **Macの場合**: `Command + Shift + 3` を同時に押す
   - **スマホの場合**: 電源ボタン＋音量下ボタンを同時に押す
3. スクリーンショットを「商品名_日付」のファイル名で保存する

このスクリーンショットがこのレッスンの成果物です。

## 確認する

次のポイントをセルフチェックしてください。すべてにチェックがつけばこのレッスンは完了です。

- [ ] AIに商品の特徴を3つ以上伝えてプロンプトを送信できた
- [ ] AIが出力した商品名に、あなたの商品の特徴が反映されている
- [ ] 実際の商品と矛盾する表現（嘘や誇張）がない
- [ ] 商品名が20文字以内に収まっている
- [ ] 見出しだけ読んで商品の魅力が伝わる
- [ ] 成果物（商品名・見出しリスト）をスクリーンショットで保存した

## よくあるつまずきと対処法

| つまずき | 原因 | 対処法 |
|---------|------|--------|
| AIの回答が一般的すぎる | 商品情報が足りない | 「成分名」「対象層」「他社との違い」を追加してもう一度依頼する |
| 文字数が長すぎる | 文字数指定が伝わっていない | 「○文字以内で」とプロンプトに明記する |
| 日本語が不自然 | トーン指定がない | 「丁寧な日本語で」「親しみやすいトーンで」を追加する |
| 同じような名前ばかり出る | 切り口が1つだけ | 「価格」「効果」「成分」など別の切り口を指定する |
| どれを選べばいいかわからない | 基準がない | ステップ3の3つの基準（検索性・具体性・正確性）で比較する |
