# 直営業でクライアントを開拓する

クラウドソーシングサイトで案件を探すのは、いわば「お店に並んでいる商品を選ぶ」ようなもの。一方、直営業は「自分から相手のドアをノックしに行く」方法です。競争が少なく、あなたの強みをそのまま伝えられるので、単価も上がりやすいのが特徴です。

このレッスンでは、AIを使って営業メッセージを作り、実際に送る直前の状態まで仕上げます。15分で「自分にもできそう」と思える営業文が1通完成します。

![直営業の全体フロー](/lesson-assets/atom.ai-freelancer.direct-outreach/diagram.png)

## 前提として知っておくこと

- ポートフォリオ（＝あなたの実績をまとめた作品集）がすでにある状態を想定しています
- 価格設定の考え方を理解していると、メッセージに説得力が出ます
- 特別な営業スキルは不要です。AIが文章の土台を作ってくれます

## ステップ1: 営業先を見つける

「どこに営業すればいいの？」がまず最初のハードルです。以下の3つの場所から探してみましょう。

### 探す場所の例

| 場所 | 特徴 | 向いている人 |
|------|------|-------------|
| X（旧Twitter） | 「○○できる人いませんか？」という投稿を検索 | SNSを普段から使う人 |
| 企業のお問い合わせフォーム | 公式サイトから直接連絡 | しっかりした提案をしたい人 |
| 知人・友人の紹介 | 信頼関係があるので話が早い | まず1件目の実績が欲しい人 |

### AIに営業先を相談するプロンプト例

以下のプロンプトをChatGPTやClaudeにそのままコピペしてみてください。

```
私はAIを使った文章作成（ブログ記事・商品説明文）を得意とするフリーランスです。
直営業で最初のクライアントを見つけたいのですが、
以下の条件で営業先の候補を5つ提案してください。

- 個人でも連絡しやすい規模の企業やサービス
- AIを使ったコンテンツ作成に需要がありそうな業界
- X（旧Twitter）や公式サイトから連絡できる相手
```

> **ポイント**: 「あなたの得意分野」の部分を自分のスキルに書き換えると、より的確な候補が出てきます。

## ステップ2: 営業メッセージをAIで作る

ここが一番大事なステップです。いきなり自分で書こうとすると手が止まってしまいますが、AIに下書きを頼めば、それを直すだけで済みます。

### 良い営業メッセージの3つのポイント

1. **相手のことに触れる**: 「御社の○○を拝見しました」と具体的に言う
2. **自分ができることを簡潔に**: 長々と書かない。3行以内で
3. **次のアクションを明確に**: 「15分お時間いただけませんか」など

### 良い例

```
○○株式会社 ご担当者様

御社のECサイトの商品説明文を拝見し、ご連絡いたしました。
AIを活用した商品説明文の作成を行っており、
月間50本の記事作成実績があります。

サンプルをお送りできますので、
15分ほどオンラインでお話しする機会をいただけないでしょうか。

ポートフォリオ: [URL]
```

### 悪い例

```
はじめまして！フリーランスの○○です！
色々なことができます！AIも使えます！
お仕事ください！よろしくお願いします！
```

> **なぜ悪いか**: 相手の名前もなく、何ができるか具体的でなく、次に何をすればいいかも不明です。

### AIに営業メッセージを作ってもらうプロンプト例

```
以下の情報をもとに、直営業用のメッセージを作ってください。

【営業先】○○株式会社（ECサイト運営）
【自分のスキル】AIを使った商品説明文・ブログ記事作成
【実績】月間50本の記事作成、ポートフォリオURL付き
【目標】15分のオンライン面談のアポイントを取る

条件:
- 300文字以内
- 相手の事業に触れる一文を入れる
- 丁寧だけど堅すぎない文体
- 最後に具体的な次のアクション（面談依頼）を入れる
```

## ステップ3: メッセージを磨く

AIが作った文章をそのまま送るのはNGです。必ず以下のチェックリストで確認しましょう。

### 送信前チェックリスト

- [ ] 相手の会社名・名前は正しいか？
- [ ] 自分の実績・スキルに嘘はないか？
- [ ] 「次に何をしてほしいか」が明確か？
- [ ] 誤字脱字はないか？
- [ ] ポートフォリオのリンクは動作するか？
- [ ] 送る時間帯は適切か？（平日10-17時が無難）

### AIにチェックしてもらうプロンプト例

```
以下の営業メッセージを添削してください。
改善点があれば具体的に指摘し、修正案も出してください。

[ここにあなたが作ったメッセージを貼り付ける]

チェック観点:
- 敬語の使い方は自然か
- 相手にとってのメリットが伝わるか
- 押し付けがましくないか
- 300文字以内に収まっているか
```

## ステップ4: 送信の記録をつける

営業は1通で終わりではありません。「誰に・いつ・何を送ったか」を記録しておくと、フォローアップ（＝後追いの連絡）がしやすくなります。

### 記録テンプレート（メモ帳やスプレッドシートに）

| 日付 | 営業先 | 連絡方法 | 送った内容の概要 | 返信状況 |
|------|--------|----------|-----------------|----------|
| 4/12 | ○○株式会社 | お問い合わせフォーム | 商品説明文作成の提案 | 未返信 |

> **コツ**: 1週間返信がなければ、1度だけフォローアップのメッセージを送りましょう。「先日ご連絡した○○です。ご検討状況を伺えれば幸いです」程度で十分です。

## つまずきやすいポイント

### 「何を書けばいいかわからない」
→ ステップ2のプロンプトをそのまま使ってください。自分のスキル部分だけ書き換えればOKです。

### 「返信が来なくて心が折れる」
→ 直営業の返信率は一般的に5〜10%です。10通送って1通返信があれば上出来。数を打つことが大事です。

### 「営業先が見つからない」
→ Xで「○○ 募集」「○○ 探してます」と検索してみてください。リアルタイムで需要が見つかります。

## 検証する：完成を確認しよう

以下がすべてクリアできていれば、このレッスンは完了です。

1. 営業先の候補を最低3つリストアップできた
2. AIを使って営業メッセージを1通作成できた
3. チェックリストで確認し、送信可能な状態に仕上げた
4. 営業記録のテンプレートを用意できた

**成果物**: 送信可能な営業メッセージ1通（Markdown形式でOK）とそのスクリーンショット

## まとめ

直営業は最初こそ緊張しますが、AIを相棒にすれば「文章を考える」というハードルがぐっと下がります。大事なのは完璧なメッセージを作ることではなく、まず1通送ってみること。このレッスンで作ったメッセージを、ぜひ実際に送ってみてください。
