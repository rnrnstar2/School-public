# 講師用ガイドをAIで生成する

## このレッスンでできるようになること

あなたが研修の講師を頼まれたとしましょう。「何を、どんな順番で、どう伝えればいいのか」をゼロから考えるのはとても大変ですよね。

このレッスンでは、AIチャットツール（＝ChatGPTやClaudeのような、文章のやり取りで使えるAI）を使って、**講師用ガイド**を15分以内で作る方法を学びます。完成したガイドはMarkdownファイル（＝見出しや箇条書きを使った読みやすいテキスト形式）として手元に残ります。

> **講師用ガイドとは？**
> 研修を進めるための「台本」のようなものです。料理教室でいうレシピカード、旅行でいうスケジュール表と同じ役割を果たします。「何を話すか」「いつ質問を投げるか」「どのくらい時間をかけるか」が一目でわかります。

## 準備する

このレッスンを進める前に、以下を用意してください。

- **AIチャットツール**：以下のいずれかを開いておきましょう
  - [ChatGPT](https://chat.openai.com/) — 無料プランでOKです
  - [Claude](https://claude.ai/) — 無料プランでOKです
  - [Cursor](https://cursor.com/) のチャット機能 — すでにインストール済みならこちらでも可
- **研修のテーマ**：「Excel基礎」「チームコミュニケーション」など、何について教えるかが決まっていること
- **対象の参加者**：「新入社員」「中堅社員」など、誰が受けるかがわかっていること

> **どのAIツールを選べばいい？**
> 結論から言うと、どれを使っても同じ手順で進められます。迷ったらChatGPTかClaudeの無料版で始めてみてください。結果に差が出るのはツールの選び方ではなく、**あなたがAIに渡す情報の具体さ**です。

## 研修の概要をまとめる（5分）

まず、AIに依頼する前に**研修の概要**をメモしておきましょう。これは料理の材料を揃えるようなものです。材料がはっきりしていれば、レシピも的確に出てきます。

次の4つの項目を箇条書きでまとめてください。

1. **研修のテーマ**（例：Excelの基本操作）
2. **対象者**（例：パソコン操作に不慣れな新入社員）
3. **研修の長さ**（例：90分）
4. **ねらい**（例：表の作成と簡単な計算ができるようになる）

### 良い例と悪い例

**良い例** ✅
- テーマ：Excelの基本操作
- 対象者：パソコン操作に不慣れな新入社員20名
- 時間：90分
- ねらい：表の作成と簡単な計算（SUM関数）ができるようになる

→ 具体的で、AIが内容を絞り込みやすいです。

**悪い例** ❌
- テーマ：Excel
- 対象者：みんな
- 時間：未定
- ねらい：わかるようになる

→ 情報があいまいで、AIが的外れなガイドを作ってしまいます。

## AIにプロンプト（＝指示文）を送る（3分）

AIチャットツールを開き、以下の形式でメッセージを送ります。かっこの中を自分の研修内容に書き換えてからコピー＆ペーストしてください。

```
あなたはプロの研修講師です。以下の条件で、講師用ガイドを作成してください。

- テーマ：（ここにテーマを入れる）
- 対象者：（ここに対象者を入れる）
- 人数：（ここに人数を入れる）
- 時間：（ここに時間を入れる）
- ねらい：（ここにねらいを入れる）

ガイドには次の内容を含めてください。
1. 研修のはじめの挨拶とアイスブレイク（＝参加者がリラックスする導入）
2. セクションごとの進め方と所要時間
3. 各セクションの具体的なセリフ例（講師が話す言葉をそのまま書く）
4. 参加者に投げかける質問（3つ以上）
5. つまずきやすいポイントとその対策
6. まとめと振り返りの進め方

出力形式：Markdown形式で、見出し・箇条書き・表を使ってください。
日本語で出力してください。
```

![AIチャット画面での入力イメージ](/lesson-assets/atom.training-designer.facilitator-guide-generation/screen_capture.png)

### プロンプトのカスタマイズ例

AIの回答をもっと自分の研修に合わせたいときは、以下のフレーズを追加してみましょう。

| やりたいこと | 追加するフレーズ |
|-------------|----------------|
| もっと短くしたい | 「A4用紙1枚に収まる長さにしてください」 |
| グループワークを入れたい | 「4人グループでのワークを2回含めてください」 |
| 参加者に配る資料も欲しい | 「講師用ガイドに加えて、参加者用の配布資料も作ってください」 |
| もっと具体的にしたい | 「講師が読み上げるセリフをそのまま書いてください」 |

## AIの回答を確認する（3分）

AIからの回答が表示されたら、**3つの観点**でチェックしましょう。

| 観点 | チェック内容 | 問題があった場合のAIへの追加指示 |
|------|-------------|-------------------------------|
| 時間の合計 | セクションごとの時間を足して、指定した時間に収まっているか | 「合計を○分に調整してください」 |
| 対象者のレベル | 参加者がわかる言葉で書かれているか。専門用語に説明がついているか | 「○○という言葉を、もっとわかりやすく言い換えてください」 |
| 抜け漏れ | はじめの挨拶からまとめまで、すべて含まれているか | 「○○のセクションが抜けています。追加してください」 |

### 良い例と悪い例

**良い例** ✅
> 「セクション1（15分）：アイスブレイクで自己紹介シートを配る。参加者が『Excelを使ったことがあるか』を〇✕で答える。講師のセリフ例：『まずは肩の力を抜いて、隣の方と自己紹介しましょう。1分間でお名前と、Excelで一番やりたいことを教えてください。』」

→ 具体的で、講師がそのまま動けます。

**悪い例** ❌
> 「セクション1：ウォームアップを行う」

→ 何をすればいいのかわかりません。このまま使うと、当日あたふたします。

> **チェックで問題を見つけたら？**
> 遠慮なくAIに「ここを直してください」と伝えましょう。AIは何度でもやり直してくれます。1回で完璧な回答を求めるよりも、「出す → 確認 → 直す」のサイクルを2-3回まわすほうが良い結果になります。

## ガイドを整える（3分）

AIの出力をそのまま使うのではなく、**あなたの現場に合わせて手直し**します。これはレストランのレシピを自宅のキッチンに合わせて調整するようなものです。

次の3つのポイントを見直してください。

1. **時間配分を調整する**：参加者の反応が遅くなりそうなセクションに5分ほど余裕を持たせる
2. **質問を自分流に言い換える**：AIが出した質問を、自分が普段使う言葉で話しやすく書き換える
3. **つまずき対策を現場に寄せる**：「〇〇がわからない参加者がいたら、『画面の左上を見てください』と声をかける」のように、具体的な場面と行動をセットで書く

### AIに手直しを手伝ってもらう

全部を自分で書き換えなくても、AIに修正を頼むことができます。

```
以下の点を修正してください。
- セクション2の時間を15分から20分に変更
- 「SUM関数」という用語に「＝数字を自動で合計する機能」という説明を追加
- アイスブレイクを「隣の人と好きな食べ物を30秒で紹介し合う」に変更
```

このように、変更したい箇所を箇条書きで送ると、AIが修正版を返してくれます。

## 完成したガイドを保存する（1分）

修正が終わったら、完成したガイドをファイルとして保存します。

1. AIの回答をすべて選択してコピーする
2. メモ帳（Windows）またはテキストエディット（Mac）を開く
3. 貼り付けて保存する。ファイル名は「研修ガイド_テーマ名_日付.md」とする
   - 例：`研修ガイド_Excel基礎_20260413.md`

> **なぜ .md で保存するの？**
> `.md` はMarkdownファイルの拡張子です。見出しや箇条書きがきれいに表示される形式で、多くのツールで読めます。`.txt` でも問題ありませんが、`.md` にしておくと後から見返したときに読みやすくなります。

## 検証する

ガイドが完成したら、最後に次の3つを確認します。

- [ ] ガイドを見ながら**実際に声を出して進行のシミュレーション**をしてみて、時間内に終わるか
- [ ] 専門用語には**すべて説明が添えてあるか**
- [ ] 参加者が**つまずきそうな場面に、講師が取る具体的な行動**が書かれているか

すべてにチェックがつけば、あなたの講師用ガイドは完成です。

## つまずきやすいポイントと対策

| つまずき | 原因 | 対策 |
|---------|------|------|
| AIが英語で回答してくる | プロンプトに言語指定がない | 「日本語で出力してください」を最後に追加する |
| ガイドが長すぎて使えない | 出力量の指定がない | 「A4用紙1枚に収まる長さで」と追加する |
| 時間配分が合計と合わない | AIが計算を間違えることがある | 「合計時間が○分になるよう調整してください」と追加して再生成する |
| 対象者に合わない難しい表現がある | 対象者の情報が不足している | 「中学生にもわかる言葉で書き直してください」と追加する |
| 同じような内容が繰り返されている | セクション分けが曖昧 | 「各セクションで扱う内容が重複しないよう整理してください」と追加する |

## まとめ

このレッスンでは、AIチャットツールを使って講師用ガイドを15分以内で作る方法を学びました。

1. 研修の概要を4項目でまとめる（5分）
2. AIに具体的なプロンプト（＝指示文）を送る（3分）
3. 回答を3つの観点で確認する（3分）
4. 自分の現場に合わせて整える（3分）
5. ファイルとして保存する（1分）

ポイントは「AIに全部任せる」のではなく、**「AIに下書きを作ってもらい、自分の知識で仕上げる」**ということです。AIは何度でもやり直してくれるので、遠慮なく修正を頼んでくださいね。

この手順を覚えておけば、初めての研修でも自信を持って準備できますよ。
