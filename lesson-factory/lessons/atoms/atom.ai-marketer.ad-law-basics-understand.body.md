# 薬機法と景表法の要点を理解する

あなたが商品のキャッチコピーを書いたとき、「これ、法律に引っかからないかな？」と不安になったことはありませんか？

このレッスンを終えると、薬機法と景表法の**基本的なルール**を理解し、AIツール（＝ChatGPTやClaudeのような、質問に答えてくれる便利なツール）を使って、自分が書いた広告文をセルフチェックできるようになります。所要時間はおよそ15分です。

## たとえ話でイメージする

想像してみてください。あなたがお菓子屋さんで「このクッキーを食べると風邪をひかない！」というポップを書きました。でも実際には医学的な証明がありません。これだとお客様に誤解を与えてしまいますよね。

国は「消費者が正しく商品を選べるように」という目的で、広告のルールを定めています。交通ルールと同じで、知っておけば安心して走れるようになります。その代表選手が**薬機法**と**景表法**の2つです。

![薬機法と景表法の関係図](/lesson-assets/atom.ai-marketer.ad-law-basics-understand/diagram.png)

## 薬機法のポイントを知る

薬機法（＝医薬品医療機器等法のことで、くすりや健康に関わる商品のルールを定める法律）は、サプリメントや化粧品、健康器具などの広告にルールを定めています。

### 覚えておくべき3つのルール

1. **効果をうたうときは証拠が必要** — 「痩せる」「病気が治る」などの表現には、科学的な根拠が求められます
2. **未承認の効果は書けない** — 国が認めていない効果・効能を広告することは禁止です
3. **特定の言葉は要注意** — 「特効薬」「万能」などの誇大（＝大げさな）表現は避けましょう

### 良い例と悪い例を比べる

| 種類 | 表現例 | 理由 |
|------|--------|------|
| ✅ 良い例 | 「お腹の調子を整えるサポートが期待できます」 | 控えめで、効果を断言していない |
| ✅ 良い例 | 「うるおいを与え、肌をすこやかに保ちます」 | 化粧品で認められた範囲の表現 |
| ❌ 悪い例 | 「このサプリを飲めばがんが治ります！」 | 医学的根拠がなく、未承認の効能 |
| ❌ 悪い例 | 「塗るだけでシミが完全に消えます」 | 化粧品で認められない効果の断言 |

## 景表法のポイントを知る

景表法（＝景品表示法のことで、商品の価格や品質についてウソや大げさな表示を禁止する法律）は、健康商品に限らず**すべての商品・サービス**が対象です。

### 覚えておくべき3つのルール

1. **優良誤認の禁止** — 実際より品質を良く見せるウソの表示はダメです（例：外国産の肉を「国産和牛」と書く）
2. **有利誤認の禁止** — 実際よりお得に見せるウソの表示もダメです（例：通常価格を吊り上げて「50%OFF」と書く）
3. **過大な景品の制限** — 商品購入でもらえる景品の上限が決まっています

### 良い例と悪い例を比べる

| 種類 | 表現例 | 理由 |
|------|--------|------|
| ✅ 良い例 | 「通常価格1,000円のところ、期間限定で800円」 | 事実にもとづく価格表示 |
| ✅ 良い例 | 「お客様満足度90%（自社調査・対象100名）」 | 根拠と調査条件を明示 |
| ❌ 悪い例 | 「業界最安値！」 | 根拠がないのに最上級表現を使用 |
| ❌ 悪い例 | 「お客様満足度No.1」 | 調査方法や対象を示していない |

## 2つの法律の違いを整理する

ここで一度、薬機法と景表法の違いを整理しましょう。

| 比較ポイント | 薬機法 | 景表法 |
|-------------|--------|--------|
| 対象商品 | 健康に関わる商品（サプリ、化粧品、医薬品など） | すべての商品・サービス |
| 禁止されること | 未承認の効果・効能をうたう広告 | 品質や価格についてのウソ・大げさな表示 |
| 管轄（＝担当する役所） | 厚生労働省 | 消費者庁 |
| 違反した場合 | 販売停止命令、罰金 | 措置命令、課徴金 |

## AIツールで広告文をチェックする

ここからが実践です。あなたが書いた広告文が法律に触れないか、AIツールを使ってセルフチェックしましょう。

### ステップ1：チェックしたい広告文を用意する

まず、あなたが今持っている広告文やキャッチコピーを1つ用意してください。まだ無い場合は、練習として次の例文を使いましょう。

```
この美容サプリを毎日飲むだけで、たった1週間でマイナス5kg！
リピート率98%、業界No.1の実力。今だけ特別価格980円！
```

### ステップ2：AIに法律チェックを依頼する

ChatGPT、Claude、またはお好みのAIツールを開き、以下のプロンプト（＝AIへの指示文）をコピーして貼り付けてください。

**プロンプト例1：基本チェック**

```
あなたは日本の広告法規に詳しいマーケティングコンサルタントです。
以下の広告文を、薬機法と景表法の観点でチェックしてください。

【チェック観点】
1. 薬機法に触れそうな表現（効果の断言、未承認効能など）
2. 景表法に触れそうな表現（優良誤認、有利誤認など）
3. 各問題点の深刻度（高・中・低）
4. 具体的な修正案

【広告文】
（ここにあなたの広告文を貼る）
```

**プロンプト例2：修正案の比較**

```
先ほどの広告文について、修正前と修正後を表にまとめてください。
表の列は「元の表現」「問題点」「修正後の表現」の3つでお願いします。
```

**プロンプト例3：別のAIツールでダブルチェック**

1つ目のAIで修正した文章を、別のAIツール（例：ChatGPTで作ったらClaudeで確認）に貼って、もう一度チェックしてもらいましょう。AIによって指摘のポイントが異なることがあり、見落としを防げます。

![AIチェックの画面例](/lesson-assets/atom.ai-marketer.ad-law-basics-understand/screen_capture.png)

### ステップ3：AIの回答を読んで修正する

- AIが「表現を控えるように」と提案したら → その部分を「〜が期待できます」「〜をサポートします」のような控えめな言い方に直す
- AIが「根拠を示してください」と言ったら → データや出典（＝情報のもとになった資料）を用意する。なければその表現自体を削る
- 修正したら、もう一度同じプロンプトでAIに確認 → 「大きな問題は見当たりません」と返ってくるまで繰り返す

⚠️ **大事な注意点**: AIはあくまで参考意見です。最終的な判断は、消費者庁や厚生労働省の公式ガイドライン、または法律の専門家に確認してください。AIが「問題ない」と言っても、実際の法律判断を保証するものではありません。

## つまずきやすいポイントを知る

| よくある疑問 | 対策 |
|-------------|------|
| 薬機法は化粧品にも適用されるの？ | はい。化粧品は薬機法の対象です。「シミが消える」「アンチエイジング」など、効果を断言する表現は避けましょう |
| 「個人の感想です」と書けばセーフ？ | いいえ。免責（＝責任逃れの）文言を添えても、誤解を招く表現自体が違反になる可能性があります |
| AIの回答をそのまま信じていい？ | AIは参考意見です。法律の最終判断は公式ガイドラインや弁護士に確認してください |
| どの程度の表現ならセーフ？ | 「〜が期待できます」「〜をサポートします」など控えめな表現を心がけると安心です |
| 景表法の「No.1表示」は全部ダメ？ | 客観的な調査データがあり、調査方法・対象を明記すれば認められる場合があります |

このレッスンの成果物として、**修正前と修正後の広告文をまとめたメモ**を作成してください。
